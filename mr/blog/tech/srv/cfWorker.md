
## CNAME बाउंड डोमेन नाव

Cloudflare Worker डोमेन नाव `SSL/TLS` → सानुकूल होस्ट नावाने बंधनकारक आहे CNAME डोमेन नाव बांधल्यानंतर, थेट प्रवेश त्रुटी `error code: 522` नोंदवेल

यावेळी, तुम्हाला HTTP राउटिंग → वर्कर्स राउटिंग वापरणे आवश्यक आहे मार्ग बांधण्यासाठी, खाली दर्शविल्याप्रमाणे
