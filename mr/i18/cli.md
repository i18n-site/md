# कमांड लाइन पॅरामीटर्सचे तपशीलवार स्पष्टीकरण

## फाइल्स स्वच्छ `-p`

`-p` `--purge` प्रत्येक अनुवाद निर्देशिकेत अस्तित्वात असलेल्या परंतु स्त्रोत भाषा निर्देशिकेत अस्तित्वात नसलेल्या फाइल्स साफ करेल.

कारण दस्तऐवज लिहिताना, मार्कडाउन फाइलची नावे अनेकदा ॲडजस्ट केली जातात, परिणामी भाषांतर निर्देशिकेत अनेक जुन्या आणि सोडलेल्या फायली असतात.

इतर भाषेच्या डिरेक्टरीमध्ये हटवल्या जाणाऱ्या फाइल्स साफ करण्यासाठी हे पॅरामीटर वापरा.

## अनुवाद निर्देशिका निर्दिष्ट `-d`

अनुवादित डिरेक्ट्री डिफॉल्ट आहे जेथे वर्तमान फाइल स्थित आहे.

`-d` `--workdir` भाषांतर निर्देशिका निर्दिष्ट करू शकते, जसे की:

```
i18 -d ~/i18n/md
```

## मदत `-h`

`-h` `--help` कमांड लाइन मदत पाहू शकता.
