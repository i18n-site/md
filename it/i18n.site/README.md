# i18n.site: Framework per siti web puramente statici multilingue

`i18n.site` generatore di siti documentali multilingue e puramente statico.

## Prefazione

`i18n.site` è un generatore di siti per la documentazione, nonché un framework per lo sviluppo web.

Un nuovo paradigma di sviluppo web centrato su `MarkDown` e che utilizza componenti front-end per integrare interattività.

Ogni componente front-end è un pacchetto installabile in modo indipendente.

Sulla base della separazione tra front-end e back-end, c'è anche una distinzione tra contenuti statici e dati dinamici.

Il sito che stai visitando [i18n.site](/) è costruito su questo framework (e include sistemi come quello degli utenti, di fatturazione, di iscrizione via email, ecc.).
