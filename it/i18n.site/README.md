# i18n.site: Framework per siti web puramente statici multilingue

`i18n.site` generatore di siti documentali multilingue e puramente statico.

## Prefazione

`i18n.site` è un generatore di siti documentali e un framework per lo sviluppo web.

Un nuovo paradigma di sviluppo web centrato su `MarkDown` e che utilizza componenti front-end per integrare interattività.

Ogni componente front-end è un pacchetto installabile in modo indipendente.

Sulla base della separazione tra front-end e back-end, vi è anche una distinzione tra contenuti statici e dati dinamici.

Il sito che stai visitando [i18n.site](/) è stato costruito su questo framework (e include sistemi come quello utente, di fatturazione, di abbonamento via email, ecc.).
