## Prefazione

`i18n.site` è una mia riflessione sul modello di sviluppo dei siti web.

È un generatore di siti documentali e un framework per lo sviluppo di siti web.

Un nuovo paradigma di sviluppo di siti web, con `MarkDown` al centro e l'uso di componenti front-end per introdurre interattività.

Ogni componente front-end è un pacchetto installabile in modo indipendente.

Sulla base della separazione tra front-end e back-end, c'è anche una separazione tra contenuti statici e dati dinamici.

Il sito che state visitando [i18n.site](/) è costruito su questo framework (e include sistemi come quello degli utenti, di fatturazione, di iscrizione via email, ecc.).
