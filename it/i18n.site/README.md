# i18n.site: framework per siti web puramente statici multilingue

`i18n.site` è un generatore di siti documentali multilingue e puramente statico.

## Prefazione

`i18n.site` è un generatore di siti per la documentazione, nonché un framework per lo sviluppo web.

Un nuovo paradigma di sviluppo web centrato su `MarkDown` e che utilizza componenti front-end per introdurre interattività.

Ogni componente front-end è un pacchetto installabile autonomamente.

Sulla base della separazione tra front-end e back-end, c'è anche una distinzione tra contenuti statici e dati dinamici.

Il sito che stai visitando [i18n.site](/) è costruito su questo framework (e include un sistema utente, un sistema di addebito, un servizio di abbonamento via email, ecc.).
