# Preparare l'ambiente

* Installare e avviare `docker` (agli utenti Mac si consiglia di utilizzare [orbstack](https://orbstack.dev) come runtime per `docker`)

* Installare [bun](https://bun.sh/docs/installation) e [nodejs](https://nodejs.org/en/download/package-manager)

## Inizializzare il progetto

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
