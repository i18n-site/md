## Backend

Il sito della documentazione richiede solo il front-end, non il backend.

Questo backend è quello del sito web i18n.site stesso, comprendente sistemi come utenti, pagamenti, notifiche push, ecc.

* Servizio [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugin [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Libreria di base [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
