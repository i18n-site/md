## Backend

La costruzione del sito dei documenti richiede solo il frontend, non il backend.

Questo backend è quello del sito web i18n.site stesso, comprendente sistemi come utenti, pagamenti, notifiche push, ecc.

* Servizi [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugin [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Librerie di base [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
