# Detalyado a Panangipalawag Kadagiti Parametro Ti Linia Ti Bilin

## `-p` Dagiti File

`-p` wenno `--purge` ket mangikkat kadagiti papeles nga adda iti tunggal maysa a direktorio ti panagipatarus ngem awan iti direktorio ti taudan a pagsasao.

Ngamin, no agsurat kadagiti dokumento, masansan a mabaliwan dagiti nagan ti file ti Markdown, a mangiturong iti adu a daan ken nabaybay-an a file iti direktorio ti panagipatarus.

Usaren daytoy a parametro tapno mangdalus kadagiti papeles a rumbeng a maikkat kadagiti sabali a direktorio ti pagsasao.

## `-d` Ket Mangibaga Ti Direktorio Ti Panagipatarus

Ti naipatarus a direktorio ket agdefault iti direktorio a pakasarakan ti agdama a papeles.

`-d` wenno `--workdir` ket mabalinna nga ikeddeng ti direktorio ti panagipatarus, a kas ti:

```
i18 -d ~/i18n/md
```

## `-h` Kitaen Ti Tulong

`-h` wenno `--help` tapno makita ti tulong ti linia ti bilin.
