
## CNAME a Naisinggalut a Nagan Ti Dominio

Cloudflare Worker ’s domain name is bound through `SSL/TLS` → custom host name CNAME a naisinggalut ti nagan ti domain, ti direkta a panagserrek ket mangipadamag ti biddut `error code: 522`

Iti daytoy a panawen, kasapulam nga usaren ti HTTP a panagruta → Panagruta ti Trabahador tapno mangisilpo ti maysa a ruta, a kas naipakita iti baba
