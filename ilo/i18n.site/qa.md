# Gagangay a Parikut

## Aksidente `Npm` Naikkat `.i18n/V`

Ti historikal a bersion ti `npm` package release ket naidulin iti `.i18n/v/ol/v.hash`

No aksidente nga i-delete `.i18n/v/ol` ti `npm` package ket saan a maipablaak.

Iti daytoy a panawen, umuna a sapulen ti numero ti bersion ti naudi a bersion ti pannakairuar ti proyektom iti [npmjs.com](//npmjs.com) `0.1.9`

Kalpasanna kitaen ti `bash` Create file iti baba.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Imutektekanyo a ti panagtarimaan iti kastoy a wagas ket mapukawna ti pakasaritaan ti papeles, a mamagbalin nga imposible ti in-inut nga i-update ti sumaganad a pannakairuar, ken amin a linaon ket maipakete manen ken mai-upload iti maminsan.
