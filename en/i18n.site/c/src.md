## Back-end

The documentation site requires only the front-end and not the back-end.

This back-end is for the i18n.site itself, including systems for users, payments, and message notifications.

* Services [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugins [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Base Libraries [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
