# i18n.site: A Pure - Static Multi - Language Website Framework

`i18n.site` is a multi - language, pure - static document site generator.

## Preface

`i18n.site` serves as both a document site generator and a website development framework.

It is a new paradigm for website development, with `MarkDown` as the core and using front - end components to introduce interactivity.

Each front - end component is an independently installable package.

Based on the separation of the front - end and the back - end, there is also a separation between static content and dynamic data.

The [i18n.site](/) you are visiting is built on this framework (including the user system, the billing system, email subscription, etc.).
