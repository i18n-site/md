# i18n.site: Pure Static Multi-Language Website Framework

`i18n.site` - A multi-language, purely static document site generator.

## Preface

`i18n.site` is not only a document site generator, but also a website development framework.

A new paradigm of website development that takes `MarkDown` as the core and uses front-end components to inject interactivity.

Each front-end component is a package that can be independently installed.

Based on the separation of the front-end and back-end, there is also a separation of static content and dynamic data.

The [i18n.site](/) that you are visiting is built based on this framework (including the user system, billing system, email subscription, etc.).
