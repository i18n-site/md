# i18n.site: A Pure Static Multilingual Website Framework

`i18n.site` – A multilingual, purely static documentation site generator.

## Preface

`i18n.site` serves as both a document site generator and a comprehensive website development framework.

An innovative website development paradigm centered around `Markdown`, enhanced with interactive front-end components.

Every front-end component is available as an independently installable package.

Beyond the separation of front-end and back-end, it also achieves a division between static content and dynamic data.

What you are visiting [i18n.site](/) is built based on this framework (including user system, billing system, email subscription, etc.).
