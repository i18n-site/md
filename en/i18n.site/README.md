## Preface

`i18n.site` is my rethinking of the website development model.

This is a document site generator and a website development framework.

A new paradigm of website development that is centered around `MarkDown` and uses front-end components to inject interactivity.

Each front-end component is a package that can be installed independently.

On the basis of the separation of front-end and back-end, there is also the separation of static content and dynamic data.

What you are visiting [i18n.site](/) is built based on this framework (including user system, billing system, email subscription, etc.).
