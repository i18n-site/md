## Preface

`i18n.site` is my rethinking of the website development model.

This is not only a document site generator, but also a website development framework.

A new paradigm of website development that centers around `MarkDown` and uses front-end components to inject interactivity.

Each front-end component is a package that can be independently installed.

Based on the separation of the front-end and back-end, there is also a separation of static content and dynamic data.

The [i18n.site](/) that you are currently visiting is built based on this framework (including the user system, billing system, email subscription, etc.).
