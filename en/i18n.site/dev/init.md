# Prepare the environment

* Install and start `docker` (It is recommended for Mac users to use [orbstack](https://orbstack.dev) as the runtime of `docker` )

* Install [bun](https://bun.sh/docs/installation) and [nodejs](https://nodejs.org/en/download/package-manager)

## Initialize the project

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
