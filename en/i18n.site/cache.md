# Front-end Cache

## Front-end Cache Update Delay

According to [MDN](https://developer.mozilla.org/docs/Web/API/Service_Worker_API) , the maximum update frequency of `service worker` is 24 hours.

i18n.site Cache CDN name in `service worker` So if `cdn` is modified in the configuration, old users will need to wait 24 hours to see the update.
