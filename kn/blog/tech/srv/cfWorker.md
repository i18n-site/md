
## CNAME ಬೌಂಡ್ ಡೊಮೇನ್ ಹೆಸರು

Cloudflare Worker ನ ಡೊಮೇನ್ ಹೆಸರನ್ನು `SSL/TLS` → ಕಸ್ಟಮ್ ಹೋಸ್ಟ್ ಹೆಸರಿನ ಮೂಲಕ ಬಂಧಿಸಲಾಗಿದೆ CNAME ಡೊಮೇನ್ ಹೆಸರನ್ನು ಬಂಧಿಸಿದ ನಂತರ, ನೇರ ಪ್ರವೇಶವು ದೋಷವನ್ನು ವರದಿ ಮಾಡುತ್ತದೆ `error code: 522`

ಈ ಸಮಯದಲ್ಲಿ, ಕೆಳಗೆ ತೋರಿಸಿರುವಂತೆ ಮಾರ್ಗವನ್ನು ಬಂಧಿಸಲು ನೀವು HTTP ರೂಟಿಂಗ್ → ವರ್ಕರ್ಸ್ ರೂಟಿಂಗ್ ಅನ್ನು ಬಳಸಬೇಕಾಗುತ್ತದೆ
