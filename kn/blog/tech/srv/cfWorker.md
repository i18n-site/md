
## CNAME ಬೌಂಡ್ ಡೊಮೇನ್ ಹೆಸರು

Cloudflare Worker ಡೊಮೇನ್ ಹೆಸರನ್ನು ರವಾನಿಸಲಾಗಿದೆ `SSL/TLS` ಕಸ್ಟಮೈಸ್ ಮಾಡಿದ ಹೋಸ್ಟ್ ಹೆಸರು CNAME ಡೊಮೇನ್ ಹೆಸರಿನ ನಂತರ, ನೇರ ಪ್ರವೇಶವು ದೋಷವನ್ನು ವರದಿ ಮಾಡುತ್ತದೆ `error code: 522`

ಈ ಸಮಯದಲ್ಲಿ, ಕೆಳಗೆ ತೋರಿಸಿರುವಂತೆ ಮಾರ್ಗವನ್ನು ಬಂಧಿಸಲು ನೀವು HTTP ರೂಟಿಂಗ್ → ವರ್ಕರ್ಸ್ ರೂಟಿಂಗ್ ಅನ್ನು ಬಳಸಬೇಕಾಗುತ್ತದೆ
