## ಪರಿಚಯ

ವಿಜ್ಞಾನ ಕಾಲ್ಪನಿಕ ಕಾದಂಬರಿ &quot;ತ್ರೀ-ಬಾಡಿ&quot; (ಚೀನೀ ಉಚ್ಚಾರಣೆ: `3Ti` ) ವಿದ್ಯುತ್ಕಾಂತೀಯ ಅಲೆಗಳ ಮೂಲಕ ಸಂವಹನ ನಡೆಸುವ ಅನ್ಯಲೋಕದ ನಾಗರಿಕತೆಯನ್ನು ಕಾಲ್ಪನಿಕಗೊಳಿಸುತ್ತದೆ, ಯಾವುದೇ ಭಾಷೆಯ ಅಡೆತಡೆಗಳಿಲ್ಲ, ಮತ್ತು ತಾಂತ್ರಿಕವಾಗಿ ಸಮೃದ್ಧವಾಗಿದೆ.

ಬೈಬಲ್ · ಜೆನೆಸಿಸ್ :

> ಪ್ರಾಚೀನರು ಆಕಾಶವನ್ನು ತಲುಪುವ, ದೇವರ ದ್ವಾರವನ್ನು ತಲುಪುವ ಮತ್ತು ಮಾನವ ಜನಾಂಗದ ಪ್ರತಿಷ್ಠೆಯನ್ನು ಹೆಚ್ಚಿಸುವ ಗೋಪುರಗಳನ್ನು ನಿರ್ಮಿಸಿದರು.
>
> ಜನರು ತಮ್ಮದೇ ಆದ ಕುಲವನ್ನು ರಚಿಸುತ್ತಾರೆ ಮತ್ತು ಒಂದೇ ಭಾಷೆಯನ್ನು ಮಾತನಾಡುತ್ತಾರೆ ಎಂದು ಭಗವಂತ ಹೇಳಿದನು. ಗೋಪುರವನ್ನು ನಿರ್ಮಿಸುವುದು ಕೇವಲ ಮುನ್ನುಡಿಯಾಗಿದೆ. ಈಗ ಅದು ಮುಗಿದ ನಂತರ, ಅವರು ಭವಿಷ್ಯದಲ್ಲಿ ಎಲ್ಲವನ್ನೂ ಮಾಡುತ್ತಾರೆ.
>
> ಆಗ, ಭಗವಂತ ಬಂದು ಒಬ್ಬರನ್ನೊಬ್ಬರು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಾಧ್ಯವಾಗದೆ ಜನರನ್ನು ವಿವಿಧ ಸ್ಥಳಗಳಲ್ಲಿ ಚದುರಿಸಿದರು.
>
> ಅಂದಿನಿಂದ, ಮಾನವ ವಿವಾದಗಳು ಮುಂದುವರಿದಿವೆ ಮತ್ತು ಜಗತ್ತಿನಲ್ಲಿ ಬಾಬೆಲ್ ಗೋಪುರವಿಲ್ಲ.

ಭೂಮಿಯ ಜನರು ಮೂರು ದೇಹಗಳಂತೆ ಇರಲು, ಭಾಷೆಗೆ ಬದ್ಧರಾಗದೆ ಸಂವಹನ ನಡೆಸಲು ಮತ್ತು ಎಲ್ಲಾ ಮನುಕುಲವನ್ನು ಮತ್ತೆ ಒಂದುಗೂಡಿಸುವ ಸಾಧನವನ್ನು ರಚಿಸಲು ನಾನು ಆಶಿಸುತ್ತೇನೆ.
