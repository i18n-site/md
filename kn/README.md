3Ti.Site, ಬಹು-ಭಾಷಾ ಸ್ಥಿರ ಸೈಟ್ ಜನರೇಟರ್, ಮಾರ್ಕ್‌ಡೌನ್ ಅನ್ನು [ನೂರಕ್ಕೂ ಹೆಚ್ಚು ವಿಭಿನ್ನ ಭಾಷೆಗಳಿಗೆ](https://github.com/i18n-site/node/blob/main/lang/src/index.js) ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅನುವಾದಿಸಬಹುದು.

<img src="//i-01.eu.org/i18n/lang.webp" alt="" />

ಕೆಲವು ಜನರು ಕೇಳಲು ಬಯಸಬಹುದು, ಈಗ ಬ್ರೌಸರ್‌ಗಳು ಅಂತರ್ನಿರ್ಮಿತ ಅನುವಾದ ಕಾರ್ಯಗಳನ್ನು ಹೊಂದಿವೆ, ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ಅಂತರರಾಷ್ಟ್ರೀಯಗೊಳಿಸುವುದು ಅನಗತ್ಯವೇ?

**ಸಂಪೂರ್ಣ ಸೈಟ್ ಅನ್ನು ಅಂತರರಾಷ್ಟ್ರೀಯಗೊಳಿಸುವುದರ ಮೂಲಕ ಮಾತ್ರ ನಾವು ಬಹು-ಭಾಷಾ ಇನ್-ಸೈಟ್ ಪೂರ್ಣ-ಪಠ್ಯ ಹುಡುಕಾಟ ಮತ್ತು ಹುಡುಕಾಟ ಎಂಜಿನ್ ಆಪ್ಟಿಮೈಸೇಶನ್ ಅನ್ನು ಬೆಂಬಲಿಸಬಹುದು ಎಂದು** ನಾನು ಹೇಳಲು ಬಯಸುತ್ತೇನೆ.

## ಪರಿಚಯ

ಬೈಬಲ್ · ಜೆನೆಸಿಸ್ :

> ಪ್ರಾಚೀನ ಕಾಲದಲ್ಲಿ, ಭಾಷೆಗಳು ಒಂದಾಗಿರುವಾಗ, ಮಾನವೀಯತೆಯು ಸ್ವರ್ಗಕ್ಕೆ ತಲುಪುವ ಗೋಪುರವನ್ನು ನಿರ್ಮಿಸಿತು, ದೇವರ ಡೊಮೇನ್‌ಗೆ ಒಂದು ಪೋರ್ಟಲ್, ಮನುಕುಲದ ಶಕ್ತಿಯನ್ನು ಘೋಷಿಸಿತು.
>
> ದೇವರು ಘೋಷಿಸಿದನು, "ಮನುಷ್ಯರು ಒಂದೇ ಬುಡಕಟ್ಟಿಗೆ, ಹಂಚಿದ ನಾಲಿಗೆಯೊಂದಿಗೆ, ಮತ್ತು ಈ ಗೋಪುರವು ಕೇವಲ ನಾಂದಿಯಾಗಿದೆ. ಈಗ ಅವರು ತಮ್ಮ ಆಸೆಯನ್ನು ಸಾಧಿಸುತ್ತಾರೆ ಮತ್ತು ಇನ್ನು ಮುಂದೆ ಯಾವುದೇ ಭಯವಿಲ್ಲ."
>
> ಹೀಗಾಗಿ, ದೇವರು ಇಳಿದು, ಜನರನ್ನು ವಿವಿಧ ಮೂಲೆಗಳಿಗೆ ಚದುರಿಸಿದನು, ವಿವಿಧ ಭಾಷೆಗಳನ್ನು ಬಳಸುತ್ತಾನೆ.
>
> ಅಂದಿನಿಂದ, ಮಾನವ ಸಂವಹನವು ಸವಾಲಾಗಿತ್ತು, ವಿವಾದಗಳು ಅಂತ್ಯವಿಲ್ಲ, ಮತ್ತು ಆಕಾಶವನ್ನು ಮುಟ್ಟುವ ಗೋಪುರವಿರಲಿಲ್ಲ.

ವಿಜ್ಞಾನ ಕಾಲ್ಪನಿಕ ಕಾದಂಬರಿ &quot;ತ್ರೀ-ಬಾಡಿ&quot; (ಚೀನೀ ಉಚ್ಚಾರಣೆ: `3Ti` ) ವಿದ್ಯುತ್ಕಾಂತೀಯ ಅಲೆಗಳ ಮೂಲಕ ಸಂವಹನ ನಡೆಸುವ ಅನ್ಯಲೋಕದ ನಾಗರಿಕತೆಯನ್ನು ಕಾಲ್ಪನಿಕಗೊಳಿಸುತ್ತದೆ, ಯಾವುದೇ ಭಾಷೆಯ ಅಡೆತಡೆಗಳಿಲ್ಲ, ಮತ್ತು ತಾಂತ್ರಿಕವಾಗಿ ಸಮೃದ್ಧವಾಗಿದೆ.

ಭೂಮಿಯ ಜನರು ಮೂರು ದೇಹಗಳಂತೆ ಇರಲು, ಭಾಷೆಗೆ ಬದ್ಧರಾಗದೆ ಸಂವಹನ ನಡೆಸಲು ಮತ್ತು ಎಲ್ಲಾ ಮನುಕುಲವನ್ನು ಮತ್ತೆ ಒಂದುಗೂಡಿಸುವ ಸಾಧನವನ್ನು ರಚಿಸಲು ನಾನು ಆಶಿಸುತ್ತೇನೆ.
