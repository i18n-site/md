>
> मनीस हो आपली एक जमात, तांची संस्कृताय आनी जाती एकूच अशें देवान म्हळां. गोपुर बांदप ही फकत एक प्रस्तावना. आतां तुमकां जाय तें साध्य करपाक मेळटा आनी तुमकां भंय दिसचें ना.
>
> तेन्ना देवान येवन एकामेकांक समजूंक शकनाशिल्ल्यान लोकांक वेगवेगळ्या सुवातींनी शिंपडायले.
>
> तेन्नासावन मनीसजातीन संवाद सादपाक कश्ट घेतले, संघर्श अशेतरेन निर्माण जाले आनी मळबांत पावपी गोपुर संवसाराक आनीक दिसलो ना.

&quot;तीन-शरीर&quot; (चीनी उच्चार: `3Ti` ) ही विज्ञान कथा कादंबरी विद्युत् चुंबकीय ल्हारां वरवीं संवाद सादपी, भाशेची आडमेळीं नाशिल्लीं आनी तंत्रीक नदरेन समृध्द अशी परकी संस्कृताय काल्पनीक करता.

ह्या साधनाच्या आदारान धर्तरेचे लोक तीन कुडीच्या मनशां भशेन जातले, भाशेन संवादाचेर मर्यादा येवची ना आनी सगळी मनीसजात परत एकवटीत जातली अशी आस्त बाळगता.
