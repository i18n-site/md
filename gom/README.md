## वळख

&quot;तीन कुडी&quot; (चीनी उच्चार: `3Tǐ` ) ही विज्ञान कथा कादंबरी पारदर्शक विचार आनी समृध्द तंत्रज्ञानान, संवाद करपाक विद्युत् चुंबकीय तरंग वापरपी परकी संस्कृतायेची काल्पनीकीकरण करता.

बायबल · उत्पत्ती :

> त्या युगांत मळबाच्या सकयल आशिल्ल्या सगळ्यांचे आवाज आनी उतरां एकवरीच आशिल्लीं.
>
> तांणी मळबांतल्यान उंचाये मेरेन पावपी, देवाच्या दारांक स्पर्श करपी, काळजां एकठांय करपी आनी आपल्या वैभवाची अभिमान बाळगून एक गोपुर बांदलो.
>
> प्रभून घोशणा केली: जेन्ना लोक भास आनी वंश वांटून घेतात तेन्ना ते एक एक जमात जातात. गोपुर हो फकत सुरवात; फुडल्या दिसांनी तांकां मेळचें ना अशें कांयच ना.
>
> अशे तरेन तो सकयल देंवलो आनी लोकांक पयस पयस शिंपडून गेले, तांचीं भास आतां एकामेकांक समजूंक शकनाशिल्लीं.
>
> तेन्नासावन झगडीं सोंपनाशिल्लीं आनी मळबाक स्पर्श करपी गोपुरां संवसाराक आनीक दिसूंक नाशिल्लीं.

धर्तरेचे लोक तीन कुडीच्या मनशां भशेन जावंक शकतले, भाशेक बांदून नासतना संवाद सादूंक शकतले आनी सगळो मनीस परतून एकवटीत जावंक शकता अशीं साधनां तयार करपाची आस्त बाळगता.
