# Kufotokozera Mwatsatanetsatane Magawo a Mzere Wa Lamulo

## `-P` Mafayilo

`-p` Kapena `--purge` idzayeretsa mafayilo omwe amapezeka muzomasulira zilizonse koma mulibe m'ndandanda wa chinenero choyambira.

Chifukwa polemba zikalata, mayina a fayilo ya Markdown nthawi zambiri amasinthidwa, zomwe zimapangitsa kuti mafayilo akale akale komanso osiyidwa m'ndandanda yomasulira.

Gwiritsani ntchito chizindikirochi kuti muyeretse mafayilo omwe akuyenera kuchotsedwa m'zilankhulo zina.

## `-D` Chikwatu Chomasulira

Chikwatu chomwe chamasuliridwa chimasinthidwa kukhala chikwatu komwe kuli fayilo yomwe ilipo.

`-d` Kapena `--workdir` ikhoza kufotokoza zomasulira, monga:

```
i18 -d ~/i18n/md
```

## Onani Thandizo `-H`

`-h` Kapena `--help` kuwona thandizo la mzere wolamula.
