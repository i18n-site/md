# i18n.site: Puur statisch meertalig websiteframeWork

`i18n.site` is een meertalige, puur statische documentatiegenerator.

## Inleiding

`i18n.site` is een documentensite-generator en tevens een framework voor websiteontwikkeling.

Een nieuw websiteontwikkelingsparadigma dat `MarkDown` als kern gebruikt en front-endcomponenten inzet om interactie toe te voegen.

Elke front-endcomponent is een afzonderlijk te installeren pakket.

Bovenop de scheiding van front-end en back-end is er ook een scheiding tussen statische content en dynamische data.

Die je momenteel bezoekt [i18n.site](/) is opgebouwd met dit frameWork (inclusief gebruikerssysteem, betalingssysteem, e-mailnieuwsbrief, enz.).
