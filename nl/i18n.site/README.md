# i18n.site: Puur statisch meertalig websiteframeWORK

`i18n.site` is een meertalige, puur statische documentatiegenerator.

## Inleiding

`i18n.site` is niet alleen een documentatiegenerator, maar ook een websiteontwikkelingsframeWORK.

Een nieuw paradigma voor websiteontwikkeling, waarbij `Markdown` centraal staat en front-endcomponenten worden gebruikt om interactie toe te voegen.

Elke front-endcomponent is een afzonderlijk te installeren pakket.

Bovenop de scheiding van front-end en back-end, is er ook een scheiding tussen statische inhoud en dynamische data.

Dit dat u momenteel bezoekt, [i18n.site](/), is opgebouwd met dit frameWORK (en bevat onder andere een gebruikerssysteem, facturatiesysteem, e-mailnieuwsbrief, enz.).
