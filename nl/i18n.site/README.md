# i18n.site: Puur statisch meertalig websiteframeWORK

`i18n.site` meertalig, puur statisch documentatiegenerator.

## Inleiding

`i18n.site` is een documentensite-generator en tevens een framework voor websiteontwikkeling.

Een nieuw paradigma voor websiteontwikkeling, met `Markdown` als kern en front-endcomponenten die interactie toevoegen.

Elke front-endcomponent is een afzonderlijk te installeren pakket.

Bovenop de scheiding van front-end en back-end, is er ook een scheiding tussen statische inhoud en dynamische data.

Wat u momenteel bezoekt, [i18n.site](/), is opgebouwd met dit framework (inclusief gebruikerssysteem, betalingssysteem, e-mailnieuwsbrief, enz.).
