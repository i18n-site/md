
## CNAME බන්ධිත වසම් නාමය

Cloudflare Worker ගේ වසම් නාමය `SSL/TLS` → අභිරුචි සත්කාරක නාමය හරහා බැඳී ඇත CNAME වසම් නාමය බැඳීමෙන් පසු, සෘජු ප්රවේශය දෝෂයක් වාර්තා කරයි `error code: 522`

මෙම අවස්ථාවේදී, පහත දැක්වෙන පරිදි, මාර්ගයක් බැඳීමට ඔබ HTTP routing → Workers routing භාවිතා කළ යුතුය.
