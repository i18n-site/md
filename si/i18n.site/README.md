`MarkDown` කේන්ද්රය ලෙස ගන්නා සහ අන්තර්ක්රියාකාරීත්වය එන්නත් කිරීම සඳහා ඉදිරිපස සංරචක භාවිතා කරන වෙබ් අඩවි සංවර්ධනයේ නව ආදර්ශයකි.

සෑම ඉදිරිපස අංගයක්ම ස්වාධීනව ස්ථාපනය කළ හැකි පැකේජයකි.
