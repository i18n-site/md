# Parámetros De Línea De Comandos Ukanakat Sum Qhanañcht’aña

## `-P` Q’uma Archivonaka

`-p` Jan ukax `--purge` sapa jaqukipañ directorio ukan utjki ukanak q’umachañapawa ukampis janiw phuqhat aru directorio ukan utjkiti.

Kunatix qillqatanaka qillqt’asax Markdown qillqatan sutinakapax walja kutiw mayjt’ayata, ukat walja nayra ukat jaytjata qillqatanakaw jaqukipañ directorio ukanx utji.

Aka parámetro ukampixa yaqha aru directorios ukanakana chhaqtayañatakixa qillqatanaka q’umachañawa.

## `-D` Jaqukipaña Directorio Uñt’ayaña

Jaqukipata directorio ukaxa nayraqata uñt’ayatawa kawkhantixa jichha qillqataxa utjki ukaru.

`-d` Jan ukax `--workdir` jaqukipañ directorio uñacht’ayaspawa, sañäni:

```
i18 -d ~/i18n/md
```

## `-H`

`-h` Jan ukax `--help` kamachin yanapt'a uñakipt'asmawa.
