
## ناوی دۆمەینی بەستراو بە CNAME

Cloudflare Worker ’s domain name is bound through `SSL/TLS` → custom host name CNAME ئەوەی ناوی دۆمەینەکە بەستراوەتەوە، دەستگەیشتن بە ڕاستەوخۆ `error code: 522` ڕاپۆرت دەکات

لەم کاتەدا پێویستە HTTP routing → Workers routing بەکاربهێنیت بۆ بەستنەوەی ڕێڕەوێک، وەک لە خوارەوە نیشان دراوە
