# کێشەی باو

## بە هەڵە سڕاوەتەوە `.i18n/v` بووە هۆی ئەوەی `npm` پاکێجەکە بڵاونەبێتەوە

`.i18n/v/ol/v.hash` مێژوویی بڵاوکردنەوەی `npm` پاکێجەکە لە !

ئەگەر بە هەڵە بسڕیتەوە `.i18n/v/ol` `npm` پاکێجەکە بڵاو نابێتەوە.

لەم کاتەدا سەرەتا ژمارەی وەشانی دوا وەشانی بڵاوکردنەوەی پڕۆژەکەت `0.1.9` [npmjs.com](//npmjs.com)

پاشان ئاماژە بە `bash` دروستکردنەکە بکە لە خوارەوە.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

تێبینی بکە کە چاککردنەوە بەم شێوەیە مێژووی پەڕگەکان لەدەست دەدات، ئەمەش وا دەکات کە نوێکردنەوەی زیادەڕۆیی وەشانی داهاتوو مەحاڵ بێت و هەموو ناوەڕۆکەکان جارێک پاکەت دەکرێنەوە و باردەکرێن.
