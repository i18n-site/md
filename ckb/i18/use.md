# دانان و بەکارهێنان

## ویندۆز سەرەتا git bash دابمەزرێنێت

windows ، تکایە [کلیک لێرە بکە بۆ دابەزاندن و دامەزراندنی سەرەتا `git bash`](https://git-scm.com/download/win)

ئۆپەراسیۆنەکانی دواتر لە `git bash`

## دامەزراندن

```sh
bash <(curl -sS https://i.i18n.site) i18
```

### ڕێکخستنی نیشانەی وەرگێڕان

سەردان [i18n.site/token](//i18n.site/token) کلیک بکە بۆ کۆپیکردنی تۆکن

<img src="https://p.3ti.site/1719911689.avif" style="max-width:400px">

دروستکردن `~/.config/i18n.site.yml` ناوەڕۆکی کۆپیکراوی تێدا لێبدە، ناوەڕۆکەکە بەم شێوەیە دەبێت:

```
token: YOUR_API_TOKEN
```

جگە لەوەش پێویستە کارتی بانکیی [i18n.site/payBill](//i18n.site/payBill) بۆ پارەدان (پێویست بە بارگاویکردنەوە ناکات، ماڵپەڕەکە بە شێوەیەکی ئۆتۆماتیکی کرێ بەپێی بەکارهێنان دەبڕێت، [بۆ زانینی نرخ سەیری ماڵپەڕی سەرەکی بکە](/#price) ).

## بەکارهێنان

### پڕۆژەی دیمۆ

تکایە سەردانی پڕۆژەی دیمۆ [github.com/i18n-site/demo.i18](//github.com/i18n-site/demo.i18) بۆ فێربوونی ڕێکخستنی وەرگێڕان `i18`

بەکارهێنەران لە چین دەتوانن کلۆن بکەن [atomgit.com/i18n/demo.i18](//atomgit.com/i18n/demo.i18)

دوای کلۆنکردن، دایرێکتۆرەکە داخڵ بکە و `i18`

### پێکهاتەی بەڕێوەبەرایەتی

پێکهاتەی بەڕێوەبەرایەتی کۆگای قاڵبەکان بەم شێوەیەی خوارەوەیە

```
┌── .i18n
│  └── conf.yml
└── en
   ├── _IgnoreDemoFile.md
   ├── i18n.yml
   └── README.md
```

بەڕێوەبەرایەتی `en`

### وەرگێڕان بەڕێوەبەرە

دایرێکتۆرەکە داخڵ بکە و `i18`

### زیادکردنی فایلەکان بۆ کۆگاکە

جگە لە وەرگێڕان، بەرنامەکە ئەم فایلانەی خوارەوەش دروست دەکات، تکایە زیادیان بکەن بۆ کۆگاکە.

```
.i18n/hash
.i18n/cache/.gitignore
```

: نێوانیاندا ناوەڕۆکی `.i18n/cache/.gitignore`

```
**/*
!**/.gitignore
```

ئەمەش واتە هەموو فایلەکانی ناو `.i18n/cache/` پشتگوێ بخە (جگە لە `.i18n/cache/.gitignore` ).

ئەگەر نەرمەکاڵای کۆنترۆڵکردنی وەشانی تۆ نییە `git` تکایە بەپێی ئەم ڕێکخستنە پشتگوێی بخە.

## فایلێکی ڕێکخستن

`.i18n/conf.yml` فایلە ڕێکخستنی ئامرازی وەرگێڕانی `i18` هێڵی فەرمانە

ناوەڕۆکەکەی بەم شێوەیەیە:

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### & سەرچاوە زمانی وەرگێڕان

لە فایلە ڕێکخستنەکەدا، ژێردەستەکانی `fromTo`

`en` سەرچاوەیە، `zh ja ko de fr` زمانی ئامانجی وەرگێڕانە.

کۆدی زمان سەیری بکە [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

بۆ نموونە ئەگەر دەتەوێت چینی وەربگێڕیت بۆ زمانی ئینگلیزی، ئەم دێڕە بنووسە `zh: en`

ئەگەر دەتەوێت وەرگێڕان بکەیت بۆ هەموو زمانە پشتگیریکراوەکان، تکایە دوای `:` بۆ نموونە

```
i18n:
  fromTo:
    en:
```

### فایلەکە پشتگوێ بخە

بە شێوەیەکی بنەڕەتی، هەموو ئەو فایلانەی بە `.md` و `.yml` دەست پێدەکەن لە ناوەڕۆکی زمانی سەرچاوەدا وەرگێڕدراون.

ئەگەر بتەوێت هەندێک فایل پشتگوێ بخەیت و وەریان نەگێڕیت (وەک ڕەشنووسی تەواونەکراو)، دەتوانیت ڕێکخستنی مەیدانی `ignore`

ڕستەسازییەکی هاوشێوەی `ignore` [globset](https://docs.rs/globset/latest/globset/#syntax) `.gitignore`

بۆ نموونە لە فایلە ڕێکخستنەکەی سەرەوەدا `_* ` واتە ئەو فایلانەی کە بە `_` دەست پێدەکەن وەرناگیرێن.

## یاساکانی وەرگێڕان

### دەستکاریکەرانی وەرگێڕان نابێت دێڕ زیاد بکەن یان بسڕنەوە

وەرگێڕانەکە دەتوانرێت دەستکاری بکرێت. دەستکاری دەقی ئەسڵی بکە و دووبارە بە ئامێر وەریبگێڕە، دەستکارییە دەستییەکانی وەرگێڕانەکە لەسەری نانووسرێتەوە (ئەگەر ئەم بڕگەیەی دەقی ئەسڵی دەستکاری نەکرابێت).

بەڵام تکایە ئاگاداربن دێڕەکانی وەرگێڕانەکە و دەقی ئەسڵی دەبێت یەک بە یەک بگونجێت. واتە لە کاتی کۆکردنەوەی وەرگێڕانەکەدا دێڕەکان زیاد مەکەن یان مەسڕنەوە. ئەگەرنا دەبێتە هۆی سەرلێشێواوی لە کاشی دەستکاریکردنی وەرگێڕان.

ئەگەر شتێک هەڵە بوو، تکایە سەردانی [پرسیارە بەردەوامەکان بکە بۆ چارەسەرەکان.](/i18/qa#H1)

### `YAML` وەرگێڕان

ئامرازی هێڵی فەرمان هەموو ئەو پەڕگانە دەدۆزێتەوە کە بە `.yml`

* تێبینی بکە کە پاشگری ناوی پەڕگە دەبێت `.yml` (نەک `.yaml` ).

ئامرازەکە تەنها بەهاکانی فەرهەنگ بە `.yml`

بۆ نموونە `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

بەم شێوەیە وەردەگێڕدرێت `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

هەروەها دەتوانرێت بە دەستی وەرگێڕانی `YAML`

لەسەر `YAML` وەرگێڕان، دەتوانیت بە ئاسانی چارەسەری نێودەوڵەتی بۆ زمانە جۆراوجۆرەکانی بەرنامەسازی دروست بکەیت.

## بەکارهێنانی پێشکەوتوو

### ژێر بەڕێوەبەرایەتی وەرگێڕان

تا ئەو کاتەی کە دروست `.i18n/conf.yml` (پێویست ناکات هەموو جارێک لە قاڵبی پڕۆژەی دیمۆوە دەست پێ بکەیت)، `i18` بە باشی کاردەکات.

ئامرازی هێڵی فەرمان ڕێکخستنی `.i18n/conf.yml` لە هەموو ژێر بەڕێوەبەرایەتییەکان دەدۆزێتەوە و وەریدەگێڕێت.

ئەو پڕۆژانەی کە بیناسازی [monorepo](//monorepo.tools)

<img alt="" src="https://p.3ti.site/1719910016.avif">

### بەڕێوەبەرایەتی دامەزراندنی تایبەتمەند

بە شێوازی پێشوەختە بۆ `/usr/local/bin`

`/usr/local/bin` مۆڵەتی نووسینی نەبێت دادەمەزرێنرێت بۆ `~/.bin` .

`TO` ژینگەییەکان دابنێ دەتوانیت بەڕێوەبەرایەتی دامەزراندنەکە پێناسە بکەیت، بۆ نموونە :

```sh
TO=/bin sudo bash <(curl -sS https://i.i18n.site) i18
```
