# Агульная Праблема

## Выпадкова Выдалены `.i18n/V` У Выніку Чаго Пакет `Npm` Не Быў Апублікаваны

Гістарычная версія `npm` выпуску пакета захавана ў `.i18n/v/ol/v.hash`

Калі вы выпадкова выдаліце `.i18n/v/ol` пакет `npm` не будзе апублікаваны.

У гэты час спачатку знайдзіце нумар апошняй версіі вашага праекта ў [npmjs.com](//npmjs.com) `0.1.9`

Затым звярніцеся да `bash` Стварыць файл ніжэй.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Звярніце ўвагу, што выпраўленне такім спосабам прывядзе да страты гісторыі файлаў, што зробіць немагчымым паступовае абнаўленне наступнага выпуску, а ўвесь кантэнт будзе перапакаваны і загружаны адзін раз.
