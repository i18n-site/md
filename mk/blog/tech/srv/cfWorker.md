
## CNAME Врзано Име На Домен

Cloudflare Worker на доменот е врзано преку `SSL/TLS` → приспособено име на домаќин CNAME името на доменот е врзано, директниот пристап ќе пријави грешка `error code: 522`

Во овој момент, треба да користите рутирање HTTP → Работници кои насочуваат за да врзат рута, како што е прикажано подолу
