# Detaillearre Útlis Fan Kommandorigelparameters

## `-P` Clean Triemmen

`-p` Of `--purge` sil bestannen opromje dy't besteane yn elke oersettingsmap, mar net bestean yn de boarnetaalmap.

Want by it skriuwen fan dokuminten wurde Markdown-bestânnammen faak oanpast, wat resulteart yn in protte âlde en ferlitten bestannen yn 'e oersettingsmap.

Brûk dizze parameter om triemmen op te romjen dy't wiske wurde moatte yn oare taalmappen.

## `-D` Spesifisearje Oersetmap

De oersette map stiet standert yn de map dêr't de aktuele triem leit.

`-d` Of `--workdir` kin de oersettingsmap opjaan, lykas:

```
i18 -d ~/i18n/md
```

## `-H` Sjoch Help

`-h` Of `--help` kin de help fan de kommandorigel besjen.
