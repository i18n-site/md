# Kongamano La Umbizo La MarkDown

## Uhakika Wa Nanga

Nakala `MarkDown` kitamaduni hutolewa kwa msingi wa maandishi katika hali ya lugha nyingi, suluhisho hili haliwezekani.

`i18n.site` Suluhisho la nanga lililokubaliwa ni kuingiza maandishi sawa na `<a rel=id href="#xxx" id="xxx"></a>` in `MarkDown` ili kuunda mwenyewe nanga ya uwekaji.

`<a rel=id href="#xxx" id="xxx"></a>` , hapa `rel=id` inafafanua mtindo wa ukurasa wa sehemu ya nanga, tafadhali badilisha `xxx` na ufupisho wako halisi wa Kiingereza.

Nanga kawaida huongezwa kwa kichwa, kama vile:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Andika `HTML` Ndani `MarkDown`

`pug` `HTML` inaweza kupachikwa kwenye msimbo.

Maudhui ndani ya kipengele cha `<pre>`

Kwa kuchanganya pointi hizi mbili, unaweza kuandika kwa urahisi `HTML` in `MarkDown`

Unaweza kurejelea utekelezaji katika [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) nambari ni kama ifuatavyo :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Kumbuka kuwa `<style>` pia imefafanuliwa katika `<pre>`
