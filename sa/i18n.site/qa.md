# सामान्यसमस्या

## आकस्मिकतया विलोपितम् `.i18n/V` येन `Npm` संकुलस्य प्रकाशनं विफलम् अभवत्

`npm` package release इत्यस्य ऐतिहासिकं संस्करणं `.i18n/v/ol/v.hash`

यदि भवान् यदृच्छया `.i18n/v/ol` ं विलोपयति तर्हि `npm` संकुलं प्रकाशितं न भविष्यति।

अस्मिन् समये `0.1.9` भवतः परियोजनायाः अन्तिमविमोचनसंस्करणस्य संस्करणसङ्ख्यां [npmjs.com](//npmjs.com)

ततः अधः `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

ध्यानं कुर्वन्तु यत् एतादृशरीत्या मरम्मतेन सञ्चिका-इतिहासः नष्टः भविष्यति, येन अग्रिम-विमोचनं क्रमेण अद्यतनीकरणं असम्भवं भवति, सर्वा सामग्री च एकवारं पुनः संकुलीकरणं अपलोड् च भविष्यति ।
