# MarkDown Formato Nisqa Hatun Huñunakuy

## Ancla Punto

Tradicional `MarkDown`

`i18n.site` Ariñisqa ancla allichayqa `<a rel=id href="#xxx" id="xxx"></a>` in `MarkDown` kaqman rikch'akuq qillqata churaymi, makiwan posición anclata ruwanapaq.

`<a rel=id href="#xxx" id="xxx"></a>` , kaypi `rel=id` ancla puntop p'anqap estilonta riqsichin, ama hina kaspa `xxx` chiqap ancla inglés abreviaturaykiwan tikray.

Yaqa llapanpim tituluman anclakunata yapanku, ahinataq:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Qillqay `HTML` in `MarkDown`

`pug` `HTML` codigo nisqapi churasqa kanman.

`<pre>`

Kay iskay puntokunata huñuspaqa, mana sasachakuspalla `HTML` in `MarkDown`

[github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) codigoqa kayhinam :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Reparay `<style>` nisqapas `<pre>` sut'inchasqa kasqanmanta.
