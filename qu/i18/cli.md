# Kamachiy Chiru Parámetros Nisqamanta Sut'inchay

## `-P` Archivokunata Pichay

`-p` Utaq `--purge` sapa tikray qillqana mayt'upi kaq willañiqikunata pichanqa ichataq mana pukyuta simi qillqana mayt'upichu.

Imaraykuchus qillqakunata qillqaspaqa Markdown willañiqi sutikunaqa sapa kuti allichasqam, chaymi achka mawk'a, saqisqa willañiqikunata tikray willañiqipi rikurin.

Kay tupuywan llamk'achiy huk simi willañiqikunapi qullusqa willañiqikunata pichanaykipaq.

## `-D` T'ikranapaq Qillqana Mayt'u Niyta

T'ikrasqa willañiqiqa maypi kunan willañiqi tarikun chay willañiqiman ñawpaqmantaraqmi.

`-d` Utaq `--workdir` tikray qillqana mayt'uta niyta atin, ahinataq:

```
i18 -d ~/i18n/md
```

## `-H` Yanapakuyta Qhaway

`-h` Utaq `--help` chiru yanapayta qhawayta atinki.
