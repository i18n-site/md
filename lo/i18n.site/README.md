## ຄໍານໍາ

`i18n.site` ແມ່ນການຄິດຄືນໃຫມ່ຂອງຂ້ອຍກ່ຽວກັບຮູບແບບການພັດທະນາເວັບໄຊທ໌.

ນີ້ແມ່ນຕົວສ້າງເວັບໄຊທ໌ເອກະສານແລະກອບການພັດທະນາເວັບໄຊທ໌.

ຮູບແບບໃໝ່ຂອງການພັດທະນາເວັບໄຊທ໌ທີ່ຕັ້ງຢູ່ໃນຈຸດໃຈກາງ `MarkDown`

ແຕ່ລະອົງປະກອບດ້ານຫນ້າແມ່ນຊຸດທີ່ສາມາດຕິດຕັ້ງໄດ້ຢ່າງເປັນເອກະລາດ.

ບົນພື້ນຖານການແຍກຕ່າງຫາກຂອງຫນ້າ - end ແລະ back-end, ຍັງມີການແຍກເນື້ອໃນ static ແລະຂໍ້ມູນແບບເຄື່ອນໄຫວ.

ສິ່ງ​ທີ່​ທ່ານ​ກໍາ​ລັງ​ໄປ​ຢ້ຽມ [i18n.site](/) ​ແມ່ນ​ສ້າງ​ຂຶ້ນ​ໂດຍ​ອີງ​ໃສ່​ໂຄງ​ການ​ນີ້ (ລວມ​ທັງ​ລະ​ບົບ​ຜູ້​ໃຊ້​, ລະ​ບົບ​ໃບ​ບິນ​, ການ​ສະ​ຫມັກ​ອີ​ເມລ​໌​, ແລະ​ອື່ນໆ​)​.
