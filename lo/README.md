## ແນະນຳ

ພຣະຄໍາພີ · Genesis :

> ໃນສະໄໝບູຮານ, ເມື່ອພາສາເປັນອັນໜຶ່ງ, ມະນຸດໄດ້ສ້າງຫໍຄອຍຂຶ້ນສູ່ສະຫວັນ, ເປັນປະຕູສູ່ດິນແດນຂອງພຣະເຈົ້າ, ປະກາດອຳນາດຂອງມະນຸດ.
>
> ພະເຈົ້າ​ປະກາດ​ວ່າ, “ຄົນ​ທັງ​ຫຼາຍ​ສາມັກຄີ​ກັນ​ເປັນ​ເຜົ່າ​ໜຶ່ງ​ດ້ວຍ​ລີ້ນ​ຮ່ວມ, ແລະ​ຫໍຄອຍ​ນີ້​ເປັນ​ແຕ່​ຄຳ​ເວົ້າ​ທີ່​ກ່າວ​ຕໍ່​ໄປ, ບັດ​ນີ້​ພວກ​ເຂົາ​ໄດ້​ບັນລຸ​ຄວາມ​ປາຖະໜາ​ຂອງ​ຕົນ ແລະ​ຈະ​ບໍ່​ມີ​ຄວາມ​ຢ້ານ​ກົວ​ອີກ​ຕໍ່​ໄປ.”
>
> ດັ່ງນັ້ນ, ພຣະເຈົ້າໄດ້ສະເດັດລົງມາ, ກະແຈກກະຈາຍຜູ້ຄົນໄປຫລາຍມຸມ, ໃຊ້ພາສາທີ່ແຕກຕ່າງກັນ.
>
> ຈາກ​ນັ້ນ​ມາ, ການ​ສື່​ສານ​ຂອງ​ມະນຸດ​ກາຍ​ເປັນ​ສິ່ງ​ທ້າ​ທາຍ, ຖົກ​ຖຽງ​ກັນ​ບໍ່​ມີ​ທີ່​ສິ້ນ​ສຸດ, ແລະ​ບໍ່​ມີ​ຫໍ​ຄອຍ​ທີ່​ສຳ​ພັດ​ກັບ​ທ້ອງ​ຟ້າ​ອີກ.

ນະວະນິຍາຍວິທະຍາສາດ &quot;ສາມຮ່າງກາຍ&quot; (ການອອກສຽງພາສາຈີນ: `3Ti` ) fictionalizes ພົນລະເຮືອນມະນຸດຕ່າງດາວທີ່ຕິດຕໍ່ສື່ສານໂດຍຜ່ານຄື້ນແມ່ເຫຼັກໄຟຟ້າ, ບໍ່ມີອຸປະສັກພາສາ, ແລະເຕັກໂນໂລຊີຈະເລີນຮຸ່ງເຮືອງ.

ຂ້າ​ພະ​ເຈົ້າ​ຫວັງ​ວ່າ​ຈະ​ສ້າງ​ເຄື່ອງ​ມື​ທີ່​ຈະ​ເຮັດ​ໃຫ້​ປະ​ຊາ​ຊົນ​ຂອງ​ແຜ່ນ​ດິນ​ໂລກ​ເປັນ​ຄື​ກັບ​ສາມ​ຮ່າງ​ກາຍ, ການ​ສື່​ສານ​ໂດຍ​ບໍ່​ມີ​ການ​ຜູກ​ມັດ​ດ້ວຍ​ພາ​ສາ, ແລະ​ສາ​ມັກ​ຄີ​ມະ​ນຸດ​ທັງ​ຫມົດ​ອີກ​ເທື່ອ​ຫນຶ່ງ.
