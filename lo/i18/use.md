### ໂຄງການສາທິດ

ກະລຸນາເບິ່ງໂຄງການສາ `i18` [github.com/i18n-site/demo.i18](//github.com/i18n-site/demo.i18)

ຜູ້​ໃຊ້​ໃນ​ປະ​ເທດ​ຈີນ​ສາ​ມາດ clone ໄດ້ [atomgit.com/i18n/demo.i18](//atomgit.com/i18n/demo.i18)

ຫຼັງ​ຈາກ cloning​, ເຂົ້າ​ລະ​ບົບ​ແລະ​ດໍາ​ເນີນ​ການ `i18`

### ໂຄງສ້າງໄດເລກະທໍລີ

ໂຄງສ້າງບັນຊີສາງແມ່ແບບມີດັ່ງນີ້

```
│  └── conf.yml

   ├── _IgnoreDemoFile.md
   ├── i18n.yml
   └── README.md
```

ໄດເລກະທໍລີ `en`

ໃສ່​ລະ​ບົບ​ແລະ​ດໍາ​ເນີນ​ການ `i18`

### ເພີ່ມໄຟລ໌ໃສ່ບ່ອນເກັບມ້ຽນ

ນອກເຫນືອຈາກການແປພາສາ, ໂຄງການຍັງຈະສ້າງໄຟລ໌ຕໍ່ໄປນີ້, ກະລຸນາເພີ່ມພວກມັນໃສ່ບ່ອນເກັບມ້ຽນ.

```
.i18n/hash
.i18n/cache/.gitignore
```

ໃນບັນດາພວກເຂົາ, ເນື້ອໃນ : `.i18n/cache/.gitignore`

```
**/*
!**/.gitignore
```

ນີ້ຫມາຍຄວາມວ່າບໍ່ສົນໃຈໄຟລ໌ທັງຫມົດໃນໄດເລກະທໍ `.i18n/cache/` (ຍົກເວັ້ນ `.i18n/cache/.gitignore` ).

ຖ້າ​ຫາກ​ວ່າ​ຊອບ​ແວ​ການ​ຄວບ​ຄຸມ​ສະ​ບັບ​ຂອງ​ທ່ານ​ບໍ່​ແມ່ນ `git`

## ໄຟລ໌ການຕັ້ງຄ່າ

`.i18n/conf.yml` ມັນແມ່ນໄຟລ໌ການຕັ້ງຄ່າຂອງເຄື່ອງມືການແປສາຍ `i18`

ເນື້ອໃນມີດັ່ງນີ້:

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### ພາສາຕົ້ນສະບັບ &

ໃນ​ໄຟລ​໌​ການ​ຕັ້ງ​ຄ່າ​, subordinates ຂອງ `fromTo`

`en` ແມ່ນພາສາທີ່ມາ, `zh ja ko de fr` ເປັນພາສາເປົ້າໝາຍຂອງການແປ.

ລະຫັດພາສາເບິ່ງ [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

ຕົວຢ່າງ: ຖ້າເຈົ້າຕ້ອງການແປພາສາຈີນເປັນພາສາອັງກິດ, ຂຽນເສັ້ນນີ້ຄືນໃໝ່ `zh: en`

ຖ້າ​ຫາກ​ວ່າ​ທ່ານ​ຕ້ອງ​ການ​ທີ່​ຈະ​ແປ​ພາ​ສາ​ທີ່​ສະ​ຫນັບ​ສະ​ຫນູນ​ທັງ​ຫມົດ​, ກະ​ລຸ​ນາ​ປ່ອຍ​ຫວ່າງ​ຫຼັງ​ຈາກ​ນັ້ນ `:` ຍົກ​ຕົວ​ຢ່າງ

```
i18n:
  fromTo:
    en:
```

### ບໍ່ສົນໃຈໄຟລ໌

ໂດຍຄ່າເລີ່ມຕົ້ນ, ໄຟລ໌ທັງໝົດເລີ່ມຕົ້ນດ້ວຍ `.md` ແລະ `.yml` ໃນໄດເຣັກທໍຣີພາສາຕົ້ນສະບັບຈະຖືກແປ.

ຖ້າ​ຫາກ​ວ່າ​ທ່ານ​ຕ້ອງ​ການ​ທີ່​ຈະ​ລະ​ເລີຍ​ບາງ​ໄຟລ​໌​ແລະ​ບໍ່​ໄດ້​ແປ​ພາ​ສາ​ໃຫ້​ເຂົາ​ເຈົ້າ (ເຊັ່ນ​: ຮ່າງ​ທີ່​ຍັງ​ບໍ່​ສໍາ​ເລັດ​)​, ທ່ານ​ສາ​ມາດ​ນໍາ​ໃຊ້​ການ​ຕັ້ງ​ຄ່າ​ພາກ​ສະ​ຫນາມ `ignore` ​.

`ignore` ໃຊ້ syntax ທີ່ຄ້າຍຄືກັບ `.gitignore` [globset](https://docs.rs/globset/latest/globset/#syntax)

ຕົວຢ່າງ, ໃນໄຟລ໌ການຕັ້ງຄ່າຂ້າງເທິງ `_* ` ຫມາຍຄວາມວ່າໄຟລ໌ທີ່ເລີ່ມຕົ້ນດ້ວຍ `_` ຈະບໍ່ຖືກແປ.

## ກົດລະບຽບການແປພາສາ

### ບັນນາທິການການແປບໍ່ຄວນເພີ່ມ ຫຼືລຶບສາຍ

ການແປສາມາດແກ້ໄຂໄດ້. ແກ້ໄຂຂໍ້ຄວາມຕົ້ນສະບັບ ແລະແປດ້ວຍເຄື່ອງຈັກອີກຄັ້ງ, ການດັດແກ້ຄູ່ມືການແປພາສາຈະບໍ່ຖືກຂຽນທັບ (ຖ້າວັກນີ້ຂອງຂໍ້ຄວາມຕົ້ນສະບັບບໍ່ໄດ້ຖືກດັດແກ້).

ແຕ່ກະລຸນາສັງເກດວ່າເສັ້ນຂອງການແປແລະຂໍ້ຄວາມຕົ້ນສະບັບຕ້ອງກົງກັນຫນຶ່ງຫາຫນຶ່ງ. ນັ້ນແມ່ນ, ຢ່າເພີ່ມຫຼືລຶບສາຍໃນເວລາທີ່ລວບລວມການແປພາສາ. ຖ້າບໍ່ດັ່ງນັ້ນ, ມັນຈະເຮັດໃຫ້ເກີດຄວາມສັບສົນໃນ cache ການແກ້ໄຂການແປພາສາ.

ຖ້າມີບາງຢ່າງຜິດພາດ, ກະລຸນາເບິ່ງ [FAQ ສໍາລັບວິທີແກ້ໄຂ.](/i18/qa#H1)

### `YAML`

ເຄື່ອງມືບັນທັດຄໍາສັ່ງຈະຊອກຫາໄຟລ໌ທັງຫມົດທີ່ລົງທ້າຍດ້ວຍ `.yml`

* ໃຫ້ສັງເກດວ່າຊື່ໄຟລ໌ຕ້ອງເປັນ `.yml` (ບໍ່ແມ່ນ `.yaml` ).

ເຄື່ອງມືພຽງແຕ່ແປຄ່າວັດຈະນານຸກົມໃນ `.yml`

ຍົກ​ຕົວ​ຢ່າງ `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

ຈະ​ແປ​ວ່າ `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

ການ​ແປ​ພາ​ສາ​ຂອງ `YAML` ຍັງ​ສາ​ມາດ​ໄດ້​ຮັບ​ການ​ແກ້​ໄຂ​ດ້ວຍ​ຕົນ​ເອງ (ແຕ່​ບໍ່​ໄດ້​ເພີ່ມ​ຫຼື​ລຶບ​ກະ​ແຈ​ຫຼື​ເສັ້ນ​ໃນ​ການ​ແປ​ພາ​ສາ​)​.

ອີງຕາມການແປ `YAML` ສາ, ທ່ານສາມາດສ້າງວິທີແກ້ໄຂສາກົນສໍາລັບພາສາການຂຽນໂປຼແກຼມຕ່າງໆ.
