# Ibisobanuro Birambuye Byumurongo Wumurongo

## Sukura `-P`

`-p` Cyangwa `--purge` izahanagura dosiye ziri muri buri gitabo cyubuhinduzi ariko ntikibaho mububiko bwururimi rwinkomoko.

Kuberako iyo wanditse inyandiko, amazina ya dosiye ya Markdown akunze guhindurwa, bikavamo amadosiye menshi ashaje kandi yataye mububiko bwubusobanuro.

Koresha iyi parameter kugirango usukure dosiye zigomba gusibwa mubindi bikoresho byindimi.

## Kugaragaza Ububiko bw'Ubusobanuro `-D`

Ububiko bwahinduwe bwahinduwe mububiko aho dosiye iri.

`-d` Cyangwa `--workdir` irashobora kwerekana ububiko bwubusobanuro, nka:

```
i18 -d ~/i18n/md
```

## Reba Reba Ubufasha `-H`

`-h` Cyangwa `--help` kureba umurongo wumurongo ubufasha.
