
## CNAME Ihuza Izina Ryizina

Izina rya domaine izina ryiziritse kuri `SSL/TLS` name izina ryabakiriye CNAME yizina rya domaine rihujwe, kwinjira bitaziguye bizatanga ikosa `error code: 522` Cloudflare Worker

Muri iki gihe, ugomba gukoresha inzira ya HTTP → Abakozi bayobora guhuza inzira, nkuko bigaragara hano hepfo
