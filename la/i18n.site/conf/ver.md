# Project Version

Demo project exemplum:

`en/demo2/v` est numerus emendationis documenti, qui ad dextram project nominis in forma laterali ostendetur.

<img src="https://p.3ti.site/1721290486.avif" width="320px">

`.i18n/conf.yml` `en/`

Si fons tuus lingua Latina non est, `v` in indice indicis fontis linguae tuae collocandus est. 

Facultas litterarum versiones historicas percurrere sub evolutione est.

Commendatur solum modificare numerum documenti versionis cum maiora updates solvens (qualis est `v1` , `v2` ) ad vitandas paginas premens per tormenta quaesita ob nimium numeros versionis.

## Uti Vacua `V`

In demo project, praeter `en/demo2/v` , etiam videre potes contenta `en/blog` et `en/demo1` directoria vacua sunt `v` lima.
