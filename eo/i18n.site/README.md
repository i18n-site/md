# i18n.site: Pura Senmova Plurlingva Retejo-Kadro

`i18n.site` Plurlingva, pure statika dokumentejo-generatoro.

Ĝi estas nuntempe disponebla komence, sed estas ankoraŭ gravaj funkcioj en disvolviĝo. Bonvolu vidi [la disvolvan planon](/i18n.site/c/roadmap) por detaloj.

Bonvolu <button onclick="mailsub()">klaki ĉi tiun retpoŝton por aboni produktajn ĝisdatigojn</button> kaj <button onclick="webpush()">ebligi retumilon push</button> Ni sciigos vin kiam produktaj ĝisdatigoj estos faritaj.

Ankaŭ bonvenon sekvi / [X.COM: @i18nSite](https://x.com/i18nSite) [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social)

## Antaŭparolo

`i18n.site` estas mia repripensado de la retejo-disvolva modelo.

Ĉi tio estas dokumenteja generatoro kaj retejo-disvolva kadro.

Nova paradigmo de retejo-disvolviĝo, kiu centras `MarkDown` kaj uzas antaŭajn komponantojn por injekti interagadon.

Ĉiu antaŭa komponanto estas pakaĵo instalebla sendepende.

Surbaze de la disiĝo de front-end kaj back-end, ekzistas ankaŭ la apartigo de statika enhavo kaj dinamika datumoj.

Kion vi vizitas [i18n.site](/) estas konstruita surbaze de ĉi tiu kadro (inkluzive de uzantsistemo, faktura sistemo, retpoŝta abono, ktp.).
