# Ravekirina Berfireh a Parametreyên Rêzika Fermanê

## Pelên Paqij `-P`

An jî `--purge` dê pelên ku di her pelrêça wergerê de hene, lê di pelrêça zimanê çavkanî de nînin, paqij bike `-p`

Ji ber ku dema nivîsandina belgeyan, navên pelên Markdown bi gelemperî têne sererast kirin, di pelrêça wergerê de gelek pelên kevn û berdayî çêdibin.

Vê pîvanê bikar bînin ku pelên ku divê di pelrêçên zimanên din de werin jêbirin paqij bikin.

## Peldanka Wergerê Diyar `-D`

Pelrêça wergerî li pelrêça ku pelê heyî lê ye, xwemalî dike.

An `-d` `--workdir` dikare pelrêça wergerê diyar bike, wek:

```
i18 -d ~/i18n/md
```

## Alîkariyê `-H`

An `-h` `--help` dikarin alîkariya xeta fermanê bibînin.
