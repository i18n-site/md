# Pirsgirêka Hevpar

## Bi Xeletî Hat Jêbirin `.i18n/V` Dibe Sedem Ku `Npm` Pakêt Neyê Weşandin

Guhertoya dîrokî ya `npm` serbestberdana pakêtê tê tomarkirin `.i18n/v/ol/v.hash`

Heke hûn bi xeletî jêbirin `.i18n/v/ol` `npm` pakêt nayê weşandin.

Di vê demê de, pêşî jimara guhertoya guhertoya paşîn a projeya xwe bibînin [npmjs.com](//npmjs.com) mînakî `0.1.9`

Dûv re `bash` li pelê li jêr biafirînin.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Bala xwe bidinê ku tamîrkirina bi vî rengî dê dîroka pelê winda bike, nekare nûvekirina paşîn bi zêdeyî nûve bike, û hemî naverok dê carekê were pakkirin û barkirin.
