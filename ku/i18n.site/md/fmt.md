# Peymana Formata MarkDown

## Xala Lengerê

Ankerên `MarkDown` li ser bingeha naveroka nivîsê têne çêkirin Di doza pirzimanî de, ev çareserî ne pêkan e.

Çareseriya `i18n.site` ya pejirandî ew e ku metnek mîna `<a rel=id href="#xxx" id="xxx"></a>` li `MarkDown` têxin da ku bi desta lengerê pozîsyonê biafirînin.

`<a rel=id href="#xxx" id="xxx"></a>` li vir `rel=id` şêwaza rûpela xala lengerê diyar dike, ji kerema xwe `xxx` bi kurtenivîsa xweya rastîn a îngilîzî biguherîne.

Anker bi gelemperî li sernavê têne zêdekirin, wekî:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## `HTML` Binivîse `MarkDown`

`pug` `HTML`

Naveroka di `<pre>` hêmanê de nayê wergerandin.

Bi berhevkirina van her du xalan, hûn dikarin bi hêsanî `HTML` binivîsin `MarkDown` ku bigihîjin bandorên cihêreng.

Hûn dikarin li ser pêkanînê binihêrin [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) kod wiha ye :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Bala xwe bidinê ku `<style>` jî li `<pre>` hatiye diyarkirin.
