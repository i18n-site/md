# Inkinga Evamile

## Isuswe Ngengozi `.i18n/V` Okubangele Ukuthi Iphakheji Engu- `Npm` Yehluleke Ukushicilelwa

`.i18n/v/ol/v.hash` Inguqulo yomlando yokukhishwa kwephakheji engu- `npm` ilondolozwe ku-!

Uma ususa ngephutha `.i18n/v/ol` iphakheji engu- `npm` ngeke ishicilelwe.

Ngalesi sikhathi, qala uthole inombolo yenguqulo yenguqulo yokugcina yephrojekthi yakho `0.1.9` [npmjs.com](//npmjs.com)

Bese ubhekisela `bash` Dala ifayela ngezansi.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Qaphela ukuthi ukulungisa ngale ndlela kuzolahlekelwa umlando wefayela, okuzokwenza kungenzeki ukubuyekeza ngokwandayo ukukhishwa okulandelayo, futhi konke okuqukethwe kuzopakishwa kabusha futhi kulayishwe kanye.
