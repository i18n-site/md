# Umhlangano Wefomethi Ye-MarkDown

## Iphoyinti Lehange

Okuvamile `MarkDown`

`i18n.site` Isixazululo okuvunyelwene ngaso siwukufaka umbhalo ofana no- `<a rel=id href="#xxx" id="xxx"></a>` ku- `MarkDown` ukuze udale mathupha ihange lokuma.

`<a rel=id href="#xxx" id="xxx"></a>` , lapha `rel=id` ichaza isitayela sekhasi lephoyinti okusetshenzwa kulo, sicela ushintshe `xxx` ngesifinyezo sakho sangempela sesiNgisi.

Amahange avamise ukungezwa esihlokweni, njengokuthi:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Bhala `HTML` Ku- `MarkDown`

`pug` `HTML` ingashumekwa kukhodi.

Okuqukethwe ngaphakathi kwe `<pre>` elementi ngeke kuhunyushwe.

Ukuhlanganisa lawa maphuzu amabili, ungabhala kalula `HTML` ku `MarkDown`

Ungabhekisa ekusetshenzisweni ku- [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) ikhodi imi kanje :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Qaphela ukuthi i- `<style>` ibuye ichazwe ku- `<pre>`
