# Detalyadong Paliwanag Ng Mga Parameter Ng Command Line

## `-P` Ng Mga File

`-p` O `--purge` ay maglilinis ng mga file na umiiral sa bawat direktoryo ng pagsasalin ngunit wala sa direktoryo ng pinagmulang wika.

Dahil kapag nagsusulat ng mga dokumento, ang mga pangalan ng Markdown file ay madalas na nababagay, na nagreresulta sa maraming luma at inabandunang mga file sa direktoryo ng pagsasalin.

Gamitin ang parameter na ito upang linisin ang mga file na dapat tanggalin sa ibang mga direktoryo ng wika.

## `-D` Tukuyin Ang Direktoryo Ng Pagsasalin

Nagde-default ang isinaling direktoryo sa direktoryo kung saan matatagpuan ang kasalukuyang file.

`-d` O `--workdir` ay maaaring tukuyin ang direktoryo ng pagsasalin, tulad ng:

```
i18 -d ~/i18n/md
```

## `-H` Ang Tulong

`-h` O `--help` mong tingnan ang tulong sa command line.
