
## CNAME Jikọtara Ngalaba Aha

Cloudflare Worker 's domain name isbound through `SSL/TLS` → custom host name CNAME aha ngalaba ahụ kechara, ohere ozugbo ga-akọ njehie `error code: 522`

N'oge a, ịkwesịrị iji HTTP routing → Ndị ọrụ na-ebugharị iji jikọta ụzọ, dị ka egosiri n'okpuru.
