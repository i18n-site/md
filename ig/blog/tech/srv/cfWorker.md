
## CNAME Jikọtara Ngalaba Aha

Cloudflare Worker ngalaba ahụ `SSL/TLS` → aha njirimara ahaziri ejikọta CNAME Mgbe aha ngalaba ahụ gachara, ịnweta ozi ga-akọ njehie `error code: 522`

N'oge a, ịkwesịrị iji HTTP routing → Ndị ọrụ na-ebugharị iji jikọta ụzọ, dị ka egosiri n'okpuru.
