# Mgbakọ Usoro MarkDown

## Ebe Arịlịka

Omenala `MarkDown` A na-emepụta arịlịka dabere na ọdịnaya ederede n'ihe gbasara ọtụtụ asụsụ, ngwọta a agaghị ekwe omume.

`i18n.site` ngwọta arịlịka ekwekọrịtara bụ itinye ederede yiri `<a rel=id href="#xxx" id="xxx"></a>` in `MarkDown` iji jiri aka mepụta arịlịka ọnọdụ.

`<a rel=id href="#xxx" id="xxx"></a>` , ebe a `rel=id` na-akọwa ụdị ibe nke ebe arịlịka, biko jiri mbiri okwu bekee dochie `xxx` .

A na-agbakwunyekarị arịlịka na aha, dị ka:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Dee `HTML` Na `MarkDown`

Enwere `pug` itinye `HTML` na koodu.

A gaghị atụgharị asụsụ ọdịnaya dị n'ime `<pre>`

Ijikọta isi ihe abụọ a, ị nwere ike dee `HTML` `MarkDown` ngwa ngwa iji nweta mmetụta ngosi dị iche iche.

Ị nwere ike na-ezo aka na mmejuputa iwu na [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) koodu dị ka ndị a :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Rịba ama na a kọwakwara `<style>` na `<pre>`
