
## Enw Parth Rhwymedig CNAME

Cloudflare Worker enw parth wedi'i rwymo trwy `SSL/TLS` → enw gwesteiwr arferol CNAME ôl i'r enw parth gael ei rwymo, bydd mynediad uniongyrchol yn adrodd gwall `error code: 522`

Ar yr adeg hon, mae angen i chi ddefnyddio llwybro HTTP → Llwybro gweithwyr i rwymo llwybr, fel y dangosir isod
