`.i18n/v/ol/v.hash` lo'o fa'asaoina le fa'asologa fa'asolopito o le `npm` afifi i totonu !

Afai e te tape fa'afuase'i `.i18n/v/ol` o le a le fa'asalalauina le `npm` afifi.

I le taimi nei, su'e muamua le numera o le fa'asalalauga mulimuli o lau poloketi i totonu [npmjs.com](//npmjs.com) mo se fa'ata'ita'iga `0.1.9`

Ona vaĘ»ai lea i le `bash` Create faila i lalo.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```
