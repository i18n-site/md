
## Igoa Ole Igoa Ole CNAME

Cloudflare Worker 's domain name is bound through `SSL/TLS` → custom host name CNAME uma ona fusifusia le igoa ole igoa, o le a lipotia sa'o se mea sese `error code: 522`

I le taimi nei, e manaʻomia lou faʻaogaina o le HTTP routing → Tagata faigaluega faʻatautaia e fusifusia se auala, e pei ona faʻaalia i lalo.
