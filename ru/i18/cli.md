# Подробное объяснение параметров командной строки

## `-p` для очистки файлов

`-p` или `--purge` удаляют файлы, которые присутствуют в каталогах переводов, но отсутствуют в каталоге исходного языка.

Потому что при написании документов имена файлов Markdown часто изменяются, что приводит к появлению множества устаревших файлов в каталогах переводов.

Используя этот параметр, можно удалить файлы, которые должны быть удалены в каталогах других языков.

## `-d` для указания каталога перевода

Каталог перевода по умолчанию совпадает с каталогом, в котором находится текущий файл.

`-d` или `--workdir` могут быть использованы для указания каталога перевода, например:

```
i18 -d ~/i18n/md
```

## `-h` для просмотра справки

`-h` или `--help` для просмотра справки по командной строке.
