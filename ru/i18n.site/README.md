# i18n.site: Чисто статический многоязычный фреймворк для веб-сайтов

`i18n.site` — многоязычный, чисто статический генератор документационных сайтов.

## Предисловие

`i18n.site` — это не только генератор документационных сайтов, но и каркас для разработки веб-сайтов.

Новая парадигма разработки веб-сайтов, основанная на `MarkDown`, с использованием фронтенд-компонентов для добавления интерактивности.

Каждый фронтенд-компонент является независимым пакетом, который можно устанавливать отдельно.

Помимо разделения фронтенда и бэкенда, также реализовано разделение статического контента и динамических данных.

Сайт [i18n.site](/), который вы посещаете, был создан на основе этой структуры (включая систему пользователей, систему выставления счетов, подписку на электронные письма и т.д.).
