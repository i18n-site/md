## Предисловие

`i18n.site` — это моя переоценка подхода к разработке веб-сайтов.

Это не только генератор документационных сайтов, но и каркас для разработки веб-сайтов.

Новая парадигма разработки веб-сайтов, где `MarkDown` является ядром, а интерактивность добавляется с помощью фронтенд-компонентов.

Каждый фронтенд-компонент является независимым пакетом для установки.

На основе разделения фронтенда и бэкенда также реализовано разделение статического контента и динамических данных.

Сайт [i18n.site](/), который вы посещаете, был создан на этой основе (включая систему пользователей, систему выставления счетов, подписку на электронную почту и т.д.).
