# Подготовьте окружение

* Установите и запустите `docker` (пользователям Mac рекомендуется использовать [orbstack](https://orbstack.dev) в качестве_runtime для `docker`)

* Установите [bun](https://bun.sh/docs/installation) и [nodejs](https://nodejs.org/en/download/package-manager)

## Инициализируйте проект

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
