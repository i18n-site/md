# Нижний Колонтитул

Продолжая использовать демонстрационный проект в качестве примера, нижний колонтитул веб-сайта `.i18n/htm/foot.pug` в каталоге `md`

<img alt="" src="https://p.3ti.site/1721286077.avif">

[`pug`](https://pugjs.org) — это язык шаблонов, генерирующий `HTML` .

[➔ Нажмите здесь, чтобы изучить грамматику pug](https://pugjs.org)

**Не пишите `css` `js` `foot.pug`** , иначе будут ошибки.

Пожалуйста, обратитесь к следующему, стиль написан соответствующим `.css` а взаимодействие достигается за счет создания веб-компонентов.

## Стиль Нижнего Колонтитула

В демо `md/.i18n/htm` проекте ниже 3 `css` файла.

* `foot.css` : Стиль нижнего колонтитула
* `import.css` : Стиль `i18n.site` для всего сайта
* `conf.css` : Значки и шрифты нижнего колонтитула

### Шрифт Значка

Значок нижнего колонтитула создается с помощью шрифта iconfont.cn ( [английская версия](https://www.iconfont.cn/?lang=en-us) /[中文版](https://www.iconfont.cn/?lang=zh)) `F`

При необходимости создайте свой собственный шрифт значков и замените : конфигурацию в `conf.css`

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
}
```

## Веб-Компоненты

`foot.pug` не может написать `js` . Если требуется взаимодействие, настройте компонент веб-страницы.

Вы можете определить [веб-компонент](https://www.freecodecamp.org/news/build-your-first-web-component/) в `md/.i18n/htm/index.js` а затем использовать его в `foot.pug` .

Создавать веб-компоненты, например пользовательские теги, легко `<x-img>`

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```

## Многоязычный Нижний Колонтитул

`.i18n/htm/foot.pug` Код в нем следующий :

```
#Ft
  b
    a.site(href="/")
    b ${I18N.C}
```

: соответствующий номер находится в `en/i18n.yml` `${I18N.C}`

```
C: Power By <a class="a" href="https://i18n.site">i18n.site</a>
```

Используя `${I18N.xxx}` аналогичный этому методу написания, в сочетании с `i18n.yml` , вы можете добиться многоязычной интернационализации нижнего колонтитула.

Добавьте `class="a"` к ссылке, чтобы избежать преобразования ссылки в `MarkDown` , см :
 [: `YAML` предотвратить преобразование `HTML` ссылки в `Markdown`](/i18/qa#H2) .
