# Общая Проблема

## Случайно Удалено, Из-За Чего Пакет `Npm` Не Удалось Опубликовать `.i18n/V`

`.i18n/v/ol/v.hash` Историческая версия выпуска `npm` пакета сохраняется в формате !

Если вы случайно удалите `.i18n/v/ol` пакет `npm` не будет опубликован.

На этом этапе сначала найдите номер версии последней версии вашего проекта [npmjs.com](//npmjs.com) например `0.1.9`

Затем обратитесь к файлу `bash` Create ниже.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Обратите внимание, что при восстановлении таким способом будет потеряна история файлов, что сделает невозможным постепенное обновление следующего выпуска, а весь контент будет переупакован и загружен один раз.
