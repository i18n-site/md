## <a rel=id href="#ha" id="ha"></a> Высокая доступность интерфейса

`i18n.site` по умолчанию публикует содержимое сайта на `npmjs.com`, используя [jsdelivr.com](//jsdelivr.com), [unpkg.com](//unpkg.com) и другие `CDN`, загруженные в `npm`.

На этой основе добавлены mirror-сервера для материкового Китая, чтобы обеспечить стабильный доступ для китайских пользователей, достигнув **высокой доступности интерфейса**.

Принцип работы таков:拦截请求通过 [`service worker`](https://developer.mozilla.org/docs/Web/API/Service_Worker_API), повторять неудачные запросы с других `CDN`, и адаптивно выбирать наиболее быстрый источник для загрузки по умолчанию.

Связанный код: [github.com/18x/serviceWorker](https://github.com/i18n-site/18x/tree/main/serviceWorker).

### Загрузка онлайн-шрифтов, поддержка китайского языка

По умолчанию на сайте используются онлайн-шрифты, такие как [Alimama Double Axial Variable Rectangle](https://www.iconfont.cn/fonts/detail?cnid=pOvFIr086ADR), [MiSans](https://hyperos.mi.com/font/zh/download/), для унификации опыта чтения пользователей на различных платформах.

Одновременно, для ускорения загрузки, шрифты разрезаются по частоте встреnчаемости символов.

Связанный код: [github.com/i18n-site/font](https://github.com/i18n-site/font).

### Автоматическое скрытие верхней навигации

При прокрутке вниз верхняя навигация автоматически скрывается.

При прокрутке вверх скрытая навигация появляется снова.

Когда мышь не двигается, навигация будет медленно исчезать.

В правом верхнем углу навигационной панели находится кнопка для полноэкранного режима, что создает погружающий опыт чтения документов.

### Синхронное выделение оглавления текущей главы

При прокрутке правого содержимого оглавление слева синхронно выделяет текущую читаемую главу.
