## Интеграция Переводов `i18`

Программа включает встроенный перевод `i18`. Подробнее см. в [документации ➔ `i18`](/i18).

## Автоматическая Подстройка Языка Под Браузер

Язык веб-сайта по умолчанию будет автоматически соответствовать языку вашего браузера.

После的手动切换语言, выбор пользователя будет запомнен.

Связанный код: [github.com/i18n-site/18x/src/lang.coffee](https://github.com/i18n-site/18x/blob/main/src/lang.coffee).

## <a rel=id href="#ha" id="ha"></a> Высокая Доступность Интерфейса

`i18n.site` по умолчанию опубликует содержимое сайта на `npmjs.com` с помощью [jsdelivr.com](//jsdelivr.com) , [unpkg.com](//unpkg.com) и другого контента `CDN` загруженного на `npm` .

На этой основе добавлены mirror-сервера для Китая, чтобы обеспечить стабильный доступ для китайских пользователей, достигая **высокого уровня доступности интерфейса**.

Принцип таков: перехватывать запросы с [`service worker`](https://developer.mozilla.org/docs/Web/API/Service_Worker_API) , повторять неудачные запросы с другими `CDN` и адаптивно включать самый быстро реагирующий исходный сайт в качестве источника загрузки по умолчанию.

Связанный код: [github.com/18x/serviceWorker](https://github.com/i18n-site/18x/tree/main/serviceWorker).

Для ускорения загрузки шрифты оптимизированы по частоте встреnчаемости символов.

Связанный код: [github.com/i18n-site/font](https://github.com/i18n-site/font).

### Автоматическое Скрытие Верхней Навигации

При прокрутке вниз верхняя навигация автоматически скроется.

При прокрутке вверх скрытая навигация появится снова.

Когда мышь не двигается, навигация будет медленно消失ать.

В правом верхнем углу панели навигации есть кнопка для полноэкранного режима, что обеспечивает погружение в процесс чтения документов.

### Синхронное Выделение Содержания Главы

При прокрутке справа содержимого, слева в контуре будет подсвечиваться текущая читаемая глава.
