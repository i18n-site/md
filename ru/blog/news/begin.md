# i18n.site · MarkDown Инструмент Перевода и Создания Веб-сайтов, Теперь В Онлайн-Доступе!

После более чем полугода разработки [https://i18n.site](//i18n.site) был запущен.

В настоящее время реализованы два开源овых командных инструмента:

* `i18` : Командный Инструмент Перевода MarkDown
* `i18n.site` : Многоязычный Генератор Статических Документов, **Оптимизированный для Удобства Чтения**

Перевод сохраняет формат `Markdown` без потерь. Он может识别ять изменения в файлах и переводить только измененные файлы.

Перевод можно редактировать; если оригинал был изменен, повторный машинный перевод не перезапишет ручные правки в переводе (если текст оригинала не был изменен).

## Истоки

В эпоху Интернета весь мир стал рынком, и многоязычие, а также локализация являются основными навыками.

Существующие инструменты для управления переводами слишком сложны, и для программистов, которые используют `git` для управления версиями, предпочтительнее командная строка.

Таким образом, я разработал инструмент для перевода `i18` и на его основе создал многоязычный генератор статических сайтов `i18n.site`.

![](https://p.3ti.site/1723777556.avif)

Это только начало, впереди还有很多 планов.

Например, интеграция статического документного сайта с социальными сетями и почтовой подпиской, чтобы своевременно информировать пользователей о новых обновлениях.

Или, например, интеграция многоязычных форумов и систем обработки заявок на любую веб-страницу, чтобы пользователи могли беспрепятственно общаться.

## Открытый Исходный Код

Весь код的前端, бэкенд и командной строки [имеет открытый исходный код](https://i18n.site/i18n.site/c/src) (модель перевода временно не开放ена).

Используемый стек технологий включает:

Веб-интерфейс: [svelte](https://svelte.dev), [stylus](https://stylus-lang.com), [pug](https://github.com/pugjs/pug), [vite](https://github.com/vitejs/vite)

Командная строка и бэкенд разработаны на языке rust.

Бэкенд использует [axum](https://github.com/tokio-rs/axum), [tower-http](https://github.com/tower-rs/tower-http/releases).

Командная строка использует [встроенный js-движок boa_engine](https://docs.rs/boa_engine), [встроенная база данных fjall](https://github.com/fjall-rs/fjall).

Сервер VPS: [contabo](https://my.contabo.com)

Базы данных: [kvrocks](https://kvrocks.apache.org), [mariadb](https://mariadb.org).

Для отправки почты используется собственный SMTP-сервер [chasquid](https://github.com/albertito/chasquid).
