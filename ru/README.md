## Предисловие

Интернет стер физическое расстояние, но языковые барьеры по-прежнему мешают единству человечества.

Хотя браузеры и имеют встроенные функции перевода, поисковые системы до сих пор не могут выполнять запросы на иностранных языках.

Социальные сети и электронная почта побуждают людей фокусироваться на информации на их родном языке.

В условиях информационного бума и глобального рынка, чтобы привлечь внимание, **поддержка нескольких языков — это базовое умение**.

Даже的个人ные проекты с открытым исходным кодом, стремящиеся повлиять на широкую аудиторию, должны с самого начала выбирать международные технологии.

### <a rel=id href="#i18" id="i18"></a> i18 : MarkDown инструмент командной строки для перевода

Командная строка для перевода `Markdown` и `YAML` на несколько языков ( [исходный код](https://github.com/i18n-site/rust/tree/main/i18) ).

Сохраняет формат `Markdown`,识别 файлы с изменениями и переводит только измененные файлы.

Перевод можно редактировать.

При повторном переводе измененного исходного текста машинный перевод не перезапишет ручные изменения (если исходный текст не был изменен).

Мы追随 `UNIX` философию "Все — это файлы", управляя переводами на сотни языков без сложных корпоративных решений.

Вы можете редактировать переводы `Markdown` с помощью знакомых инструментов (но не добавлять или удалять абзацы) и использовать привычные методы управления версиями.

Для языковых файлов можно создать репозиторий开源 кода, используя процесс `Pull Request` для привлечения全球参与者 к оптимизации переводов. Гладкая интеграция с [github](//github.com) и другими开源 сообществами.

[→ руководства по использованию см. в документации проекта](/i18).

## Лучшее качество машинного перевода

Мы разработали новое поколение технологий перевода, сочетающее advantages традиционных моделей машинного перевода и больших языковых моделей, делая перевод точным, плавным и элегантным.

Результаты盲测ов показывают, что качество наших переводов значительно лучше, чем у аналогичных сервисов.

Для достижения того же качества人工ного редактирования, необходимого Google Translate и `ChatGPT`, в `2.67` и `3.15` раз больше, чем у нас соответственно.

### i18n.site : генератор многоязычных статических сайтов

Инструмент командной строки для создания многоязычных статических сайтов ( [исходный код](https://github.com/i18n-site/rust/tree/main/i18n-site) ).

Полностью статический, оптимизирован для чтения, интегрирован с переводом [i18](#i18), ideal для создания документационных сайтов проектов.

Основа фронтендной архитектуры использует модульную структуру, легко поддается二次ной разработке и может быть интегрирована с бэкендными функциями по мере необходимости.

Этот сайт разработан на этой структуре, включая функции пользователей и платежей ( [исходный код](/i18n.site/src) ), подробное руководство будет выпущено позже.

[→ руководства по использованию см. в документации проекта](/i18n.site).
