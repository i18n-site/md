
## CNAME Შეკრული Დომენის Სახელი

დომენის სახელი შეკრულია `SSL/TLS` -ით → მორგებული ჰოსტის სახელი CNAME დომენის სახელის შეკვრის შემდეგ, პირდაპირი წვდომა შეატყობინებს შეცდომას `error code: 522` Cloudflare Worker

ამ დროს, თქვენ უნდა გამოიყენოთ HTTP მარშრუტი → მარშრუტის მუშები მარშრუტის დასაკავშირებლად, როგორც ეს ნაჩვენებია ქვემოთ
