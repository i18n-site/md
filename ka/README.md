>
> უფალმა გამოაცხადა: როდესაც ხალხი იზიარებს ენას და საგვარეულოს, ისინი იქცევიან ცალკეულ ტომად. კოშკი მხოლოდ დასაწყისია; უახლოეს დღეებში ისინი ვერაფერს მიაღწევენ.
