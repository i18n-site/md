# Განვითარების Დაგეგმვა

შემდეგი მნიშვნელოვანი ფუნქციები დამუშავების პროცესშია:

- [ ] გადართვა დოკუმენტების მრავალ ვერსიას შორის
- [ ] სუფთა ფრონტის სრული ტექსტის ძიება
- [ ] RSS არხი
- [ ] საძიებო სისტემის ოპტიმიზაცია
- [ ] ერთდროულად გადაიტანეთ სტატიების განახლებები მთავარ სოციალურ მედიაში
- [ ] ელფოსტა ხელმისაწვდომია / `web push` გამოიწერეთ პროდუქტის განახლებები
- [ ] კომენტარი სტატიის პუნქტებზე
- [ ] ადგილობრივი დოკუმენტების ინტეგრირება ცვლილებების მონიტორინგისა და გადახედვის გენერირების მიზნით, ყოველ ჯერზე ხელით შედგენის საჭიროების აღმოფხვრა და NGINX-ის გამოყენებით
