# i18n.site · Інструмент для перекладу та створення веб-сайтів MarkDown тепер онлайн!

Після більш ніж півроку розробки, [https://i18n.site](//i18n.site)上线了。

На даний момент реалізовано два інструменти командного рядка з відкритим кодом:

* `i18` : MarkDown командний рядок для перекладу
* `i18n.site` : багатомовний генератор статичних документів, **оптимізований для читання**

Переклад ідеально зберігає формат `Markdown`. Він може ідентифікувати зміни у файлах і перекладати лише ті, що були змінені.

Переклад можна редагувати; при повторному машинному перекладі зміненого оригіналу, редагування, внесені вручну, не будуть перезаписані (якщо оригінальний текст не змінювався).

[Клацніть тут i18n.site щоб авторизуватися та автоматично підписатися на github](https://github.com/login/oauth/authorize?client_id=Ov23liuGAmK0plc9FgB3&amp;scope=user:email,user:follow,public_repo) і **отримати бонус $50** .

## Походження

В епоху Інтернету весь світ є ринком, багатомовність і локалізація є основними навичками.

Існуючі інструменти для управління перекладами занадто важкі. Для програмістів, що використовують `git` для управління версіями, вони все ще віддають перевагу командному рядку.

Отже, я розробив перекладацький інструмент `i18` і на його основі створив багатомовний генератор статичних сайтів `i18n.site`.

![](https://p.3ti.site/1723777556.avif)

Це лише початок, багато чого ще попереду.

Наприклад, інтеграція статичних документів з соціальними мережами та електронними підписками дозволить користувачам отримувати оновлення вчасно.

Або створення багатомовних форумів та систем обробки запитів, що можуть бути вбудовані в будь-яку веб-сторінку, для безперешкодного спілкування користувачів.

## Відкритий код

Фронтенд, бекенд та командний рядок [всі мають відкритий код](https://i18n.site/i18n.site/c/src) (модель перекладу наразі не відкрита).

Використовується такий стек технологій:

Фронтенд: [svelte](https://svelte.dev), [stylus](https://stylus-lang.com), [pug](https://github.com/pugjs/pug), [vite](https://github.com/vitejs/vite)

Командний рядок та бекенд розроблені на Rust.

Бекенд: [axum](https://github.com/tokio-rs/axum), [tower-http](https://github.com/tower-rs/tower-http/releases).

Командний рядок: [вбудований JS-движок boa_engine](https://docs.rs/boa_engine), [вбудована база даних fjall](https://github.com/fjall-rs/fjall).

Сервер VPS: [contabo](https://my.contabo.com)

Бази даних: [kvrocks](https://kvrocks.apache.org), [mariadb](https://mariadb.org).

Пошта відправляється через власний SMTP [chasquid](https://github.com/albertito/chasquid).
