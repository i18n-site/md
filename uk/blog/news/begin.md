# i18n.site · Інструмент для перекладу та створення веб-сайтів Markdown тепер онлайн!

Після більш ніж півроку розробки, [https://i18n.site](//i18n.site) запущено.

На даний момент реалізовано два інструменти командного рядка з відкритим кодом:

* `i18`: інструмент для перекладу з командного рядка Markdown
* `i18n.site`: багатомовний генератор статичних веб-документів, **оптимізований для читання**

Переклад ідеально підтримує формат `Markdown`, не перекладаючи вбудовані коди. Він може виявляти зміни в файлах та перекладати лише ті файли, які були змінені.

Переклад можна редагувати; при повторному машинному перекладі зміненої частини тексту, ручні зміни не будуть перезаписані (якщо оригінальний текст цього фрагменту залишився без змін).

## Походження

У добу Інтернету весь світ є ринком, де багатомовність і локалізація є основними навичками.

Існуючі інструменти для управління перекладами занадто складні для програмістів, які використовують `git` для управління версіями, і вони віддають перевагу командному рядку.

Отже, я розробив інструмент для перекладу `i18` і на його основі створив багатомовний генератор статичних веб-сайтів `i18n.site`.

![](https://p.3ti.site/1723777556.avif)

Це лише початок, перед нами ще багато планів.

Наприклад, інтеграція статичних веб-документів з соціальними мережами та електронними підписками, щоб користувачі могли отримувати оновлення вчасно.

Або створення багатомовних форумів та систем обробки запитів, які можна вбудувати в будь-яку веб-сторінку, щоб користувачі могли спілкуватися без перешкод.

## Відкритий код

Фронтенд, бекенд та код командного рядка [всі відкриті](https://i18n.site/i18n.site/c/src) (модель для перекладу наразі не відкрита).

Використовується такий стек технологій:

[svelte](https://svelte.dev), [stylus](https://stylus-lang.com), [pug](https://github.com/pugjs/pug), [vite](https://github.com/vitejs/vite)

Командний рядок і бекенд розроблені на мові Rust.

Бекенд використовує [axum](https://github.com/tokio-rs/axum), [tower-http](https://github.com/tower-rs/tower-http/releases).

Командний рядок використовує [вбудований JavaScript-движок boa_engine](https://docs.rs/boa_engine), [вбудовану базу даних fjall](https://github.com/fjall-rs/fjall).

Використовується VPS від [contabo](https://my.contabo.com)

Бази даних: [kvrocks](https://kvrocks.apache.org), [mariadb](https://mariadb.org).

Відправка електронної пошти здійснюється через самовстановлений SMTP [chasquid](https://github.com/albertito/chasquid).
