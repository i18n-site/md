# i18n.site · Інструмент для перекладу та створення веб-сайтів на Markdown тепер онлайн!

Після більш ніж півроку розробки, [https://i18n.site](//i18n.site) запущено.

Наразі реалізовано два інструменти командного рядка з відкритим кодом:

* `i18`: Інструмент для перекладу Markdown з командного рядка
* `i18n.site`: Генератор багатомовних статичних веб-документів, **оптимізований для покращення читальницького досвіду**

Переклад може ідеально підтримувати формат `Markdown`, ідентифікувати зміни в файлах та перекладати лише ті файли, які були змінені.

Переклад є редагованим; змініть оригінальний текст, і при повторному машинному перекладі, ручні зміни перекладу не будуть перезаписані (якщо цей абзац оригінального тексту не був змінений).

[Клацніть тут, щоб авторизуватися та автоматично підписатися github i18n.site](https://github.com/login/oauth/authorize?client_id=Ov23liuGAmK0plc9FgB3&amp;scope=user:email,user:follow,public_repo) і **отримати бонус $50** .

## Походження

У епоху Інтернету весь світ є ринком, багатомовність і локалізація — це базові навички.

Існуючі інструменти для управління перекладами занадто складні. Програмісти, що використовують `git` для управління версіями, віддають перевагу командному рядку.

Отже, я розробив перекладацький інструмент `i18` та на його основі створив багатомовний генератор статичних веб-сайтів `i18n.site`.

![](https://p.3ti.site/1723777556.avif)

Це лише початок, перед нами ще багато планів.

Наприклад, інтеграція статичних веб-документів з соціальними мережами та електронними підписками, щоб користувачі могли отримувати оновлення вчасно.

Або створення багатомовних форумів та систем обробки запитів, які можна вбудувати в будь-яку веб-сторінку для безперешкодного спілкування користувачів.

## Відкритий код

Фронтенд, бекенд та код командного рядка [є відкритими](https://i18n.site/i18n.site/c/src) (модель перекладу наразі не відкрита).

Використовується такий стек технологій:

[svelte](https://svelte.dev), [stylus](https://stylus-lang.com), [pug](https://github.com/pugjs/pug), [vite](https://github.com/vitejs/vite)

Командний рядок і бекенд розроблені на основі Rust.

Бекенд використовує [axum](https://github.com/tokio-rs/axum), [tower-http](https://github.com/tower-rs/tower-http/releases).

Командний рядок використовує [вбудований JS-движок boa_engine](https://docs.rs/boa_engine), [вбудовану базу даних fjall](https://github.com/fjall-rs/fjall).

Сервер VPS [contabo](https://my.contabo.com)

Бази даних [kvrocks](https://kvrocks.apache.org), [mariadb](https://mariadb.org).

Відправка електронної пошти через власний SMTP [chasquid](https://github.com/albertito/chasquid).
