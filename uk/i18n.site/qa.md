# Поширена Проблема

## Випадково Видалено `.i18n/V` Через Що Пакет `Npm` Не Вдалося Опублікувати

`.i18n/v/ol/v.hash` Історичну версію випуску `npm` пакета збережено в !

Якщо ви випадково видалите `.i18n/v/ol` пакет `npm` не буде опубліковано.

У цей час спочатку знайдіть номер версії останнього випуску вашого проекту [npmjs.com](//npmjs.com) наприклад `0.1.9`

Тоді зверніться до файлу `bash` Create нижче.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Зауважте, що відновлення таким чином призведе до втрати історії файлів, що унеможливить поступове оновлення наступного випуску, а весь вміст буде переупаковано та завантажено один раз.
