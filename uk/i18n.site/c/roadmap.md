# Планування розвитку

У процесі розробки перебувають такі важливі функції:

- [ ] Перемикання між кількома версіями документів
- [ ] Повнотекстовий пошук на pure frontend
- [ ] RSS-канал
- [ ] Оптимізація для пошукових систем
- [ ] Синхронізація та推送 оновлень статей до великих соціальних мереж
- [ ] Користувачам доступна підписка через електронну пошту / `web push` на оновлення продукту
- [ ] Залишення коментарів до абзаців статей
- [ ] Інтеграція monitoringu місцевих документів для створення попередніх переглядів без необхідності ручної компіляції та використання NGINX
