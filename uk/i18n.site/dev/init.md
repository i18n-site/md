# Підготуйте середовище

* Встановіть та запустіть `docker` (для користувачів Mac рекомендується використовувати [orbstack](https://orbstack.dev) як середовище виконання для `docker`)

* Встановіть [bun](https://bun.sh/docs/installation) та [nodejs](https://nodejs.org/en/download/package-manager)

## Ініціалізуйте проект

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
