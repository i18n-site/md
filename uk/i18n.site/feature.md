## <a rel=id href="#ha" id="ha"></a> Висока доступність інтерфейсу

`i18n.site` за замовчуванням публікує вміст сайту на `npmjs.com`, використовуючи [jsdelivr.com](//jsdelivr.com), [unpkg.com](//unpkg.com) та інші `CDN` для завантаження вмісту, розміщеного на `npm`.

На цій основі додано дзеркальні джерела для материкового Китаю, щоб забезпечити стабільний доступ для китайських користувачів, досягнувши **високої доступності інтерфейсу**.

Принцип полягає в тому, щоб перехоплювати запити за допомогою [`service worker`](https://developer.mozilla.org/docs/Web/API/Service_Worker_API), повторювати невдалі запити на інших `CDN`, та адаптивно встановлювати як джерело завантаження той сайт, який відповідає найкраще за швидкістю.

Код: [github.com/18x/serviceWorker](https://github.com/i18n-site/18x/tree/main/serviceWorker).

## Односторінковий додаток, швидке завантаження

Веб-сайт використовує архітектуру односторінкового додатка, що забезпечує швидке завантаження без刷新у при перемиканні сторінок.

### Завантаження онлайн-шрифтів, підтримка китайської мови

За замовчуванням на веб-сторінці використовуються [двовісні змінні прямокутні шрифти](https://www.iconfont.cn/fonts/detail?cnid=pOvFIr086ADR), [MiSans](https://hyperos.mi.com/font/zh/download/) та інші онлайн-шрифти для уніфікації читального досвіду користувачів на різних платформах.

Для підвищення швидкості завантаження шрифти наріжані відповідно до статистики частот слів.

Код: [github.com/i18n-site/font](https://github.com/i18n-site/font).

### Автоматичне приховування верхньої навігації

При прокручуванні вниз верхня навігація автоматично ховається.

При прокручуванні вгору прихована навігація знову з'являється.

Вона зникає, коли миша не рухається.

У верхньому правому куті навігаційної панелі є кнопка для повноекранного режиму, що створює занурений читальний досвід.

### Синхронізоване підсвічування поточної глави

При прокручуванні правого вмісту лівий контур автоматично підсвічує поточну главу, яку ви читаєте.
