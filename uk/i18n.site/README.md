# i18n.site: Чисто статична багатомовна структура веб-сайту

`i18n.site` — багатомовний генератор суто статичних документів.

## Пролог

`i18n.site` — це генератор для створення документативних сайтів, а також фреймворк для розробки веб-сайтів.

Нова парадигма розробки веб-сайтів, де `MarkDown` є центром, а інтерактивність забезпечують фронтенд-компоненти.

Кожен фронтенд-компонент є незалежним пакунком для встановлення.

На основі розділення фронтенду та бекенду також відбувається розділення статичного контенту та динамічних даних.

Веб-сайт [i18n.site](/), який ви відвідуєте, побудований на основі цього фреймворку (включаючи систему користувачів, систему支付, підписку на електронну пошту тощо).
