# i18n.site: Чисто статична багатомовна структура веб-сайту

`i18n.site` — багатомовний генератор суто статичних документів.

Наразі він є初步 доступним, але ще є важливі функції, які знаходяться у розробці. Докладніше у [плані розробки](/i18n.site/c/roadmap).

<button onclick="mailsub()">Клацніть цей електронний лист, щоб підписатися на оновлення продукту</button> та <button onclick="webpush()">ввімкнути функцію push у веб-переглядачі</button> . Ми повідомимо вас, коли будуть зроблені оновлення продукту.

Також просимо стежити за нашими акаунтами в / [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) [X.COM: @i18nSite](https://x.com/i18nSite)

## Передмова

`i18n.site` — це моя переосмислена модель розробки веб-сайту.

Це не лише генератор документів, але й структура для розробки веб-сайтів.

Нова парадигма розробки веб-сайтів, де `MarkDown` є центром, а інтерфейсні компоненти додають інтерактивність.

Кожен інтерфейсний компонент є незалежним пакетом для встановлення.

На основі розділення фронтенду та бекенду також відбувається розділення статичного контенту та динамічних даних.

Веб-сайт [i18n.site](/), який ви відвідуєте, побудований на цій架构ції (включаючи систему користувачів, платіжну систему, підписку на електронні листи тощо).
