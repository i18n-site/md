# i18n.site: Чисто статична багатомовна структура веб-сайту

`i18n.site` — багатомовний генератор суто статичних документів.

## Передмова

`i18n.site` — це генератор для створення документативних сайтів, а також фреймворк для розробки веб-сайтів.

Нова парадигма розробки веб-сайтів, яка має `MarkDown` як центр і використовує фронтенд-компоненти для впровадження інтерактивності.

Кожен фронтенд-компонент є незалежним пакунком для встановлення.

На основі розділення фронтенду та бекенду також існує розділення статичного контенту та динамічних даних.

Сайт, який ви відвідуєте [i18n.site](/), створено на основі цього фреймворку (включаючи систему користувачів, платіжну систему, підписку на електронну пошту тощо).
