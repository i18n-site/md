# Детальне пояснення параметрів командного рядка

## `-p` для очищення файлів

`-p` або `--purge` видаляє файли, що знаходяться в каталогах перекладів, але відсутні в каталозі вихідної мови.

Оскільки під час написання документів часто змінюються назви файлів Markdown, це призводить до появи багатьох застарілих файлів у каталогах перекладів.

Використовуючи цей параметр, ви можете видалити файли, які більше не потрібні в інших мовних каталогах.

## `-d` для вказання каталогу перекладу

Каталог для перекладу за замовчуванням — це каталог, де знаходиться поточний файл.

`-d` або `--workdir` дозволяє вказати каталог для перекладу, наприклад:

```
i18 -d ~/i18n/md
```

## `-h` для перегляду довідки

`-h` або `--help` для перегляду довідки командного рядка.
