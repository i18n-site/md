# Детальне пояснення параметрів командного рядка

## `-p` для очищення файлів

`-p` або `--purge` видаляє файли, що знаходяться в каталогах перекладів, але відсутні в каталозі вихідної мови.

Оскільки під час написання документів назви файлів Markdown часто змінюються, що призводить до появи багатьох застарілих файлів у каталогах перекладів.

Використовуючи цей параметр, ви можете видалити файли, які мають бути видалені в інших мовних каталогах.

## `-d` для вказання каталогу перекладу

Каталог перекладу за замовчуванням — це каталог, де знаходиться поточний файл.

`-d` або `--workdir` можуть вказати каталог для роботи, наприклад:

```
i18 -d ~/i18n/md
```

## `-h` для перегляду довідки

`-h` або `--help` для перегляду довідки командного рядка.
