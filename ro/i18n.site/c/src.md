## Backend

Construirea site-ului de documentație necesită doar front-end, fără backend.

Acest backend este backend-ul site-ului i18n.site în sine, incluzând sisteme pentru utilizatori, plăți, notificări și altele.

* Servicii [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugin-uri [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Bibliotecă de bază [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
