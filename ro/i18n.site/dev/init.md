# Pregătiți mediul de lucru

* Instalați și porniți `docker` (pentru utilizatorii de Mac, se recomandă utilizarea [orbstack](https://orbstack.dev) ca mediu de ejecție pentru `docker`)

* Instalați [bun](https://bun.sh/docs/installation) și [nodejs](https://nodejs.org/en/download/package-manager)

## Inicializați proiectul

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
