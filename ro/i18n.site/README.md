# i18n.site: Cadru de site-uri web static pur în mai multe limbi

`i18n.site` - generator de site-uri de documente, multiplu în limbi și pur static.

## Prefață

`i18n.site` este un generator de site-uri pentru documente și, în același timp, un cadru pentru dezvoltarea web.

O nouă paradigmă de dezvoltare a site-urilor web, centrată pe `MarkDown`, care utilizează componente front-end pentru a adăuga interactivitate.

Fiecare componentă front-end este un pachet care poate fi instalat independent.

Pe lângă separarea front-endului de back-end, există și o separare a conținutului static de datele dinamice.

Site-ul pe care îl vizitați [i18n.site](/) este construit pe baza acestui cadru (inclusiv sistemul de utilizatori, sistemul de facturare, abonamentele prin e-mail etc.).
