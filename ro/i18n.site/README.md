# i18n.site: Cadru Static Pur pentru Site-uri Web Multilingve

`i18n.site` - generator de site-uri de documente multilingve, pur static.

Deja funcțional într-o fază inițială, dar încă sunt în dezvoltare funcții esențiale. Consultați [planul de dezvoltare](/i18n.site/c/roadmap) pentru mai multe detalii.

Vă rugăm <button onclick="mailsub()">să faceți clic pe acest e-mail pentru a vă abona la actualizările de produs</button> și <button onclick="webpush()">pentru a activa notificările browserului</button>, vă vom anunța la actualizările produsului.

## Prefață

`i18n.site` reprezintă o reflecție asupra modului de dezvoltare a site-urilor web.

Acesta este atât un generator de site-uri de documente, cât și un cadru de dezvoltare pentru site-uri web.

O nouă paradigmă de dezvoltare a site-urilor web, centrată pe `MarkDown`, care folosește componente front-end pentru a adăuga interactivitate.

Fiecare componentă front-end este un pachet care poate fi instalat independent.

Pe lângă separarea front-endului de back-end, există și separarea conținutului static de datele dinamice.

Site-ul pe care îl vizitați [i18n.site](/) este construit pe baza acestui cadru (inclusiv sistemul de utilizatori, sistemul de facturare, abonamentele prin e-mail, etc.).
