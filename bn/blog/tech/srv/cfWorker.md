
## CNAME আবদ্ধ ডোমেন নাম

Cloudflare Worker ডোমেইন নাম `SSL/TLS` → কাস্টম হোস্ট নেম দ্বারা আবদ্ধ হয় CNAME ডোমেন নাম আবদ্ধ হওয়ার পরে, সরাসরি অ্যাক্সেস একটি ত্রুটি `error code: 522` রিপোর্ট করবে

এই সময়ে, আপনাকে HTTP রাউটিং ব্যবহার করতে হবে
