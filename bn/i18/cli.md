# কমান্ড লাইন প্যারামিটারের বিস্তারিত ব্যাখ্যা

## পরিষ্কার ফাইল `-p`

`-p` `--purge` ফাইলগুলি পরিষ্কার করবে যা প্রতিটি অনুবাদ ডিরেক্টরিতে বিদ্যমান কিন্তু উৎস ভাষা ডিরেক্টরিতে বিদ্যমান নেই।

কারণ নথি লেখার সময়, মার্কডাউন ফাইলের নামগুলি প্রায়শই সামঞ্জস্য করা হয়, যার ফলে অনুবাদ ডিরেক্টরিতে অনেকগুলি পুরানো এবং পরিত্যক্ত ফাইল থাকে।

অন্যান্য ভাষার ডিরেক্টরিতে মুছে ফেলা উচিত এমন ফাইলগুলি পরিষ্কার করতে এই প্যারামিটারটি ব্যবহার করুন।

## অনুবাদ ডিরেক্টরি উল্লেখ `-d`

অনুবাদিত ডিরেক্টরি ডিফল্ট যে ডিরেক্টরিতে বর্তমান ফাইলটি অবস্থিত।

`-d` `--workdir` অনুবাদ ডিরেক্টরি উল্লেখ করতে পারে, যেমন:

```
i18 -d ~/i18n/md
```

## সাহায্য `-h`

`-h` `--help` কমান্ড লাইন সাহায্য দেখতে পারেন।
