# Gemeinsame Problem

## Zoufälleg Geläscht `.i18n/V` Wouduerch De `Npm` Package Net Publizéiert Gëtt

`.i18n/v/ol/v.hash` Déi historesch Versioun vum `npm` Package Release gëtt gespäichert!

Wann Dir zoufälleg läscht `.i18n/v/ol` gëtt de `npm` Package net publizéiert.

Zu dëser Zäit fannt Dir d'Versiounsnummer vun der leschter Verëffentlechungsversioun vun Ärem Projet zum Beispill [npmjs.com](//npmjs.com) `0.1.9`

Da kuckt op `bash` Erstelle Datei hei ënnen.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Bedenkt datt d'Reparatur op dës Manéier d'Dateigeschicht verléiert, sou datt et onméiglech ass déi nächst Verëffentlechung inkrementell ze aktualiséieren, an all Inhalt gëtt ëmgepackt an eemol eropgelueden.
