# Editor Empfehlung

Jiddereen huet hiren eegene Liiblingsredaktor Loosst eis eis bescht Praktiken hei deelen `Markdown`

Mir [vscode](https://code.visualstudio.com/) Schreiwen `MarkDown` installéieren de Plug-in [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) fir an Echtzäit ze kucken!

Wéi hei ënnendrënner, klickt riets am Editor fir d'Virschaufenster opzemaachen.

<img alt="" src="https://p.3ti.site/1720775216.avif">
<img alt="" src="https://p.3ti.site/1720775043.avif">

`code xxx.md` op der Kommandozeil fir `vscode` opzemaachen `Markdown` Datei.

Benotzt [PicList](https://github.com/Kuingsmile/PicList) fir Biller eropzelueden, erënnert un d'Ofkierzungsschlëssel ze setzen an de Screenshot mat engem Klick eropzelueden.

Et ënnerstëtzt automatesch Kopie `Markdown` Dateinumm nom Eroplueden, wat d'Effizienz staark verbessert!

Zur selwechter Zäit kënnt Dir op déi folgend Konfiguratioun referenzéieren, d'Datei änneren an déi eropgeluede Datei konfiguréieren fir automatesch kompriméiert `avif` ginn fir d'Bildgréisst ze reduzéieren.
