
## CNAME Gebonnen Domain Numm

Cloudflare Worker Den Domain Numm ass `SSL/TLS` → Benotzerdefinéiert Hostnumm CNAME Nom Domain Numm gëtt den direkten Zougang e Feeler bericht `error code: 522`

Zu dëser Zäit musst Dir HTTP Routing benotzen → Aarbechter Routing fir eng Streck ze binden, wéi hei ënnendrënner
