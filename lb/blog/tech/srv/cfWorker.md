
## CNAME Gebonnen Domain Numm

Cloudflare Worker Domain Numm ass gebonnen duerch `SSL/TLS` → Benotzerdefinéiert Hostnumm CNAME den Domain Numm gebonnen ass, mellt den direkten Zougang e Feeler `error code: 522`

Zu dëser Zäit musst Dir HTTP Routing benotzen → Aarbechter Routing fir eng Streck ze binden, wéi hei ënnendrënner
