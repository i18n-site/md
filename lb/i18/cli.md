# Detailléiert Erklärung Vun Kommando Linn Parameteren

## `-P` Propper Dateien

`-p` `--purge` wäert Dateien botzen, déi an all Iwwersetzungsverzeechnes existéieren, awer net am Quellsproochverzeechnes.

Well wann Dir Dokumenter schreift, ginn Markdown Dateinumm dacks ugepasst, wat zu villen alen a verloossenen Dateien am Iwwersetzungsverzeechnes resultéiert.

Benotzt dëse Parameter fir Dateien ze botzen déi an anere Sproochverzeichnisser geläscht solle ginn.

## `-D` Gitt Iwwersetzungsverzeechnes

Den iwwersaten Verzeechnes gëtt als Standard an de Verzeechnes wou déi aktuell Datei läit.

`-d` `--workdir` kann den Iwwersetzungsverzeechnes spezifizéieren, wéi:

```
i18 -d ~/i18n/md
```

## `-H` Kuckt Hëllef

`-h` `--help` kënnt d'Hëllef vun der Kommandozeil kucken.
