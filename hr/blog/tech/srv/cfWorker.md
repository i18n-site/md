
## CNAME Vezan Naziv Domene

Cloudflare Worker Naziv domene je vezan kroz `SSL/TLS` → prilagođeni naziv hosta CNAME što je naziv domene vezan, izravni pristup će prijaviti pogrešku `error code: 522`

U ovom trenutku morate koristiti HTTP usmjeravanje → Radničko usmjeravanje za vezanje rute, kao što je prikazano u nastavku
