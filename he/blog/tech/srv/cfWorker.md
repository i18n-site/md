
## שם דומיין קשור ל-CNAME

שם הדומיין של Cloudflare Worker קשור ל `SSL/TLS` → שם מארח מותאם אישית CNAME ששם הדומיין נקשר, גישה ישירה תדווח על שגיאה `error code: 522`

בשלב זה, עליך להשתמש בניתוב HTTP → ניתוב עובדים כדי לאגד מסלול, כפי שמוצג להלן
