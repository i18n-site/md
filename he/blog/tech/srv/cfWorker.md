
## שם דומיין קשור ל-CNAME

שם הדומיין `SSL/TLS` → מחייב שם מארח מותאם CNAME לאחר שם הדומיין, גישה ישירה תדווח Cloudflare Worker שגיאה `error code: 522`

בשלב זה, עליך להשתמש בניתוב HTTP → ניתוב עובדים כדי לאגד מסלול, כפי שמוצג להלן
