אני רוצה לומר שרק **על ידי אינטרנציונל של האתר כולו נוכל לתמוך בחיפוש טקסט מלא רב-לשוני באתר ובאופטימיזציה למנועי חיפוש** .

## סדר פעולות

תנ"ך · בראשית :

> בימי קדם, כאשר השפות היו אחת, האנושות בנתה מגדל המגיע עד השמים, שער לתחום אלוהים, והכריז על כוחה של האנושות.
>
> אלוהים הכריז: "בני אדם מתאחדים כשבט אחד, עם לשון משותפת, והמגדל הזה אינו אלא פרולוג. כעת הם משיגים את רצונם, ולא תישאר יראת שמים".
>
> כך, אלוהים ירד, פיזר אנשים לפינות שונות, השתמש בשפות שונות.
>
> מכאן ואילך, התקשורת האנושית הפכה למאתגרת, אין סוף מחלוקות, וכבר לא היה מגדל שנגע בשמים.

רומן המדע הבדיוני &quot;שלושה גופים&quot; (הגייה בסינית: `3Ti` ) מציג ציוויליזציה חייזרית המתקשרת באמצעות גלים אלקטרומגנטיים, אין לה מחסומי שפה ומשגשגת מבחינה טכנולוגית.

אני מקווה ליצור כלי שיאפשר לאנשי כדור הארץ להיות כמו אנשים בעלי שלושה גופים, לתקשר בלי להיות כבולים לשפה, ולאחד שוב את כל האנושות.

תומך כרגע בתרגום הערות עבור rust, c, cpp, java, js, coffee, python ו-bash.

ערוך את [tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) כדי להוסיף תמיכת תרגום להערות בשפות נוספות.
