# בעיה נפוצה

## נמחק בטעות `.i18n/v` מה שגורם לחבילה `npm` להיכשל בפרסום

הגרסה ההיסטורית של מהדורת חבילה `npm` נשמרת ב `.i18n/v/ol/v.hash`

אם תמחק בטעות את `.i18n/v/ol` החבילה `npm` לא תפורסם.

בשלב זה, מצא תחילה את מספר הגרסה של גרסת ההפצה האחרונה של הפרויקט שלך ב- [npmjs.com](//npmjs.com) `0.1.9`

לאחר מכן עיין `bash` צור למטה.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

שים לב שתיקון בדרך זו יאבד את היסטוריית הקבצים, מה שלא יאפשר לעדכן בהדרגה את המהדורה הבאה, וכל התוכן ייארז מחדש ויועלה פעם אחת.
