# הסבר מפורט על פרמטרים של שורת הפקודה

## `-p` תיקיות נקיות

`-p` או `--purge` ינקה קבצים שקיימים בכל ספריית תרגום אך אינם קיימים בספריית שפת המקור.

כי בעת כתיבת מסמכים, שמות קבצי Markdown מותאמים לעתים קרובות, וכתוצאה מכך קבצים ישנים ונטושים רבים בספריית התרגום.

השתמש בפרמטר זה כדי לנקות קבצים שיש למחוק בספריות שפות אחרות.

## `-d` ספריית תרגום

ברירת המחדל של הספרייה המתורגמת היא הספרייה שבה נמצא הקובץ הנוכחי.

`-d` או `--workdir` יכול לציין את ספריית התרגום, כגון:

```
i18 -d ~/i18n/md
```

## `-h` צפה בעזרה

`-h` או `--help` תוכל להציג את העזרה של שורת הפקודה.
