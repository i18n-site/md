# برنامه ریزی توسعه

ویژگی های مهم زیر در حال توسعه هستند:

- [ ] جابجایی بین چندین نسخه از اسناد
- [ ] جستجوی متن کامل جلویی خالص
- [ ] فید RSS
- [ ] بهینه سازی موتور جستجو
- [ ] به‌روزرسانی‌های مقاله را همزمان به رسانه‌های اجتماعی اصلی فشار دهید
- [ ] کاربران می توانند `web push` ایمیل مشترک به روز رسانی محصول شوند /
- [ ] در مورد پاراگراف های مقاله نظر دهید
- [ ] ادغام اسناد محلی برای نظارت بر تغییرات و ایجاد پیش نمایش، بدون نیاز به کامپایل دستی در هر بار و استفاده از NGINX
