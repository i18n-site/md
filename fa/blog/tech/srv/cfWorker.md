
## نام دامنه محدود به CNAME

نام دامنه ارسال `SSL/TLS` → نام میزبان سفارشی شده CNAME پس از نام دامنه Cloudflare Worker یک خطای `error code: 522` را گزارش می‌کند!

در این زمان، همانطور که در زیر نشان داده شده است، باید از مسیریابی HTTP → Workers routing برای اتصال یک مسیر استفاده کنید.
