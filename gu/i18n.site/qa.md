# સામાન્ય સમસ્યા

## આકસ્મિક રીતે કાઢી નાખવામાં આવ્યું છે `.i18n/v` જેના કારણે `npm` પેકેજ પ્રકાશિત થવામાં નિષ્ફળ થયું છે

`npm` પેકેજ રીલીઝનું ઐતિહાસિક સંસ્કરણ સાચવેલ છે `.i18n/v/ol/v.hash`

જો તમે આકસ્મિક રીતે કાઢી નાખો `.i18n/v/ol` તો `npm` પેકેજ પ્રકાશિત કરવામાં આવશે નહીં.

આ સમયે, પ્રથમ તમારા પ્રોજેક્ટના છેલ્લા રિલીઝ સંસ્કરણનો સંસ્કરણ નંબર શોધો [npmjs.com](//npmjs.com) ઉદાહરણ તરીકે `0.1.9`

પછી નીચેની `bash` બનાવો.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

નોંધ કરો કે આ રીતે રિપેર કરવાથી ફાઈલનો ઈતિહાસ ખોવાઈ જશે, જેનાથી આગલી રીલીઝને વધતા જતા અપડેટ કરવાનું અશક્ય બનશે, અને બધી સામગ્રી એકવાર ફરીથી પેકેજ અને અપલોડ કરવામાં આવશે.
