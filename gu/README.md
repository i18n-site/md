કેટલાક લોકો પૂછવા માગે છે કે, હવે જ્યારે બ્રાઉઝર્સમાં બિલ્ટ-ઇન ટ્રાન્સલેશન ફંક્શન્સ છે, તો શું વેબસાઇટનું આંતરરાષ્ટ્રીયકરણ કરવું બિનજરૂરી છે?

હું કહેવા માંગુ છું કે **સમગ્ર સાઇટનું આંતરરાષ્ટ્રીયકરણ કરીને જ આપણે બહુભાષી ઇન-સાઇટ ફુલ-ટેક્સ્ટ સર્ચ અને સર્ચ એન્જિન ઓપ્ટિમાઇઝેશનને સમર્થન આપી શકીએ છીએ** .

## ક્રમ

"બાઇબલ · ઉત્પત્તિ":

> પ્રાચીન સમયમાં, જ્યારે ભાષા એક હતી, ત્યારે માનવતાનો ગર્વ થયો હતો. તેઓએ એક ઉંચી ઈમારત બાંધવાની કોશિશ કરી, એક ટાવર જે સ્વર્ગને સ્પર્શે.
>
> જો કે, દૈવી, તેમના ઘમંડથી નારાજ થઈને, લોકોને પૃથ્વી પર વેરવિખેર કરી દીધા, તેમની જીભ એકબીજા માટે અસ્પષ્ટ રેન્ડર કરી.
>
> ત્યારથી, માનવજાતે વાતચીત કરવા માટે સંઘર્ષ કર્યો, સંઘર્ષો અવિરતપણે ઉભા થયા, અને વિશ્વને આકાશ સુધી પહોંચે તેવા ટાવરને વધુ જોયા નહીં.

વિજ્ઞાન સાહિત્ય નવલકથા &quot;થ્રી-બોડી&quot; (ચીની ઉચ્ચારણ: `3Ti` ) એક એલિયન સંસ્કૃતિને કાલ્પનિક બનાવે છે જે ઇલેક્ટ્રોમેગ્નેટિક તરંગો દ્વારા સંચાર કરે છે, તેને ભાષામાં કોઈ અવરોધો નથી અને તે તકનીકી રીતે સમૃદ્ધ છે.

હું આશા રાખું છું કે આ સાધનની મદદથી, પૃથ્વીના લોકો ત્રણ શરીરના લોકો જેવા બનશે, સંદેશાવ્યવહાર ભાષા દ્વારા પ્રતિબંધિત રહેશે નહીં, અને સમગ્ર માનવજાત ફરીથી એક થઈ જશે.
