## પરિચય

વિજ્ઞાન સાહિત્ય નવલકથા &quot;થ્રી બોડી&quot; (ચીની ઉચ્ચારણ: `3Tǐ` ) પારદર્શક વિચારસરણી અને સમૃદ્ધ ટેકનોલોજી સાથે, વાતચીત કરવા માટે ઇલેક્ટ્રોમેગ્નેટિક તરંગોનો ઉપયોગ કરતી એલિયન સંસ્કૃતિને કાલ્પનિક બનાવે છે.

બાઇબલ · ઉત્પત્તિ :

> એ જમાનામાં આકાશની નીચે બધાના અવાજો અને શબ્દો એક જ હતા.
>
> તેઓએ એક ટાવર બનાવ્યો જે આકાશમાં પહોંચે છે, ભગવાનના દરવાજાને સ્પર્શે છે, હૃદયને એક કરે છે અને તેમના ગૌરવની બડાઈ કરે છે.
>
> ભગવાને જાહેર કર્યું: જ્યારે લોકો ભાષા અને વંશ વહેંચે છે, ત્યારે તેઓ એકવચન આદિજાતિ બની જાય છે. ટાવર એ શરૂઆત છે; આવનારા દિવસોમાં, એવું કંઈ નથી જે તેઓ હાંસલ કરશે નહીં.
>
> આમ, તે નીચે ઉતર્યો, જેના કારણે લોકો દૂર દૂર સુધી વિખેરાઈ ગયા, તેમની ભાષાઓ હવે એકબીજા માટે અસ્પષ્ટ છે.
>
> ત્યારથી, ઝઘડો અનંત હતો, અને વિશ્વએ આકાશને સ્પર્શતા ટાવર જોયા નથી.

હું સાધનો બનાવવાની આશા રાખું છું જેથી પૃથ્વીના લોકો ત્રણ શરીરના લોકો જેવા બની શકે, ભાષાના બંધન વિના વાતચીત કરી શકે અને સમગ્ર માનવજાત ફરી એક થઈ શકે.

તેથી, ત્યાં [`3Ti.Site`](//3Ti.Site) હતી.
