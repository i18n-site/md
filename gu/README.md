3Ti.Site, એક બહુ-ભાષા સ્થિર સાઇટ જનરેટર, માર્કડાઉનનું [સો કરતાં વધુ વિવિધ ભાષાઓમાં](https://github.com/i18n-site/node/blob/main/lang/src/index.js) આપમેળે અનુવાદ કરી શકે છે.

<img src="//i-01.eu.org/i18n/lang.webp" alt="" />

કેટલાક લોકો પૂછવા માગે છે કે, હવે જ્યારે બ્રાઉઝર્સમાં બિલ્ટ-ઇન ટ્રાન્સલેશન ફંક્શન્સ છે, તો શું વેબસાઇટનું આંતરરાષ્ટ્રીયકરણ કરવું બિનજરૂરી છે?

હું કહેવા માંગુ છું કે **સમગ્ર સાઇટનું આંતરરાષ્ટ્રીયકરણ કરીને જ આપણે બહુભાષી ઇન-સાઇટ ફુલ-ટેક્સ્ટ સર્ચ અને સર્ચ એન્જિન ઓપ્ટિમાઇઝેશનને સમર્થન આપી શકીએ છીએ** .

## પરિચય

બાઇબલ · ઉત્પત્તિ :

> પ્રાચીન સમયમાં, જ્યારે ભાષાઓ એક હતી, ત્યારે માનવજાતે સ્વર્ગ સુધી પહોંચતો એક ટાવર બનાવ્યો, જે ભગવાનના ડોમેન માટેનું એક પોર્ટલ હતું, જે માનવજાતની શક્તિનો ઘોષણા કરતું હતું.
>
> ભગવાને જાહેર કર્યું, "પુરુષો એક આદિજાતિ તરીકે, એક સહિયારી જીભ સાથે એક થાય છે, અને આ ટાવર માત્ર એક પ્રસ્તાવના છે. હવે તેઓ તેમની ઇચ્છા પ્રાપ્ત કરે છે, અને હવે કોઈ ભય રહેશે નહીં."
>
> આમ, ભગવાન નીચે ઉતર્યા, લોકોને જુદા જુદા ખૂણામાં વિખેર્યા, વિવિધ ભાષાઓનો ઉપયોગ કરો.
>
> ત્યારથી, માનવ સંદેશાવ્યવહાર પડકારરૂપ બન્યો, વિવાદો અનંત બન્યા, અને આકાશને સ્પર્શે એવો ટાવર હવે રહ્યો નહીં.

વિજ્ઞાન સાહિત્ય નવલકથા &quot;થ્રી-બોડી&quot; (ચીની ઉચ્ચારણ: `3Ti` ) એક એલિયન સંસ્કૃતિને કાલ્પનિક બનાવે છે જે ઇલેક્ટ્રોમેગ્નેટિક તરંગો દ્વારા સંચાર કરે છે, તેને ભાષામાં કોઈ અવરોધો નથી અને તે તકનીકી રીતે સમૃદ્ધ છે.

હું એક એવું સાધન બનાવવાની આશા રાખું છું જે પૃથ્વીના લોકોને ત્રણ શરીરના લોકો જેવા બનવા, ભાષાથી બંધાયેલા વિના વાતચીત કરવા અને સમગ્ર માનવજાતને ફરીથી એક કરવા સક્ષમ બનાવે.
