# આદેશ વાક્ય પરિમાણોની વિગતવાર સમજૂતી

## ફાઈલો સાફ `-p`

`-p` `--purge` ફાઇલોને સાફ કરશે જે દરેક અનુવાદ નિર્દેશિકામાં અસ્તિત્વમાં છે પરંતુ સ્રોત ભાષા નિર્દેશિકામાં અસ્તિત્વમાં નથી.

કારણ કે દસ્તાવેજો લખતી વખતે, માર્કડાઉન ફાઇલના નામો ઘણીવાર એડજસ્ટ કરવામાં આવે છે, પરિણામે અનુવાદ નિર્દેશિકામાં ઘણી જૂની અને ત્યજી દેવાયેલી ફાઇલો આવે છે.

અન્ય ભાષાની ડિરેક્ટરીઓમાં ડિલીટ થવી જોઈએ તેવી ફાઇલોને સાફ કરવા માટે આ પેરામીટરનો ઉપયોગ કરો.

## અનુવાદ નિર્દેશિકા સ્પષ્ટ `-d`

અનુવાદિત ડિરેક્ટરી ડિફૉલ્ટ ડિરેક્ટરીમાં છે જ્યાં વર્તમાન ફાઇલ સ્થિત છે.

`-d` `--workdir` અનુવાદ નિર્દેશિકાનો ઉલ્લેખ કરી શકે છે, જેમ કે:

```
i18 -d ~/i18n/md
```

## મદદ `-h`

`-h` `--help` કમાન્ડ લાઇન મદદ જોઈ શકો છો.
