
## CNAME બાઉન્ડ ડોમેન નામ

Cloudflare Worker નામ પાસ `SSL/TLS` `error code: 522` કસ્ટમાઇઝ્ડ હોસ્ટ નેમ CNAME

આ સમયે, તમારે રૂટને બાંધવા માટે HTTP રૂટીંગ → વર્કર્સ રૂટીંગનો ઉપયોગ કરવાની જરૂર છે, નીચે બતાવ્યા પ્રમાણે
