
## CNAME બાઉન્ડ ડોમેન નામ

Cloudflare Worker નું ડોમેન નામ `SSL/TLS` → કસ્ટમ હોસ્ટ નામ દ્વારા બંધાયેલું છે CNAME ડોમેન નામ બંધાયા પછી, ડાયરેક્ટ એક્સેસ ભૂલ `error code: 522` ની જાણ કરશે

આ સમયે, તમારે રૂટને બાંધવા માટે HTTP રૂટીંગ → વર્કર્સ રૂટીંગનો ઉપયોગ કરવાની જરૂર છે, નીચે બતાવ્યા પ્રમાણે
