# Buyruqlar Qatori Parametrlarini Batafsil Tushuntirish

## Fayllarni `-P`

`-p` `--purge` har bir tarjima katalogida mavjud bo'lgan, lekin manba til katalogida mavjud bo'lmagan fayllarni tozalaydi.

Chunki hujjatlarni yozishda Markdown fayl nomlari ko'pincha o'zgartiriladi, natijada tarjima katalogida ko'plab eski va tashlab ketilgan fayllar paydo bo'ladi.

Boshqa til kataloglarida o'chirilishi kerak bo'lgan fayllarni tozalash uchun ushbu parametrdan foydalaning.

## Tarjima Katalogini `-D`

Tarjima qilingan katalog sukut bo'yicha joriy fayl joylashgan katalogga o'rnatiladi.

`-d` `--workdir` tarjima katalogini belgilashi mumkin, masalan:

```
i18 -d ~/i18n/md
```

## `-H` Ko'ring

`-h` `--help` buyruq qatori yordamini ko'rishi mumkin.
