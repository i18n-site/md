# വികസന ആസൂത്രണം

ഇനിപ്പറയുന്ന പ്രധാന സവിശേഷതകൾ വികസിപ്പിച്ചുകൊണ്ടിരിക്കുന്നു:

- [ ] പ്രമാണങ്ങളുടെ ഒന്നിലധികം പതിപ്പുകൾക്കിടയിൽ മാറുന്നു
- [ ] ശുദ്ധമായ ഫ്രണ്ട്-എൻഡ് ഫുൾ-ടെക്സ്റ്റ് തിരയൽ
- [ ] RSS ഫീഡ്
- [ ] തിരയൽ എഞ്ചിൻ ഒപ്റ്റിമൈസേഷൻ
- [ ] പ്രധാന സോഷ്യൽ മീഡിയകളിലേക്ക് ലേഖന അപ്ഡേറ്റുകൾ ഒരേസമയം പുഷ് ചെയ്യുക
- [ ] ഉപയോക്താക്കൾക്ക് ഇമെയിൽ / `web push` ഉൽപ്പന്ന അപ്ഡേറ്റുകൾ സബ്സ്ക്രൈബ് ചെയ്യുക
- [ ] ലേഖന ഖണ്ഡികകളിൽ അഭിപ്രായമിടുക
- [ ] മാറ്റങ്ങൾ നിരീക്ഷിക്കുന്നതിനും പ്രിവ്യൂകൾ സൃഷ്ടിക്കുന്നതിനും പ്രാദേശിക പ്രമാണങ്ങൾ സംയോജിപ്പിക്കുക, ഓരോ തവണയും സ്വമേധയാ സമാഹരിക്കുന്നതിൻ്റെ ആവശ്യകത ഇല്ലാതാക്കുകയും NGINX ഉപയോഗിക്കുകയും ചെയ്യുക
