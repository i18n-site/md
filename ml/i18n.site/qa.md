# സാധാരണ പ്രശ്നം

## ആകസ്മികമായി ഇല്ലാതാക്കി `.i18n/v` `npm` പാക്കേജ് പ്രസിദ്ധീകരിക്കുന്നതിൽ പരാജയപ്പെടുന്നു

`npm` പാക്കേജ് റിലീസിൻ്റെ ചരിത്ര പതിപ്പ് സംരക്ഷിച്ചു `.i18n/v/ol/v.hash`

നിങ്ങൾ അബദ്ധവശാൽ `.i18n/v/ol` `npm` പാക്കേജ് പ്രസിദ്ധീകരിക്കില്ല.

ഈ സമയത്ത്, നിങ്ങളുടെ പ്രോജക്റ്റിൻ്റെ അവസാന പതിപ്പിൻ്റെ പതിപ്പ് നമ്പർ ആദ്യം കണ്ടെത്തുക [npmjs.com](//npmjs.com) ഉദാഹരണത്തിന് `0.1.9`

തുടർന്ന് താഴെയുള്ള `bash` ഫയൽ കാണുക.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

ഈ രീതിയിൽ റിപ്പയർ ചെയ്യുന്നത് ഫയൽ ചരിത്രം നഷ്‌ടപ്പെടുത്തും, ഇത് അടുത്ത പതിപ്പ് വർദ്ധിപ്പിച്ച് അപ്‌ഡേറ്റ് ചെയ്യുന്നത് അസാധ്യമാക്കുന്നു, കൂടാതെ എല്ലാ ഉള്ളടക്കവും വീണ്ടും പാക്കേജുചെയ്‌ത് ഒരിക്കൽ അപ്‌ലോഡ് ചെയ്യപ്പെടും.
