
## CNAME ബന്ധിപ്പിച്ച ഡൊമെയ്ൻ നാമം

Cloudflare Worker ൻ്റെ ഡൊമെയ്ൻ നാമം `SSL/TLS` → ഇഷ്ടാനുസൃത ഹോസ്റ്റ് നാമത്തിലൂടെ ബന്ധിപ്പിച്ചിരിക്കുന്നു CNAME ഡൊമെയ്ൻ നാമം ബന്ധിപ്പിച്ചതിന് ശേഷം, നേരിട്ടുള്ള ആക്സസ് ഒരു പിശക് റിപ്പോർട്ട് ചെയ്യും `error code: 522`

ഈ സമയത്ത്, താഴെ കാണിച്ചിരിക്കുന്നതുപോലെ ഒരു റൂട്ട് ബന്ധിപ്പിക്കുന്നതിന് നിങ്ങൾ HTTP റൂട്ടിംഗ് → വർക്കേഴ്സ് റൂട്ടിംഗ് ഉപയോഗിക്കേണ്ടതുണ്ട്
