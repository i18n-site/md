## ആമുഖം

ശാസ്ത്ര ഫിക്ഷൻ നോവൽ &quot;ത്രീ-ബോഡി&quot; (ചൈനീസ് ഉച്ചാരണം: `3Ti` ) വൈദ്യുതകാന്തിക തരംഗങ്ങളിലൂടെ ആശയവിനിമയം നടത്തുന്ന, ഭാഷാ തടസ്സങ്ങളില്ലാത്ത, സാങ്കേതികമായി സമ്പന്നമായ ഒരു അന്യഗ്രഹ നാഗരികതയെ സാങ്കൽപ്പികമാക്കുന്നു.

ബൈബിൾ · ഉല്പത്തി :

> മനുഷ്യരാശിയുടെ അന്തസ്സ് ഉയർത്തിപ്പിടിച്ച്, ദൈവത്തിന്റെ കവാടത്തിലെത്തി, ആകാശത്ത് എത്തുന്ന ഗോപുരങ്ങൾ പുരാതനന്മാർ നിർമ്മിച്ചു.
>
> ആളുകൾ സ്വന്തം വംശം രൂപീകരിക്കുകയും ഒരേ ഭാഷ സംസാരിക്കുകയും ചെയ്യുന്നുവെന്ന് ഭഗവാൻ പറഞ്ഞു. ഒരു ഗോപുരം പണിയുന്നത് ഒരു ആമുഖം മാത്രമാണ്. ഇപ്പോൾ അത് ചെയ്തുകഴിഞ്ഞാൽ, അവർ ഭാവിയിൽ എല്ലാം ചെയ്യും.
>
> അപ്പോൾ കർത്താവ് വന്ന് പരസ്പരം മനസ്സിലാക്കാൻ കഴിയാതെ ആളുകളെ പലയിടങ്ങളിലായി ചിതറിച്ചു.
>
> അതിനുശേഷം, മാനുഷിക തർക്കങ്ങൾ തുടർന്നു, ലോകത്ത് ബാബേൽ ഗോപുരം ഇല്ല.

ഭൂമിയിലെ മനുഷ്യരെ മൂന്ന് ശരീരങ്ങളെപ്പോലെ ആകാനും ഭാഷയിൽ ബന്ധിതരാകാതെ ആശയവിനിമയം നടത്താനും എല്ലാ മനുഷ്യരെയും വീണ്ടും ഒന്നിപ്പിക്കാനും കഴിയുന്ന ഒരു ഉപകരണം സൃഷ്ടിക്കുമെന്ന് ഞാൻ പ്രതീക്ഷിക്കുന്നു.
