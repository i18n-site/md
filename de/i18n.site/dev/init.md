# Umgebung vorbereiten

* Installieren und starten Sie `docker` (Mac-Benutzer werden empfohlen, [orbstack](https://orbstack.dev) als Runtime für `docker` zu verwenden)

* Installieren Sie [bun](https://bun.sh/docs/installation) und [nodejs](https://nodejs.org/en/download/package-manager)

## Projekt initialisieren

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
