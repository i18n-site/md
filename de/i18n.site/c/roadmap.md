# Entwicklungsplanung

Die folgenden wichtigen Funktionen sind in der Entwicklung:

- [ ] Wechseln zwischen mehreren Versionen von Dokumenten
- [ ] Reine Frontend-Volltextsuche
- [ ] RSS-Feed
- [ ] Suchmaschinenoptimierung
- [ ] Veröffentlichen Sie gleichzeitig Artikelaktualisierungen in den wichtigsten sozialen Medien
- [ ] Benutzer können Produktaktualisierungen `web push` E-Mail abonnieren /
- [ ] Kommentieren Sie Artikelabsätze
- [ ] Integrieren Sie lokale Dokumente, um Änderungen zu überwachen und Vorschauen zu erstellen, sodass nicht jedes Mal eine manuelle Kompilierung erforderlich ist und NGINX verwendet wird
