# कमाण्ड लाइन प्यारामिटरहरूको विस्तृत व्याख्या

## फाइलहरू सफा `-p`

वा `--purge` प्रत्येक अनुवाद डाइरेक्टरीमा अवस्थित तर स्रोत भाषा डाइरेक्टरीमा अवस्थित नभएका फाइलहरूलाई सफा गर्नेछ `-p`

किनभने कागजातहरू लेख्दा, मार्कडाउन फाइल नामहरू प्रायः समायोजित हुन्छन्, परिणामस्वरूप अनुवाद डाइरेक्टरीमा धेरै पुराना र परित्याग गरिएका फाइलहरू।

अन्य भाषा डाइरेक्टरीहरूमा मेटिने फाइलहरू सफा गर्न यो प्यारामिटर प्रयोग गर्नुहोस्।

## अनुवाद निर्देशिका निर्दिष्ट `-d`

हालको फाइल अवस्थित डाइरेक्टरीमा अनुवाद गरिएको डाइरेक्टरी पूर्वनिर्धारित हुन्छ।

`-d` `--workdir` अनुवाद निर्देशिका निर्दिष्ट गर्न सक्नुहुन्छ, जस्तै:

```
i18 -d ~/i18n/md
```

## मद्दत `-h`

`-h` `--help` कमाण्ड लाइन मद्दत हेर्न सक्नुहुन्छ।
