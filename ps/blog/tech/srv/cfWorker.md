
## د CNAME پابند ډومین نوم

د ډومین نوم د `SSL/TLS` → دودیز کوربه نوم پورې تړلی دی CNAME له دې Cloudflare Worker د ډومین نوم تړل شوی، مستقیم لاسرسی به د `error code: 522` غلطی راپور ورکړي

په دې وخت کې، تاسو اړتیا لرئ چې د HTTP روټینګ وکاروئ → کارګر روټینګ د یوې لارې تړلو لپاره، لکه څنګه چې لاندې ښودل شوي
