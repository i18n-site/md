# عام ستونزه

## په ناڅاپه توګه ړنګ `npm` `.i18n/v`

د `npm` بسته بندۍ تاریخي نسخه په کې خوندي شوې ده `.i18n/v/ol/v.hash`

که تاسو په ناڅاپي ډول حذف کړئ `.i18n/v/ol` `npm` ​​کڅوړه به خپره نشي.

پدې وخت کې، لومړی د خپلې پروژې د وروستي خپریدو نسخه شمیره ومومئ [npmjs.com](//npmjs.com) د بیلګې په توګه `0.1.9`

بیا لاندې فایل ته مراجعه وکړئ `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

په یاد ولرئ چې پدې طریقه ترمیم کول به د فایل تاریخ له لاسه ورکړي، دا به ناشونی وي چې راتلونکی ریلیز په تدریجي ډول تازه کړي، او ټول مینځپانګې به یو ځل بیا پیک او اپلوډ شي.
