# د کمانډ لاین پیرامیټونو تفصيلي توضیحات

## فایلونه پاک `-p`

`-p` `--purge` به هغه فایلونه پاک کړي چې د هرې ژباړې لارښود کې شتون لري مګر د سرچینې ژبې لارښود کې شتون نلري.

ځکه چې کله د اسنادو لیکلو سره، د مارک ډاون فایل نومونه ډیری وختونه تنظیم کیږي، په پایله کې د ژباړې په لارښود کې ډیری زاړه او پریښودل شوي فایلونه.

دا پیرامیټر د فایلونو پاکولو لپاره وکاروئ کوم چې باید د نورو ژبو لارښودونو کې حذف شي.

## د ژباړې لارښود مشخص `-d`

ژباړل شوی ډایرکټر هغه ډایرکټر ته ډیفالټ کوي چیرې چې اوسنی فایل موقعیت لري.

`-d` `--workdir` کولی شي د ژباړې لارښود مشخص کړي، لکه:

```
i18 -d ~/i18n/md
```

## مرسته `-h`

`-h` `--help` کولی شئ د کمانډ لاین مرسته وګورئ.
