# Λεπτομερής Επεξήγηση των Παραμέτρων της Γραμμής Εντολών

## `-p` Καθαρισμός Αρχείων

`-p` ή `--purge` θα καθαρίσει αρχεία που υπάρχουν στους καταλόγους των μεταφράσεων, αλλά δεν υπάρχουν στον κατάλογο της πηγαίας γλώσσας.

Λόγω της συχνής προσαρμογής των ονομάτων αρχείων Markdown κατά τη συγγραφή τεκμηρίωσης, προκύπτουν πολλά παλαιά και περιττά αρχεία στους καταλόγους των μεταφράσεων.

Χρησιμοποιώντας αυτήν την παράμετρο, μπορείτε να καθαρίσετε αρχεία που θα πρέπει να�除ηθούν στους καταλόγους άλλων γλωσσών.

## `-d` Ορισμός Καταλόγου Μετάφρασης

Ο κατάλογος μετάφρασης είναι προεπιλεγμένος στον κατάλογο όπου βρίσκεται το τρέχον αρχείο.

`-d` ή `--workdir` μπορεί να καθορίσει τον κατάλογο μετάφρασης, π.χ.:

```
i18 -d ~/i18n/md
```

## `-h` Προβολή Βοήθειας

`-h` ή `--help` για να δείτε τη βοήθεια της γραμμής εντολών.
