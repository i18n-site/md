<img src="//i-01.eu.org/i18n/lang.webp" alt="" />

Μερικοί άνθρωποι μπορεί να ρωτήσουν, τώρα που τα προγράμματα περιήγησης έχουν ενσωματωμένες λειτουργίες μετάφρασης, είναι περιττό να διεθνοποιηθεί ο ιστότοπος;

Θέλω να πω ότι **μόνο με τη διεθνοποίηση ολόκληρου του ιστότοπου μπορούμε να υποστηρίξουμε πολυγλωσσική αναζήτηση πλήρους κειμένου εντός του ιστότοπου και βελτιστοποίηση μηχανών αναζήτησης** .

## Εισαγωγή

Βίβλος · Γένεση :

> Στην αρχαιότητα, όταν οι γλώσσες ήταν μία, η ανθρωπότητα έχτισε έναν πύργο που έφτανε στους ουρανούς, μια πύλη στην κυριαρχία του Θεού, διακηρύσσοντας τη δύναμη της ανθρωπότητας.
>
> Ο Θεός δήλωσε, "Οι άνθρωποι ενώνονται ως μία φυλή, με κοινή γλώσσα, και αυτός ο πύργος δεν είναι παρά ένας πρόλογος. Τώρα πετυχαίνουν την επιθυμία τους και δεν θα υπάρχει πια φόβος."
>
> Έτσι, ο Θεός κατέβηκε, σκορπίζοντας ανθρώπους σε διαφορετικές γωνιές, χρησιμοποιούν διαφορετικές γλώσσες.
>
> Από τότε, η ανθρώπινη επικοινωνία έγινε προκλητική, οι διαμάχες ατελείωτες και δεν υπήρχε πλέον πύργος που άγγιξε τον ουρανό.

Το μυθιστόρημα επιστημονικής φαντασίας &quot;Three-Body&quot; (Κινεζική προφορά: `3Ti` ) μυθιστορεί έναν εξωγήινο πολιτισμό που επικοινωνεί μέσω ηλεκτρομαγνητικών κυμάτων, δεν έχει γλωσσικά εμπόδια και είναι τεχνολογικά ευημερούν.

Ελπίζω να δημιουργήσω ένα εργαλείο που θα επιτρέψει στους ανθρώπους της γης να είναι σαν άνθρωποι με τρία σώματα, να επικοινωνούν χωρίς να δεσμεύονται από τη γλώσσα και να ενώσουν ξανά όλη την ανθρωπότητα.

### Τροποποίηση Μετάφρασης

Αφού τροποποιήσετε τη μετάφραση, πρέπει να εκτελέσετε ξανά `./i18n.sh` για να ενημερώσετε την προσωρινή μνήμη.

### Σημειώσεις Μετάφρασης

Τα σχόλια μετάφρασης πρέπει να υποδεικνύουν τη γλώσσα μετά το \```, όπως το ` ```rust` .

Αυτήν τη στιγμή υποστηρίζεται η μετάφραση σχολίων για rust, c, cpp, java, js, καφέ, python και bash.

Επεξεργαστείτε [το tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) για να προσθέσετε υποστήριξη μετάφρασης για σχόλια σε περισσότερες γλώσσες.

### Διαμόρφωση Διακομιστή Μεσολάβησης

Η ρύθμιση των ακόλουθων μεταβλητών περιβάλλοντος επιτρέπει στις κλήσεις API της Μετάφρασης Google να περνούν μέσω του διακομιστή μεσολάβησης.

```bash
export https_proxy=http://127.0.0.1:7890
```
