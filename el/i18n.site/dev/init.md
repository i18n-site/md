# Προετοιμάστε το περιβάλλον

* Εγκαταστήστε και ξεκινήστε το `docker` (οι χρήστες Mac προτείνουν να χρησιμοποιήσετε το [orbstack](https://orbstack.dev) ως runtime για το `docker`)

* Εγκαταστήστε το [bun](https://bun.sh/docs/installation) και το [nodejs](https://nodejs.org/en/download/package-manager)

## Αρχικοποιήστε το έργο

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
