# i18n.site: Καθαρό Στατικό Πολυγλωσσικό Πλαίσιο Ιστότοπου

`i18n.site` πολυγλωσσική, καθαρά στατική δημιουργία τοποθεσίας εγγράφων.

## Πρόλογος

`i18n.site` είναι ένας δημιουργός ιστότοπων για έγγραφα, καθώς και ένα πλαίσιο ανάπτυξης ιστοσελίδων.

Ένας νέος τρόπος ανάπτυξης ιστού με το `MarkDown` στο κέντρο και την ενσωμάτωση διαδραστικών στοιχείων μέσω front-end组件.

Κάθε front-end στοιχείο είναι ένα ανεξάρτητο πακέτο που μπορεί να εγκατασταθεί.

Με βάση τον διαχωρισμό του front-end και του back-end, υπάρχει επίσης διαχωρισμός μεταξύ στατικού περιεχομένου και δυναμικών δεδομένων.

Ο ιστότοπος που επισκέπτεστε, [i18n.site](/), είναι χτισμένος με βάση αυτό το πλαίσιο (περιλαμβάνει σύστημα χρηστών, σύστημα χρεώσεων, συνδρομή μέσω email κ.λπ.).
