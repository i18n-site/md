## Πρόλογος

`i18n.site` η επανεξέταση του μοντέλου ανάπτυξης ιστοτόπου.

Αυτό είναι ένα πρόγραμμα δημιουργίας τοποθεσιών εγγράφων και ένα πλαίσιο ανάπτυξης ιστότοπου.

Ένα νέο παράδειγμα ανάπτυξης ιστοτόπων που επικεντρώνεται γύρω από το `MarkDown`

Κάθε στοιχείο διεπαφής είναι ένα πακέτο που μπορεί να εγκατασταθεί ανεξάρτητα.

Με βάση τον διαχωρισμό του front-end και του back-end, υπάρχει επίσης ο διαχωρισμός στατικού περιεχομένου και δυναμικών δεδομένων.

Αυτό που [i18n.site](/) είναι χτισμένο με βάση αυτό το πλαίσιο (συμπεριλαμβανομένου του συστήματος χρήστη, του συστήματος χρέωσης, της συνδρομής ηλεκτρονικού ταχυδρομείου κ.λπ.).
