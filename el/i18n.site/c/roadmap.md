# Αναπτυξιακός Σχεδιασμός

Τα εξής σημαντικά χαρακτηριστικά βρίσκονται σε ανάπτυξη:

- [ ] Αντι交换 μεταξύ πολλαπλών εκδόσεων των εγγράφων
- [ ] Πλήρης αναζήτηση κειμένου σε καθαρά frontend
- [ ] Προσθήκη ροής RSS
- [ ] βελτιστοποίηση μηχανών αναζήτησης
- [ ] Συγχρονισμένη προώθηση ενημερώσεων άρθρων στις μεγάλες πλατφόρμες κοινωνικής δικτύωσης
- [ ] Email διαθέσιμο στους χρήστες / `web push` για εγγραφή σε ενημερώσεις προϊόντων
- [ ] Δυνατότητα σχολιασμού των παραγράφων των άρθρων
- [ ] Ενσωμάτωση τοπικής παρακολούθησης εγγράφων για την αυτόματη δημιουργία προεπισκοπήσεων, χωρίς την ανάγκη για χειροκίνητη 编译 και τη χρήση του NGINX
