# Υποσέλιδο

Άλλωστε, παίρνοντας ως παράδειγμα το έργο παρουσίασης, το αρχείο `.i18n/htm/foot.pug` στον κατάλογο `md` ορίζει το υποσέλιδο της ιστοσελίδας.

![](https://p.3ti.site/1721286077.avif)

[`pug`](https://pugjs.org) είναι μια γλώσσα προτύπων που δημιουργεί `HTML`.

[➔ Κάντε κλικ εδώ για να μάθετε τη γραμματική του pug](https://pugjs.org)

**Μην γράφετε `css` και `js` στο `foot.pug`**, αλλιώς θα προκύψει σφάλμα.

Ανατρέξτε στις παρακάτω οδηγίες, το στυλ γράφεται στο αντίστοιχο `.css`, ενώ η αλληλεπίδραση επιτυγχάνεται μέσω της δημιουργίας στοιχείων ιστοσελίδας.

## Στυλ Υποσέλιδου

Στο έργο παρουσίασης υπάρχουν τρία αρχεία `css` στον κατάλογο `md/.i18n/htm`.

* `foot.css`: στυλ υποσέλιδου
* `import.css` : `i18n.site` για ολόκληρο τον ιστότοπο
* `conf.css`: εικονίδια και γραμματοσειρές του υποσέλιδου

### Εικονίδιο Γραμματοσειράς

Τα εικονίδια του υποσέλιδου δημιουργούνται από μια γραμματοσειρά με το όνομα `F` μέσω του iconfont.cn ([Αγγλική έκδοση](https://www.iconfont.cn/?lang=en-us) / [Κινεζική έκδοση](https://www.iconfont.cn/?lang=zh)).

Δημιουργήστε τα δικά σας εικονίδια γραμματοσειράς ανάλογα με τις ανάγκες σας και αντικαταστήστε τις παρακάτω ρυθμίσεις στο `conf.css`:

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
}
```

## Στοιχεία Ιστού

Δεν μπορείτε να γράψετε `js` στο `foot.pug`. Αν χρειάζεστε αλληλεπίδραση, δημιουργήστε προσαρμοσμένα στοιχεία ιστοσελίδας.

[Ένα στοιχείο web](https://www.freecodecamp.org/news/build-your-first-web-component/) μπορεί να οριστεί στο `md/.i18n/htm/index.js` και στη συνέχεια να χρησιμοποιηθεί στο `foot.pug`.

Η δημιουργία προσαρμοσμένων στοιχείων ιστού είναι εύκολη, όπως για παράδειγμα το προσαρμοσμένο ετικέτα `<x-img>`.

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```

## Πολυγλωσσικό Υποσέλιδο

Ο κώδικας στο `.i18n/htm/foot.pug` είναι ο εξής:

```
#Ft
  b
    a.site(href="/")
    b ${I18N.C}
```

Εδώ `${I18N.C}` αντιστοιχεί σε αυτό που υπάρχει στο `en/i18n.yml`:

```
C: Power By <a class="a" href="https://i18n.site">i18n.site</a>
```

Χρησιμοποιώντας τη μορφή `${I18N.xxx}`, σε συνδυασμό με το `i18n.yml`, μπορείτε να επιτύχετε την πολυγλωσσική διεθνοποίηση του υποσέλιδου.

Η προσθήκη `class="a"` στους συνδέσμους γίνεται για να αποτρέψει τη μετατροπή τους σε μορφή `Markdown`. Δείτε:
 [➔ `YAML` Πώς να αποτρέψετε τη μετατροπή ενός συνδέσμου `HTML` σε `Markdown`](/i18/qa#H2).
