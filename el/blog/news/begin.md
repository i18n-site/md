# i18n.site · Το εργαλείο μετάφρασης και δημιουργίας ιστοτόπων MarkDown είναι πλέον διαθέσιμο!

Μετά από περισσότερο από μισό χρόνο ανάπτυξης, το [https://i18n.site](//i18n.site) είναι διαθέσιμο.

Προς το παρόν, έχουν υλοποιηθεί δύο εργαλεία γραμμής εντολών ανοιχτού κώδικα:

* `i18` : Εργαλείο γραμμής εντολών για τη μετάφραση MarkDown
* `i18n.site` : Δημιουργός πολυγλωσσικών στατικών εγγράφων, **βελτιστοποιημένος για την εμπειρία ανάγνωσης**

Η μετάφραση διατηρεί τέλεια τη μορφή του `Markdown`. Μπορεί να αναγνωρίσει τις αλλαγές στα αρχεία και να μεταφράσει μόνο τα αρχεία που έχουν τροποποιηθεί.

Η μετάφραση είναι επεξεργάσιμη· αν διορθώσετε το πρωτότυπο και μεταφράσετε ξανά, οι διορθώσεις δεν θα αντικατασταθούν (αν η εν λόγω παράγραφος δεν έχει τροποποιηθεί).

## Προέλευση

Στην εποχή του Διαδικτύου, όλος ο κόσμος είναι αγορά, και η πολυγλωσσία και η τοπικοποίηση είναι βασικές δεξιότητες.

Τα υπάρχοντα εργαλεία διαχείρισης μετάφρασης είναι πολύ βαριά για προγραμματιστές που χρησιμοποιούν `git` για τη διαχείριση εκδόσεων, και εξακολουθούν να προτιμούν τη γραμμή εντολών.

Επομένως, αναπτύξα το εργαλείο μετάφρασης `i18` και δημιούργησα τον δημιουργό πολυγλωσσικών στατικών ιστοσελίδων `i18n.site` με βάση το εργαλείο μετάφρασης.

![](https://p.3ti.site/1723777556.avif)

Αυτό είναι μόνο η αρχή, υπάρχουν πολλά ακόμα που θέλουμε να κάνουμε.

Για παράδειγμα, να συνδέσουμε τον στατικό ιστότοπο εγγράφων με κοινωνικά δίκτυα και συνδρομές email, ώστε να μπορούμε να ενημερώνουμε τους χρήστες εγκαίρως όταν δημοσιεύονται ενημερώσεις.

Για παράδειγμα, να ενσωματώσουμε πολυγλωσσικά φόρουμ και συστήματα εργασιών σε οποιαδήποτε ιστοσελίδα, επιτρέποντας στους χρήστες να επικοινωνούν χωρίς εμπόδια.

## Ανοιχτού Κώδικα

[Οι κώδικες για το front-end, το back-end και τη γραμμή εντολών είναι όλοι διαθέσιμοι ως ανοιχτός κώδικας](https://i18n.site/i18n.site/c/src) (το μοντέλο μετάφρασης δεν έχει ακόμη ανοιχτεί για κοινή χρήση).

Η τεχνολογική στοίβα που χρησιμοποιούμε είναι η εξής:

Frontend [svelte](https://svelte.dev), [stylus](https://stylus-lang.com), [pug](https://github.com/pugjs/pug), [vite](https://github.com/vitejs/vite)

Η γραμμή εντολών και το back-end αναπτύσσονται με τη χρήση rust.

Back-end [axum](https://github.com/tokio-rs/axum), [tower-http](https://github.com/tower-rs/tower-http/releases).

Γραμμή εντολών [嵌入式js引擎 boa_engine](https://docs.rs/boa_engine), [嵌入式数据库 fjall](https://github.com/fjall-rs/fjall).

Διακομιστής VPS [contabo](https://my.contabo.com)

Βάση δεδομένων [kvrocks](https://kvrocks.apache.org), [mariadb](https://mariadb.org).

Αποστολή email μέσω αυθεντικού SMTP [chasquid](https://github.com/albertito/chasquid).
