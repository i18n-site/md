
## Όνομα Τομέα Δεσμευμένου CNAME

Το όνομα τομέα `SSL/TLS` → Προσαρμοσμένο όνομα κεντρικού υπολογιστή Cloudflare Worker Μετά το όνομα τομέα, η άμεση πρόσβαση θα αναφέρει ένα σφάλμα `error code: 522` CNAME

Αυτή τη στιγμή, πρέπει να χρησιμοποιήσετε τη δρομολόγηση HTTP → Εργαζόμενοι που δρομολογούν για να συνδέσετε μια διαδρομή, όπως φαίνεται παρακάτω
