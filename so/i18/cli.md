# Sharaxaad Faahfaahsan Oo Ku Saabsan Cabbiraadaha Khadka Taliska

## `-P` Faylasha

`-p` `--purge` waxay nadiifin doontaa faylasha ku jira hagaha tarjumaadda laakiin aan ku jirin tusaha luqadda.

Sababtoo ah marka dukumiintiyada la qorayo, faylasha Markdown badanaa waa la hagaajiyaa, taasoo keenta faylal badan oo duug ah iyo kuwa la dayacay oo ku jira hagaha tarjumaada.

Isticmaal cabbirkan si aad u nadiifiso faylasha ay tahay in lagu tirtiro hagayaasha luqadaha kale.

## `-D` Tusaha Tarjumaada

Tusaha la turjumay waxa uu ku dhici doonaa hagaha halka uu ku yaallo faylka hadda jira.

`-d` `--workdir` waxay qeexi kartaa tusaha tarjumaada, sida:

```
i18 -d ~/i18n/md
```

## `-H` Caawimada

`-h` `--help` arki kartaa caawinta khadka taliska.
