- [ ] U beddelashada noocyo badan oo dukumeenti ah
- [ ] Raadinta qoraal-buuxa ee hore-dhamaadka saafiga ah
- [ ] Akhriska RSS
- [ ] hagaajinta mashiinka raadinta

- [ ] Iimayl diyaar u ah / `web push` Ku biir wixii cusboonaysiinta alaabta
