# Habka MarkDown

## Barta Barroosinka

Barroosinno `MarkDown` ayaa la soo saaray iyadoo lagu salaynayo nuxurka qoraalka.

Xalka barroosinka `i18n.site` heshiiyey waa in la geliyo qoraal la mid ah `<a rel=id href="#xxx" id="xxx"></a>` in `MarkDown` si gacanta loogu sameeyo barroosinka.

`<a rel=id href="#xxx" id="xxx"></a>` halkan `rel=id` waxay qeexaysaa qaabka bogga barta barroosinka, fadlan `xxx` ku beddel soo gaabintaada dhabta ah ee Ingiriisiga.

Barroosinnada waxaa badanaa lagu daraa cinwaanka, sida:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Ku Qor `HTML` Gudaha `MarkDown`

`pug` `HTML` lagu dhejin karaa koodka.

Waxa ku jira `<pre>` lama tarjumi doono.

Isku darka labadan qodob, waxaad si fudud u qori kartaa `HTML` gudaha `MarkDown`

Waxaad tixraaci kartaa hirgelinta gudaha [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) koodka waa sida soo socota :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Ogow in `<style>` sidoo kale lagu qeexay `<pre>`
