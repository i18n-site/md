# i18n.site: Qaabka Mareegaha Ee Luqadaha Badan Ee Saafiga Ah

`i18n.site` luqado badan, oo kaliya madhalaysa dukumeenti goobta.

Hadda waa la heli karaa markii hore, laakiin weli waxaa jira sifooyin muhiim ah oo ku hoos jira horumarinta Fadlan eeg [qorshaha horumarinta](/i18n.site/c/roadmap) .

Fadlan <button onclick="mailsub()">dhagsii iimaylkan si aad isu diiwaan geliso cusboonaysiinta badeecada</button> oo <button onclick="webpush()">aad awood u yeelato riixitaanka browser-ka</button> Waxaan ku ogaysiin doonaa marka alaabta cusub la sameeyo.

Sidoo kale ku soo dhawoow inaad raacdo [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) / [X.COM: @i18nSite](https://x.com/i18nSite)

## Hordhac

`i18n.site` waa dib u fakarkayga qaabka horumarinta shabakada.

Kani waa dukumeenti dhaliye goobta ah iyo qaabka horumarinta mareegaha.

Qaab cusub oo horumarinta mareegaha oo qaata `MarkDown` xarun ahaan oo adeegsata qaybaha hore-u-dhamaadka si loo durayo isdhexgalka.

Qayb kasta oo hore-dhamaadka ah waa xirmo si madaxbanaan loo rakibo.

Iyada oo ku saleysan kala-soocidda hore-dhamaadka iyo dhabarka-dhamaadka, waxaa sidoo kale jira kala-soocidda nuxurka taagan iyo xogta firfircoon.

Waxa aad [i18n.site](/) waxaa lagu dhisay qaab-dhismeedkan (oo ay ku jiraan nidaamka isticmaale, nidaamka biilasha, is-diiwaangelinta iimaylka, iwm.).
