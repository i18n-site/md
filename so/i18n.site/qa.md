# Dhibaatada Guud

## Si Lama Filaan Ah Ayaa Loo Tirtiray `.i18n/V` Taasoo Keentay in Xirmada `Npm` Fashilanto in La Daabaco

Nooca taariikhiga ah ee `npm` xirmada ayaa lagu keydiyay gudaha `.i18n/v/ol/v.hash`

Haddii aad si lama filaan ah u tirtirto `.i18n/v/ol` xirmada `npm` lama daabici doono.

Wakhtigan, marka hore hel nambarka nooca ugu dambeeya ee mashruucaaga [npmjs.com](//npmjs.com) tusaale ahaan `0.1.9`

Kadibna tixraac `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Ogsoonow in hagaajinta habkan ay lumin doonto taariikhda feylka, taasoo ka dhigaysa mid aan suurtagal ahayn in la kordhiyo siideynta soo socota, dhammaan macluumaadka ayaa dib loo baakayn doonaa oo la gelin doonaa hal mar.
