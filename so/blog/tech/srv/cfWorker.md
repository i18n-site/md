
## CNAME Waxay Ku Xidhan Tahay Magaca Domain

Magaca domainka wuxuu ku xiran yahay `SSL/TLS` → magaca martida loo yahay CNAME Cloudflare Worker domainka la xiro, gelitaanka tooska ah ayaa soo sheegi doona qaladka `error code: 522`

Waqtigaan, waxaad u baahan tahay inaad isticmaasho HTTP routing → Shaqaaluhu marinayaan si aad u xirto waddo, sida hoos ku cad
