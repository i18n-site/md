## Sözbaşy

web sahypasynyň ösüş modeline täzeden garamagym `i18n.site`

Bu resminama sahypasyny dörediji we web sahypasyny döretmek çarçuwasy.

Daş-töwereginde jemlenen we interaktiwlige sanjym etmek üçin öňdäki komponentleri ulanýan täze bir paradigma `MarkDown`

Her bir öňdäki bölek özbaşdak gurnalyp bilinýän bukjadyr.

Öň tarapy we arka tarapy bölünişiniň esasynda statiki mazmunyň we dinamiki maglumatlaryň bölünişi hem bar.

Girýän [i18n.site](/) şu çarçuwanyň esasynda gurulýar (ulanyjy ulgamy, töleg ulgamy, e-poçta abunalygy we ş.m.).
