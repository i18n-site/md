<img src="//i-01.eu.org/i18n/lang.webp" alt="" />

အချို့သောလူများက ယခုအခါ ဘရောက်ဆာများတွင် ထည့်သွင်းထားသော ဘာသာပြန်လုပ်ဆောင်ချက်များ ရှိနေပြီဖြစ်သော်လည်း၊ ဝဘ်ဆိုဒ်ကို နိုင်ငံတကာသို့ ပြောင်းလဲရန် မလိုအပ်ဘူးလားဟု မေးကောင်းမေးနိုင်ပါသည်။

**ဝဘ်ဆိုက်တစ်ခုလုံးကို နိုင်ငံတကာသို့ ပြောင်းလဲခြင်းဖြင့်သာ ကျွန်ုပ်တို့သည် ဘာသာစကားပေါင်းစုံဖြင့် ဆိုက်တွင်း စာသားအပြည့်အစုံရှာဖွေမှုနှင့် ရှာဖွေရေးအင်ဂျင်ကို ပိုမိုကောင်းမွန်အောင်ပြုလုပ်ခြင်းကို ပံ့ပိုးပေးနိုင်သည်ဟု** ကျွန်တော်ဆိုလိုပါသည်။

## နိဒါန်း

သမ္မာကျမ်းစာ · ကမ္ဘာဦးကျမ်း :

> ရှေးခေတ်က ဘာသာစကားများ တစ်သားတည်းဖြစ်သောအခါ၊ လူသားများသည် ကောင်းကင်ဘုံသို့တက်လှမ်းနိုင်သော မျှော်စင်တစ်ခု၊ လူသားတို့၏တန်ခိုးကို ကြွေးကြော်ကာ ဘုရားသခင်ပိုင်နက်ဆိုင်ရာတံခါးပေါက်ကို တည်ဆောက်ခဲ့ကြသည်။
>
> ဘုရားသခင်က "လူတို့သည် တမျိုးတည်း၊ အချင်းချင်း နှုတ်ဖြင့် ပေါင်းစည်းကြ၍၊ ဤရဲတိုက်သည် နှုတ်ကပတ်တော်သာ ဖြစ်ပေ၏။ ယခု သူတို့သည် မိမိတို့၏ ဆန္ဒကို ပြီးမြောက်ပြီး ကြောက်ရွံ့ခြင်း မရှိကြပေ။"
>
> ထို့ကြောင့် ဘုရားသခင်သည် ဆင်းသက်လာကာ လူများကို မတူညီသောထောင့်များသို့ ဖြန့်ခွဲကာ ဘာသာစကားအမျိုးမျိုးကို အသုံးပြုခဲ့သည်။
>
> ထိုအချိန်မှစ၍ လူသားတို့၏ ဆက်သွယ်မှုမှာ စိန်ခေါ်မှုများ ရှိလာခဲ့ပြီး အငြင်းအခုံများ အဆုံးမရှိ ဖြစ်လာကာ ကောင်းကင်ကိုထိသော မျှော်စင်တစ်ခု ရှိတော့မည်မဟုတ်ပေ။

သိပ္ပံစိတ်ကူးယဉ်ဝတ္ထု &quot;Three-Body&quot; (တရုတ်အသံထွက်- `3Ti` ) သည် လျှပ်စစ်သံလိုက်လှိုင်းများမှတစ်ဆင့် ဆက်သွယ်သည့် ဂြိုလ်သားယဉ်ကျေးမှုကို စိတ်ကူးယဉ်ကာ ဘာသာစကားအတားအဆီးမရှိ၊ နည်းပညာအရ ကြွယ်ဝသည်။

ကမ္ဘာပေါ်ရှိ လူများကို ကိုယ်ခန္ဓာသုံးမျိုးကဲ့သို့ဖြစ်အောင်၊ ဘာသာစကားဖြင့် ချည်နှောင်ခြင်းမပြုဘဲ ဆက်သွယ်ပြောဆိုနိုင်ပြီး လူသားအားလုံးကို ပြန်လည်စည်းလုံးစေမည့် ကိရိယာတစ်ခု ဖန်တီးရန် မျှော်လင့်ပါသည်။

## လုပ်ဆောင်ချက်မိတ်ဆက်

### Markdown ဖော်မတ်ထားရှိပါ။

### ဘာသာပြန်ခြင်းကို ပြုပြင်ပါ။

ဘာသာပြန်ခြင်းကို မွမ်းမံပြီးနောက်၊ ကက်ရှ်ကို အပ်ဒိတ်လုပ်ရန် `./i18n.sh` ပြန်လည်လုပ်ဆောင်ရန် လိုအပ်သည်။

### ဘာသာပြန်မှတ်စုများ

ဘာသာပြန်မှတ်ချက်များသည် \``` ကဲ့သို့သော ` ```rust` ပြီးနောက် ဘာသာစကားကို ညွှန်ပြရန် လိုအပ်သည်။

လောလောဆယ်တွင် rust၊ c၊ cpp၊ java၊ js၊ coffee၊ python နှင့် bash အတွက် မှတ်ချက်ဘာသာပြန်ခြင်းကို ပံ့ပိုးပေးပါသည်။

နောက်ထပ်ဘာသာစကားများဖြင့် မှတ်ချက်များအတွက် ဘာသာပြန်ခြင်းဆိုင်ရာ ပံ့ပိုးကူညီမှုပေါင်းထည့်ရန် [tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) ကို တည်းဖြတ်ပါ။

### Proxy ကို စီစဉ်သတ်မှတ်ပါ။

အောက်ပါပတ်ဝန်းကျင် ကိန်းရှင်များကို သတ်မှတ်ခြင်းသည် Google Translate API ခေါ်ဆိုမှုများကို ပရောက်စီမှတစ်ဆင့် ဖြတ်သန်းခွင့်ပြုသည်။

```bash
export https_proxy=http://127.0.0.1:7890
```
