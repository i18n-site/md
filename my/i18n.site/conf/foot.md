# အောက်ခြေမှတ်တိုင်

သရုပ်ပြပရောဂျက်ကို နမူနာအဖြစ် ယူနေသေးသည်၊ ဝဘ်ဆိုက်၏ အောက်ခြေကို လမ်းညွှန်ချက်ထဲတွင် `.i18n/htm/foot.pug` `md`

<img alt="" src="https://p.3ti.site/1721286077.avif">

[`pug`](https://pugjs.org) သည် `HTML` ထုတ်ပေးသည့် နမူနာဘာသာစကားတစ်ခုဖြစ်သည်။

[➔ သဒ္ဒါကိုလေ့လာရန် ဤနေရာကိုနှိပ်ပါ pug](https://pugjs.org)

**`css` နှင့် `js` မရေးပါ `foot.pug`** ၊ မဟုတ်ပါက အမှားများ ရှိလိမ့်မည်။

ကျေးဇူးပြု၍ အောက်ပါတို့ကို ကိုးကားပါ၊ စတိုင်ကို သက်ဆိုင်ရာ `.css` ဝဘ်အစိတ်အပိုင်းများကို ဖန်တီးခြင်းဖြင့် အပြန်အလှန် ဆက်သွယ်မှုကို ရရှိပါသည်။

## အောက်ခြေပုံစံ

သရုပ်ပြပရောဂျက် `md/.i18n/htm` အောက်တွင် ဖိုင် ၃ `css` ရှိသည်။

* `foot.css` အောက်ခြေ :
* `import.css` ဆိုက်တစ်ခုလုံးအတွက် `i18n.site` :
* `conf.css` အောက်ခြေသင်္ကေတ : နှင့် ဖောင့်များ

### အိုင်ကွန်ဖောင့်

အောက်ခြေသင်္ကေတကို iconfont.cn ( [အင်္ဂလိပ်ဗားရှင်း](https://www.iconfont.cn/?lang=en-us) /[中文版](https://www.iconfont.cn/?lang=zh)) ဟု အမည်ပေးထားသည့် ဖောင့်တစ်ခုကို ဖန်တီးထားသည် `F`

ကျေးဇူးပြု၍ သင့်ကိုယ်ပိုင်အိုင်ကွန်ဖောင့်ကို လိုအပ်သလိုဖန်တီးပြီး : ဖွဲ့စည်းမှုပုံစံကို အစားထိုးပါ `conf.css`

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
}
```

## ဝဘ်အစိတ်အပိုင်းများ

အပြန်အလှန်တုံ့ပြန်မှု လိုအပ်ပါက၊ ကျေးဇူးပြု၍ ဝဘ်စာမျက်နှာ အစိတ်အပိုင်း `js` စိတ်ကြိုက်ပြင်ဆင် `foot.pug` ။

[ဝဘ်အစိတ်အပိုင်းတစ်ခုကို](https://www.freecodecamp.org/news/build-your-first-web-component/) `md/.i18n/htm/index.js` သတ်မှတ်နိုင်ပြီး `foot.pug` တွင် အစိတ်အပိုင်းကို အသုံးပြုနိုင်သည်။

စိတ်ကြိုက်တဂ်များကဲ့သို့ ဝဘ်အစိတ်အပိုင်းများကို ဖန်တီးရန် လွယ်ကူသည် `<x-img>`

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```

## ဘာသာစုံ အောက်ခြေမှတ်စု

`.i18n/htm/foot.pug` :

```
#Ft
  b
    a.site(href="/")
    b ${I18N.C}
```

`en/i18n.yml` `${I18N.C}` :

```
C: Power By <a class="a" href="https://i18n.site">i18n.site</a>
```

`i18n.yml` နှင့် ပေါင်းစပ်ထားသော ဤစာရေးနည်းနှင့် ဆင်တူသည် `${I18N.xxx}` အသုံးပြုခြင်းဖြင့်၊ သင်သည် ဘာသာစကားပေါင်းစုံ နိုင်ငံတကာသို့ ပြောင်းလဲခြင်းကို ရရှိနိုင်သည်။
