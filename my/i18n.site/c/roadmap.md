# ဖွံ့ဖြိုးရေးစီမံကိန်း

အောက်ဖော်ပြပါ အရေးကြီးသောအင်္ဂါရပ်များသည် ဖွံ့ဖြိုးဆဲဖြစ်သည်။

- [ ] စာရွက်စာတမ်းများ၏ ဗားရှင်းများစွာကြားသို့ ပြောင်းခြင်း။
- [ ] စာသားအပြည့်အစုံ ရှာဖွေမှု
- [ ] RSS feed
- [ ] search engine optimization
- [ ] ဆောင်းပါးအပ်ဒိတ်များကို အဓိက ဆိုရှယ်မီဒီယာသို့ တပြိုင်နက် တွန်းအားပေးပါ။
- [ ] အသုံးပြုသူများထံ အီးမေးလ်ရရှိနိုင်ပါ / `web push` ထုတ်ကုန်အပ်ဒိတ်များအတွက် စာရင်းသွင်းပါ။
- [ ] ဆောင်းပါးစာပိုဒ်များအပေါ် မှတ်ချက်ပေးသည်။
- [ ] အပြောင်းအလဲများကို စောင့်ကြည့်ရန်နှင့် အစမ်းကြည့်ရှုမှုများ ပြုလုပ်ရန် ဒေသဆိုင်ရာစာရွက်စာတမ်းများကို ပေါင်းစည်းပြီး အကြိမ်တိုင်းတွင် ကိုယ်တိုင်စုစည်းမှုနှင့် NGINX ကို အသုံးပြုခြင်းတို့ကို ဖယ်ရှားပေးသည်။
