# i18n.site- သန့်ရှင်းသော အငြိမ်ဘာသာစကားပေါင်းစုံ ဝဘ်ဆိုဒ်ဘောင်

ဘာသာစကား `i18n.site` ၊ အငြိမ်စာရွက်စာတမ်းဆိုဒ်မီးစက်သက်သက်။

၎င်းကို လောလောဆယ်တွင် ကနဦးရရှိနိုင်သော်လည်း ဖွံ့ဖြိုးတိုးတက်မှုအောက်တွင် အရေးကြီးသောအင်္ဂါရပ်များရှိနေသေးကြောင်း အသေးစိတ်အချက်အလက်များအတွက် ကျေးဇူးပြု၍ [ဖွံ့ဖြိုးရေးအစီအစဉ်ကို](/i18n.site/c/roadmap) ကြည့်ပါ။

ကျေးဇူးပြု၍ <button onclick="mailsub()">ထုတ်ကုန်အပ်ဒိတ်များကို စာရင်းသွင်းရန်</button> နှင့် <button onclick="webpush()">ဘရောက်ဆာတွန်းအား ဖွင့်ရန်</button> ဤအီးမေးလ်ကို နှိပ်ပါ ထုတ်ကုန်အပ်ဒိတ်များ ပြုလုပ်သည့်အခါ သင့်ကို အကြောင်းကြားပါမည်။

ကျွန်ုပ်တို့၏လူမှုရေးအကောင့်များကို / [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) လိုက်နာရန်လည်းကြိုဆိုပါသည် [X.COM: @i18nSite](https://x.com/i18nSite)

## နိမိတ်ဖတ်

`i18n.site` သည် ကျွန်ုပ်၏ ဝဘ်ဆိုဒ် ဖွံ့ဖြိုးတိုးတက်မှု ပုံစံကို ပြန်လည်စဉ်းစားခြင်း ဖြစ်ပါသည်။

၎င်းသည် စာရွက်စာတမ်းဆိုက် ဂျင်နရေတာနှင့် ဝဘ်ဆိုဒ် ဖွံ့ဖြိုးတိုးတက်မှု မူဘောင်တစ်ခုဖြစ်သည်။

`MarkDown` ဗဟိုအဖြစ်ယူကာ အပြန်အလှန်အကျိုးသက်ရောက်မှုကိုထည့်သွင်းရန် ရှေ့ဆုံးအစိတ်အပိုင်းများကိုအသုံးပြုသည့် ဝဘ်ဆိုက်ဖွံ့ဖြိုးတိုးတက်မှုဆိုင်ရာ ပါရာဒိုင်းအသစ်။

Front-end အစိတ်အပိုင်းတစ်ခုစီသည် သီးခြားထည့်သွင်းနိုင်သည့် ပက်ကေ့ခ်ျတစ်ခုဖြစ်သည်။

front-end နှင့် back-end တို့ကို ခွဲထုတ်ခြင်း၏ အခြေခံတွင်၊ static content နှင့် dynamic data တို့ကို ခွဲခြားထားသည်။
