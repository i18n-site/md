# i18n.site- သန့်ရှင်းသော အငြိမ်ဘာသာစကားပေါင်းစုံ ဝဘ်ဆိုဒ်ဘောင်

`i18n.site` ဘာသာစကားပေါင်းစုံ၊ ငြိမ်သက်နေသော စာရွက်စာတမ်းဆိုက် မီးစက်သက်သက်။

၎င်းကို လောလောဆယ်တွင် ကနဦးရရှိနိုင်သော်လည်း ဖွံ့ဖြိုးတိုးတက်မှုအောက်တွင် အရေးကြီးသောအင်္ဂါရပ်များရှိနေသေးကြောင်း အသေးစိတ်အချက်အလက်များအတွက် ကျေးဇူးပြု၍ [ဖွံ့ဖြိုးတိုးတက်မှုအစီအစဉ်ကို](/i18n.site/c/roadmap) ကြည့်ပါ။

ကျေးဇူးပြု၍ <button onclick="mailsub()">ထုတ်ကုန်အပ်ဒိတ်များကို စာရင်းသွင်းရန်</button> နှင့် <button onclick="webpush()">ဘရောက်ဆာတွန်းအား ဖွင့်ရန်</button> ဤအီးမေးလ်ကို နှိပ်ပါ ထုတ်ကုန်အပ်ဒိတ်များ ပြုလုပ်သည့်အခါ သင့်ကို အကြောင်းကြားပါမည်။

ကျွန်ုပ်တို့၏လူမှုရေးအကောင့်များကို / [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) လိုက်နာရန်လည်းကြိုဆိုပါသည် [X.COM: @i18nSite](https://x.com/i18nSite)

## နိမိတ်ဖတ်

`i18n.site` ဝဘ်ဆိုဒ် ဖွံ့ဖြိုးတိုးတက်မှု ပုံစံကို ကျွန်ုပ် ပြန်လည် စဉ်းစားခြင်း ဖြစ်ပါသည်။

၎င်းသည် စာရွက်စာတမ်းဆိုက် ဂျင်နရေတာနှင့် ဝဘ်ဆိုဒ်ဖွံ့ဖြိုးတိုးတက်မှု မူဘောင်တစ်ခုဖြစ်သည်။

`MarkDown`

Front-end အစိတ်အပိုင်းတစ်ခုစီသည် သီးခြားထည့်သွင်းနိုင်သည့် ပက်ကေ့ခ်ျတစ်ခုဖြစ်သည်။

Front-end နှင့် Back-end တို့ကို ခွဲထုတ်ခြင်းအပေါ် အခြေခံ၍ static content နှင့် dynamic data တို့ကို ခွဲခြားထားသည်။

သင်သွားရောက်ကြည့်ရှုနေသည့်အရာသည် ဤမူဘောင် (အသုံးပြုသူစနစ်၊ ငွေပေးချေစနစ်၊ အီးမေးလ်စာရင်းသွင်းမှု စသည်ဖြင့်) ပေါ်တွင် အခြေခံတည်ဆောက်ထားသည် [i18n.site](/)
