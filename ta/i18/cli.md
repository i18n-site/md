# கட்டளை வரி அளவுருக்கள் பற்றிய விரிவான விளக்கம்

## கோப்புகளை சுத்தம் `-p`

`-p` அல்லது `--purge` ஒவ்வொரு மொழிபெயர்ப்பு கோப்பகத்திலும் இருக்கும் ஆனால் மூல மொழி கோப்பகத்தில் இல்லாத கோப்புகளை சுத்தம் செய்யும்.

ஏனெனில் ஆவணங்களை எழுதும் போது, ​​மார்க் டவுன் கோப்புப் பெயர்கள் அடிக்கடி சரிசெய்யப்படுகின்றன, இதன் விளைவாக பல பழைய மற்றும் கைவிடப்பட்ட கோப்புகள் மொழிபெயர்ப்பு கோப்பகத்தில் இருக்கும்.

பிற மொழி கோப்பகங்களில் நீக்கப்பட வேண்டிய கோப்புகளை சுத்தம் செய்ய இந்த அளவுருவைப் பயன்படுத்தவும்.

## மொழிபெயர்ப்பு கோப்பகத்தைக் `-d`

மொழிபெயர்க்கப்பட்ட கோப்பகம் தற்போதைய கோப்பு இருக்கும் கோப்பகத்திற்கு இயல்புநிலையாக இருக்கும்.

`-d` அல்லது `--workdir` போன்ற மொழிபெயர்ப்பு கோப்பகத்தைக் குறிப்பிடலாம்:

```
i18 -d ~/i18n/md
```

## உதவியைப் `-h`

`-h` அல்லது `--help` கட்டளை வரி உதவியைப் பார்க்கலாம்.
