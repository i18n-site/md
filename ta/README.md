## அறிமுகம்

பைபிள் · ஆதியாகமம் :

> பண்டைய காலங்களில், மொழிகள் ஒன்றாக இருந்தபோது, ​​​​மனிதகுலம் மனிதகுலத்தின் வல்லமையை பறைசாற்றும் ஒரு கோபுரத்தை, கடவுளின் களத்திற்கு ஒரு நுழைவாயிலை உருவாக்கியது.
>
> கடவுள் அறிவித்தார், "ஆண்கள் ஒரே கோத்திரமாக, பகிரப்பட்ட நாவுடன் ஒன்றுபடுகிறார்கள், இந்த கோபுரம் ஒரு முன்னுரை மட்டுமே. இப்போது அவர்கள் தங்கள் விருப்பத்தை அடைகிறார்கள், மேலும் பயம் இருக்காது."
>
> இவ்வாறு, கடவுள் இறங்கி, மக்களை வெவ்வேறு மூலைகளுக்கு சிதறடித்து, வெவ்வேறு மொழிகளைப் பயன்படுத்துகிறார்.
>
> அப்போதிருந்து, மனித தொடர்பு சவாலானது, தகராறுகள் முடிவில்லாதது, மேலும் வானத்தைத் தொடும் ஒரு கோபுரம் இல்லை.

அறிவியல் புனைகதை நாவலான &quot;த்ரீ-பாடி&quot; (சீன உச்சரிப்பு: `3Ti` ) ஒரு வேற்றுகிரக நாகரீகத்தை கற்பனையாக்குகிறது, அது மின்காந்த அலைகள் மூலம் தொடர்பு கொள்கிறது, மொழி தடைகள் இல்லை, மற்றும் தொழில்நுட்ப ரீதியாக செழிப்பானது.

பூமியில் வாழும் மக்கள் மூன்று உடல் மனிதர்களைப் போல இருக்கவும், மொழிக்கு கட்டுப்படாமல் தொடர்பு கொள்ளவும், மீண்டும் மனிதகுலத்தை ஒன்றிணைக்கவும் ஒரு கருவியை உருவாக்க நம்புகிறேன்.
