# செருகுநிரல்

செருகுநிரல்களை `.i18n/conf.yml` இல் கட்டமைக்க முடியும், அவை:

```yml
addon:
  - i18n.addon/toc
```

## அதிகாரப்பூர்வ செருகுநிரல்

* `i18n.addon/toc` :
  `TOC` அடிப்படையாகக் கொண்டு `json` இன் அடைவுக் குறியீட்டை உருவாக்கவும், முன்னிருப்பாக இயக்கப்பட்டது

* `i18n.addon/mouse` :

## கோப்பு பெயர் மரபு

செருகுநிரல்கள் அனைத்தும் `npm` தொகுப்புகள்.

மேலே உள்ள `i18n.addon/toc` உடன் தொடர்புடைய தொகுப்பு [https://www.npmjs.com/package/@i18n.addon/toc](https://www.npmjs.com/package/@i18n.addon/toc)

செருகுநிரல் இயல்பாகவே சமீபத்திய பதிப்பைப் பயன்படுத்துகிறது மற்றும் வாரந்தோறும் புதுப்பிப்புகளைச் சரிபார்க்கிறது.

நீங்கள் பதிப்பை சரிசெய்ய விரும்பினால், நீங்கள் `i18n.addon/toc@0.1.3` எழுதலாம்.

மொழிபெயர்ப்பு கட்டளை வரி `i18n.site` ஆனது செருகுநிரல் தொகுப்பின் மாநாட்டு கோப்பை நிறுவி பின்னர் அதை இயக்கும்.

ஒப்புக் கொள்ளப்பட்ட கோப்பு பெயர்கள் பின்வருமாறு

### htmIndex.js

`.i18n/htm/index.js` இன் இறுதி வரை `htmIndex.js` செலுத்தப்படும்.

`__CONF__` என்பது தற்போதைய கட்டமைப்பின் பெயரால் மாற்றப்படும் ( `dev` அல்லது `ol` போன்றவை).

### afterTran.js

மொழிபெயர்ப்பு முடிந்ததும் அது அழைக்கப்படும், மேலும் உள்ள அளவுருக்கள் பின்வருமாறு.

* `lang_li` : மொழி பட்டியல், முதல் மொழி மூல மொழி
* `changed` :
* `root` :

திரும்ப மதிப்பு ஒரு அகராதி, போன்ற

```json
{
  file:{
    //  path: txt, for example :
    // "_.json": "[]"
  }
}
```

`file` என்பது வெளியீட்டு கோப்பு பட்டியல், `path` கோப்பு பாதை மற்றும் `txt` கோப்பு உள்ளடக்கம்.

## உள்ளமைக்கப்பட்ட செயல்பாடுகள்

உள்ளமைக்கப்பட்ட `js` இயக்க நேரம் [boa](https://github.com/boa-dev/boa) இன் இரண்டாம் நிலை வளர்ச்சியை அடிப்படையாகக் கொண்டது :

* `wPath(path, txt)` :
* `rTxt(path)` :
* `rBin(path)` : கோப்பைப் படியுங்கள்
* `rDir(dirpath)` : கோப்பகத்தைப் படிக்கவும், திரும்பும் மதிப்பு ஒரு வரிசை : பட்டியல், கோப்பு பட்டியல்
