# பொதுவான பிரச்சனை

## தற்செயலாக நீக்கப்பட்டது `.i18n/v` `npm` தொகுப்பு வெளியிடப்படுவதில் தோல்வி

`npm` தொகுப்பு வெளியீட்டின் வரலாற்று பதிப்பு சேமிக்கப்பட்டது `.i18n/v/ol/v.hash`

நீங்கள் தற்செயலாக நீக்கினால் `.i18n/v/ol` `npm` தொகுப்பு வெளியிடப்படாது.

இந்த நேரத்தில், எடுத்துக்காட்டாக `0.1.9` உங்கள் திட்டத்தின் கடைசி வெளியீட்டின் பதிப்பு எண்ணைக் கண்டறியவும் [npmjs.com](//npmjs.com)

பின்னர் கீழே உள்ள `bash` கோப்பை பார்க்கவும்.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

இந்த வழியில் பழுதுபார்ப்பது கோப்பு வரலாற்றை இழக்கும், மேலும் அடுத்த வெளியீட்டை மேம்படுத்துவது சாத்தியமற்றது, மேலும் அனைத்து உள்ளடக்கங்களும் மீண்டும் தொகுக்கப்பட்டு ஒரு முறை பதிவேற்றப்படும்.
