
## CNAME பிணைக்கப்பட்ட டொமைன் பெயர்

Cloudflare Worker இன் டொமைன் பெயர் `SSL/TLS` → தனிப்பயன் ஹோஸ்ட் பெயர் மூலம் பிணைக்கப்பட்டுள்ளது CNAME டொமைன் பெயர் பிணைக்கப்பட்ட பிறகு, நேரடி அணுகல் ஒரு பிழையைப் புகாரளிக்கும் `error code: 522`

இந்த நேரத்தில், கீழே காட்டப்பட்டுள்ளபடி, ஒரு வழியை இணைக்க HTTP ரூட்டிங் → பணியாளர்கள் ரூட்டிங் பயன்படுத்த வேண்டும்
