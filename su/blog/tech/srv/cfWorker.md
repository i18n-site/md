
## CNAME Ngaran Domain Kabeungkeut

Cloudflare Worker 's ngaran domain kabeungkeut ngaliwatan `SSL/TLS` → custom ngaran host CNAME ngaran domain kabeungkeut, aksés langsung bakal ngalaporkeun kasalahan `error code: 522`

Dina waktos ayeuna, anjeun kedah nganggo HTTP routing → Workers routing pikeun ngabeungkeut rute, sapertos anu dipidangkeun di handap.
