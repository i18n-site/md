# Penjelasan Rinci Ngeunaan Parameter Garis Paréntah

## `-P` Beresih

`-p` Atawa `--purge` bakal ngabersihan file anu aya dina unggal diréktori tarjamahan tapi henteu aya dina diréktori basa sumber.

Kusabab nalika nyerat dokumén, nami file Markdown sering disaluyukeun, nyababkeun seueur file anu lami sareng ditinggalkeun dina diréktori tarjamahan.

Anggo parameter ieu pikeun ngabersihan file anu kedah dipupus dina diréktori basa sanés.

## `-D` Sebutkeun Diréktori Tarjamahan

Diréktori anu ditarjamahkeun sacara standar kana diréktori dimana file ayeuna aya.

`-d` Atawa `--workdir` bisa nangtukeun diréktori tarjamah, kayaning:

```
i18 -d ~/i18n/md
```

## `-H`

`-h` Atawa `--help` tiasa ningali pitulung baris paréntah.
