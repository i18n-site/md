# Vérsi Proyék

Candak proyék demo salaku conto:

`en/demo2/v` nyaéta nomer vérsi proyék ayeuna, anu bakal dipintonkeun di sisi katuhu nami proyék dina garis sisi sidebar.

<img src="https://p.3ti.site/1721290486.avif" width="320px">

`en/` mangrupikeun kode basa anu cocog sareng `.i18n/conf.yml` basa sumber tarjamahan anu dikonpigurasikeun.

Upami basa sumber anjeun sanés Inggris, file `v` kedah disimpen dina diréktori proyék basa sumber anjeun. 

Kamampuhan pikeun ngotéktak vérsi sajarah dokumén nuju dikembangkeun.

Disarankeun pikeun ngan ukur ngarobih nomer vérsi dokumén nalika ngaluarkeun apdet utama (sapertos `v1` , `v2` ) pikeun ngahindarkeun kaca-kaca anu diindeks ku mesin pencari kusabab seueur teuing nomer versi.

## Anggo File `v` Pikeun Ngabagi Indéks File Pikeun Proyék-Proyék Anu Béda

Dina proyék demo, salian ti `en/demo2/v` , anjeun ogé tiasa ningali yén eusi diréktori `en/blog` sareng `en/demo1` kosong `v` file.

Kosong `v` moal dipintonkeun dina outline sidebar, tapi salami aya `v` file, hiji indéks bebas bakal dihasilkeun pikeun file dina diréktori jeung subdirectories.

Ku ngabagi indéks tina proyék anu béda, anjeun tiasa ngahindarkeun aksés anu laun disababkeun ku ngamuat indéks sadaya file dina sadaya situs sakaligus.
