# 常见问题

## 不小心删除`.i18n/v`，导致`npm`包无法发布

`.i18n/v/ol/v.hash`中保存了`npm`包发布的历史版本。

如果你不小心删除了`.i18n/v/ol`会导致`npm`包无法发布。

这时候，先在[npmjs.com](//npmjs.com)找到你项目最后一次发布版的版本号，比如是`0.1.9`。

然后参考下面的`bash`创建文件。

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

注意，采用此方式修复会丢失文件历史，导致下一次发布时无法增量更新，所有内容会全部重新打包上传一次。
