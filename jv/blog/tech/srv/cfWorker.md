
## Jeneng Domain Kaiket CNAME

Cloudflare Worker 's domain name is bound through `SSL/TLS` → custom host name CNAME jeneng domain diikat, akses langsung bakal nglaporake kesalahan `error code: 522`

Ing wektu iki, sampeyan kudu nggunakake routing HTTP → Routing buruh kanggo ngiket rute, kaya sing kapacak ing ngisor iki
