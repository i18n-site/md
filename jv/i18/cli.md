# Panjelasan Rinci Paramèter Baris Printah

## `-P` Resik

`-p` Utawa `--purge` bakal ngresiki file sing ana ing saben direktori terjemahan nanging ora ana ing direktori basa sumber.

Amarga nalika nulis dokumen, jeneng file Markdown asring diatur, nyebabake akeh file lawas lan ditinggalake ing direktori terjemahan.

Gunakake parameter iki kanggo ngresiki file sing kudu dibusak ing direktori basa liyane.

## `-D` Nemtokake Direktori Terjemahan

Direktori sing diterjemahake dadi standar menyang direktori ing ngendi file saiki ana.

`-d` Utawa `--workdir` bisa nemtokake direktori terjemahan, kayata:

```
i18 -d ~/i18n/md
```

## `-H` Ndeleng Bantuan

`-h` Utawa `--help` bisa ndeleng bantuan baris perintah.
