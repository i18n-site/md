
## Ainm Fearainn Faoi Cheangal CNAME

Tá ainm fearainn faoi cheangal trí `SSL/TLS` → ainm óstaigh saincheaptha CNAME Cloudflare Worker t-ainm fearainn a cheangal, tuairisceoidh rochtain dhíreach earráid `error code: 522`

Ag an am seo, ní mór duit ródú HTTP → Ródú oibrithe a úsáid chun bealach a cheangal, mar a thaispeántar thíos
