# 명령줄 매개변수의 자세한 설명

## `-p` 파일 정리

`-p` 또는 `--purge`는 각 번역 디렉토리에 존재하지만 소스 언어 디렉토리에는 존재하지 않는 파일들을 정리합니다.

문서 작성 시 Markdown 파일 이름이 자주 변경되어 번역 디렉토리에 많은 폐기된旄文件이 쌓이게 됩니다.

이 매개변수를 사용하여 다른 언어 디렉토리에서 제거해야 할 파일들을 정리할 수 있습니다.

## `-d` 번역 디렉토리 지정

번역 디렉토리의 기본값은 현재 파일이 위치한 디렉토리입니다.

`-d` 또는 `--workdir`를 사용하여 다음과 같이 번역 디렉토리를 지정할 수 있습니다:

```
i18 -d ~/i18n/md
```

## `-h` 도움말 보기

명령줄 도움말을 보려면 `-h` 또는 `--help`를 사용하세요.
