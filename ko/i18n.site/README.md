# i18n.site: 순수 정적 다국어 웹사이트 프레임워크

`i18n.site` 다중 언어, 순수 정적 문서 사이트 생성기.

## 서론

`i18n.site`는 문서 사이트 생성기이자, 웹 사이트 개발을 위한 프레임워크입니다.

`MarkDown`을 중심으로, 프론트엔드 컴포넌트를 통해 상호작용을 주입하는 새로운 웹 개발 패러다임입니다.

각 프론트엔드 컴포넌트는 독립적으로 설치할 수 있는 패키지입니다.

프론트엔드와 백엔드 분리의 기반 위에, 정적 콘텐츠와 동적 데이터의 분리를 더합니다.

당신이 방문하고 있는 [i18n.site](/)는 이 프레임워크를 기반으로 구축되었으며 (사용자 시스템, 결제 시스템, 이메일 구독 등 포함).
