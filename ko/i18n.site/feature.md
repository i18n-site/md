# 제품 특성

## `i18` 번역 통합

프로그램 내에 `i18` 번역 기능이 내장되어 있습니다. 구체적인 사용법은 [➔ `i18` 문서](/i18) 참조 바랍니다.

## 브라우저 언어 자동 매칭

웹사이트의 기본 언어는 브라우저의 언어와 자동으로 매칭됩니다.

사용자가 수동으로 언어를 전환한 후, 그 선택이 기억됩니다.

관련 코드: [github.com/i18n-site/18x/src/lang.coffee](https://github.com/i18n-site/18x/blob/main/src/lang.coffee).

## <a rel=id href="#ha" id="ha"></a> 프론트엔드 고가용성

`i18n.site` 기본적으로 사이트 콘텐츠를 `npmjs.com`에 배포하며, [jsdelivr.com](//jsdelivr.com) 및 [unpkg.com](//unpkg.com)와 같은 여러 `CDN`을 통해 `npm`에 로드된 콘텐츠를 활용합니다.

이를 바탕으로 중국 사용자도 안정적으로 접근할 수 있도록 중국 본토의 미러 소스를 추가하여 **프론트엔드 고가용성**을 실현했습니다.

원리는: [`service worker`](https://developer.mozilla.org/docs/Web/API/Service_Worker_API)를 통해 요청을 가로채고, 요청이 다른 `CDN`에서 실패하면 다시 시도하며, 가장 빠르게 응답하는 원본 사이트를 기본 로딩 소스로 자동으로 설정합니다.

관련 코드: [github.com/18x/serviceWorker](https://github.com/i18n-site/18x/tree/main/serviceWorker).

### 온라인 글꼴 로드, 중국어 지원

웹페이지는 기본적으로 [Alimama 이중축 가변 직사각형 글꼴](https://www.iconfont.cn/fonts/detail?cnid=pOvFIr086ADR), [MiSans](https://hyperos.mi.com/font/zh/download/) 등의 온라인 글꼴을 활성화하여 다양한 플랫폼에서 일관된 독서 경험을 제공합니다.

또한 로딩 속도를 높이기 위해 단어 빈도 통계에 따라 글꼴을 분할합니다.

관련 코드: [github.com/i18n-site/font](https://github.com/i18n-site/font).

### 상단 导航 自动隐藏

아래로 스크롤하면 상단 导航이 자동으로 숨겨집니다.

위로 스크롤하면 숨겨진 导航이 다시 나타납니다.

마우스가 움직이지 않으면 점차 사라집니다.

右上角的导航栏中还有全屏按钮，提供沉浸式的文档阅读体验。

### 현재 장을 동기화하여 개요 강조 표시

오른쪽의 콘텐츠를 스크롤할 때 왼쪽의 개요가 현재 읽고 있는 장을 강조 표시합니다.
