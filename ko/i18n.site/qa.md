# 자주 묻는 질문

## 실수로 `.i18n/v`를 삭제하여 `npm` 패키지를 发布할 수 없게 되었습니다

`.i18n/v/ol/v.hash`에 `npm` 패키지 发布의 역사 버전이 저장되어 있습니다.

실수로 `.i18n/v/ol`을 삭제하면 `npm` 패키지를 发布할 수 없습니다.

이 경우, 먼저 [npmjs.com](//npmjs.com)에서 프로젝트의 마지막 发布된 버전 번호를 찾아보세요. 예를 들어 `0.1.9`입니다.

그런 다음 아래의 `bash` 명령을 참조하여 파일을 생성하십시오.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

이 방법을 사용하여 복구하면 파일의 기록이 손실되어 다음 发布 시 점진적인 업데이트가 불가능해지며 모든 콘텐츠가 다시 패키징되어 업로드됩니다.
