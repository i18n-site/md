# 개발 계획

다음과 같은 중요한 기능이 개발 중입니다:

- [ ] 문서의 다양한 버전 간 전환이 가능합니다
- [ ] 순수 프론트엔드 기반의 전체 텍스트 검색 기능입니다
- [ ] RSS 구독 피드입니다
- [ ] 검색 엔진 최적화
- [ ] 대형 소셜 미디어 플랫폼에 동시에 글 업데이트를 푸시합니다
- [ ] 사용자에게 이메일 제공 `web push` 제품 업데이트를 구독하세요 /
- [ ] 기사의 각段落에 댓글을 달 수 있습니다
- [ ] 로컬 문서를 통합하여 변경 사항을 모니터링하고 미리 보기를 생성할 수 있어, 매번 수동으로 컴파일하거나 NGINX를 사용할 필요가 없습니다
