# Ibsa Bal'aa Qajoojiiwwan Sarara Ajajaa

## `-P` Faayilota Qulqulleessi

`-p` Ykn `--purge` faayilii galmee hiikkaa tokkoon tokkoo keessa jiran garuu galmee afaan madda keessa hin jirre ni qulqulleessa.

Sababni isaas, yeroo galmeewwan barreessinu, maqaan faayilii Markdown yeroo baay'ee sirreeffama, kunis faayilii durii fi gatame hedduu galmee hiikkaa keessatti argamsiisa.

Faayilota galmee afaanota biroo keessatti haqamuu qaban qulqulleessuuf qaree kana fayyadami.

## `-D` Galmee Hiikkaa Ifteessi

Galmeen hiikame durtii galmee bakka faayilii ammee jirutti ta'a.

`-d` Yookiin `--workdir` galmee hiikkaa ifteessuu danda'a, kan akka:

```
i18 -d ~/i18n/md
```

## `-H` Ilaali

`-h` Yookiin `--help` sarara ajajaa ilaaluu dandeessu.
