# Gorsa Gulaalaa

Namni kamiyyuu gulaalaa mataa isaa kan jaallatu qaba Muuxannoo keenya isa gaarii asitti haa qoodnu `Markdown`

`MarkDown` [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) [vscode](https://code.visualstudio.com/)

Akkuma armaan gaditti mul'atu, foddaa durtii banuuf gulaalaa keessatti mirga cuqaasi.

Faayilii `vscode` banaa `Markdown` waamuuf sarara ajaja irratti `code xxx.md` galchi.

Suuraalee olkaa'uuf [PicList](https://github.com/Kuingsmile/PicList) fayyadami, furtuu karaa gabaabaa olkaa'uu saaguu yaadadhu, fi suuraa iskiriinii cuqaasuu tokkoon olkaa'i.

Erga olkaa'amee booda bifa maqaa `Markdown` ofumaan waraabuu ni deeggara, kunis gahumsa baay'ee fooyyessa.

Kanuma waliin, qindeessaa armaan gadii ilaaluu, faayilii fooyyessuu, fi faayilii olkaa'ame ofumaan gara `avif`
