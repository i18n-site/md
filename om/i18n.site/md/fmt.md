# Konveenshinii Bifa MarkDown

## Qabxii Anchor

Aadaa `MarkDown` Anchors kan uumaman qabiyyee barruu irratti hundaa'uun yoo ta'e furmaanni kun hin danda'amu.

`i18n.site` `MarkDown` `<a rel=id href="#xxx" id="xxx"></a>`

`<a rel=id href="#xxx" id="xxx"></a>` asitti `rel=id` akkaataa fuula qabxii anchor ibsa, maaloo `xxx` gabaabduu Ingiliffaa anchor kee isa dhugaatiin bakka buusi.

Anchors yeroo baayyee mata duree irratti dabalamu, kanneen akka:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## `HTML` Keessa Barreessi `MarkDown`

`pug` `HTML` koodii keessatti ukkaamfamuu danda'a.

Qabiyyeen elementii `<pre>`

Qabxiilee lamaan kana walitti makuun, bu'aa agarsiisaa adda addaa galmaan ga'uuf salphaatti `HTML` `MarkDown` barreessuun ni danda'ama.

Hojiirra oolmaa [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) koodii akka armaan gadiitti :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Hubadhaa `<style>` armaan olitti `<pre>`
