# Vershinii Pirojektii

Pirojektii demoo akka fakkeenyaatti fudhadhaa:

`en/demo2/v`

<img src="https://p.3ti.site/1721290486.avif" width="320px">

`en/` koodii afaanii afaan madda hiikkaa `.i18n/conf.yml` qindaa'ee wajjin walsimu.

Yoo afaan madda kee Ingiliffa hin taane, kana booda faayilii `v` 

Dandeettiin sanadoota seenaa qaban daawwachuu hojjetamaa jira.

Fuulota motora barbaachaan indeksii ta'an sababa lakkoofsa gosa baay'ee irraa kan ka'e akka hin jeeqneef yeroo fooyya'iinsa gurguddoo (kan akka `v1` , `v2` ) gadhiifamu lakkoofsa gosa galmee qofa fooyyessuutu gorfama.

## Indeeksii Faayilii Pirojektoota Adda Addaatiif Qooduuf Faayilii Duwwaa `V`

Pirojektii agarsiisaa keessatti, `en/demo2/v` dabalatee, qabiyyeen galmeewwan `en/blog` fi `en/demo1` duwwaa ta'uu `v` arguu dandeessa.

`v` `v`

Indeeksii pirojektoota adda addaa qoqqooduudhaan, indeeksii faayilii hunda guutummaa saayitii keessa jiran yeroo tokkotti fe'uudhaan qaqqabummaa suuta jedhu irraa fagaachuu dandeessa.

Fakkeenyaaf, pirojektii demo `blog` faayilii indeksii walgitu [https://unpkg.com/i18n.site/en/blog.json](https://unpkg.com/i18n.site/en/blog.json) dha :
