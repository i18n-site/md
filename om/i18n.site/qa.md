# Rakkoo Waliigalaa

## Akka Tasaa Haqame `.i18n/V` `Npm` Paakeejiin Akka Hin Maxxanfamne Taasiseera

Version seena qabeessa `npm` package release keessatti save ta'eera `.i18n/v/ol/v.hash`

Yoo akka tasaa haqxe `.i18n/v/ol` `npm` paakeejii hin maxxanfamu.

Yeroo kanatti `0.1.9` jalqaba lakkoofsa gosa gosa gadhiifame isa dhumaa pirojektii keetii [npmjs.com](//npmjs.com)

Sana booda faayilii `bash` Uumi armaan gadii ilaali.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Hubadhu, haala kanaan suphuun seenaa faayilii ni dhaba, kunis gadhiifama itti aanu dabalataan haaromsuun akka hin danda'amne taasisa, qabiyyeen hundis al tokko irra deebi'amee kan kuufamee fi olkaa'ama.
