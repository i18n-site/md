## Seensa

`i18n.site`

Kun jenereetarii saayitii sanadaa fi bu’uura misooma weebsaayitii ti.

Paradigm haaraa guddina marsariitii kan naannoo `MarkDown`

Tokkoon tokkoon qaama fuulduraa paakeejii of danda'ee fe'amuu danda'uudha.

Bu’uura addaan ba’iinsa fuulduraa fi duubaa irratti qabiyyee istaatiksii fi deetaa daayinamikii addaan bahuunis ni jira.
