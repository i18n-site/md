
## Maqaa Domeenii CNAME Hidhamee

Cloudflare Worker ’s domain name is bound through `SSL/TLS` → custom host name CNAME domeenii erga hidhamee booda, kallattiin qaqqabummaan dogongora `error code: 522` ni gabaasa

Yeroo kanatti, daandii hidhuudhaaf HTTP routing → Workers routing fayyadamuu qabda, akkuma armaan gadiitti mul'atu
