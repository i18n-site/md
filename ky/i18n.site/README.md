# i18n.site: Таза Статикалык Көп Тилдүү Веб-Сайт Алкагы

`i18n.site` тилдүү, таза статикалык документ сайтынын генератору.

Ал учурда жеткиликтүү, бирок дагы эле иштеп [жаткан](/i18n.site/c/roadmap) маанилүү функциялар бар.

<button onclick="mailsub()">Өнүмдүн жаңыртууларына жазылуу</button> жана <button onclick="webpush()">браузерди иштетүү үчүн</button> бул электрондук почтаны басыңыз.

Ошондой [X.COM: @i18nSite](https://x.com/i18nSite) биздин коомдук эсептерге кош келиңиз / [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) .

## Кириш Сөз

`i18n.site` бул менин веб-сайтты өнүктүрүү моделин кайра карап чыгуу.

Бул документ сайтынын генератору жана веб-сайтты иштеп чыгуу негизи.

Вебсайтты өнүктүрүүнүн жаңы парадигмасы, анын айланасында жайгашкан жана интерактивдүүлүктү киргизүү үчүн алдыңкы компоненттерди колдонот `MarkDown`

Ар бир алдыңкы компонент өз алдынча орнотула турган пакет.

Front-end жана back-end бөлүүнүн негизинде, ошондой эле статикалык мазмунду жана динамикалык маалыматтарды бөлүү бар.

Сиз барган [i18n.site](/) ушул алкактын негизинде курулган (анын ичинде колдонуучу системасы, эсеп-кысап системасы, электрондук почтага жазылуу ж.б.).
