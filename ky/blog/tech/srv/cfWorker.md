
## CNAME Менен Байланышкан Домен Аты

Домен аталышы `SSL/TLS` → Ыңгайлаштырылган хост атын байланыштыруу CNAME Домендик аталыштан кийин түз жетүү ката жөнүндө `error code: 522` кабарлайт Cloudflare Worker

Бул учурда, төмөндө көрсөтүлгөндөй, маршрутту туташтыруу үчүн HTTP багыттоо → Жумушчулар багыттоосун колдонушуңуз керек.
