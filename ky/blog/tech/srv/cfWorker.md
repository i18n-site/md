
## CNAME Менен Байланышкан Домен Аты

Домендин аталышы `SSL/TLS` аркылуу байланган → ыңгайлаштырылган хост аты CNAME Домендин аталышы байлангандан кийин, түз жетүү `error code: 522` ката Cloudflare Worker кабарлайт!

Бул учурда, төмөндө көрсөтүлгөндөй, маршрутту туташтыруу үчүн HTTP багыттоо → Жумушчулар багыттоосун колдонушуңуз керек.
