## Introduction

Библия · Башталыш :

> Байыркы убакта, тилдер бир болгондо, адамзат асманга чейин жеткен мунара курган, адамзаттын кудуретин жар салган Кудайдын бийлигинин дарбазасы болгон.
>
> Кудай: "Адамдар бир уруу болуп биригет, тилдери орток, бул мунара жөн гана пролог. Эми алар каалоолоруна жетишип, мындан ары коркуу болбойт" деп жар салган.
>
> Ошентип, Кудай түшүп, адамдарды ар кайсы бурчтарга чачыратып, ар кандай тилдерди колдонушат.
>
> Ошондон тартып адамдардын баарлашуусу кыйын болуп, талаш-тартыштар бүтпөй, асманга тийген мунара жок болуп калды.

&quot;Үч дене&quot; (кытайча айтылышы: `3Ti` ) илимий фантастикалык романы электромагниттик толкундар аркылуу байланышып, тилдик тоскоолдуктары жок жана технологиялык жактан гүлдөп-өнүккөн бөтөн цивилизацияны ойдон чыгарат.

Жер жүзүндөгү адамдарды үч денелүү адамдардай кылып, тилге байланбай баарлашууга жана бүткүл адамзатты кайрадан бириктирүүгө мүмкүндүк бере турган куралды түзүүгө үмүттөнөм.
