# Буйрук Сабынын Параметрлеринин Деталдуу Түшүндүрмөсү

## Файлдарды `-P`

`-p` `--purge` ар бир котормо каталогунда бар, бирок баштапкы тил каталогунда жок файлдарды тазалайт.

Анткени документтерди жазууда Markdown файл аттары көбүнчө туураланып, котормо каталогунда көптөгөн эски жана ташталган файлдар пайда болот.

Башка тил каталогдорунан жок кылынышы керек болгон файлдарды тазалоо үчүн бул параметрди колдонуңуз.

## Котормо Каталогун `-D`

Которулган каталог демейки боюнча учурдагы файл жайгашкан каталогго келет.

`-d` `--workdir` котормо каталогун көрсөтө алат, мисалы:

```
i18 -d ~/i18n/md
```

## Жардамды `-H`

`-h` `--help` буйрук сабынын жардамын көрө аласыз.
