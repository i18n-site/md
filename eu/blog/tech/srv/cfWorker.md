
## CNAME Loturiko Domeinu-Izena

Cloudflare Worker Domeinuaren izena gainditu `SSL/TLS` → Ostalari-izena pertsonalizatua CNAME Domeinuaren izenaren ondoren, zuzeneko sarbideak errore bat emango du `error code: 522`

Une honetan, HTTP bideratzea → Langileen bideratzea erabili behar duzu ibilbide bat lotzeko, behean erakusten den moduan
