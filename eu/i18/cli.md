# Komando-Lerroko Parametroen Azalpen Zehatza

## `-P` Garbitu Fitxategiak

`-p` `--purge` itzulpen-direktorio bakoitzean dauden baina iturburu-hizkuntzaren direktorioan ez dauden fitxategiak garbituko ditu.

Dokumentuak idaztean, Markdown fitxategi-izenak sarritan moldatzen direlako, itzulpen-direktorioan fitxategi zahar eta abandonatu asko sortzen direlako.

Erabili parametro hau beste hizkuntza bateko direktorioetan ezabatu behar diren fitxategiak garbitzeko.

## `-D` Zehaztu Itzulpen Direktorioa

Itzulitako direktorioa lehenetsita dago uneko fitxategia dagoen direktorioan.

`-d` `--workdir` itzulpen-direktorioa zehaztu dezake, adibidez:

```
i18 -d ~/i18n/md
```

## `-H` Ikusi Laguntza

`-h` `--help` lerroko laguntza ikus dezakezu.
