# Udviklingsplanlægning

Følgende vigtige funktioner er under udvikling:

- [ ] Skift mellem flere versioner af dokumenter
- [ ] Ren frontend fuldtekstsøgning
- [ ] RSS feed
- [ ] Søgemaskine optimering
- [ ] Skub samtidig artikelopdateringer til store sociale medier
- [ ] Brugere kan abonnere på produktopdateringer `web push` e-mail /
- [ ] Kommentar til artikelafsnit
- [ ] Integrer lokale dokumenter for at overvåge ændringer og generere forhåndsvisninger, hvilket eliminerer behovet for manuel kompilering hver gang og brug af NGINX
