* Hovedside [github](https://github.com/i18n-site/site) / [atomgit](https://atomgit.com/i18n/proto)
* Dokumentation [github](https://github.com/i18n-site/md) / [atomgit](https://atomgit.com/i18n/md)
* Indépendante komponenter [github](https://github.com/i18n-site/18x) / [atomgit](https://atomgit.com/i18n/18x)
* Svelte-komponenter [github](https://github.com/i18n-site/plugin) / [atomgit](https://atomgit.com/i18n/plugin)
* Grænseflader [github](https://github.com/i18n-site/proto) / [atomgit](https://atomgit.com/i18n/proto)
* Almindelig grundbibliotek [github](https://github.com/i18n-site/lib) / [atomgit](https://atomgit.com/i18n/lib)
* Browser-bibliotek [github](https://github.com/i18n-site/ie) / [atomgit](https://atomgit.com/i18n/ie)
* Frontend-bibliotek, der kan refereres direkte uden pakning [github](https://github.com/i18n-site/x) / [atomgit](https://atomgit.com/i18n/x)
  Nemt at referere direkte i markdown-initialiseringscriptet og samtidig bruge det på den pakrede hjemmeside, hvilket undgår gentagen kode.

## Backend

Dokumentationsstedet kræver kun frontend, ingen backend.

Denne backend er i18n.site-webstedets egen backend, som inkluderer bruger-, betalings-, beskedpush- og andre systemer.

* Service [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugins [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Grundbibliotek [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
