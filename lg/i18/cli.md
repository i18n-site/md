# Ennyinyonnyola Enzijuvu Ku Parameters Za Layini Y'ekiragiro

## `-P` Yoza Fayiro

`-p` Oba `--purge` ejja kuyonja fayiro eziriwo mu buli dayirekita y'okuvvuunula naye nga teziri mu dayirekita y'olulimi ensibuko.

Kubanga bwe bawandiika ebiwandiiko, amannya ga fayiro za Markdown gatera okutereezebwa, ekivaamu fayiro nnyingi enkadde n’ezisuuliddwa mu dayirekita y’okuvvuunula.

Kozesa parameter eno okuyonja fayiro ezirina okusangulwa mu dayirekita z'ennimi endala.

## `-D` Laga Ekitabo Ky'okuvvuunula

Dayirekita evvuunuddwa esookera ku dayirekita fayiro eriwo kati gy'eri.

`-d` Oba `--workdir` asobola okulaga dayirekita y'okuvvuunula, gamba nga:

```
i18 -d ~/i18n/md
```

## `-H` Laba Obuyambi

`-h` Oba `--help` okulaba obuyambi bwa layini y'ekiragiro.
