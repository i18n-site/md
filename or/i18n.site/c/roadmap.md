# ବିକାଶ ଯୋଜନା

ନିମ୍ନଲିଖିତ ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ବ features ଶିଷ୍ଟ୍ୟଗୁଡିକ ବିକାଶ ଅଧୀନରେ ଅଛି:

- [ ] ଡକ୍ୟୁମେଣ୍ଟ୍ ର ଏକାଧିକ ସଂସ୍କରଣ ମଧ୍ୟରେ ସୁଇଚ୍ କରିବା |
- [ ] ଶୁଦ୍ଧ ଫ୍ରଣ୍ଟ-ଏଣ୍ଡ ପୂର୍ଣ୍ଣ-ପାଠ୍ୟ ସନ୍ଧାନ |
- [ ] RSS ଫିଡ୍ |
- [ ] ସର୍ଚ୍ଚ ଇଞ୍ଜିନ୍ ଅପ୍ଟିମାଇଜେସନ୍ |
- [ ] ଏକାସାଙ୍ଗରେ ପ୍ରମୁଖ ସୋସିଆଲ୍ ମିଡିଆରେ ଆର୍ଟିକିଲ୍ ଅପଡେଟ୍ କୁ ଠେଲିଦିଅ |
- [ ] ଉପଭୋକ୍ତାମାନଙ୍କ ପାଇଁ ଇମେଲ୍ ଉପଲବ୍ଧ `web push` ଉତ୍ପାଦ ଅଦ୍ୟତନକୁ ସବସ୍କ୍ରାଇବ କରନ୍ତୁ /
- [ ] ପ୍ରବନ୍ଧ ଅନୁଚ୍ଛେଦ ଉପରେ ମନ୍ତବ୍ୟ ଦିଅନ୍ତୁ |
- [ ] ପରିବର୍ତ୍ତନଗୁଡିକ ଉପରେ ନଜର ରଖିବା ଏବଂ ପୂର୍ବାବଲୋକନ ସୃଷ୍ଟି କରିବା, ପ୍ରତ୍ୟେକ ଥର ମାନୁଆଲ ସଂକଳନର ଆବଶ୍ୟକତାକୁ ଦୂର କରିବା ଏବଂ NGINX ବ୍ୟବହାର କରିବା ପାଇଁ ସ୍ଥାନୀୟ ଡକ୍ୟୁମେଣ୍ଟଗୁଡ଼ିକୁ ଏକତ୍ର କରନ୍ତୁ |
