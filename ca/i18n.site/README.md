# i18n.site: Marc De Lloc Web Multi-Idioma Estàtic Pur

`i18n.site` generador de llocs de documents multiidioma purament estàtic.

## Pròleg

`i18n.site` és un generador de llocs de documents i un marc de desenvolupament de llocs web.

Un nou paradigma de desenvolupament de llocs web que pren `MarkDown` com a centre i utilitza components front-end per injectar interactivitat.

Cada component frontal és un paquet que es pot instal·lar de manera independent.

Sobre la base de la separació de front-end i back-end, també hi ha la separació de contingut estàtic i dades dinàmiques.

El que esteu [i18n.site](/) es basa en aquest marc (incloent el sistema d'usuari, el sistema de facturació, la subscripció per correu electrònic, etc.).
