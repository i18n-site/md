
## Nom De Domini Vinculat a CNAME

El nom Cloudflare Worker domini s'ha `SSL/TLS` → Vinculació de nom d'amfitrió personalitzada CNAME Després del nom de domini, l'accés directe informarà d'un error `error code: 522`

En aquest moment, heu d'utilitzar l'encaminament HTTP → Enrutament de treballadors per vincular una ruta, tal com es mostra a continuació
