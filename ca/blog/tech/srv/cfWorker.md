
## Nom De Domini Vinculat a CNAME

El nom de domini s'enllaça mitjançant `SSL/TLS` → nom d'amfitrió personalitzat CNAME Cloudflare Worker el nom de domini estigui vinculat, l'accés directe informarà d'un error `error code: 522`

En aquest moment, heu d'utilitzar l'encaminament HTTP → Enrutament de treballadors per vincular una ruta, tal com es mostra a continuació
