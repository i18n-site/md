
## Ime Domene, Vezano Na CNAME

Cloudflare Worker Ime domene je vezano prek `SSL/TLS` → ime gostitelja po meri CNAME je ime domene vezano, bo neposredni dostop sporočil napako `error code: 522`

Trenutno morate za povezovanje poti uporabiti usmerjanje HTTP → usmerjanje delavcev, kot je prikazano spodaj
