
## CNAME Байланысты Домен Атауы

Cloudflare Worker ның домен атауы `SSL/TLS` арқылы байланыстырылған → пайдаланушы хост атауы CNAME Домен атауы байланыстырылғаннан кейін, тікелей қатынас `error code: 522` қате туралы хабарлайды.

Қазіргі уақытта төменде көрсетілгендей маршрутты байланыстыру үшін HTTP маршруттау → Жұмысшылар маршруттауын пайдалану қажет.
