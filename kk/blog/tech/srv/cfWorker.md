
## CNAME Байланысты Домен Атауы

Cloudflare Worker атауы `SSL/TLS` → Реттелетін хост атауын байланыстыру CNAME Домен атауынан кейін тікелей қатынас қате туралы хабарлайды `error code: 522`

Бұл уақытта төменде көрсетілгендей маршрутты байланыстыру үшін HTTP маршруттау → Жұмысшылар маршрутын пайдалану қажет.
