# Ортақ Мәселе

## Кездейсоқ Жойылды `.i18n/V` Бұл `Npm` Буманың Жарияланбауына Себеп Болды

`npm` пакет шығарылымының тарихи нұсқасы сақталады `.i18n/v/ol/v.hash`

Байқамай жойсаңыз `.i18n/v/ol` `npm` бума жарияланбайды.

Бұл кезде алдымен жобаңыздың соңғы шығарылым нұсқасының нұсқа нөмірін табыңыз [npmjs.com](//npmjs.com) мысалы `0.1.9`

Содан кейін төмендегі файлды `bash` .

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Осы жолмен жөндеу файл тарихын жоғалтып, келесі шығарылымды біртіндеп жаңарту мүмкін болмайтынын және барлық мазмұн бір рет қайта оралып, жүктеп салынатынын ескеріңіз.
