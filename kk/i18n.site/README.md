# i18n.site: Таза Статикалық Көп Тілді Веб-Сайт Құрылымы

`i18n.site` Көптілді, таза статикалық құжат сайтының генераторы.

Ол бастапқыда қол жетімді, бірақ әлі де маңызды мүмкіндіктер әзірленуде. Толық ақпарат алу үшін [әзірлеу жоспарын](/i18n.site/c/roadmap) қараңыз.

<button onclick="mailsub()">Өнім жаңартуларына жазылу</button> және <button onclick="webpush()">шолғышты қосу үшін</button> осы электрондық поштаны басыңыз. Өнім жаңартулары жасалғанда сізге хабарлаймыз.

Сондай [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) ақ біздің әлеуметтік аккаунттарымызға жазылуға қош / [X.COM: @i18nSite](https://x.com/i18nSite)

## Алғы Сөз

`i18n.site` - бұл менің веб-сайтты әзірлеу моделін қайта қарау.

Бұл құжат сайтының генераторы және веб-сайтты әзірлеу негізі.

Интерактивтілікті енгізу үшін алдыңғы қатарлы компоненттерді пайдаланатын веб-сайтты дамытудың жаңа парадигмасы `MarkDown`

Әрбір алдыңғы бөлік компоненті дербес орнатылатын пакет болып табылады.

Фронт пен бэк-соңды бөлу негізінде статикалық мазмұн мен динамикалық деректерді бөлу де бар.

Сіз баратын [i18n.site](/) осы негізге негізделген (оның ішінде пайдаланушы жүйесі, есепшот жүйесі, электрондық пошта жазылымы және т.б.).
