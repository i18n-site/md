
## ชื่อโดเมนที่ผูกกับ CNAME

Cloudflare Worker ชื่อโดเมนผ่าน `SSL/TLS` → การเชื่อมโยงชื่อโฮสต์ที่กำหนดเอง CNAME หลังจากชื่อโดเมน การเข้าถึงโดยตรงจะรายงานข้อผิดพลาด `error code: 522`

ในขณะนี้ คุณต้องใช้การกำหนดเส้นทาง HTTP → การกำหนดเส้นทางผู้ปฏิบัติงานเพื่อผูกเส้นทาง ดังที่แสดงด้านล่าง
