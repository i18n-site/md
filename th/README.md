## การแนะนำ

นิยายวิทยาศาสตร์เรื่อง &quot;Three Body&quot; (การออกเสียงภาษาจีน: `3Tǐ` ) สวมบทบาทเป็นอารยธรรมของมนุษย์ต่างดาวที่ใช้คลื่นแม่เหล็กไฟฟ้าในการสื่อสาร ด้วยความคิดที่โปร่งใสและเทคโนโลยีที่เจริญรุ่งเรือง

พระคัมภีร์ · ปฐมกาล :

> ในยุคนั้นเสียงและคำพูดของทุกคนใต้ฟ้าเป็นหนึ่งเดียวกัน
>
> พวกเขาสร้างหอคอยสูงเสียดฟ้า สัมผัสประตูของพระเจ้า จิตใจที่เป็นน้ำหนึ่งใจเดียวกัน และอวดพระสิริของพวกเขา
>
> พระเจ้าทรงประกาศว่า: เมื่อผู้คนแบ่งปันภาษาและเชื้อสาย พวกเขาจะกลายเป็นชนเผ่าเดียว หอคอยนี้เป็นเพียงจุดเริ่มต้นเท่านั้น ในอีกไม่กี่วันข้างหน้า ไม่มีอะไรที่พวกเขาจะทำไม่ได้
>
> ดังนั้นพระองค์จึงเสด็จลงมา ทำให้ผู้คนกระจัดกระจายไปไกล ภาษาของพวกเขาจึงไม่สามารถเข้าใจซึ่งกันและกันได้
>
> ตั้งแต่นั้นเป็นต้นมา ความขัดแย้งก็ไม่มีที่สิ้นสุด และโลกก็ไม่เห็นหอคอยที่แตะท้องฟ้าอีกต่อไป

ฉันหวังว่าจะสร้างเครื่องมือเพื่อให้ผู้คนในโลกสามารถเป็นเหมือนคนสามร่างสื่อสารได้โดยไม่ถูกผูกมัดด้วยภาษาและมนุษยชาติทั้งหมดสามารถกลับมารวมกันได้อีกครั้ง

ดังนั้นจึงมี [`3Ti.Site`](//3Ti.Site)

### แก้ไขการแปล

หลังจากแก้ไขการแปล คุณจะต้องรัน `./i18n.sh` อีกครั้งเพื่ออัปเดตแคช

### หมายเหตุการแปล

ความคิดเห็นเกี่ยวกับคำแปลต้องระบุภาษาที่ตามหลัง \``` เช่น ` ```rust`

ปัจจุบันรองรับการแปลความคิดเห็นสำหรับrust, c, cpp, java, js, coffee, python และ bash

แก้ไข [tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) เพื่อเพิ่มการสนับสนุนการแปลความคิดเห็นในภาษาต่างๆ มากขึ้น

### กำหนดค่าพร็อกซี

การตั้งค่าตัวแปรสภาพแวดล้อมต่อไปนี้ทำให้การเรียก Google Translate API ผ่านพร็อกซีได้

```bash
export https_proxy=http://127.0.0.1:7890
```
