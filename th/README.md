3Ti.Site ซึ่งเป็นเครื่องมือสร้างเว็บไซต์คงที่หลายภาษา สามารถแปล Markdown เป็น [ภาษาต่างๆ มากกว่าร้อยภาษา](https://github.com/i18n-site/node/blob/main/lang/src/index.js) ได้โดยอัตโนมัติ

<img src="//i-01.eu.org/i18n/lang.webp" alt="" />

บางคนอาจอยากถามว่าตอนนี้เบราว์เซอร์มีฟังก์ชั่นการแปลในตัวแล้ว ไม่จำเป็นต้องทำให้เว็บไซต์เป็นสากลหรือไม่?

ฉันอยากจะบอกว่า **เฉพาะการทำให้ไซต์ทั้งหมดเป็นสากลเท่านั้นที่เราสามารถรองรับการค้นหาข้อความแบบเต็มในไซต์หลายภาษาและการเพิ่มประสิทธิภาพกลไกค้นหา**

## ลำดับ

"พระคัมภีร์·ปฐมกาล":

> ในสมัยโบราณ เมื่อภาษาเป็นหนึ่ง มนุษยชาติเกิดมาอย่างภาคภูมิใจ พวกเขาพยายามสร้างอาคารสูงตระหง่าน ซึ่งเป็นหอคอยที่แตะสวรรค์
>
> อย่างไรก็ตาม พระเจ้าไม่ทรงพอพระทัยกับความเย่อหยิ่งของพวกเขา ทรงทำให้ผู้คนกระจัดกระจายไปทั่วโลก ส่งผลให้ลิ้นของพวกเขาไม่สามารถเข้าใจซึ่งกันและกันได้
>
> ตั้งแต่นั้นเป็นต้นมา มนุษยชาติพยายามดิ้นรนในการสื่อสาร ความขัดแย้งเกิดขึ้นอย่างไม่มีที่สิ้นสุด และโลกก็ไม่เห็นหอคอยที่ขึ้นไปบนท้องฟ้าอีกต่อไป

นวนิยายวิทยาศาสตร์เรื่อง &quot;Three-Body&quot; (การออกเสียงภาษาจีน: `3Ti` ) สวมบทบาทเป็นอารยธรรมของมนุษย์ต่างดาวที่สื่อสารผ่านคลื่นแม่เหล็กไฟฟ้า ไม่มีอุปสรรคด้านภาษา และมีความเจริญรุ่งเรืองทางเทคโนโลยี

ฉันหวังว่าด้วยความช่วยเหลือของเครื่องมือนี้ ผู้คนในโลกจะเป็นเหมือนคนสามคน การสื่อสารจะไม่ถูกจำกัดด้วยภาษา และมนุษยชาติทั้งหมดจะกลับมารวมกันอีกครั้ง

### แก้ไขการแปล

หลังจากแก้ไขการแปล คุณจะต้องรัน `./i18n.sh` อีกครั้งเพื่ออัปเดตแคช

### หมายเหตุการแปล

ความคิดเห็นเกี่ยวกับคำแปลต้องระบุภาษาที่ตามหลัง \``` เช่น ` ```rust`

ปัจจุบันรองรับการแปลความคิดเห็นสำหรับrust, c, cpp, java, js, coffee, python และ bash

แก้ไข [tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) เพื่อเพิ่มการสนับสนุนการแปลความคิดเห็นในภาษาต่างๆ มากขึ้น

### กำหนดค่าพร็อกซี

การตั้งค่าตัวแปรสภาพแวดล้อมต่อไปนี้ทำให้การเรียก Google Translate API ผ่านพร็อกซีได้

```bash
export https_proxy=http://127.0.0.1:7890
```
