# คำอธิบายโดยละเอียดของพารามิเตอร์บรรทัดคำสั่ง

## `-p` ทำความสะอาดไฟล์

`-p` หรือ `--purge` จะล้างไฟล์ที่มีอยู่ในไดเร็กทอรีการแปลแต่ละรายการ แต่ไม่มีอยู่ในไดเร็กทอรีภาษาต้นทาง

เพราะเมื่อเขียนเอกสาร ชื่อไฟล์ Markdown มักจะถูกปรับเปลี่ยน ส่งผลให้มีไฟล์เก่าและไฟล์ที่ถูกละทิ้งจำนวนมากในไดเร็กทอรีการแปล

ใช้พารามิเตอร์นี้เพื่อล้างไฟล์ที่ควรลบในไดเร็กทอรีภาษาอื่น

## `-d` ระบุไดเรกทอรีการแปล

ไดเร็กทอรีที่แปลแล้วจะมีค่าเริ่มต้นเป็นไดเร็กทอรีที่มีไฟล์ปัจจุบันอยู่

`-d` หรือ `--workdir` สามารถระบุไดเร็กทอรีการแปล เช่น:

```
i18 -d ~/i18n/md
```

## `-h` ดูความช่วยเหลือ

`-h` หรือ `--help` สามารถดูวิธีใช้บรรทัดคำสั่งได้
