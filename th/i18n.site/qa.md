# ปัญหาทั่วไป

## ลบโดยไม่ได้ตั้งใจ `.i18n/v` ทำให้ไม่สามารถเผยแพร่แพ็คเกจ `npm` ได้

`.i18n/v/ol/v.hash` เวอร์ชันประวัติของ `npm` การเปิดตัวแพ็คเกจถูกบันทึกไว้ใน !

หากคุณลบ `.i18n/v/ol` โดยไม่ได้ตั้งใจ แพ็คเกจ `npm` จะไม่ถูกเผยแพร่

ในตอนนี้ ให้ค้นหาหมายเลขเวอร์ชันของเวอร์ชันล่าสุดของโปรเจ็กต์ของคุณ `0.1.9` [npmjs.com](//npmjs.com)

จากนั้นอ้างอิงถึงไฟล์ `bash` Create ด้านล่าง

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

โปรดทราบว่าการซ่อมแซมด้วยวิธีนี้จะสูญเสียประวัติไฟล์ ทำให้ไม่สามารถอัปเดตรุ่นถัดไปแบบค่อยเป็นค่อยไป และเนื้อหาทั้งหมดจะถูกแพ็กเกจใหม่และอัปโหลดเพียงครั้งเดียว
