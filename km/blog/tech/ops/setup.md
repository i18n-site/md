contabo бһӮбһәбһҮбһ¶бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶ VPS бһҗбҹ„бһҖбһ“бҹ…бһ”бҹ’бһҡбһ‘бҹҒбһҹбһўбһ¶бһЎбһәбһҳбҹүбһ„бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҷбһјбһҡбһўбһ„бҹ’бһңбҹӮбһ„ (бһҮбһ¶бһ„ 20 бһҶбҹ’бһ“бһ¶бҹҶ) бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ 200,000) бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ–бҹ’бһҡбһҪбһҷбһ”бһ¶бһҡбһҳбҹ’бһ—бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһҸбҹӢбһ…бҹҒбһүбһ“бҹ„бҹҮбһ‘бҹҒбҹ”

бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҡбһ”бһҹбҹӢбһңбһ¶бһ–бҹ’бһҷбһҪбһҡбһҳбҹ’бһҸбһ„бһҳбҹ’бһҖбһ¶бһӣ бһҠбҹ„бһҷбһ’бһ¶бһ“бһ¶бһ“бһјбһңбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ“бҹғбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһ—бһ¶бһ–бһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһҒбҹ’бһ–бһҹбҹӢбҹ”

бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбҹ’бһҳбһёбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһҮбһјбһ“бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһ”бҹҗбһҺбҹ’бһҺбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ–бһ·бһҸбһ“бҹ…бһ–бҹҒбһӣбһ‘бһ·бһүбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ

бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ“бһ¶бһ‘бһёбһҖбҹ’бһҡбҹ„бһҷбһҳбһҖ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҹбһҖбһҳбҹ’бһҳбҹ”

бһҡбһҝбһ„бһҠбҹҶбһ”бһјбһ„бһӮбһәбһҸбҹ’бһҡбһјбһңбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ `~/.ssh/config` бһҖбҹҶбһҺбһҸбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһҖбҹ’бһӣбҹӮбһ„бһҖбҹ’бһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ IP бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бҹ’бһҡбһҫ `ssh-copy-id root@c2` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбҹ”

<img alt="" src="https://i-01.eu.org/1718287198.avif">

бһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһҖбҹӮбһ”бҹ’бһҡбҹӮ [ssh-copy-id.sh](https://github.com/i18n-ops/ubuntu/blob/main/ssh-copy-id.sh) вҖӢбһ“бҹҒбҹҮвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бһүбҹ’бһ…бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’вҖӢбһҖбһ¶бһҡвҖӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳвҖӢбһ”бһ¶бһ“вҖӢбһӣбһҝбһ“вҖӢбһҮбһ¶бһ„бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёвҖӢбһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’вҖӢбһҳбһ·бһ“вҖӢбһ”бҹ’бһҡбһҫвҖӢбһӣбҹҒбһҒвҖӢбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһ”бһүбҹ’бһ…бһ”бҹӢ бһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһ”бҹ’бһҡбһҫвҖӢбһңбһ¶вҖӢбһ”бһ¶бһ“ [pdsh](https://github.com/chaos/pdsh)

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҸбһ¶бһҳбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҸбһ¶бһҳбһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶ бһҠбһјбһ…бһҮбһ¶ [mise](https://github.com/jdx/mise) , [direnv](https://direnv.net/) бһҮбһ¶бһҠбһҫбһҳбҹ” бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһҳбҹ’бһӣбһ„бһғбҹ’бһӣбһ¶бҹҶбһ„ [i18n-ops/ubuntu](https://github.com/i18n-ops/ubuntu) бһ бһҫбһҷбһҖбҹӮбһ”бҹ’бһҡбҹӮбһңбһ¶бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”

бһҮбҹҶбһ бһ¶бһ“вҖӢбһ“бҹҒбҹҮвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ…бҹҶбһҺбһ¶бһҷвҖӢбһ–бҹҒбһӣвҖӢбһҷбһјбһҡ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёвҖӢбһҡбһ„бҹӢбһ…бһ¶бҹҶвҖӢбһҖбһ“бҹ’бһӣбҹҮвҖӢбһҳбҹүбҹ„бһ„ бһҖбһ¶бһҡвҖӢбһҠбҹҶбһЎбһҫбһ„вҖӢбһҖбҹҶбһҺбҹӮ бһ“бһ·бһ„вҖӢбһҠбҹҶбһЎбһҫбһ„вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҠбҹӮбһӣвҖӢбһ”бҹ’бһҡбһҫвҖӢбһҮбһ¶вҖӢбһ‘бһјбһ‘бҹ…вҖӢбһ“бһ№бһ„вҖӢбһ‘бһ‘бһҪбһӣвҖӢбһ”бһ¶бһ“вҖӢбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһңбһ¶бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

## mosh бһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“ ssh

бһ”бһҺбҹ’бһҸбһ¶бһүвҖӢбһ–бһёвҖӢбһўбһәбһҡбҹүбһ»бһ”вҖӢбһ‘бҹ…вҖӢбһ”бҹ’бһҡбһ‘бҹҒбһҹвҖӢбһ…бһ·бһ“вҖӢбһҳбһ·бһ“вҖӢбһҳбһ¶бһ“вҖӢбһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–вҖӢбһӮбһҪбһҡ `ssh` [mosh](//mosh.org)

`PATH` `c0` script бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ local machine's `~/.bin`

```bash
#!/usr/bin/env bash

if [ -z "$1" ]; then
arg=/root/.tmux_default
else
arg=${@:1:$#}
fi

vps=$(basename $0)
set -ex
mosh $vps -p 60000:60009 $arg
```

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ”бһ¶бһ“ бһ“бһ·бһ„ Soft- `c2` `c1`

```bash
chmod +x c0
ln -s c0 c1
ln -s c0 c2
```

бһ“бҹ…бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ `c0` , `c1` бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ `mosh` бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“ бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣ [tmux](https://www.ruanyifeng.com/blog/2019/10/tmux.html)

## бһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ—бһ¶бһӮбһҗбһ¶бһҹбһ–бһё ext4 бһ‘бҹ… btrfs

[btrfs](https://wikipedia.org/wiki/Btrfs) бһўбһ¶бһ…бһҗбһҸбһҡбһјбһ”бһҗбһ¶бһҹ бһҠбһҫбһҳбҹ’бһ”бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбҹ”

бһҠбҹҶбһ”бһјбһ„бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ

<img alt="" src="https://i-01.eu.org/1718300448.avif">

бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Debian Rescue

<img alt="" src="https://i-01.eu.org/1718300753.avif">

бһҡбһ„бҹӢбһ…бһ¶бҹҶбһұбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһү

бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `ssh-keygen -R ip` бһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһ“бһ№бһ„бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣ ssh

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ—бһ¶бһӮбһҗбһ¶бһҹ

```bash
nohup bash -c "bash <(curl -s https://raw.githubusercontent.com/i18n-ops/ops/main/contabo/ext4_btrfs.sh) >/tmp/nohup.out 2>&1 && reboot" &
sleep 1 && tail -f /tmp/nohup.out
```

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбҹ”

### бһҳбһҫбһӣбһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢ btrfs

бһңбһ¶вҖӢбһ“бһ№бһ„вҖӢбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢвҖӢбһҠбҹ„бһҷвҖӢбһҹбҹ’бһңбҹҗбһҷвҖӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·вҖӢбһҸбһ¶бһҳвҖӢбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ бһ–бһ¶бһҖбҹ’бһҷвҖӢбһ”бһүбҹ’бһҮбһ¶вҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷвҖӢбһҳбһҫбһӣвҖӢбһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢвҖӢбһҳбһ¶бһ“вҖӢбһҠбһјбһ…вҖӢбһҒбһ¶бһ„вҖӢбһҖбҹ’бһҡбҹ„бһҳбҹ–

```bash
if ! command -v compsize &>/dev/null; then
apt install -y  btrfs-compsize
fi
compsize -x /
```

### бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢбһҠбҹ„бһҷбһҠбҹғ

бһ”бҹ’бһҡбһҹбһ·бһ“вҖӢбһ”бһҫвҖӢбһўбҹ’бһ“бһҖвҖӢбһҮбһ¶вҖӢбһўбҹ’бһ“бһҖвҖӢбһ’бҹ’бһңбһҫвҖӢбһҖбһ¶бһҡвҖӢбһ–бҹҒбһүвҖӢбһӣбҹҒбһү бһўбҹ’бһ“бһҖвҖӢбһҖбҹҸвҖӢбһўбһ¶бһ…вҖӢбһ’бҹ’бһңбһҫвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢвҖӢбһҗбһ¶бһҹвҖӢбһ–бҹҒбһүвҖӢбһӣбҹҒбһүвҖӢбһҠбҹ„бһҷвҖӢбһҠбҹғвҖӢбһ•бһ„вҖӢбһҠбҹӮбһҡ бһ–бһ¶бһҖбҹ’бһҷвҖӢбһ”бһүбҹ’бһҮбһ¶вҖӢбһҳбһ¶бһ“вҖӢбһҠбһјбһ…вҖӢбһҒбһ¶бһ„вҖӢбһҖбҹ’бһҡбҹ„бһҳ

```bash
btrfs filesystem defragment -r -czstd /
```

бһҠбһјбһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҗбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢбһҠбҹ„бһҷбһҠбҹғбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҗбҹ’бһҳбһё бһҮбһ·бһҸ 100MB бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһ”бҹӢбҹ”

<img alt="" src="https://i-01.eu.org/2023/12/8NWlr4c.webp">

### бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҡбһјбһ”бһҗбһҸ Btrfs бһҮбһ¶бһҳбһҪбһҷ snapper

[snapper](https://manual.siduction.org/sys-admin-btrfs-snapper_en.html) бһӮбһәбһҮбһ¶бһҠбҹғбһӮбһјбһҠбҹҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖ Btrfs бҹ”

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸ `~/ubuntu` бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```
~/ubuntu/snapper/init.sh
```

бһҮбһҪбһ”бһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҹбҹ’бһӮбҹ’бһҡбһёбһ” [github.com/i18n-ops/ubuntu/blob/main/snapper/init.sh](https://github.com/i18n-ops/ubuntu/blob/main/snapper/init.sh)

бһ”бҹ’бһҡбһҫ `snapper list-configs` бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһ—бһ¶бһӮбһҡбһ„бһҠбҹӮбһӣбһҡбһјбһ”бһҗбһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ”

`/etc/snapper/configs/mnt` бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹғбһ—бһ¶бһӮбһҡбһ„бһ“бҹҒбҹҮбҹ”

бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ–

## бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ

бһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ 22 бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бҹ’бһҡбһҖбһӮбһә [tcping](https://github.com/paradiseduo/tcping) бһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһңбһ¶бһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ 2-3 бһ“бһ¶бһ‘бһёбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҡбһ„бҹӢбһ…бһ¶бҹҶбһ”бһ“бҹ’бһҸбһ·бһ…бҹ”

```
tcping ip 22 -c 9999
```

бһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҖбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”

### бһҡбһ”бҹҖбһ”бһ…бһјбһӣбһ”бҹ’бһҡбһҫ hard drive бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бһјбһӣбһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ &

```
mkdir -p /mnt/os
mount  /dev/sda3 /mnt/os -t btrfs -o "defaults,ssd,discard,noatime,compress=zstd:3,space_cache=v2"
mount --bind /dev /mnt/os/dev
mount --bind /proc /mnt/os/proc
mount --bind /sys /mnt/os/sys
mount --bind /run /mnt/os/run
chroot /mnt/os zsh -c "mount -a && dbus-daemon --system 2>/dev/null; exec zsh"
```

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһҡбһјбһ”бһҗбһҸ

```
snapper -c etc list
```

бһҳбһҫбһӣбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҜбһҖбһҹбһ¶бһҡбһҡбһјбһ”бһҗбһҸ

```
snapper -c etc status 5..6
```

бһҳбһҫбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ

```
snapper -c etc diff 3..5
```

бһҹбҹ’бһҸбһ¶бһҡбһҡбһјбһ”бһҗбһҸбһЎбһҫбһ„бһңбһ·бһүбҹ– бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҳбһ·бһ“бһ’бҹ’бһңбһҫбһңбһ·бһүбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҡбһјбһ”бһҗбһҸ 5

```
snapper -c etc undochange 1..0
snapper -c disk undochange 1..0
```

### бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»

бһҖбһ¶бһҡбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһҠбҹғ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“бҹ–

chroot бһҠбҹҶбһ”бһјбһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ

```
mkdir -p /var/log/journal
chown root:systemd-journal /var/log/journal
chmod 2755 /var/log/journal
sed -i 's/^#\?\s*Storage=.*/Storage=persistent/' /etc/systemd/journald.conf
```

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҠбһјбһ”бҹҠбһ»бһ“бһ‘бһјбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһЎбһҫбһ„бһңбһ·бһүбҹ– бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҒбһ¶бһ„бһӣбһҫ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһЎбһҫбһ„бһңбһ·бһүбһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶ бһ бһҫбһҷбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҠбһјбһ”бҹҠбһ»бһ“бһ‘бһјбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбҹ”

бһҸбһ¶бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮ systemd-journald бһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҗбҹ’бһҳбһё бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ“бҹ…бһӣбһҫбһҗбһ¶бһҹбһҡбһ№бһ„бҹ”

бһҹбһјбһҳбһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҒбһ¶бһ„бһӣбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бһ№бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһӣбҹӮбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бҹҖбһ”бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбҹ”

бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ·бһ“бһғбһҫбһүбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҺбһ¶бһҳбһҪбһҷбһ…бҹҶбһ–бҹ„бҹҮбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бһ“бҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһңбһ¶бһ“бһ№бһ„бһ”бһ“бҹ’бһҸбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбһўбҹҶбһ–бһёбһҮбҹҶбһ„бһәбһ–бһ·бһ”бһ¶бһҖ бһ“бһ·бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

### бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ…бҹҶбһӣбҹӮбһҖбҹ– бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһ·бһ“бһўбһ¶бһ… ssh бһ”бһ¶бһ“бһ‘бҹҒбҹ”

бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– github бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһёбһҳбһ»бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһҗбһ¶бһҹбҹҒбһңбһ¶бһҠбһ¶бһҖбҹӢбһ–бһ„бҹ’бһҡбһ¶бһҷбһҳбһ·бһ“бһўбһ¶бһ…бһҮбһ¶ ssh бһ”бһ¶бһ“бһ‘бҹҒбҹ”

бһҖбһ¶бһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮ бһ“бһ·бһ„бһҳбһҫбһӣбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһӮбһәбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ–

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё tar бһҳбҹ’бһҸбһ„бһ бһҫбһҷбһҳбҹ’бһҸбһ„ `uid` бһӮбһә `1001` github action

tar бһ–бһ“бҹ’бһӣбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ’бҹ’бһңбһҫбһЎбһҫбһ„бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `--no-same-owner` бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһҳбһ‘бҹ…бһҮбһ¶ `uid` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `1001` user

бһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣ rsync бһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбҹ’бһҡбһ»бһҳбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹғбһҗбһҸ root бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ ssh бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһүбҹ”

бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҹбҹ’бһӮбҹ’бһҡбһёбһ” decompression бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```bash
tar --no-same-owner -xvf $TZT
rm $TZT

if [ -f "init.sh" ]; then
  ./init.sh
  rm init.sh
fi

rsync --remove-source-files -av . /
```
