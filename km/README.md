## សេចក្តីផ្តើម

ប្រលោមលោកបែបវិទ្យាសាស្ត្រ &quot;Three-Body&quot; (ការបញ្ចេញសំឡេងចិន៖ `3Ti` ) ប្រឌិតអំពីអារ្យធម៌របស់ជនបរទេសដែលទំនាក់ទំនងតាមរយៈរលកអេឡិចត្រូម៉ាញេទិក មិនមានឧបសគ្គភាសា និងមានភាពរីកចម្រើនខាងបច្ចេកវិទ្យា។

> បុរាណ​បាន​សង់​ប៉ម​ឡើង​ដល់​មេឃ ដល់​ទ្វារ​ព្រះ និង​លើក​តម្កើង​កិត្យានុភាព​នៃ​ពូជ​មនុស្ស។
>
> ព្រះអម្ចាស់​មាន​ព្រះបន្ទូល​ថា មនុស្ស​បង្កើត​ត្រកូល​របស់​ខ្លួន ហើយ​និយាយ​ភាសា​ដូច​គ្នា។ ការ​សង់​ប៉ម​គឺ​គ្រាន់​តែ​ជា​ការ​លើក​ដំបូង​ប៉ុណ្ណោះ។ ពេល​នេះ​វា​ត្រូវ​បាន​គេ​ធ្វើ​ហើយ ពួក​គេ​នឹង​ធ្វើ​គ្រប់​យ៉ាង​នៅ​ពេល​អនាគត។
>
> បន្ទាប់​មក ព្រះ​អម្ចាស់​បាន​យាង​មក ហើយ​ធ្វើ​ឲ្យ​ប្រជាជន​នៅ​កន្លែង​ផ្សេង​ៗ​គ្នា​មិន​អាច​យល់​ចិត្ត​គ្នា​បាន។
>
> តាំង​ពី​ពេល​នោះ​មក វិវាទ​របស់​មនុស្ស​បាន​បន្ត​រហូត​ដល់​លែង​មាន Tower of Babel ក្នុង​លោក​ទៀត​ហើយ។
