## бһ”бһ»бһ–бҹ’бһңбһ”бһ‘

бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ”бһ¶бһ“бһӣбһ»бһ”бһ”бҹҶбһ”бһ¶бһҸбҹӢбһ…бһҳбҹ’бһ„бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ  бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹғбһ—бһ¶бһҹбһ¶бһ“бҹ…бһҸбҹӮбһҡбһ¶бһҡбһ¶бҹҶбһ„бһҠбһӣбҹӢбһҖбһ¶бһҡбһҡбһҪбһ”бһҡбһҪбһҳбһҡбһ”бһҹбҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбҹ”

бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһҖбҹҸбһҠбҹ„бһҷ бһҖбҹҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹ…бһҸбҹӮбһҳбһ·бһ“бһўбһ¶бһ…бһҹбһҪбһҡбһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһ¶бһ“бһЎбһҫбһҷбҹ”

бһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһ„бҹ’бһӮбһҳ бһ“бһ·бһ„бһўбҹҠбһёбһҳбҹӮбһӣ бһҳбһ“бһ»бһҹбҹ’бһҹбһҳбһ¶бһ“бһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ”бҹ’бһҡбһ—бһ–бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбҹҶбһҺбһҫбһҸбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ”

бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бҹҮбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бһ·бһ„бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһҖбҹ’бһ“бһ»бһ„бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ”бҹ’бһҡбһҖбһҪбһҸбһ”бҹ’бһҡбһҮбҹӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ **бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** бҹ”

бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһңбһ¶бһҮбһ¶бһӮбһҳбҹ’бһҡбҹ„бһ„бһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖбһ…бҹҶбһ бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹӮбһӣбһ…бһ„бҹӢбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһ‘бһҹбҹ’бһҹбһ“бһ·бһҖбһҮбһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹҸбһҠбҹ„бһҷ бһңбһ¶бһӮбһҪбһҡбһҸбҹӮбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһўбһ“бҹ’бһҸбһҡбһҮбһ¶бһҸбһ·бһҸбһ¶бҹҶбһ„бһ–бһёбһҠбҹҶбһ”бһјбһ„бҹ”

### <a rel=id href="#i18" id="i18"></a> i18бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһҖбһ”бҹ’бһҡбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ MarkDown

бһ”бһҖбһ”бҹ’бһҡбҹӮ `Markdown` бһ“бһ·бһ„ `YAML` бһ‘бҹ…бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ ( [бһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ–](https://github.com/i18n-site/rust/tree/main/i18) ) бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ—бһ¶бһҹбһ¶бҹ”

бһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһ“бһјбһңбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹғ `Markdown` бһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһ”бһҖбһ”бҹ’бһҡбҹӮбһҸбҹӮбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

## бһўбҹҶбһ–бһёбһ–бһҪбһҖбһҷбһҫбһ„

> бһ–бһҪбһҖбһӮбҹҒбһ“бһ·бһҷбһ¶бһҷбһҗбһ¶бҹ– бһ…бһјбһҡбһҳбһҖбһҹбһ¶бһ„бһҹбһ„бҹӢбһ”бҹүбһҳбһҠбҹӮбһӣбһЎбһҫбһ„бһҠбһӣбҹӢбһҳбҹҒбһғ бһ бһҫбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ–бһјбһҮбһҳбһ“бһ»бһҹбҹ’бһҹбһӣбҹ’бһ”бһёбһӣбҹ’бһ”бһ¶бһүбҹ”
>
> бһ–бҹ’бһҡбҹҮбһўбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ‘бһҸбһғбһҫбһүбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһ бһҫбһҷбһҳбһ¶бһ“бһ–бҹ’бһҡбҹҮбһ”бһ“бҹ’бһ‘бһјбһӣбһҗбһ¶бҹ– В«бһҳбһ“бһ»бһҹбҹ’бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһҮбһ¶бһҸбһ·бһҹбһ¶бһҹбһ“бҹҚбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бһ бһҫбһҷ бһўбҹ’бһңбһёбҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бһ№бһ„бһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…В»бҹ”
>
> бһ–бҹҒбһӣбһ“бҹ„бҹҮбһңбһ¶бһ”бһ¶бһ“бһҳбһҖбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ“бһ»бһҹбҹ’бһҹбһҳбһ·бһ“бһўбһ¶бһ…бһҷбһӣбҹӢбһ—бһ¶бһҹбһ¶бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖ бһ бһҫбһҷбһ”бҹӮбһҖбһҒбҹ’бһүбҹӮбһҖбһӮбҹ’бһ“бһ¶бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

<p style="text-align:right">в”Җв”Җбһ–бҹ’бһҡбҹҮбһӮбһҳбҹ’бһ–бһёбһҡ В· бһӣбҹ„бһҖбһ»бһ”бҹ’бһ”бһҸбҹ’бһҸбһ·</p>

бһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҠбҹ„бһҷбһҳбһ·бһ“бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһ—бһ¶бһҹбһ¶бҹ”
бһҷбһҫбһ„бһҹбһ„бҹ’бһғбһ№бһҳбһҗбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбһҮбһ¶бһҸбһ·бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бһ№бһ„бһҳбһҖбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҹбһ»бһ”бһ·бһ“бһҡбһҪбһҳбҹ”

бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ Markdown бһӮбһәбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһ¶бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
бһ’бҹ’бһңбһҫбһҹбһҳбһҖбһ¶бһӣбһҖбһҳбҹ’бһҳбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһ„бҹ’бһӮбһҳ;

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ…бҹ’бһҡбһҫбһ“бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹӢбһҡбһ„бҹ’бһңбһ¶бһ“бҹӢ;
бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһӣбһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһҒбһ¶бһ„бһҳбһ»бһҒбһўбһ“бҹ’бһҸбһҡбһҮбһ¶бһҸбһ·;
