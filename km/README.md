## សេចក្តីផ្តើម

>
> ដូច្នេះ ទ្រង់​បាន​យាង​ចុះ​មក ធ្វើ​ឲ្យ​ប្រជាជន​ខ្ចាត់ខ្ចាយ​ទៅ​កាន់​ទី​ឆ្ងាយ ភាសា​របស់​ពួក​គេ​ឥឡូវ​មិន​អាច​យល់​គ្នា​ទៅ​វិញ​ទៅ​មក។
>
> ចាប់​ពី​ពេល​នោះ​មក ជម្លោះ​ក៏​មិន​ចេះ​ចប់ ហើយ​ពិភពលោក​ក៏​លែង​ឃើញ​ប៉ម​ដែល​ប៉ះ​លើ​មេឃ​ទៀត​ផង។
