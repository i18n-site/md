# ការណែនាំអំពីកម្មវិធីនិពន្ធ

អ្នកគ្រប់គ្នាមានកម្មវិធីនិពន្ធដែលខ្លួនចូលចិត្ត សូមចែករំលែកការអនុវត្តន៍ដ៏ល្អបំផុតរបស់យើងនៅទីនេះ `Markdown`

យើងប្រើ [vscode](https://code.visualstudio.com/) Write `MarkDown` និងដំឡើងកម្មវិធីជំនួយ [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) ដើម្បីមើលជាមុនក្នុងពេលកំពុងសរសេរ។

ដូចដែលបានបង្ហាញខាងក្រោម ចុចកណ្ដុរស្ដាំលើកម្មវិធីនិពន្ធ ដើម្បីបើកបង្អួចមើលជាមុន។

បញ្ចូល `code xxx.md` នៅលើបន្ទាត់ពាក្យបញ្ជាដើម្បីហៅ `vscode` បើកឯកសារ `Markdown` ។

ប្រើ [PicList](https://github.com/Kuingsmile/PicList) ដើម្បីអាប់ឡូតរូបភាព កុំភ្លេចកំណត់គ្រាប់ចុចផ្លូវកាត់បង្ហោះ ហើយបង្ហោះរូបថតអេក្រង់ដោយចុចតែម្តង។

វាគាំទ្រការចម្លងឈ្មោះ `Markdown` ដោយស្វ័យប្រវត្តិបន្ទាប់ពីផ្ទុកឡើង ដែលបង្កើនប្រសិទ្ធភាពយ៉ាងខ្លាំង

ក្នុងពេលជាមួយគ្នានេះ អ្នកអាចយោងទៅលើការកំណត់រចនាសម្ព័ន្ធខាងក្រោម កែប្រែឯកសារ និងកំណត់រចនាសម្ព័ន្ធឯកសារដែលបានផ្ទុកឡើងដែលត្រូវបង្ហាប់ដោយស្វ័យប្រវត្តិដើម្បីកាត់បន្ថយទំហំរូបភាព `avif`
