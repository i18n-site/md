# бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…

## бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһӣбһ»бһ”вҖӢбһҠбҹ„бһҷвҖӢбһ…бҹғбһҠбһ“бҹ’бһҷ `.i18n/v` бһҠбҹӮбһӣвҖӢбһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷвҖӢбһҖбһүбҹ’бһ…бһ”бҹӢ `npm` бһҳбһ·бһ“вҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ

`.i18n/v/ol/v.hash` бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹғбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҖбһүбҹ’бһ…бһ”бҹӢ `npm` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ !

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһӣбһ»бһ”бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷ `.i18n/v/ol` бһҖбһүбҹ’бһ…бһ”бҹӢ `npm` бһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ‘бҹҒбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһ“бҹҒбҹҮ бһҠбҹҶбһ”бһјбһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӣбҹҒбһҒбһҖбҹҶбһҺбҹӮбһ“бҹғбһҖбҹҶбһҺбҹӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [npmjs.com](//npmjs.com) `0.1.9`

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһҳбһҖвҖӢбһҹбһјбһҳвҖӢбһҷбҹ„бһ„вҖӢбһ‘бҹ…вҖӢбһӣбһҫвҖӢбһҜбһҖбһҹбһ¶бһҡ `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

бһ…бҹҶбһҺбһ¶бҹҶбһҗбһ¶бһҖбһ¶бһҡбһҮбһҪбһҹбһҮбһ»бһӣбһҸбһ¶бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҜбһҖбһҹбһ¶бһҡ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһ бһҫбһҷбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбҹҒбһ…бһҒбҹ’бһ…бһ”бҹӢбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбҹ’бһҸбһ„бҹ”
