## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ

`i18n.site` бһ‘бһ‘бһҪбһӣбһҷбһҖбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ‘бҹҶбһ–бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҹбҹ’бһҡбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ·бһ”бһ·бһҖбҹ’бһҡбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ‘бҹҶбһ–бҹҗбһҡбһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖ бһ“бһ·бһ„ `sitemap.xml` бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ»бһҖбһҡбһҖбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ `User-Agent` бһ“бҹғбһҹбҹҶбһҺбһҫбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһҹбҹҶбһҺбһҫбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ”бһ“бҹ’бһҸбһ‘бҹ…бһ‘бҹҶбһ–бҹҗбһҡбһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸбһҸбһ¶бһҳбһҡбһҷбҹҲ `302` бҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһңбһҸбҹ’бһҗбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһҜбһҖбһҹбһ¶бһҡбһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸ

бһҜбһҖбһҹбһ¶бһҡбһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ‘бһјбһ‘бҹ…бһҳбһҪбһҷбһӮбһәбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖбһңбһҸбҹ’бһҗбһ»бҹ”

бһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ `.i18n/htm/ol.yml` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ

```yml
host:
seo: true
out:
  - s3
v: //unpkg.com/i18n.site
x: 18x
importmap:
  i/: //unpkg.com/@i18n.site/
```

бһҠбҹҶбһ”бһјбһ„ бһҹбһјбһҳбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбһҳбҹ’бһӣбҹғ `host:` бһҒбһ¶бһ„бһӣбһҫбһ‘бҹ…бһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһјбһ…бһҮбһ¶ `i18n.site` .

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ `~/.config/i18n.site.yml` бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ :

```yml
site:
  i18n.site:
    s3:
      - endpoint: s3.eu-central-003.backblazeb2.com
        ak: # access key
        sk: # secret key
        bucket: # bucket name
        # region:
```

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһҹбһјбһҳбһ”бҹ’бһҸбһјбһҡбһӣбҹҒбһҒ `i18n.site` бһ‘бҹ…бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ `host:` бһҖбҹ’бһ“бһ»бһ„ `.i18n/htm/ol.yml` бһ бһ¶бһ„бһӣбһҖбҹӢбһңбһҸбҹ’бһҗбһ»бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһҡбҹ„бһҳ `s3` бһ бһҫбһҷбһңбһ¶бһӣ `region` бһӮбһәбһҹбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ (бһ бһ¶бһ„бһӣбһҖбҹӢбһңбһҸбҹ’бһҗбһ»бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бһӣбһ“бҹҒбҹҮбһ‘бҹҒ)бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ `i18n.site -n` бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһӮбһҳбҹ’бһҡбҹ„бһ„бһЎбһҫбһ„бһңбһ·бһүбҹ”

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮ `~/.config/i18n.site.yml` бһ бһҫбһҷбһ…бһ„бҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһЎбһҫбһ„бһңбһ·бһү бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһўбһ¶бһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ :

```
rm -rf .i18n/data/seo .i18n/data/public
```
