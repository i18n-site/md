**бһҖбһ»бҹҶбһҹбһҡбһҹбҹҒбһҡ `css` бһ“бһ·бһ„ `js` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `foot.pug`** бһ”бһҫбһҳбһ·бһ“бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ‘бҹҒбһ“бһ№бһ„бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹбҹ”

бһҹбһјбһҳбһҷбҹ„бһ„бһ‘бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶ `.css` бһ бһҫбһҷбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

## бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бһ¶бһҸбһҖбһҗбһ¶

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ `md/.i18n/htm` бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ 3 `css` бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

* `foot.css` : бһ”бһ¶бһҸбһҖбһҗбһ¶
* `import.css` : бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ `i18n.site` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ
* `conf.css` бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ”бһ¶бһҸбһҖбһҗбһ¶ : бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ

### бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ

бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ”бһ¶бһҸбһҖбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷ iconfont.cn ( [бһҖбҹҶбһҺбҹӮбһҮбһ¶бһ—бһ¶бһҹбһ¶бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹ](https://www.iconfont.cn/?lang=en-us) /[дёӯж–ҮзүҲ](https://www.iconfont.cn/?lang=zh)) бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҲбҹ’бһҳбҹ„бҹҮ `F`

бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ бһҫбһҷбһҮбҹҶбһ“бһҪбһҹ : бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `conf.css`

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
```

## бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһ”бһҺбҹ’бһҸбһ¶бһү

`foot.pug` бһҳбһ·бһ“бһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ `js` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ¶бһ”бһ¶бһ“бһ‘бҹҒ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ бһҹбһјбһҳбһ”бҹ’бһҸбһјбһҡбһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһ‘бҹҶбһ–бҹҗбһҡбһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢ [бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһ”бһҺбҹ’бһҸбһ¶бһү](https://www.freecodecamp.org/news/build-your-first-web-component/) `foot.pug` `md/.i18n/htm/index.js`

бһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһҠбһјбһ…бһҮбһ¶бһҹбҹ’бһӣбһ¶бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ `<x-img>`

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```
