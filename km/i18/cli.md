# ការពន្យល់លម្អិតនៃប៉ារ៉ាម៉ែត្របន្ទាត់ពាក្យបញ្ជា

## សម្អាត `-p`

`-p` ឬ `--purge` នឹងសម្អាតឯកសារដែលមាននៅក្នុងថតការបកប្រែនីមួយៗ ប៉ុន្តែមិនមាននៅក្នុងថតភាសាប្រភពទេ។

ដោយសារតែនៅពេលសរសេរឯកសារ ឈ្មោះឯកសារ Markdown ជារឿយៗត្រូវបានកែសម្រួល ដែលបណ្តាលឱ្យមានឯកសារចាស់ៗ និងត្រូវបានបោះបង់ចោលជាច្រើននៅក្នុងថតការបកប្រែ។

ប្រើប៉ារ៉ាម៉ែត្រនេះដើម្បីសម្អាតឯកសារដែលគួរតែត្រូវបានលុបនៅក្នុងថតភាសាផ្សេងទៀត។

## `-d` ​បញ្ជី​បកប្រែ

ថតដែលបានបកប្រែលំនាំដើមទៅថតដែលឯកសារបច្ចុប្បន្នស្ថិតនៅ។

`-d` ឬ `--workdir` អាចបញ្ជាក់ថតការបកប្រែដូចជា៖

```
i18 -d ~/i18n/md
```

## មើល `-h`

`-h` ឬ `--help` អាចមើលជំនួយបន្ទាត់ពាក្យបញ្ជា។
