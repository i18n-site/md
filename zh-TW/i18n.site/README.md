# i18n.site : 純靜態多語言網站框架

`i18n.site`多語言、純靜態的文檔站生成器。

## 序言

`i18n.site`是文檔站生成器，更是網站開发框架。

一個以`MarkDown`為中心，用前端組件來注入交互的網站開发新范式。

每個前端組件都是可以獨立安裝的包。

在前后端分離基礎上，更有靜態內容和動態數據的分離。

您正在訪問的 [i18n.site](/) 就是基于此框架搭建(包含了用戶系統、扣費系統、郵件訂閱等等)。
