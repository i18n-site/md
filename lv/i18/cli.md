# Detalizēts Komandrindas Parametru Skaidrojums

## Notīriet `-P`

`-p` `--purge` iztīrīs failus, kas ir katrā tulkošanas direktorijā, bet neeksistē avota valodas direktorijā.

Jo, rakstot dokumentus, Markdown failu nosaukumi bieži tiek koriģēti, kā rezultātā tulkošanas direktorijā ir daudz vecu un pamestu failu.

Izmantojiet šo parametru, lai notīrītu failus, kas jādzēš citu valodu direktorijos.

## `-D` Tulkošanas Direktoriju

Tulkotajā direktorijā pēc noklusējuma ir direktorija, kurā atrodas pašreizējais fails.

`-d` `--workdir` var norādīt tulkošanas direktoriju, piemēram:

```
i18 -d ~/i18n/md
```

## `-H` Palīdzību

`-h` `--help` skatīt komandrindas palīdzību.
