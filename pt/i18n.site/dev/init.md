# Preparar o ambiente

* Instalar e inicializar o `docker` (Usuários de Mac são aconselhados a usar [orbstack](https://orbstack.dev) como runtime para o `docker`)

* Instalar [bun](https://bun.sh/docs/installation) e [nodejs](https://nodejs.org/en/download/package-manager)

## Inicializar o projeto

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
