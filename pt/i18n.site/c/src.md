## Backend

A construção do site de documentos requer apenas o front-end, não o backend.

Este backend é do próprio site i18n.site, compreendendo sistemas de usuários, pagamentos, envio de mensagens, etc.

* Serviço [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Plugin [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Biblioteca Básica [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
