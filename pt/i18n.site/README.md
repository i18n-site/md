## Prefácio

`i18n.site` é uma reflexão sobre o modelo de desenvolvimento de sites.

É um gerador de sites de documentos e também um framework para desenvolvimento de sites.

Um novo paradigma de desenvolvimento de sites, com `MarkDown` como centro e componentes front-end para injetar interatividade.

Cada componente front-end é um pacote que pode ser instalado de forma independente.

Com base na separação entre front-end e back-end, há também a separação entre conteúdo estático e dados dinâmicos.

O site que você está visitando, [i18n.site](/), foi construído com base neste framework (incluindo sistema de usuários, sistema de cobrança, assinatura de e-mail, etc.).
