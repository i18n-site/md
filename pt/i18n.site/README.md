# i18n.site: Estrutura de site multilíngue estática pura

`i18n.site` gerador de sites de documentos estáticos e multilíngues.

## Prefácio

`i18n.site` é um gerador de sites de documentos e uma framework para desenvolvimento web.

Um novo paradigma de desenvolvimento de sites, com `Markdown` no centro e componentes front-end para injetar interatividade.

Cada componente front-end é um pacote que pode ser instalado de forma independente.

Com base na separação entre front-end e back-end, há também a separação entre conteúdo estático e dados dinâmicos.

Você está visitando [i18n.site](/), que foi construído com base nesta framework (incluindo sistema de usuários, sistema de cobrança, assinatura de e-mails, etc.).
