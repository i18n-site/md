# i18n.site: Estrutura de site multilíngue puramente estática

`i18n.site` gerador de sites de documentos multilíngues e puramente estáticos.

## Prefácio

`i18n.site` é um gerador de sites para documentos e, além disso, uma framework para o desenvolvimento de websites.

Uma nova abordagem de desenvolvimento de sites centrada em `Markdown`, utilizando componentes front-end para integrar interatividade.

Cada componente front-end é um pacote que pode ser instalado de forma independente.

Com base na separação entre front-end e back-end, há também a divisão entre conteúdo estático e dados dinâmicos.

O site que você está visitando, [i18n.site](/), foi construído sobre esta framework (e inclui sistemas de usuários, de cobrança, de assinatura de e-mail, etc.).
