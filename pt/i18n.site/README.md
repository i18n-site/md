# i18n.site: Framework de site multilíngue estático puro

`i18n.site` é um gerador de sites de documentos multilíngues e estáticos puros.

## Prefácio

`i18n.site` é um gerador de sites para documentos e também uma framework para desenvolvimento web.

Um novo paradigma de desenvolvimento de sites, com `MarkDown` no centro e que utiliza componentes front-end para injectar interatividade.

Cada componente front-end é um pacote que pode ser instalado de forma independente.

Com base na separação entre front-end e back-end, há também a separação entre conteúdo estático e dados dinâmicos.

Você está visitando [i18n.site](/), que é construído com base neste framework (incluindo sistema de usuários, sistema de cobrança, inscrição por e-mail, etc.).
