
## Názov Domény Viazanej Na CNAME

Názov domény je viazaný cez `SSL/TLS` → vlastný názov hostiteľa CNAME Cloudflare Worker názvu domény bude priamy prístup hlásiť chybu `error code: 522`

V tejto chvíli musíte použiť smerovanie HTTP → Smerovanie pracovníkov na naviazanie trasy, ako je znázornené nižšie
