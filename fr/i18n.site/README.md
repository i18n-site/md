# i18n.site : Cadre de site web multilingue纯静态

`i18n.site` : Générateur de documentation multilingue et纯静态.

## Préface

`i18n.site` est non seulement un générateur de sites de documentation, mais également un framework de développement pour sites Web.

Un nouveau paradigme de développement de sites web centré sur `MarkDown` et utilisant des composants front-end pour injecter de l'interactivité.

Chaque composant front-end est un paquet pouvant être installé indépendamment.

Sur la base de la séparation du front-end et du back-end, il y a également une séparation entre le contenu statique et les données dynamiques.

L'[i18n.site](/) que vous visitez est construit sur ce cadre (et inclut un système d'utilisateurs, un système de facturation, un abonnement par email, etc.).
