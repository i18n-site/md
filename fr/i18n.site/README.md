## Préface

`i18n.site` est une réflexion sur le mode de développement des sites web.

C'est à la fois un générateur de sites documentaires et un cadre de développement web.

Un nouveau paradigme de développement web centré sur `MarkDown` et utilisant des composants front-end pour injecter de l'interactivité.

Chaque composant front-end est un paquet pouvant être installé indépendamment.

Basé sur la séparation du front-end et du back-end, il y a également une séparation entre le contenu statique et les données dynamiques.

Le site que vous visitez [i18n.site](/) est construit sur ce cadre (et inclut un système d'utilisateurs, un système de facturation, un abonnement par email, etc.).
