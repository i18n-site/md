## Backend

La construction du site de documentation nécessite uniquement le front-end, pas de backend.

Ce backend est celui du site Web i18n.site lui-même, incluant les systèmes d'utilisateurs, de paiement, de notifications, etc.

* Services [GitHub](https://github.com/i18n-api/srv) / [AtomGit](https://atomgit.com/i18n-api/srv)
* Plugins [GitHub](https://github.com/i18n-api/pub) / [AtomGit](https://atomgit.com/i18n-api/pub)
* Bibliothèques de base [GitHub](https://github.com/i18n-site/rust) / [AtomGit](https://atomgit.com/i18n/rust)
