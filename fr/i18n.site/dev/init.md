# Préparer l'environnement

* Installer et lancer `docker` (les utilisateurs Mac sont invités à utiliser [orbstack](https://orbstack.dev) comme environnement d'exécution pour `docker`)

* Installer [bun](https://bun.sh/docs/installation) et [nodejs](https://nodejs.org/en/download/package-manager)

## Initialiser le projet

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
