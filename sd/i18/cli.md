# ڪمانڊ لائن پيٽرولر جي تفصيلي وضاحت

## فائلون صاف `-p`

`-p` `--purge` فائلن کي صاف ڪندو جيڪي هر ترجمي جي ڊاريڪٽري ۾ موجود آهن پر ماخذ ٻولي ڊاريڪٽري ۾ موجود نه آهن.

ڇاڪاڻ ته جڏهن دستاويز لکندا آهن، مارڪ ڊائون فائل جا نالا اڪثر ترتيب ڏئي رهيا آهن، نتيجي ۾ ترجمو ڊاريڪٽري ۾ ڪيتريون ئي پراڻيون ۽ ختم ٿيل فائلون.

فائلن کي صاف ڪرڻ لاءِ ھي پيٽرول استعمال ڪريو جيڪي ٻين ٻولين جي ڊاريڪٽريز ۾ ڊھي وڃن.

## ترجمو ڊاريڪٽري بيان `-d`

ترجمو ٿيل ڊاريڪٽري ڊاريڪٽري کي ڊفالٽ ڪري ٿي جتي موجوده فائل واقع آهي.

`-d` `--workdir` ترجمو ڊاريڪٽري بيان ڪري سگھي ٿو، جيئن ته:

```
i18 -d ~/i18n/md
```

## مدد `-h`

`-h` `--help` ڏسي سگهو ٿا ڪمانڊ لائن مدد.
