# انسٽال ڪريو ۽ استعمال ڪريو

## ونڊوز پهريون ڀيرو گٽ بش انسٽال ڪريو

windows ، مهرباني ڪري [هتي ڪلڪ ڪريو ڊائون لوڊ ۽ انسٽال ڪريو `git bash`](https://git-scm.com/download/win)

اڳتي هلي آپريشن ۾ `git bash`

## انسٽال ڪريو

```sh
bash <(curl -sS https://i.i18n.site) i18
```

### ترجمي جي ٽوڪن کي ترتيب ڏيو

ٽوڪن کي نقل ڪرڻ لاءِ ڪلڪ [i18n.site/token](//i18n.site/token)

<img src="https://p.3ti.site/1719911689.avif" style="max-width:400px">

ٺاهيو `~/.config/i18n.site.yml` ڪاپي ٿيل مواد کي ان ۾ پيسٽ ڪريو، مواد هن ريت آهي:

```
token: YOUR_API_TOKEN
```

اضافي طور تي، توهان کي ادائگي لاء ڪريڊٽ ڪارڊ پابند [i18n.site/payBill](//i18n.site/payBill) ضرورت آهي (ڪنهن به ريچارج جي ضرورت ناهي، ويب سائيٽ خودڪار طور تي استعمال جي مطابق فيس ڪٽيندي، [قيمت لاء هوم صفحو ڏسو](/#price) ).

## استعمال ڪرڻ

### ڊيمو پروجيڪٽ

`i18` جي ترتيب سکڻ لاءِ مهرباني ڪري ڊيمو پروجيڪٽ ڏانهن رجوع ڪريو [github.com/i18n-site/demo.i18](//github.com/i18n-site/demo.i18)

چين ۾ استعمال ڪندڙ کلون ڪري سگهن ٿا [atomgit.com/i18n/demo.i18](//atomgit.com/i18n/demo.i18)

ڪلوننگ کان پوء، ڊاريڪٽري داخل ڪريو ۽ ترجمو مڪمل ڪرڻ لاء `i18`

### ڊاريڪٽري جي جوڙجڪ

ٽيمپليٽ گودام ڊاريڪٽري جي جوڙجڪ هن ريت آهي

```
┌── .i18n
│  └── conf.yml
└── en
   ├── _IgnoreDemoFile.md
   ├── i18n.yml
   └── README.md
```

`en` ۾ ترجمو ٿيل ڊيمو فائلون شامل آهن، جيڪي صرف هڪ مثال آهن ۽ ختم ڪري سگهجن ٿيون.

### ترجمو ڪريو

ڊاريڪٽري داخل ڪريو ۽ ترجمو ڪرڻ لاء `i18`

### ذخيرو ۾ فائلون شامل ڪريو

ترجمي کان علاوه، پروگرام ھيٺيون فائلون پڻ ٺاھيندو، مھرباني ڪري انھن کي مخزن ۾ شامل ڪريو.

```
.i18n/hash
.i18n/cache/.gitignore
```

انهن مان، جو مواد هن ريت : `.i18n/cache/.gitignore`

```
**/*
!**/.gitignore
```

هن جو مطلب آهي `.i18n/cache/` ۾ سڀني فائلن کي نظر انداز ڪريو (سواء `.i18n/cache/.gitignore` ).

جيڪڏهن توهان جو نسخو ڪنٽرول سافٽ ويئر نه آهي `git` مهرباني ڪري هن ترتيب جي مطابق ان کي نظر انداز ڪريو.

## ٺاھ جوڙ فائيل

اهو `i18` لائن ترجمي واري اوزار جي ترتيب واري فائل `.i18n/conf.yml`

مواد هن ريت آهي:

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### ماخذ ٻولي &

ترتيب واري فائل ۾، ماتحت `fromTo`

ماخذ ٻولي آهي `en` `zh ja ko de fr` ترجمي جي ٽارگيٽ ٻولي آهي.

ٻولي ڪوڊ ڏسو [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

مثال طور، جيڪڏھن توھان چاھيو ٿا چيني ترجمو انگريزيءَ ۾، وري لکو ھن لائن کي `zh: en`

جيڪڏھن توھان چاھيو ٿا ترجمو ڪيل سڀني ٻولين ۾، مھرباني ڪري خالي ڇڏي ڏيو بعد ۾ `:` مثال طور

```
i18n:
  fromTo:
    en:
```

### فائل کي نظر انداز ڪريو

ڊفالٽ طور، سڀني فائلن کي شروع ڪيو ويندو `.md` ۽ `.yml` سان ذريعو ٻولي ڊاريڪٽري ۾.

جيڪڏهن توهان ڪجهه فائلن کي نظرانداز ڪرڻ چاهيو ٿا ۽ انهن جو ترجمو نه ڪرڻ چاهيو ٿا (جهڙوڪ نامڪمل ڊرافٽ)، توهان استعمال ڪري سگهو ٿا `ignore` جي ترتيب.

`.gitignore` سان ملندڙ هڪ نحو استعمال [globset](https://docs.rs/globset/latest/globset/#syntax) ٿو `ignore`

مثال طور، مٿي ڏنل ترتيب واري فائل ۾ `_* ` مطلب ته `_` سان شروع ٿيندڙ فائلون ترجمو نه ٿينديون.

## ترجمي جا ضابطا

### ترجمي جي ايڊيٽرن کي لائنن کي شامل يا ختم نه ڪرڻ گھرجي

ترجمو قابل تدوين آهي. اصل متن کي تبديل ڪريو ۽ مشين-ان کي ٻيهر ترجمو ڪريو، ترجمي ۾ دستي ترميمون اوور رائٽ نه ٿينديون (جيڪڏهن اصل متن جي هن پيراگراف ۾ ترميم نه ڪئي وئي آهي).

پر مهرباني ڪري نوٽ ڪريو ته ترجمي جون لائينون ۽ اصل متن هڪ ٻئي سان ملندڙ جلندڙ هجڻ گهرجن. اهو آهي، ترجمي کي گڏ ڪرڻ وقت لائين شامل يا ختم نه ڪريو. ٻي صورت ۾، اهو ترجمي جي ترميمي ڪيش ۾ مونجهارو پيدا ڪندو.

جيڪڏھن ڪجھھ غلط ٿي وڃي، مھرباني ڪري حوالو ڏيو [FAQ حل لاءِ.](/i18/qa#H1)

### `YAML`

ڪمانڊ لائن ٽول سڀني فائلن کي ڳولهيندو `.yml` سان ختم ٿئي ٿو سورس ٻولي فائل ڊاريڪٽري ۽ انهن کي ترجمو ڪريو.

* نوٽ ڪريو ته فائل جو نالو لازمي `.yml` (نه `.yaml` ).

اوزار صرف ڊڪشنري جي قدرن جو ترجمو ڪري ٿو `.yml` لغت جي چاٻين کي نه.

مثال طور `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

جيئن ترجمو ڪيو ويندو `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

`YAML` ترجمي کي دستي طور تي تبديل ڪري سگھجي ٿو (پر ترجمي ۾ ڪيز يا لائين شامل نه ڪريو).

ترجمي `YAML` بنياد تي، توهان آساني سان مختلف پروگرامنگ ٻولين لاءِ بين الاقوامي حل ٺاهي سگهو ٿا.

## ترقي يافته استعمال

### ترجمي جي ذيلي ڊائريڪٽري

جيستائين توهان ٺاهيندا `.i18n/conf.yml` (هر وقت ڊيمو پروجيڪٽ ٽيمپليٽ کان شروع ڪرڻ جي ضرورت ناهي)، `i18` ٺيڪ ڪم ڪندو.

ڪمانڊ لائن ٽول تمام ذيلي ڊائريڪٽرن ۾ ترتيب `.i18n/conf.yml` ڳوليندو ۽ ان کي ترجمو ڪندو.

پروجيڪٽ [monorepo](//monorepo.tools) استعمال ڪن ٿا آرڪيٽيڪچر ٻولي فائلن کي ذيلي ڊائريڪٽرن ۾ ورهائي سگهن ٿا.

<img alt="" src="https://p.3ti.site/1719910016.avif">

### ڪسٽم انسٽاليشن ڊاريڪٽري

اهو ڊفالٽ طور تي نصب ڪيو ويندو `/usr/local/bin`

`/usr/local/bin` لکڻ جي اجازت نه آهي ته ان کي انسٽال ڪيو ويندو `~/.bin` .

سيٽ ڪريو ماحوليات متغير `TO` توھان انسٽاليشن ڊاريڪٽري جي وضاحت ڪري سگھو ٿا، مثال طور :

```sh
TO=/bin sudo bash <(curl -sS https://i.i18n.site) i18
```
