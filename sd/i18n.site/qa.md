# عام مسئلو

## اتفاقي طور ختم ٿي ويو `.i18n/v` جنهن ڪري `npm` پيڪيج شايع ٿيڻ ۾ ناڪام ٿي ويو

`npm` پيڪيج رليز جو تاريخي نسخو محفوظ ڪيو ويو آهي `.i18n/v/ol/v.hash`

جيڪڏهن توهان اتفاقي طور تي حذف ڪيو `.i18n/v/ol` `npm` پيڪيج شايع نه ڪيو ويندو.

هن وقت، پهريون ڀيرو توهان جي پروجيڪٽ جي آخري ورزن جو نسخو نمبر ڳوليو [npmjs.com](//npmjs.com) مثال طور `0.1.9`

پوءِ ھيٺ ڏنل `bash` ٺاھيو ڏانھن وڃو.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

نوٽ ڪريو ته ھن طريقي سان مرمت ڪرڻ سان فائل جي تاريخ ختم ٿي ويندي، جيڪا ايندڙ رليز کي واڌاري سان اپڊيٽ ڪرڻ ناممڪن بڻائيندي، ۽ سڀ مواد ھڪ ڀيرو ٻيهر پيڪيج ۽ اپلوڊ ڪيو ويندو.
