### ޕޯސްޓް ކޮންޓެންޓް

`i18n.site`[​](https://developer.mozilla.org/docs/Glossary/SPA)

މަތީގައިވާ ތަރުޖަމާ ހިންގުމަށްފަހު، `htm` އަދި `v` ޑައިރެކްޓަރީތައް `md/out/dev`

މިތަނުގައި، `dev` ދައްކަނީ އެއީ `.i18n/htm/dev.yml` ފައިލްގެ މައްޗަށް ބިނާކޮށް ބިނާކޮށްފައިވާ އެއްޗެކެވެ.

ކޮންޓެންޓްސްގެ : `dev`

`htm`

`v`

ލޯކަލް ޕްރިވިއު އިން ވަރޝަން ނަންބަރަށް ބެލުމެއްނެތި ހުރިހާ ފައިލްތަކެއް `out/dev/v/0.1.0`

ރަސްމީކޮށް ރިލީޒް ކުރުމަށްޓަކައި ބަދަލުކޮށްފައިވާ ފައިލްތައް އާ ވަރޝަން ނަންބަރު ޑައިރެކްޓަރީއަށް ކޮޕީ ކުރެވޭނެއެވެ.

#### ކޮންފިގްރޭޝަން ފައިލް ކަނޑައެޅުމަށް -c ބޭނުން ކުރާށެވެ

އެކި ކޮންފިގްރޭޝަން ފައިލްތަކުން `out`

`out/ol` `.i18n/htm/ol.yml`

`dev.yml` އަދި `ol.yml` އަކީ ޑިފޯލްޓް ކޮންފިގްރޭޝަންތަކެވެ.

`dev` `development`
އޮންލައިން މާހައުލު ތަމްސީލުކުރާ `online` ގެ ކުރުކުރުން `ol` ރަސްމީކޮށް ރިލީޒް ކުރުމަށް ބޭނުންކުރާ، އަދި `npm` ކޮމާންޑް ލައިން ޕެރެމިޓަރ `-n` ބޭނުންކޮށްގެން) އަށް ޝާއިއުކުރާއިރު ޑިފޯލްޓް ކޮންފިގްރޭޝަން ފައިލް ވެސް މެއެވެ.

އަދި އެހެން ކޮންފިގްރޭޝަން ފައިލްތައް ވެސް އުފެއްދިދާނެ ކޮންފިގްރޭޝަން ފައިލް ނަން ކަނޑައެޅުމަށް ކޮމާންޑް ލައިންގައި `--htm_conf`

މިސާލަކަށް:
```
i18n.site --htm_conf yourConfig --save
```

މިތަނުގައި `--save` އަޕްޑޭޓް ރިލީޒް ވާޝަން ނަންބަރު ދައްކައެވެ.

#### <a rel=id href="#npm" id="npm"></a> npmjs.com އަށް ކޮންޓެންޓް ޝާއިއުކުރުން

[npmjs.com](//npmjs.com) ކޮންޓެންޓް ޝާއިއުކުރުމަކީ ރަނގަޅު ޑިފޯލްޓް ހައްލެއް ( [ފްރަންޓް-އެންޑް ހައި އެވޭލަބިލިޓީ](/i18n.site/feature#ha) ބައްލަވާ ).

##### npm login & ދޫކުރުން

އިންސްޓޯލް `nodejs` ލޮގްއިން ކުރުމަށް `npm login` ބޭނުން ކުރާށެވެ.

އެޑިޓް `md/.i18n/htm/ol.yml` `i18n.site` އިން `v: //unpkg.com/i18n.site` އަށް ބަދަލުކޮށް އަމިއްލަ `npm` ޕެކޭޖް ނަން.

ހަމައެކަނި އޮކިއުޕޭޓް ޕެކޭޖް ނަން ބޭނުން ކުރާށެވެ [npmjs.com](//npmjs.com)

`npm` `v:` **`//unpkg.com/`** `i18n.site` `/.v`

ތަރުޖަމާކޮށް ޝާއިޢުކުރުމަށްޓަކައި `md` ޑައިރެކްޓަރީގައި `i18n.site --npm` ނުވަތަ `i18n.site -n` ހިންގާށެވެ.

ޕަބްލިޝް ކުރުމަށް ކޮންޓިނިއުއަސް އިންޓަގްރޭޝަން `~/.npmrc` ބޭނުންކުރާނަމަ، އިންސްޓޯލް ކުރަން ނުޖެހޭނެ `nodejs`

`v:` `ol.yml` ގައި ޕެކޭޖް ނަން ބަދަލުކޮށްފިނަމަ، **ފުރަތަމަ `.i18n/v/ol` ޑިލީޓް ކުރުމަށްފަހު ޝާއިއުކުރުން ޔަގީން** ކުރައްވާށެވެ.

##### ޕްރޮކްސީ ސަރވަރ ޕަބްލިޝްކޮށްފައިވަނީ npm އިންނެވެ

ޗައިނާގެ ބިންގަނޑުގައި ތިބި ޔޫޒަރުންނަށް ނެޓްވޯކް މައްސަލަތަކެއް ދިމާވެ `npm` ޕެކޭޖް ޝާއިއު ނުކުރެވިއްޖެނަމަ، ޕްރޮކްސީ ސަރވަރ ކޮންފިގްރޭޓް ކުރުމަށް އެންވަޔަރަންމަންޓް ވެރިއޭބަލް `https_proxy` ސެޓް ކުރެވިދާނެއެވެ.

ތިބާގެ ޕްރޮކްސީ ސަރވަރ ޕޯޓަކީ `7890` ލިޔެވިދާނެއެވެ:

```
https_proxy=http://127.0.0.1:7890 i18n.site -n
```

#### އަމިއްލައަށް ހޯސްޓް ކުރާ ކޮންޓެންޓެވެ

ކޮންޓެންޓް އަމިއްލައަށް ހޯސްޓް ކުރަން ބޭނުންނަމަ، ފުރަތަމަ އެޑިޓްކޮށް `md/.i18n/htm/ol.yml` `v: //unpkg.com/i18n.site` `v: //i18n-v.xxx.com` ޔޫ.އާރ.އެލް ޕްރިފިކްސް އަށް ބަދަލުކޮށްލާށެވެ.

`md` ޑައިރެކްޓަރީއަށް ވަދެ ދުއްވާލާށެވެ

```
i18n.site --htm_conf ol --save
```

ނުވަތަ ކުރުކުރުން

```
i18n.site -c ol -s
```

ދެން، ޑައިރެކްޓަރީގައިވާ ކޮންޓެންޓް `v:` ގައި ސެޓްކޮށްފައިވާ `md/out/ol/v` ޕްރިފިކްސް މަގަށް ކޮންފިގްރޭޓް ކުރާށެވެ.

އެންމެ ފަހުން **`/.v` އިން ނިމޭ މަގުގެ ކެޝް ޓައިމް `1s` އަށް ކޮންފިގްރޭޓް ކުރާށެވެ** ، އެހެން ނޫންނަމަ އަލަށް ނެރޭ ކޮންޓެންޓަށް ވަގުތުން އެކްސެސް ނުކުރެވޭނެއެވެ.

އެހެން މަގުތަކުގެ ކެޝް ޓައިމް އެއް އަހަރު ނުވަތަ އެއަށްވުރެ ގިނަ އަށް ސެޓްކޮށްގެން ކޮންމެހެން ބޭނުން ނުވާ ރިކުއެސްޓްތައް މަދުކުރެވިދާނެ އެވެ.

##### s3 އަށް ކޮންޓެންޓް ހޯސްޓްކުރުން

އަމިއްލައަށް ކޮންޓެންޓް ހޯސްޓް ކުރުމަށްޓަކައި + އަމިއްލަ ސަރވަރ ބޭނުން ކުރުމުގެ އިތުރުން، އާންމު އެހެން `CDN` `S3`

`S3` [rclone](https://rclone.org) `S3`

```bash
i18n.site -c ol -s
s3=your-s3
bucket=your-bucket
ver=$(head -1 .i18n/v/ol/v.hash | cut -c 2-)
rclone copy --overwrite-dir out/ol/htm/v/$ver $s3:/$bucket/$ver
rclone copy out/ol/v/.v "$s3:/$bucket/"
```

`/.v` އިން ނިމޭ މަގުގެ ކެޝް ޓައިމް `1s` އަށް ވާގޮތަށް `CDN` ކޮންފިގްރޭޓް ކުރަން ހަނދާން ކުރާށެވެ، އެހެން ނޫންނަމަ އަލަށް ނެރޭ ކޮންޓެންޓަށް ވަގުތުން އެކްސެސް ނުކުރެވޭނެއެވެ.

### ވެބްސައިޓް ޝާއިއުކުރުން

ވެބްސައިޓް ކޮންމެ ތަނެއްގައި ވެސް ޑިޕްލޮއި ކުރެވޭނެ [github page](https://pages.github.com) އަދި [cloudflare page](https://pages.cloudflare.com) އަކީ ރަނގަޅު ޗޮއިސްތަކެކެވެ.

ވެބްސައިޓުން [އެއް ޞަފްޙާގެ އެޕްލިކޭޝަނެއްގެ](https://developer.mozilla.org/docs/Glossary/SPA) އަރކިޓެކްޗަރ ގެންގުޅޭތީ، `. ` `index.html` ނުހިމެނޭ ޔޫ.އާރ.އެލް މަގު އަލުން ލިޔަން ހަނދާން ކުރާށެވެ.

ވެބްސައިޓް އެންޓްރީ ޕޭޖް ޑިޕްލޮއި ކުރަން ޖެހެނީ އެންމެ ފަހަރަކު ކަމަށާއި، އޭގެ ފަހުން ކޮންޓެންޓް އަޕްޑޭޓް ކުރުމަށް ވެބްސައިޓް އެންޓްރީ ޕޭޖް އަލުން ޑިޕްލޮއި ކުރަން ނުޖެހޭ ކަމަށެވެ.
