# އާންމު މައްސަލައެކެވެ

## އެކްސިޑެންޓްވެގެން ޑިލީޓް ވުމުން `.i18n/v` `npm` ޕެކޭޖް ޝާއިއު ނުކުރެވިގެންނެވެ

`npm` ޕެކޭޖް ރިލީޒްގެ ތާރީޚީ ވަރޝަން ސޭވް ކުރެވިފައިވަނީ `.i18n/v/ol/v.hash`

އަނިޔާވެރިކަމާއެކު `.i18n/v/ol` ޑިލީޓް ކޮށްފިނަމަ `npm` ޕެކޭޖް ޝާއިއު ނުކުރެވޭނެއެވެ.

މިވަގުތު ފުރަތަމަ `0.1.9` ޕްރޮޖެކްޓްގެ އެންމެ ފަހުގެ ރިލީޒް ވާޝަންގެ ވަރޝަން ނަންބަރު [npmjs.com](//npmjs.com)

ދެން ތިރީގައިވާ `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

މިގޮތަށް މަރާމާތު ކުރުމުން ފައިލް ހީސްޓްރީ ގެއްލި، ދެން އަންނަ ރިލީޒް އިންކްރިމެންޓަލީ އަޕްޑޭޓް ކުރުން މުސްތަޙީލުވެ، ހުރިހާ ކޮންޓެންޓެއް ރީޕެކޭޖްކޮށް އެއްފަހަރު އަޕްލޯޑް ކުރެވޭނެކަން ފާހަގަކޮށްލެވެއެވެ.
