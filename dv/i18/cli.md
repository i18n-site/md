# ކޮމާންޑް ލައިން ޕެރެމިޓަރތަކުގެ ތަފްސީލީ ބަޔާން

## `-p`

`-p` ނުވަތަ `--purge` ކޮންމެ ތަރުޖަމާ ޑައިރެކްޓަރީއެއްގައި ހުރި ނަމަވެސް ސޯސް ލޭންގުއޭޖް ޑައިރެކްޓަރީގައި ނެތް ފައިލްތައް ސާފުކުރާނެއެވެ.

އެހެނީ ލިޔެކިޔުންތައް ލިޔާއިރު ގިނަ ފަހަރަށް މާކްޑައުން ފައިލް ނަންތައް އެޖެސްޓް ކުރެވޭތީ، އޭގެ ނަތީޖާއެއްގެ ގޮތުން ތަރުޖަމާ ޑައިރެކްޓަރީގައި ގިނަ ދުވަސްވީ އަދި ދޫކޮށްލާފައިވާ ފައިލްތައް އުފެދެއެވެ.

މި ޕެރެމިޓަރ ބޭނުންކޮށްގެން އެހެން ބަހުގެ ޑައިރެކްޓަރީތަކުގައި ޑިލީޓްކުރަންޖެހޭ ފައިލްތައް ސާފުކުރުން.

## `-d` ތަރުޖަމާ ޑައިރެކްޓަރީ ކަނޑައެޅުން

ތަރުޖަމާ ކުރެވިފައިވާ ޑައިރެކްޓަރީ ޑިފޯލްޓް ވާނީ މިހާރު ހުރި ފައިލް ހުރި ޑައިރެކްޓަރީ އަށެވެ.

`-d` ނުވަތަ `--workdir` ތަރުޖަމާ ޑައިރެކްޓަރީ ކަނޑައެޅިދާނެ، ފަދައިން:

```
i18 -d ~/i18n/md
```

## `-h`

`-h` ނުވަތަ `--help` ކޮމާންޑް ލައިން އެހީ ބަލާލެވޭނެއެވެ.
