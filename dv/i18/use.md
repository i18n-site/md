# އިންސްޓޯލްކޮށް ބޭނުންކުރުން

## ވިންޑޯޒް ފުރަތަމަ އިންސްޓޯލް ކުރަނީ ގިޓް ބެޝް އެވެ

windows ، [ފުރަތަމަ ޑައުންލޯޑްކޮށް އިންސްޓޯލް ކުރުމަށް މިތަނަށް ފިތާލާށެވެ `git bash`](https://git-scm.com/download/win)

ގައި އޭގެ ފަހުން ހިންގާ އޮޕަރޭޝަންތައް ހިންގާށެވެ `git bash`

އުފެއްދުން `~/.config/i18n.site.yml` ކޮޕީކޮށްފައިވާ ކޮންޓެންޓް އޭގެ ތެރެއަށް ޕޭސްޓްކޮށް، ކޮންޓެންޓް ތިރީގައި މިވަނީއެވެ.

މީގެ އިތުރުން ފައިސާ ދެއްކުމަށް ކްރެޑިޓް ކާޑެއް ބައިންޑް [i18n.site/payBill](//i18n.site/payBill) ޖެހޭނެ (ރީޗާޖް ކުރަން ނުޖެހޭ، ވެބްސައިޓުން ބޭނުން ކުރާ ގޮތަށް އޮޓޮމެޓިކުން ފީ އުނިކުރާނެ، [އަގު ކަނޑައެޅުމަށް ހޯމްޕޭޖް ބައްލަވާ](/#price) ).

## ބޭނުންކުރުން

### ޑެމޯ ޕްރޮޖެކްޓް

`i18` ކޮންފިގްރޭޝަން ދަސްކުރުމަށް ޑެމޯ ޕްރޮޖެކްޓަށް ރިއާޔަތް ކުރައްވާ [github.com/i18n-site/demo.i18](//github.com/i18n-site/demo.i18)

ޗައިނާގައި ތިބި ޔޫޒަރުންނަށް ކްލޯން ކުރެވޭނެ [atomgit.com/i18n/demo.i18](//atomgit.com/i18n/demo.i18)

ކްލޯން ކުރުމަށްފަހު ޑައިރެކްޓަރީއަށް ވަދެ ތަރުޖަމާ ފުރިހަމަ ކުރުމަށް `i18`

### ޑައިރެކްޓަރީ ސްޓްރަކްޗަރ

ޓެމްޕްލޭޓް ގުދަން ޑައިރެކްޓަރީގެ އޮނިގަނޑު ތިރީގައި މިވަނީއެވެ

```
┌── .i18n
│  └── conf.yml
└── en
   ├── _IgnoreDemoFile.md
   ├── i18n.yml
   └── README.md
```

### ތަރުޖަމާ ހިންގާށެވެ

ޑައިރެކްޓަރީއަށް ވަދެ ތަރުޖަމާ ކުރަން `i18`

### ރިޕޮޒިޓަރީއަށް ފައިލްތައް އިތުރުކުރުން

ތަރުޖަމާ ކުރުމުގެ އިތުރުން މި ޕްރޮގްރާމްގެ ދަށުން ތިރީގައިވާ ފައިލްތައް ވެސް އުފައްދާނެ ކަމަށާއި، އެ ފައިލްތައް ރިޕޮޒިޓަރީއަށް އެޑް ކުރައްވާށެވެ.

```
.i18n/hash
.i18n/cache/.gitignore
```

އޭގެ ތެރޭގައި : ކޮންޓެންޓް ތިރީގައި މިވަނީ `.i18n/cache/.gitignore`

```
**/*
!**/.gitignore
```

މާނައަކީ `.i18n/cache/` ޑައިރެކްޓަރީގައިވާ ހުރިހާ ފައިލްތަކެއް ( `.i18n/cache/.gitignore` ފިޔަވައި) އަޅާނުލާށެވެ.

ތިޔަބޭފުޅުންގެ ވަރޝަން ކޮންޓްރޯލް ސޮފްޓްވެއަރ އަކީ ނޫން ނަމަ `git` މި ކޮންފިގްރޭޝަން އާއި އެއްގޮތަށް އަޅާނުލާށެވެ.

## ކޮންފިގްރޭޝަން ފައިލް

`.i18n/conf.yml` އެއީ `i18` ލައިން ތަރުޖަމާ ޓޫލްގެ ކޮންފިގްރޭޝަން ފައިލް އެވެ

ކޮންޓެންޓް ތިރީގައި މިވަނީއެވެ.

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### މަސްދަރު ބަސް &

ކޮންފިގްރޭޝަން ފައިލްގައި، `fromTo`

`en` `zh ja ko de fr`

ބަހުގެ ކޯޑް ބައްލަވާ [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

މިސާލަކަށް ޗައިނީސް އިނގިރޭސި ބަހަށް ތަރުޖަމާ ކުރަން ބޭނުންނަމަ މި ސަފުހާ އަލުން ލިޔެލާށެވެ `zh: en`

ސަޕޯޓްކުރާ ހުރިހާ ބަސްބަހަކަށް ތަރުޖަމާ ކުރަން ބޭނުންނަމަ `:` މިސާލަކަށް

```
i18n:
  fromTo:
    en:
```

### ފައިލް އަޅާނުލާށެވެ

ބައި ޑިފޯލްޓްކޮށް ސޯސް ލޭންގުއޭޖް ޑައިރެކްޓަރީގައިވާ `.md` އަދި `.yml` އިން ފެށޭ ހުރިހާ ފައިލްތަކެއް ތަރުޖަމާ ކުރެވޭނެއެވެ.

ވަކި ފައިލްތަކަށް އަޅާނުލާ ތަރުޖަމާ ނުކުރަން ބޭނުންނަމަ (ނުނިމި ހުރި ޑްރާފްޓްތައް ފަދަ) `ignore`

`ignore` `.gitignore` އެއްގޮތް ސިންޓެކްސްއެއް ބޭނުންކުރެއެވެ [globset](https://docs.rs/globset/latest/globset/#syntax)

މިސާލަކަށް މަތީގައިވާ ކޮންފިގްރޭޝަން ފައިލްގައި `_* ` ގެ މާނައަކީ `_` އިން ފެށޭ ފައިލްތައް ތަރުޖަމާ ނުކުރެވޭނެއެވެ.

## ތަރުޖަމާ ކުރުމުގެ އުސޫލުތައް

### ތަރުޖަމާ އެޑިޓަރުން ލައިންތައް އިތުރުކޮށް ނުވަތަ ޑިލީޓްކޮށްގެން ނުވާނެއެވެ

ތަރުޖަމާއަކީ އެޑިޓް ކުރެވޭ ތަރުޖަމާއެކެވެ. އަސްލު ލިޔުން ބަދަލުކޮށް އަލުން މެޝިން ތަރުޖަމާ ކުރުމުން، ތަރުޖަމާއަށް މެނުއަލްކޮށް ގެންނަ ބަދަލުތައް އޯވަރރައިޓް ނުކުރެވޭނެ (އަސްލު ލިޔުމުގެ މި ޕެރެގްރާފް ބަދަލުކޮށްފައި ނުވާނަމަ) އެވެ.

އެކަމަކު ފާހަގަކޮށްލަންޖެހޭ ކަމަކީ ތަރުޖަމާގެ ސަފުހާތަކާއި އަސްލު ލިޔުމަކީ އެއް ސަފުހާއެއް އެއް ސަފުހާއަކަށް ދިމާވާންޖެހޭނެ ކަމެވެ. އެއީ، ތަރުޖަމާ އެކުލަވާލާއިރު ސަފުހާތައް އިތުރު ނުކުރުން ނުވަތަ އުނި ނުކުރުމެވެ. އެހެން ނޫންނަމަ ތަރުޖަމާ އެޑިޓްކުރާ ކެޝްގައި އޮޅުން ބޮޅުން އުފެދިގެންދާނެ އެވެ.

ކަމެއް ގޯސްވެއްޖެނަމަ [ހައްލު ހޯދުމަށް އެފްއޭކިއު އަށް ރިއާޔަތް ކުރައްވާށެވެ.](/i18/qa#H1)

### `YAML` ތަރުޖަމާކުރުން

ކޮމާންޑް ލައިން ޓޫލް އިން ސޯސް ލޭންގުއޭޖް ފައިލް ޑައިރެކްޓަރީގައިވާ `.yml`

* ފައިލް ނަމުގެ ސުފިކްސް ވާންޖެހޭނީ `.yml` ( `.yaml` ނޫން ).

މި ޓޫލް ތަރުޖަމާ ކުރަނީ ހަމައެކަނި ބަސްފޮތުގެ އަގުތައް `.yml`

މިސާލަކަށް `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

ތަރުޖަމާ ކުރާނީ `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

## ކުރިއަރާފައިވާ ބޭނުންކުރުން

### ތަރުޖަމާ ސަބްޑައިރެކްޓަރީ

`.i18n/conf.yml` ހިނދަކު (ކޮންމެ ފަހަރަކު ޑެމޯ ޕްރޮޖެކްޓް ޓެމްޕްލޭޓުން ފެށުމުގެ ބޭނުމެއް ނެތް) `i18` ރަނގަޅަށް މަސައްކަތް ކުރާނެއެވެ.

ކޮމާންޑް ލައިން ޓޫލް އިން ހުރިހާ ސަބްޑައިރެކްޓަރީތަކެއްގައި ހުރި `.i18n/conf.yml`

### ކަސްޓަމް އިންސްޓޯލްމަންޓް ޑައިރެކްޓަރީ

އެއީ ޑިފޯލްޓްކޮށް އިންސްޓޯލް ކުރެވޭނެ އެއްޗެކެވެ `/usr/local/bin`

ލިޔުމުގެ ހުއްދަ `/usr/local/bin` `~/.bin` އަށް އިންސްޓޯލް ކުރެވޭނެއެވެ.

އެންވަޔަރަންމަންޓް ވެރިއޭބަލްސް `TO` އިންސްޓޯލްމަންޓް ޑައިރެކްޓަރީ ޑެފިނިޝް ކުރެވިދާނެ، މިސާލަކަށް :

```sh
TO=/bin sudo bash <(curl -sS https://i.i18n.site) i18
```
