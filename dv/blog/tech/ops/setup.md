# ސަރވަރ އިނިޝިއަލައިޒްކުރުން (ބީޓީއާރުއެފްއެސް ޕާރޓީޝަން، ސްނެޕަރ ޝެޑިއުލްޑް ސްނެޕްޝޮޓް ބޭނުންކުރުން)

## ކޮންޓަބޯ ޕާޗޭސް ސަރވަރ އެވެ

contabo އަކީ ޖަރުމަނުގައި ދިގު ތާރީހެއް (20 އަހަރަށް ވުރެ ގިނަ) އަދި ގިނަ ޔޫޒަރުން (200،000 އަށް ވުރެ ގިނަ) ލިބިފައިވާ އަގުހެޔޮ ވީޕީއެސް ޕްރޮވައިޑަރެއް ކަމުން ފިލައިގެން ދިއުމާ މެދު ކަންބޮޑުވާކަށް ނުޖެހެ އެވެ.

VPS [contabo.com](//contabo.com) ޔޫރޯ އަގު ހޮވުން ހަނދާން ކުރާށެވެ، އެއީ އަގުހެޔޮ އެއްޗެކެވެ.

![](https://i-01.eu.org/1718285782.avif)

ކޮންޓަބޯ އަށް އެއްފަހަރު އިންސްޓޯލް ފީއެއް ނަގަ އެވެ.
ނަމަވެސް ކޮންމެ މަހަކު ވެސް އިންސްޓޯލްމަންޓް ފީ ހިލޭ ލިބޭ މަދު ދުވަސްކޮޅެއް ވެސް ރަޖިސްޓްރީކޮށް ޕްރޮމޯޝަނަލް އީމެއިލްތަކަށް ސަމާލުކަން ދޭށެވެ.

ޔޫރަޕްގައި ހިލޭ އޭރިއާ ފީގެ އިތުރުން އެހެން ސަރަހައްދުތަކުގައި ވެސް ތަފާތު ފެންވަރުގެ އަގު ބޮޑުވެ އެވެ.

އަހަރެން ޔޫރަޕްގެ ސަރވަރެއް ހޮވީ ކޮންމެ ގޮތެއްވިޔަސް ކްލައުޑްފްލޭއާ ބޭނުންކުރުމަށްފަހު ސަރވަރތަކުގެ މެދުގައި މާބޮޑު ތަފާތެއް ނެތެވެ.

ފައިނަލް ކޮންފިގްރޭޝަން ތިރީގައި މިވަނީ، 16ޖީބީ މެމޮރީ، 6ސީޕީޔޫ، 200 ޖީބީ އެންވީއެމްއީ، މަހަކު 9.50 ޔޫރޯ، 74.43 އާރުއެމްބީއާ އެއްވަރެވެ.

![](https://i-01.eu.org/1718356956.avif)

އަޅުގަނޑު [Yet-Another-Bench-Script](https://github.com/masonr/yet-another-bench-script) ޕާފޯމަންސް އަދި ޕެރެމިޓަރސް ހިންގުމަށް:

```
curl -sL yabs.sh | bash -s -- -s https://www.vpsbenchmarks.com/yabs/upload
```

ދައްކަނީ މޮޑެލް `AMD EPYC 9224 24-Core Processor` , ސިންގަލް ކޯރ ޕާފޯމަންސް ސްކޯރ 1285 CPU
, މަލްޓި ކޯރ ސްކޯރ 4277 އިތުރު ނަތީޖާތައް ބައްލަވާ [vpsbenchmarks.com](https://www.vpsbenchmarks.com/yabs/contabo-6c-16gb-3fb7d8)

އަޅުގަނޑު އެއްފަހަރާ ތިން ސަރވަރ ގަނެގެން އެއްފަހަރާ އެކްޓިވޭޓް ކުރިން މިކަމުގެ ސަބަބުން ސަރވަރތައް އެއް ކޮމްޕިއުޓަރ ރޫމެއްގައި ހުރިކަން ކަށަވަރުވެ، ކުރިއަށް އޮތްތަނުގައި ހައި އެވޭލަބިލިޓީ ބިލްޑް ކުރުމަށް ފަސޭހަވެގެންދެއެވެ.

އޭގެ ސާވަރުތައް ދޭތެރެ ދޭތެރެއިން ފޯނު ކަނޑާލާ، ހައި އެވޭލަބިލިޓީ މެދުވެރިކޮށް ވެބްސައިޓަށް އެކްސެސިބިލިޓީ ކަށަވަރުކޮށްދެ އެވެ.

އަލަށް ބޭނުންކުރާ ފަރާތްތަކުން ގަންނައިރު ރިއަލް ނޭމް އޮތެންޓިކޭޝަން އަށް އައިޑީ ކާޑު ހުށަހަޅަން ޖެހެ އެވެ.

## ޕާސްވޯޑް ފްރީ ލޮގިން ކޮންފިގްރޭޓް ކުރުން

ދެތިން މިނެޓެއް ފަހުން ސާވަރތައް ހުރިހާ އެއްޗެއް އެކްޓިވޭޓް ވެގެން ދިޔައެވެ.

ފުރަތަމަ IP ލޯކަލް މެޝިންގެ `~/.ssh/config` އެޑިޓްކޮށްގެން އެލިއާސްއެއް ސެޓްކޮށް، ދެން `ssh-copy-id root@c2` ބޭނުންކޮށްގެން ޕާސްވޯޑް ފްރީ ލޮގިން ކޮންފިގްރޭޓް ކުރުމެވެ.

![](https://i-01.eu.org/1718287198.avif)

މި [ssh-copy-id.sh](https://github.com/i18n-ops/ubuntu/blob/main/ssh-copy-id.sh) ބަދަލުކޮށްގެން އިނިޝިއަލައިޒޭޝަން ކޮންފިގްރޭޝަން އަވަހަށް ފުރިހަމަ ކުރެވިދާނެއެވެ.

ޕާސްވޯޑް ފްރީ ކޮންފިގްރޭޝަން ނިމުމުން ބޭނުން ކުރެވޭނެ [pdsh](https://github.com/chaos/pdsh)

ފުރަތަމަ ކަމަކީ މެނުއަލްކޮށް ipv6 އެނެބަލްކޮށް ޓެސްޓް ކުރުމެވެ

```bash
pdsh -w 'c0 c1 c2' 'bash -i -c enable_ipv6'
pdsh -w 'c0 c1 c2' 'curl -s 6.ipw.cn;echo'
```

![](https://i-01.eu.org/1718287969.avif)

ދެން އެއް ކްލިކުން ސިސްޓަމް އުބުންޓޫ 24 އަށް އަޕްގްރޭޑްކޮށް އާންމުކޮށް ބޭނުންކުރާ ސޮފްޓްވެއާ އިންސްޓޯލް ކުރާށެވެ.

```bash
pdsh -w 'c0 c1 c2' 'curl -s https://raw.githubusercontent.com/i18n-ops/ubuntu/main/boot.sh | CN=1 bash'
```

CN=1 ގެ މާނައަކީ ޗައިނީސް ބަހަށް އަދި ޓައިމް ޒޯން ބެއިޖިން އަށް ބަދަލުވުމެވެ.

ޑިފޯލްޓްކޮށް އިންސްޓޯލް ކުރެވިފައިވާ ސޮފްޓްވެއަރ ކޮންފިގްރޭޓް ކުރެވިފައިވަނީ އަޅުގަނޑުގެ [i18n-ops/ubuntu](https://github.com/i18n-ops/ubuntu) ބޭނުންކުރުމުގެ އާދަތަކަށް ބަލައިގެން، [direnv](https://direnv.net/) [mise](https://github.com/jdx/mise)

މި ފިޔަވަޅަކީ ގިނަ ވަގުތު ނަގާ ފިޔަވަޅަށް ބައިގަޑިއިރު މަޑުކުރުމަށްފަހު އާންމުކޮށް ބޭނުންކުރާ ސޮފްޓްވެއަރ އަޕްގްރޭޑްކޮށް އިންސްޓޯލް ކުރުން ކާމިޔާބުވާނެއެވެ. މިކަން ކަށަވަރު ކުރެވޭނީ ތިރީގައިވާ ކޮމާންޑުންނެވެ.

```bash
pdsh -w 'c0 c1 c2' 'cat /etc/issue'
```

## މޮޝް ސްޕީޑްސް އަޕް ސްޝް

ޔޫރަޕުން ޗައިނާއަށް ދާ ނެޓްވޯކް ސްޓޭބަލް [mosh](//mosh.org) ސްޕީޑް `ssh` އަށް ބޭނުން ކުރުން ރަނގަޅެވެ.

ލޯކަލް ޑައިރެކްޓަރީ `~/.bin` (މައިގަނޑު ގޮތެއްގައި އިތުރުކުރެވިފައިވަނީ އެންވަޔަރަންމަންޓް ވެރިއޭބަލް `PATH` އަށް) ގައި ތިރީގައިވާ ގޮތަށް `c0` ސްކްރިޕްޓް އުފެއްދުން:

```bash
#!/usr/bin/env bash

if [ -z "$1" ]; then
arg=/root/.tmux_default
else
arg=${@:1:$#}
fi

vps=$(basename $0)
set -ex
mosh $vps -p 60000:60009 $arg
```

އުފެއްދުމަށްފަހު އެގްޒެކެޓެބަލް ހުއްދަތަކާއި ސޮފްޓް-ލިންކް `c1` އަދި `c2` ދިނުމެވެ.

```bash
chmod +x c0
ln -s c0 c1
ln -s c0 c2
```

ކުރިއަށް އޮތްތަނުގައި ސީދާ `c0` އަދި `c1` ޖަހައިގެން `mosh` ހައިސްޕީޑް ޑައިރެކްޓް ކަނެކްޝަން ބޭނުންކޮށް [tmux](https://www.ruanyifeng.com/blog/2019/10/tmux.html)

## ޕާރޓީޝަން ފޯމެޓް ext4 އިން btrfs އަށް ބަދަލުކުރުން

[btrfs](https://wikipedia.org/wiki/Btrfs)

ފުރަތަމަ ރެސްކިއު ސިސްޓަމަށް ވަދެވޭނެއެވެ

![](https://i-01.eu.org/1718300448.avif)

ޑެބިއަން ރެސްކިއު ހޮވާށެވެ

![](https://i-01.eu.org/1718300753.avif)

ސަރވަރ އަލުން ސްޓާޓް ވާންދެން މަޑުކޮށްލާށެވެ

ssh `ssh-keygen -R ip`

ދެން ޕާރޓީޝަން ބަދަލުކުރުމަށްޓަކައި ތިރީގައިވާ ސްކްރިޕްޓް ދުއްވާލާށެވެ

```bash
nohup bash -c "bash <(curl -s https://raw.githubusercontent.com/i18n-ops/ops/main/contabo/ext4_btrfs.sh) >/tmp/nohup.out 2>&1 && reboot" &
sleep 1 && tail -f /tmp/nohup.out
```

ބަދަލުކުރުން ކާމިޔާބުވުމުން ސްކްރިޕްޓް އޮޓޮމެޓިކުން ސިސްޓަމް އަލުން ސްޓާޓް ކުރާނެއެވެ.

### btrfs ކޮމްޕްރެޝަން ރޭޝިއޯ ބަލާށެވެ

އެއީ ޑިފޯލްޓްކޮށް އޮޓޮމެޓިކުން ކޮމްޕްރެސް ކުރާނެ އެއްޗެކެވެ.

```bash
if ! command -v compsize &>/dev/null; then
apt install -y  btrfs-compsize
fi
compsize -x /
```

### މެނުއަލް ކޮމްޕްރެޝަން

ޕާފެކްޝަނިސްޓެއްނަމަ ފުލް ޑިސްކް ކޮމްޕްރެޝަން މެނުއަލްކޮށް ވެސް ކުރެވިދާނެ އެވެ.

```bash
btrfs filesystem defragment -r -czstd /
```

ތިރީގައިވާ ފޮޓޯއިން ފެންނަ ގޮތުގައި އާ ސިސްޓަމްގައި މެނުއަލް ކޮމްޕްރެޝަނަށްފަހު ގާތްގަނޑަކަށް 100އެމްބީ އިތުރަށް ކޮމްޕްރެސް ކުރެވުނުކަން ފާހަގަކުރެވެއެވެ.

![](https://i-01.eu.org/2023/12/8NWlr4c.webp)

### ސްނެޕަރ އާއި އެކު Btrfs ސްނެޕްޝޮޓް ބެކަޕް އެއް އުފެއްދުން

Btrfs ބެކަޕް އަށް އެންމެ ރަނގަޅު ޕާޓްނަރ [snapper](https://manual.siduction.org/sys-admin-btrfs-snapper_en.html) .

އިނިޝިއަލައިޒޭޝަން ސްކްރިޕްޓް އިން `~/ubuntu` ޑައިރެކްޓަރީ އުފައްދާނެ، ހަމައެކަނި ތިރީގައިވާ ކޮމާންޑް ހިންގާށެވެ.

```
~/ubuntu/snapper/init.sh
```

ސްކްރިޕްޓް ކޮންޓެންޓްގައި ބައްދަލުވާނެ [github.com/i18n-ops/ubuntu/blob/main/snapper/init.sh](https://github.com/i18n-ops/ubuntu/blob/main/snapper/init.sh)

ސްނެޕްޝޮޓްތައް ސެޓްކޮށްފައިވާ ސަބްވޮލިއުމްތައް ބަލާލުމަށް `snapper list-configs` ބޭނުންކުރާށެވެ.

![](https://i-01.eu.org/1718302689.avif)

ސަބްވޮލިއުމްގެ ސްނެޕްޝޮޓެއް ބަލާލުމަށް `snapper -c mnt list` ބޭނުން ކުރާށެވެ

`/etc/snapper/configs/mnt` އަށް މި ސަބްވޮލިއުމްގެ ވަކި ބެކަޕް ސެޓިންގސްތައް ބަލާލެވޭނެއެވެ

އިތުރު ރިފަރެންސްތައް:

[ފަސޭހައިން Btrfs ގެ ސްނެޕްޝޮޓް ފަންކްޝަން އާއި ކުޅެވޭނެއެވެ snapper](https://zhuanlan.zhihu.com/p/31082518)

އާންމުކޮށް ބީޓީއާރުއެފްއެސް ސްނެޕްޝޮޓް ބޭނުންކުރުމަކީ މާ ފަސޭހަ ކަމެކެވެ.

## ރެސްކިއު ސިސްޓަމްގެ ތަފްސީލް

ރެސްކިއު ސިސްޓަމަށް ވަނުން 22 [tcping](https://github.com/paradiseduo/tcping) އެކްސެސިބަލް ވުމަށްފަހު 2-3 މިނިޓް ވަންދެން ސެޓް ޕާސްވޯޑާއެކު އެކްސެސް ނުކުރެވޭނެއެވެ.

```
tcping ip 22 -c 9999
```

ބަނދަރަށް ވަދެވޭނެ ގޮތް މޮނިޓަރ ކުރެވޭނެއެވެ.

### ރެސްކިއު އަށް ވަނުމަށްފަހު ހާޑް ޑްރައިވް އަށް ވަދެވޭނެ ގޮތް ރިސްޓޯރ ސްނެޕްޝޮޓް &

```
mkdir -p /mnt/os
mount  /dev/sda3 /mnt/os -t btrfs -o "defaults,ssd,discard,noatime,compress=zstd:3,space_cache=v2"
mount --bind /dev /mnt/os/dev
mount --bind /proc /mnt/os/proc
mount --bind /sys /mnt/os/sys
mount --bind /run /mnt/os/run
chroot /mnt/os zsh -c "mount -a && dbus-daemon --system 2>/dev/null; exec zsh"
```

މިސާލަކަށް ސްނެޕްޝޮޓް ބަލާލެވޭނެއެވެ

```
snapper -c etc list
```

ސްނެޕްޝޮޓް ފައިލްއަށް އަންނަ ބަދަލުތައް ބަލާށެވެ

```
snapper -c etc status 5..6
```

ވަކި ޑިފް ޓެކްސްޓެއް ބަލާށެވެ

```
snapper -c etc diff 3..5
```

ސްނެޕްޝޮޓް ރިސްޓޯރކުރުން: މިސާލަކަށް ސްނެޕްޝޮޓް 5 އަށްފަހު ހުރިހާ ބަދަލުތަކެއް އަންޑޯކުރުން

```
snapper -c etc undochange 1..0
snapper -c disk undochange 1..0
```

### ލޮގް ޕަރސިސްޓެންސް އެނެބަލް ކުރުން

ރެސްކިއު ސިސްޓަމެއްގެ ދަށުން ލޮގް ޕަރސިސްޓެންސް އެނެބަލް ކުރުމަށްޓަކައި މެނުއަލްކޮށް އެއަށް ގުޅޭ ޑައިރެކްޓަރީ އުފައްދައި އިތުރު ބައެއް ފިޔަވަޅުތައް އަޅަން ޖެހެއެވެ. މިއީ ތިބާއަށް ޓްރައި ކުރެވިދާނެ ފިޔަވަޅުތަކެކެވެ.

ފުރަތަމަ ކްރޫޓް، ދެން

```
mkdir -p /var/log/journal
chown root:systemd-journal /var/log/journal
chmod 2755 /var/log/journal
sed -i 's/^#\?\s*Storage=.*/Storage=persistent/' /etc/systemd/journald.conf
```

އުބުންޓޫ ސިސްޓަމަށް ރީބޫޓްކުރުން: މަތީގައިވާ ބަދަލުތައް ގެނައުމަށްފަހު، ރެސްކިއު ސިސްޓަމްގެ ބަދަލުގައި އާންމުކޮށް މެޝިން އަދި އުބުންޓޫ ސިސްޓަމަށް ރީބޫޓް ކުރަން ޖެހެއެވެ.

މިގޮތަށް ސިސްޓަމްޑް-ޖާނަލްޑް އާ ކޮންފިގްރޭޝަން އިން ފެށިގެން ހާޑް ޑިސްކްގައި ލޮގްތައް ޕަރސިސްޓެންޓްކޮށް ރައްކާކުރަން ފަށާނެއެވެ.

މަތީގައިވާ އޮޕަރޭޝަންތައް ކުރިއަށް ގެންދަނީ ރެސްކިއު ސިސްޓަމެއްގައިކަން ހަނދާނުގައި ބަހައްޓަންވީ، އެހެންކަމުން އެ ބަދަލުތަކަށް އަމަލު ނުކުރާނީ ސިސްޓަމް ރީބޫޓްވެ، ރެސްކިއު މޯޑުގައި ނުހުރެއެވެ.

މާނައަކީ މިހާރު ހުރި ލޮގްތަކަށް ޕަރސިސްޓެންޓް ވެގެންދާ އެއްވެސް ބަދަލެއް ވަގުތުން ނުފެންނާނެ ކަމަށާއި، ނަމަވެސް އޭގެ ފަހުން އުފައްދާ ހުރިހާ ލޮގްތަކެއް ޕަރސިސްޓެންޓް ވާނެ ކަމަށެވެ.

މިގޮތަށް ssh ސާވިސްގެ ލޮގްތައް ބަލާލެވޭނެއެވެ

```
journalctl --directory=/var/log/journal  _SYSTEMD_UNIT=ssh.service
```

## އުނދަގޫ އަދި މުއްސަނދި ބަލިތަކާ ބެހޭ ނޯޓްސް

### އާދަޔާ ހިލާފު މައްސަލަ ހައްލުކުރުން: ސަރވަރަށް ssh ނުކުރެވޭނެއެވެ

އަޅުގަނޑު ކުރިން އޮޓޮމެޓިކް ކޮމްޕައިލޭޝަން އަދި ޑިޕްލޮއިމަންޓް ސްކްރިޕްޓެއް ހެދުމަށް ގިތަބް އެކްޝަން ބޭނުންކުރި ނަމަވެސް ޑިޕްލޮއިމަންޓް ސާވިސް އަކީ ssh އަކަށް ނުވެވޭނެކަން އެނގުނެވެ.

ރެސްކިއު ސިސްޓަމަށް ވަދެ ލޮގްތައް ބަލާލުމަކީ ހުއްދަގެ މައްސަލައެއް:

`Missing privilege separation directory: /run/sshd`

މި ޑައިރެކްޓަރީ ތިރީގައިވާ ކޮމާންޑުން އުފެއްދިދާނެއެވެ

```
chown root:root /
systemd-tmpfiles --create
```

ތަކުރާރުކޮށް ތަހުގީގު ކުރުމަށްފަހު އެއީ tar ގެ ޔޫޒަރ `uid` އަކީ `1001` ކަމަށް github action

tar `--no-same-owner` `uid` `1001`

ނަތީޖާއެއްގެ ގޮތުން rsync އިން ރޫޓް ޑައިރެކްޓަރީގެ ޔޫޒަރ ގްރޫޕް ބަދަލުވެ، ދެން ރީސްޓާޓް ކުރުމަށްފަހު ssh އަށް ލޮގްއިން ނުކުރެވޭނެއެވެ.

ޑިކޮމްޕްރެޝަން ސްކްރިޕްޓް އަންނަނިވި ގޮތަށް ބަދަލުކުރުން.

```bash
tar --no-same-owner -xvf $TZT
rm $TZT

if [ -f "init.sh" ]; then
  ./init.sh
  rm init.sh
fi

rsync --remove-source-files -av . /
```
