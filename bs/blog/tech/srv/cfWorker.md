
## Ime Domene Vezano Za CNAME

Ime domene Cloudflare Worker `SSL/TLS` → Prilagođeno povezivanje imena hosta CNAME Nakon imena domene, direktan pristup će prijaviti grešku `error code: 522`

U ovom trenutku morate koristiti HTTP rutiranje → Rutiranje radnika da povežete rutu, kao što je prikazano ispod
