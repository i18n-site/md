
## CNAME Бәйләнгән Домен Исеме

Домен исеме `SSL/TLS` → махсус хост исеме белән бәйләнгән CNAME Домен исеме бәйләнгәннән соң, туры керү хата турында хәбәр итә `error code: 522` Cloudflare Worker

Бу вакытта сезгә HTTP маршрутлау → Маршрутны бәйләү өчен эшчеләр маршрутын кулланырга кирәк, аста күрсәтелгәнчә
