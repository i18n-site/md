# Команда Сызыгы Параметрларын Җентекләп Аңлату

## Чиста Файлларны `-P`

`-p` `--purge` һәр тәрҗемә каталогында булган, ләкин чыганак тел каталогында булмаган файлларны чистартачак.

Чөнки документлар язганда, Маркдаун файл исемнәре еш көйләнә, нәтиҗәдә тәрҗемә каталогындагы күп иске һәм ташланган файллар барлыкка килә.

Башка тел каталогларында бетерелергә тиешле файлларны чистарту өчен бу параметрны кулланыгыз.

## Тәрҗемә Каталогын Күрсәтегез `-D`

Тәрҗемә ителгән каталог хәзерге файл урнашкан каталогка килешенми.

`-d` `--workdir` тәрҗемә каталогын күрсәтә ала, мәсәлән:

```
i18 -d ~/i18n/md
```

## Ярдәмне Карау `-H`

`-h` `--help` командование ярдәмен карый аласыз.
