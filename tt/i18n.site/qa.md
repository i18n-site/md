# Гомуми Проблема

## Көтмәгәндә Бетерелде, `Npm` Пакет Бастырылмады `.i18n/V`

`npm` Пакетның тарихи версиясе сакланган `.i18n/v/ol/v.hash`

Әгәр сез очраклы рәвештә бетерсәгез `.i18n/v/ol` `npm` Пакет бастырылмас.

Бу вакытта башта `0.1.9` проектның соңгы чыгарылыш версиясенең версия номерын табыгыз [npmjs.com](//npmjs.com)

Аннары түбәндәге файл ясагыз `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Игътибар итегез, бу рәвешле ремонтлау файл тарихын югалтачак, киләсе чыгарылышны яңартып булмый, һәм барлык эчтәлек бер тапкыр йөкләнәчәк һәм йөкләнәчәк.
