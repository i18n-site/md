Хәзерге вакытта ул бар, ләкин [үсеш](/i18n.site/c/roadmap) вакытында мөһим үзенчәлекләр бар.

## Кереш Сүз

Вебсайтны үстерү моделе турында минем яңадан уйлануым `i18n.site`

Бу документ сайты генераторы һәм вебсайтны үстерү нигезе.

Вебсайтны үстерүнең яңа парадигмасы, ул тирә-якка тупланган һәм интерактив инъекция өчен фронтовик компонентларны куллана `MarkDown`

Eachәрбер фронт компоненты - мөстәкыйль урнаштырыла торган пакет.

Алгы һәм арткы очны аеру нигезендә статик эчтәлекне һәм динамик мәгълүматны аеру бар.

Сез барган [i18n.site](/) бу база нигезендә төзелгән (кулланучылар системасы, исәп-хисап системасы, электрон почтага язылу һ.б.).
