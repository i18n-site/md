* _uavhengige komponenter_ [github](https://github.com/i18n-site/18x) / [atomgit](https://atomgit.com/i18n/18x)
* Svelte-komponenter [github](https://github.com/i18n-site/plugin) / [atomgit](https://atomgit.com/i18n/plugin)

* Generelle grunnbiblioteker [github](https://github.com/i18n-site/lib) / [atomgit](https://atomgit.com/i18n/lib)
* Nettlesergrunnbiblioteker [github](https://github.com/i18n-site/ie) / [atomgit](https://atomgit.com/i18n/ie)
* Frontend-biblioteker som kan refereres direkte uten pakking [github](https://github.com/i18n-site/x) / [atomgit](https://atomgit.com/i18n/x)
  Praktisk for direkte referanse i markdown-initialiseringsskriptet, og samtidig bruk på den pakkede nettsiden, for å unngå dupliserende kode.

## Bakende

Dokumentasjonssiden krever kun frontend, ingen bakende.

Denne bakenden er for i18n.site-nettstedet selv, og inkluderer brukere, betaling, meldingsutsendelser og andre systemer.

* Tjenester [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
* Programtillegg [github](https://github.com/i18n-api/pub) / [atomgit](https://atomgit.com/i18n-api/pub)
* Grunnbiblioteker [github](https://github.com/i18n-site/rust) / [atomgit](https://atomgit.com/i18n/rust)
