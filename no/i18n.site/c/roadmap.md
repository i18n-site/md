# Utviklingsplan

Følgende viktige funksjoner er under utvikling:

- [ ] Dokumentversjonsbytte
- [ ] Pure front-end fulltekst-søk
- [ ] RSS-abonnement
- [ ] 搜索引擎优化
- [ ] Synkronisert pushing av artikkeloppdateringer til større sosiale medier
- [ ] 用户可用邮件 /`web push`订阅产品更新
- [ ] Kommentarer til artikkelavsnitt
- [ ] Integrasjon av lokal dokumentovervåking for å generere forhåndsvisninger uten behov for manuell kompilering og bruk av NGINX
