# Utviklingsplanlegging

Følgende viktige funksjoner er under utvikling:

- [ ] Bytte mellom flere versjoner av dokumenter
- [ ] Rent front-end fulltekstsøk
- [ ] RSS-feed
- [ ] Søkemotoroptimalisering
- [ ] Push artikkeloppdateringer til store sosiale medier samtidig
- [ ] Brukere kan abonnere på produktoppdateringer `web push` e-post /
- [ ] Kommentar til artikkelavsnitt
- [ ] Integrer lokale dokumenter for å overvåke endringer og generere forhåndsvisninger, eliminere behovet for manuell kompilering hver gang og bruke NGINX
