# Detalyadong Pagpatin-Aw Sa Mga Parameter Sa Command Line

## Limpyo Nga Mga `-P`

`-p` O `--purge` maglimpyo sa mga file nga anaa sa matag translation directory apan wala sa source language directory.

Tungod kay kung nagsulat sa mga dokumento, ang mga ngalan sa file sa Markdown kanunay nga gi-adjust, nga nagresulta sa daghang mga daan ug gibiyaan nga mga file sa direktoryo sa paghubad.

Gamita kini nga parametro aron limpyohan ang mga file nga kinahanglang tangtangon sa ubang mga direktoryo sa pinulongan.

## `-D`

Ang gihubad nga direktoryo nag-default sa direktoryo diin ang kasamtangan nga file nahimutang.

`-d` O `--workdir` mahimong magtakda sa direktoryo sa paghubad, sama sa:

```
i18 -d ~/i18n/md
```

## Tan `-H` Awa Ang Tabang

`-h` `--help` nimong tan-awon ang tabang sa linya sa mando.
