# Долен колонтитул

Въпреки това, вземайки за пример демонстрационния проект, в директорията `md` файлът `.i18n/htm/foot.pug` дефинира долната част на страницата на уебсайта.

![](https://p.3ti.site/1721286077.avif)

[`pug`](https://pugjs.org) е шаблонен език за генериране на `HTML`.

[➔ Кликнете тук, за да научите синтаксиса на pug](https://pugjs.org)

**Не пишете `css` и `js` в `foot.pug`**,否则 ще има грешка.

Моля, спрете се на следващото, стилът трябва да бъде записан в съответния `.css`, а взаимодействието се осъществява чрез създаване на уеб компоненти.

## Стил на долния колонтитул

В демонстрационния проект под `md/.i18n/htm` има три `css` файла.

* `foot.css`: стил на долния колонтитул
* `import.css`: стандартен стил за целия уебсайт `i18n.site`
* `conf.css`: икони и шрифтове за долния колонтитул

### Иконен шрифт

Иконите в долния колонтитул се генерират чрез създаване на шрифт с име `F` на iconfont.cn ([английска версия](https://www.iconfont.cn/?lang=en-us) / [китайска версия](https://www.iconfont.cn/?lang=zh)).

Моля, създайте свой собствен иконен шрифт, ако е необходимо, и заменете следната конфигурация в `conf.css`:

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
}
```

## Уеб компоненти

Не пишете `js` в `foot.pug`; ако е необходимо взаимодействие, създайте персонализиран уеб компонент.

Можете да дефинирате [уеб компонент](https://www.freecodecamp.org/news/build-your-first-web-component/) в `md/.i18n/htm/index.js`, след което да го използвате в `foot.pug`.

Създаването на уеб компоненти е лесно, например персонализиран етикет `<x-img>`.

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```

## Многоезичен долен колонтитул

Кодът в `.i18n/htm/foot.pug` е следния:

```
#Ft
  b
    a.site(href="/")
    b ${I18N.C}
```

Тук `${I18N.C}` съответства на `en/i18n.yml`:

```
C: Power By <a class="a" href="https://i18n.site">i18n.site</a>
```

Използвайки `${I18N.xxx}` подобен начин на писане, в комбинация с `i18n.yml`, можете да осъществите многоезична интернационализация на долния колонтитул.

Добавянето на `class="a"` към връзката е за да се предотврати преобразуването на връзката в `Markdown`. Вижте:
 [➔ `YAML`: Как да предотвратим преобразуването на `HTML` връзка в `Markdown`](/i18/qa#H2).
