# Долен Колонтитул

Като вземем за пример демонстрационния проект, долният колонтитул на уебсайта `.i18n/htm/foot.pug` дефиниран в директорията `md`

<img alt="" src="https://p.3ti.site/1721286077.avif">

[`pug`](https://pugjs.org) е шаблонен език, който генерира `HTML` .

[➔ Щракнете тук, за да научите граматиката на pug](https://pugjs.org)

**Не пишете `css` и `js` `foot.pug`** , в противен случай ще има грешки.

Моля, вижте следното, стилът е написан в съответния `.css` а взаимодействието се постига чрез създаване на уеб компоненти.

## Стил На Долния Колонтитул

В демонстрационния `md/.i18n/htm` има 3 `css` файла по-долу.

* `foot.css` Стил : долния колонтитул
* `import.css` Стил `i18n.site` : целия сайт
* Икони и шрифтове в : `conf.css`

### Шрифт На Икона

Иконата на долния колонтитул се генерира от iconfont.cn ( [английска версия](https://www.iconfont.cn/?lang=en-us) /[中文版](https://www.iconfont.cn/?lang=zh)) създаване на шрифт с име `F`

Моля, създайте свой собствен шрифт на иконата, ако е необходимо, и заменете : конфигурация в `conf.css`

```
@font-face {
  font-family: "F";
  src: url("//at.alicdn.com/t/c/font_4281991_urfar2m2zce.woff2?t=1716188208767") format("woff2");
}
```

## Уеб Компоненти

`foot.pug` не може да пише `js` в него, моля, персонализирайте компонента на уеб страницата.

Можете да дефинирате [уеб компонент](https://www.freecodecamp.org/news/build-your-first-web-component/) в `md/.i18n/htm/index.js` и след това да използвате компонента в `foot.pug` .

Лесно е да създавате уеб компоненти, като потребителски етикети `<x-img>`

```js
customElements.define(
  'x-img',
  class extends HTMLElement {
    constructor() {
      super();
      var img = document.createElement('img');
      img.src = '//p.3ti.site/i18n.svg';
      img.style = "height:99px;width:99px;";
      this.append(img);
    }
  }
)
```

## Многоезичен Долен Колонтитул

`.i18n/htm/foot.pug` Кодът в него е както следва :

```
#Ft
  b
    a.site(href="/")
    b ${I18N.C}
```

: съответния е в `en/i18n.yml` `${I18N.C}`

```
C: Power By <a class="a" href="https://i18n.site">i18n.site</a>
```

Използвайки `${I18N.xxx}` подобно на този метод на писане, в комбинация с `i18n.yml` , можете да постигнете многоезична интернационализация на долния колонтитул.

Добавете `class="a"` към връзката, за да избегнете преобразуването й в `MarkDown` , вижте :
 [: `YAML` да предотвратим преобразуването на `HTML` на връзката в `Markdown`](/i18/qa#H2) .
