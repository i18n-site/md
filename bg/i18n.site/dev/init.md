# Подгответе околната среда

* Инсталирайте и стартирайте `docker` (за потребителите на Mac се препоръчва да използват [orbstack](https://orbstack.dev) като RuntimeObject за `docker`)

* Инсталирайте [bun](https://bun.sh/docs/installation) и [nodejs](https://nodejs.org/en/download/package-manager)

## Инициализирайте проекта

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
