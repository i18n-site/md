
## CNAME پابند ڈومین نام

کا ڈومین نام `SSL/TLS` → کسٹم ہوسٹ نام کے ذریعے پابند ہے CNAME ڈومین نام کے پابند ہونے کے بعد، براہ راست رسائی غلطی `error code: 522` کی اطلاع دے گی Cloudflare Worker

اس وقت، آپ کو روٹ کو باندھنے کے لیے HTTP روٹنگ → ورکرز روٹنگ استعمال کرنے کی ضرورت ہے، جیسا کہ ذیل میں دکھایا گیا ہے۔
