# ترقیاتی منصوبہ بندی

مندرجہ ذیل اہم خصوصیات ترقی کے تحت ہیں:

- [ ] دستاویزات کے متعدد ورژن کے درمیان سوئچنگ
- [ ] خالص فرنٹ اینڈ فل ٹیکسٹ سرچ
- [ ] آر ایس ایس فیڈ
- [ ] تلاش کے انجن کی اصلاح
- [ ] اس کے ساتھ ساتھ بڑے سوشل میڈیا پر آرٹیکل اپ ڈیٹس کو آگے بڑھائیں۔
- [ ] صارفین کے لیے ای میل دستیاب ہے `web push` پروڈکٹ اپ ڈیٹس کو سبسکرائب کریں /
- [ ] مضمون کے پیراگراف پر تبصرہ کریں۔
- [ ] تبدیلیوں کی نگرانی اور پیش نظارہ تیار کرنے کے لیے مقامی دستاویزات کو مربوط کریں، ہر بار دستی تالیف کی ضرورت کو ختم کرتے ہوئے اور NGINX استعمال کریں۔
