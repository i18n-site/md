
## CNAME बाउंड डोमेन नाम

डोमेन नाम पारित Cloudflare Worker → अनुकूलित मेजबान नाम बाइंडिंग CNAME डोमेन नामक बाद, सीधा पहुँच एकटा त्रुटि `error code: 522` रिपोर्ट करत `SSL/TLS`

अइ समय, अहां कें कोनों मार्ग कें बाइंड करय कें लेल HTTP रूटिंग → वर्कर्स रूटिंग कें उपयोग करय कें जरूरत छै, जैना की नीचा दिखायल गेल छै
