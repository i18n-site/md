# Förbered miljön

* Installera och starta `docker` (mac-användare rekommenderas att använda [orbstack](https://orbstack.dev) som körningstid för `docker`)

* Installera [bun](https://bun.sh/docs/installation) och [nodejs](https://nodejs.org/en/download/package-manager)

## Initiera projektet

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
