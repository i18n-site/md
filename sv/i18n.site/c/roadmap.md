# Utvecklingsplanering

Följande viktiga funktioner är under utveckling:

- [ ] Växla mellan flera versioner av dokument
- [ ] Ren front-end fulltextsökning
- [ ] RSS-flöde
- [ ] sökmotoroptimering
- [ ] Skicka samtidigt artikeluppdateringar till stora sociala medier
- [ ] Användare kan prenumerera på produktuppdateringar `web push` e-post /
- [ ] Kommentera artikelstycken
- [ ] Integrera lokala dokument för att övervaka ändringar och generera förhandsvisningar, eliminera behovet av manuell kompilering varje gång och använda NGINX
