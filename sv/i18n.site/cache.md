# Front-end Cache

## Front-end Cacheuppdateringsfördröjning

Enligt [MDN](https://developer.mozilla.org/docs/Web/API/Service_Worker_API) `service worker` den maximala uppdateringsfrekvensen 24 timmar.

i18n.site Cachelagra CDN i `service worker` Så om `cdn` ändras i konfigurationen måste gamla användare vänta 24 timmar för att se uppdateringen.
