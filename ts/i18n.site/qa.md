# Xiphiqo Lexi Tolovelekeke

## `Npm` Xihoxo Yi Susiwile `.i18n/V`

Vuhundzuluxi bya matimu bya `npm` package release byi hlayisiwile eka `.i18n/v/ol/v.hash`

Loko hi xihoxo u susa `.i18n/v/ol` `npm` phasela a ri nge kandziyisiwi.

Hi nkarhi lowu, rhanga `0.1.9` kuma nomboro ya vuhundzuluxi bya vuhundzuluxi byo hetelela bya ku humesiwa ka phurojeke ya wena eka [npmjs.com](//npmjs.com)

Kutani languta eka `bash` Endla fayili leyi nga laha hansi.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Xiya leswaku ku lunghisa hi ndlela leyi swi ta lahlekeriwa hi matimu ya fayili, leswi nga ta endla leswaku swi nga koteki ku pfuxeta nkandziyiso lowu landzelaka hakatsongo-tsongo, naswona swilo hinkwaswo swi ta pakiwa nakambe ni ku layicha kan’we.
