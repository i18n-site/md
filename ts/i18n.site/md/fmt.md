# Ntwanano Wa Xivumbeko Xa MarkDown

## Nhlokomhaka Ya Anchor

Ti-anchor `MarkDown` ndhavuko ti endliwa hi ku ya hi nhundzu ya matsalwa Eka mhaka ya tindzimi to tala, ntlhantlho lowu a wu koteki.

`i18n.site` Ntlhantlho wa anchor lowu pfumelelaniweke i ku nghenisa tsalwa leri fanaka na `<a rel=id href="#xxx" id="xxx"></a>` eka `MarkDown` ku tumbuluxa hi voko anchor yo veka.

`<a rel=id href="#xxx" id="xxx"></a>` , laha `rel=id` hlamusela xitayili xa pheji ya anchor point, hi kombela u siva `xxx` hi xifunengeto xa wena xa xiviri xa anchor English.

Hi ntolovelo ku engeteriwa ti- anchor eka xihloko, to tanihi:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Tsala `HTML` Eka `MarkDown`

`pug` `HTML` swi nga nghenisiwa eka khodi.

Swilo leswi nga endzeni ka xiaki xa `<pre>`

Loko u hlanganisa tinhla leti timbirhi, u nga tsala `HTML` in `MarkDown` ku olova ku fikelela switandzhaku swo hambana swa nkombiso.

U nga kongomisa eka ku tirhisiwa eka [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) khodi yi le ka xiyimo lexi landzelaka :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Xiya leswaku `<style>` yi tlhela yi hlamuseriwa eka `<pre>`
