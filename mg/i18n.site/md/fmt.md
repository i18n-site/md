# MarkDown Endrika Fivoriambe

## Teboka Vatofantsika

Ny vatofantsika nentim `MarkDown` paharazana dia noforonina mifototra amin'ny atiny amin'ny fiteny maro, ity vahaolana ity dia tsy azo atao.

`i18n.site` `MarkDown` `<a rel=id href="#xxx" id="xxx"></a>`

`<a rel=id href="#xxx" id="xxx"></a>` , eto `rel=id` no mamaritra ny endriky ny pejin'ny teboka vatofantsika, azafady soloy `xxx` fanafohezanao amin'ny teny anglisy.

Matetika ny vatofantsika dia ampiana amin'ny lohateny, toy ny:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Soraty `HTML` in `MarkDown`

`pug` `HTML` dia azo ampidirina amin'ny kaody.

Ny votoaty ao anatin'ny singa `<pre>`

Raha manambatra ireo teboka roa ireo ianao dia afaka manoratra mora foana `HTML` in `MarkDown`

Azonao atao ny manondro ny fampiharana amin'ny [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) ny code dia toy izao manaraka izao :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Mariho fa ny `<style>` dia voafaritra ao amin'ny `<pre>`
