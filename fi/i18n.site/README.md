## Esipuhe

`i18n.site`

Tämä on asiakirjasivustojen luontiohjelma ja verkkosivustojen kehityskehys.

Uusi verkkosivustokehityksen paradigma, joka `MarkDown` ja käyttää käyttöliittymäkomponentteja vuorovaikutteisuuden lisäämiseen.

Jokainen käyttöliittymäkomponentti on paketti, joka voidaan asentaa itsenäisesti.

Etu- ja takapään erottelun perusteella erotetaan myös staattinen sisältö ja dynaaminen data.

Vierailemasi [i18n.site](/) perustuu tähän kehykseen (mukaan lukien käyttäjäjärjestelmä, laskutusjärjestelmä, sähköpostitilaus jne.).
