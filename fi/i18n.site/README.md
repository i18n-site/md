# i18n.site: Puhdas staattinen monikielinen verkkosivustokehys

Se on tällä hetkellä alustavasti käytettävissä, mutta tärkeitä ominaisuuksia on vielä kehityksessä. Katso lisätietoja [kehityssuunnitelmasta](/i18n.site/c/roadmap).

## Esipuhe

`i18n.site` on verkkosivustokehitysmallin uudelleenarviointi.

Tämä on sekä asiakirjasivustojen luontiohjelma että verkkosivustokehys.

Uusi verkkosivustokehityksen paradigma, jonka ytimessä on `MarkDown` ja joka hyödyntää käyttöliittymäkomponentteja vuorovaikutuksen lisäämiseen.

Jokainen käyttöliittymäkomponentti on itsenäisesti asennettava paketti.

Perustuen etu- ja takapään erotteluun, myös staattinen sisältö ja dynaaminen data erotetaan toisistaan.

Vierailemasi [i18n.site](/) on rakennettu tälle kehykselle (sisältäen käyttäjäjärjestelmän, laskutusjärjestelmän, sähköpostitilauksen jne.).
