## Ñe’ẽ Ñepyrũrã

`i18n.site` ha’e che pensamiento jey pe modelo desarrollo sitio web rehegua.

Kóva haꞌehína peteĩ kuatiahaipyre tenda apoha ha peteĩ ñanduti renda ñemoheñói rembiaporã.

Peteĩ paradigma pyahu desarrollo página web rehegua oñembohapéva `MarkDown` ha oiporúva componente front-end oinyecta hag̃ua interactividad.

Káda componente front-end haꞌehína peteĩ paquete ikatúva oñemboguapy ijehegui.

Pe base de separación front-end ha back-end rehe, oĩ avei separación contenido estático ha dato dinámico rehegua.
