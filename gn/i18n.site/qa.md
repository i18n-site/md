# Problema Común Rehegua

## `.i18n/V` Oñembogue Accidentalmente , Upévare Ndojekuaaukái `Npm` Paquete

`.i18n/v/ol/v.hash` Pe versión histórica `npm` paquete ñeguenohẽ rehegua oñeñongatu !

Rembogue ramo accidentalmente `.i18n/v/ol` pe `npm` paquete noñemoherakuãmo’ãi.

Ko’áĝa `0.1.9` eheka raẽ versión número nde proyecto versión ñeguenohẽ paha rehegua [npmjs.com](//npmjs.com)

Upéi ehecha pe `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Ñañamindu’u ñamyatyrõramo péicha operdetaha vore rembiasakue, ndaikatumo’ãiha oñembopyahu ohóvo ambue ñeguenohẽ, ha opaite mba’ekuaarã oñembohyru jeýta ha ojeguerahaukáta peteĩ jey.
