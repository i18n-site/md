
## CNAME Bound Domain Name

Ang pangalan ng domain ng Cloudflare Worker ay nakatali sa `SSL/TLS` → custom na pangalan ng host CNAME ang domain name ay nakatali, direktang pag-access ay mag-uulat ng isang error `error code: 522`

Sa oras na ito, kailangan mong gumamit ng pagruruta ng HTTP → Pagruruta ng mga manggagawa upang itali ang isang ruta, tulad ng ipinapakita sa ibaba
