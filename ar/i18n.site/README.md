# i18n.site: إطار عمل موقع ويب ثابت متعدد اللغات

`i18n.site` منشئ مستندات ثابت ومتعدد اللغات.

## مقدمة

`i18n.site` هو منشئ مستندات، ويعمل كإطار عمل لتطوير مواقع الويب.

نموذج تطوير مواقع ويب جديد يركز على `MarkDown` ويستخدم مكونات واجهة المستخدم الأمامية لإدخال التفاعل.

كل مكون واجهة أمامية هو حزمة يمكن تثبيتها بشكل مستقل.

بني على فكرة الفصل بين واجهة المستخدم الأمامية والخلفية، ويقدم أيضًا فصلًا بين المحتوى الثابت والبيانات الديناميكية.

الموقع الذي تزوره [i18n.site](/) هو بناءً على هذا الإطار (ويشمل نظام المستخدم، نظام الفوترة، اشتراك البريد الإلكتروني، وما إلى ذلك).
