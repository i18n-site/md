# i18n.site: إطار عمل موقع ويب ثابت متعدد اللغات

`i18n.site` منشئ مستندات ثابت متعدد اللغات.

## مقدمة

`i18n.site` هو منشئمواقع المستندات، ويعد أيضًا إطارًا عمل لتطوير مواقع الويب.

نموذج تطوير مواقع ويب جديد يركز على `MarkDown` ويستخدم مكونات واجهة المستخدم الأمامية لإدخال التفاعل.

كل مكون واجهة أمامية هو حزمة يمكن تثبيتها بشكل مستقل.

بجانب فصل واجهة المستخدم الأمامية عن الخلفية، هناك أيضًا فصل بين المحتوى الثابت والبيانات الديناميكية.

الموقع الذي تزوره الآن [i18n.site](/) هو بناءً على هذا الإطار (ويشمل نظام المستخدم، نظام الفوترة، اشتراك البريد الإلكتروني، وما إلى ذلك).
