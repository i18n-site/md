# التخطيط التطويري

الميزات المهمة التالية قيد التطوير:

- [ ] تبديل إصدارات المستندات المتعددة
- [ ] بحث كامل النص في واجهة المستخدم الأمامية البحتة
- [ ] مصادر الاشتراك via RSS
- [ ] تحسين محركات البحث
- [ ] تزامن إرسال تحديثات المقالات إلى وسائل التواصل الاجتماعي الرئيسية
- [ ] إمكانية الاشتراك في تحديثات المنتج عبر البريد الإلكتروني / `web push`
- [ ] إضافة تعليقات على فقرات المقال
- [ ] التكامل مع مراقبة المستندات المحلية لإنشاء معاينات التغييرات دون الحاجة إلى تجميع يدوي باستخدام NGINX
