# Cov Lus Piav Qhia Ntxaws Txog Kab Hais Kom Ua Kab

## `-P` Cov Ntaub Ntawv

`-p` Los yog `--purge` yuav ntxuav cov ntaub ntawv uas muaj nyob hauv txhua phau ntawv txhais lus tab sis tsis muaj nyob hauv phau ntawv qhia lus.

Vim tias thaum sau cov ntaub ntawv, Markdown cov npe cov ntaub ntawv feem ntau raug kho, ua rau ntau cov ntaub ntawv qub thiab tso tseg hauv phau ntawv txhais lus.

Siv qhov ntsuas no los ntxuav cov ntaub ntawv uas yuav tsum tau muab tshem tawm hauv lwm hom lus.

## `-D` Qhia Phau Ntawv Txhais Lus

Phau ntawv txhais lus tsis raug rau cov npe uas cov ntaub ntawv tam sim no nyob.

`-d` Los yog `--workdir` tuaj yeem qhia cov npe txhais lus, xws li:

```
i18 -d ~/i18n/md
```

## `-H` Saib Kev Pab

`-h` Los yog `--help` tuaj yeem saib cov kab hais kom pab.
