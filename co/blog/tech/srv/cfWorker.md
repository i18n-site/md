
## Nome Di Duminiu CNAME Legatu

U nome di u duminiu hè `SSL/TLS` → Ubligatoriu di u nome d'ospitu persunalizatu CNAME Dopu à u nome di u duminiu Cloudflare Worker l'accessu direttu signalà un errore `error code: 522`

À questu tempu, avete bisognu di utilizà HTTP routing → Workers routing per ligà una strada, cum'è mostra quì sottu
