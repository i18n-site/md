
## Nome Di Duminiu CNAME Legatu

U nome di duminiu hè ligatu à traversu `SSL/TLS` → nome d'ospitu persunalizatu CNAME chì u nome di duminiu hè ligatu, l'accessu direttu signalà un errore `error code: 522` Cloudflare Worker

À questu tempu, avete bisognu di utilizà HTTP routing → Workers routing per ligà una strada, cum'è mostra quì sottu
