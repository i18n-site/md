## ক্ৰম

"বাইবেল·আদিপুস্তক":

> প্ৰাচীন কালত যেতিয়া ভাষা এক আছিল, তেতিয়া মানৱতা জন্ম হৈছিল গৌৰৱান্বিত। তেওঁলোকে এটা উচ্চ অট্টালিকা নিৰ্মাণ কৰিবলৈ বিচাৰিছিল, আকাশ স্পৰ্শ কৰা এটা টাৱাৰ।
>
> কিন্তু তেওঁলোকৰ অহংকাত অসন্তুষ্ট হৈ ঈশ্বৰে মানুহবোৰক পৃথিৱীত সিঁচৰতি কৰি পেলালে, যাৰ ফলত তেওঁলোকৰ জিভা ইজনে সিজনৰ বাবে বুজিব নোৱাৰা হৈ পৰিল।
>
> তেতিয়াৰ পৰাই মানৱ জাতিয়ে যোগাযোগৰ বাবে সংগ্ৰাম কৰিছিল, অন্তহীনভাৱে সংঘাতৰ সৃষ্টি হৈছিল আৰু পৃথিৱীখনে আৰু আকাশলৈ যোৱা এটা টাৱাৰ দেখা নাপালে।

কল্পবিজ্ঞান উপন্যাস &quot;তিনি-শৰীৰ&quot; (চীনা উচ্চাৰণ: `3Ti` ) বিদ্যুৎচুম্বকীয় তৰংগৰ জৰিয়তে যোগাযোগ কৰা, ভাষাৰ কোনো বাধা নথকা আৰু প্ৰযুক্তিগতভাৱে সমৃদ্ধিশালী বিদেশী সভ্যতাক কাল্পনিক কৰি তুলিছে।

আশা কৰোঁ এই আহিলাটোৰ সহায়ত পৃথিৱীৰ মানুহবোৰ তিন শৰীৰৰ মানুহৰ দৰে হ’ব, যোগাযোগ ভাষাৰ দ্বাৰা বাধাগ্ৰস্ত নহ’ব, আৰু সকলো মানৱজাতি পুনৰ একত্ৰিত হ’ব।
