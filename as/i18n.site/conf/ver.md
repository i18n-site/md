# প্ৰকল্প সংস্কৰণ

ডেমো প্ৰকল্পটোক উদাহৰণ হিচাপে লওক:

`en/demo2/v` প্ৰকল্পৰ বৰ্তমান সংস্কৰণ নম্বৰ, যি প্ৰকল্পৰ নামৰ সোঁফালে চাইডবাৰৰ ৰূপৰেখাত প্ৰদৰ্শিত হ'ব।

<img src="https://p.3ti.site/1721290486.avif" width="320px">

ইয়াত `.i18n/conf.yml` কনফিগাৰ কৰা অনুবাদ উৎস ভাষাৰ সৈতে সংগতি ৰাখি ভাষা ক'ড দিয়া হৈছে `en/`

যদি আপোনাৰ উৎস ভাষা ইংৰাজী নহয়, তেন্তে `v` 

নথিপত্ৰৰ ঐতিহাসিক সংস্কৰণ ব্ৰাউজ কৰাৰ ক্ষমতা বিকাশৰ কাম চলি আছে।

অত্যধিক সংস্কৰণ সংখ্যাৰ বাবে সন্ধান ইঞ্জিনসমূহে সূচীভুক্ত কৰা পৃষ্ঠাসমূহত অগোছালতা এৰাই `v1` ডাঙৰ আপডেইটসমূহ মুকলি কৰাৰ সময়ত কেৱল দস্তাবেজৰ সংস্কৰণ সংখ্যা পৰিবৰ্তন কৰাটো উপদেশিত `v2`

## বিভিন্ন প্ৰকল্পৰ বাবে নথিপত্ৰ সূচীসমূহ বিভাজিত কৰিবলে খালী `v`

`en/blog` `v` `en/demo1` `en/demo2/v`

`v` `v`

বিভিন্ন প্ৰকল্পৰ সূচীসমূহ বিভাজিত কৰি, আপুনি সম্পূৰ্ণ চাইটৰ সকলো নথিপত্ৰৰ সূচী একেলগে লোড কৰাৰ ফলত হোৱা লেহেমীয়া অভিগম এৰাই চলিব পাৰিব।

উদাহৰণস্বৰূপে, ডেমো প্ৰকল্পত সংশ্লিষ্ট সূচী ফাইলটো [https://unpkg.com/i18n.site/en/blog.json](https://unpkg.com/i18n.site/en/blog.json) : `blog`
