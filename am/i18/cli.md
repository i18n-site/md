# የትእዛዝ መስመር መለኪያዎች ዝርዝር ማብራሪያ

## ፋይሎችን አጽዳ `-p`

`-p` `--purge` በእያንዳንዱ የትርጉም ማውጫ ውስጥ ያሉትን ነገር ግን በምንጭ ቋንቋ ማውጫ ውስጥ የሌሉ ፋይሎችን ያጸዳል።

ምክንያቱም ሰነዶችን በሚጽፉበት ጊዜ የማርክዳው ፋይል ስሞች ብዙ ጊዜ ይስተካከላሉ፣ በዚህም ምክንያት በትርጉም ማውጫ ውስጥ ብዙ ያረጁ እና የተተዉ ፋይሎች።

በሌሎች የቋንቋ ማውጫዎች ውስጥ መሰረዝ ያለባቸውን ፋይሎች ለማፅዳት ይህን ግቤት ይጠቀሙ።

## የትርጉም ማውጫ `-d`

የተተረጎመው ማውጫ አሁን ያለው ፋይል ወደሚገኝበት ማውጫ ነባሪው ነው።

`-d` `--workdir` የትርጉም ማውጫውን መግለጽ ይችላል፣ ለምሳሌ፡-

```
i18 -d ~/i18n/md
```

## እገዛ `-h`

`-h` `--help` የትእዛዝ መስመር እገዛን ማየት ትችላላችሁ።
