# Детаљно Објашњење Параметара Командне Линије

## `-P` Фајлове

`-p` ће `--purge` очистити датотеке које постоје у сваком преводном директоријуму, али не постоје у директоријуму изворног језика.

Зато што се приликом писања докумената имена датотека Маркдовн често прилагођавају, што доводи до многих старих и напуштених датотека у директоријуму за превод.

Користите овај параметар да бисте очистили датотеке које треба избрисати у директоријумима других језика.

## `-D` Именик Превода

Преведени директоријум подразумевано подразумева директоријум у коме се налази тренутна датотека.

`-d` `--workdir` може навести директориј за превођење, као што је:!

```
i18 -d ~/i18n/md
```

## `-H`

`-h` `--help` Можете погледати помоћ командне линије.
