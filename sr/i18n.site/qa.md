# Заједнички Проблем

## Случајно Избрисан, Због Чега Пакет `Npm` Није Објављен `.i18n/V`

Историјска верзија `npm` пакета је сачувана у `.i18n/v/ol/v.hash`

Ако случајно избришете `.i18n/v/ol` `npm` пакет неће бити објављен.

У овом тренутку, прво пронађите број верзије последње верзије вашег пројекта у [npmjs.com](//npmjs.com) `0.1.9`

Затим погледајте датотеку `bash`

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Имајте на уму да ће поправка на овај начин изгубити историју датотека, што ће онемогућити постепено ажурирање следећег издања, а сав садржај ће бити поново упакован и отпремљен једном.
