# Мушкилоти Умумӣ

## Тасодуфан Нест Карда Шуд `.i18n/V` Ки `Npm` Баста Нашр Нашуд

Варианти таърихии `npm` барориши баста дар `.i18n/v/ol/v.hash`

Агар шумо тасодуфан нест кунед `.i18n/v/ol` бастаи `npm` нашр намешавад.

Дар айни замон, аввал рақами версияи версияи охирини лоиҳаи худро дар [npmjs.com](//npmjs.com) , масалан, пайдо кунед `0.1.9`

Пас ба `bash` зер нигаред.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Аҳамият диҳед, ки таъмир бо ин роҳ таърихи файлро аз даст медиҳад, ки тадриҷан навсозии версияи навбатиро ғайриимкон месозад ва ҳама мундариҷа як маротиба аз нав баста ва бор карда мешаванд.
