# Шарҳи Муфассали Параметрҳои Сатри Фармон

## Файлҳоро Тоза `-P`

Ё `--purge` файлҳоеро, ки дар ҳар як феҳристи тарҷума вуҷуд доранд, вале дар феҳристи забони манбаъ мавҷуд нестанд, тоза мекунад `-p`

Зеро ҳангоми навиштани ҳуҷҷатҳо номи файлҳои Markdown аксар вақт танзим карда мешаванд, ки дар натиҷа файлҳои зиёди кӯҳна ва партофташуда дар феҳристи тарҷума ба вуҷуд меоянд.

Ин параметрро барои тоза кардани файлҳое, ки бояд дар директорияҳои забонҳои дигар тоза карда шаванд, истифода баред.

## Феҳристи Тарҷумаро Муайян `-D`

Феҳристи тарҷумашуда ба директорияе, ки дар он файли ҷорӣ ҷойгир аст, муқаррар карда мешавад.

`-d` `--workdir` метавонад директорияи тарҷумаро муайян кунад, масалан:

```
i18 -d ~/i18n/md
```

## Кӯмакро `-H`

`-h` `--help` метавонед кӯмаки сатри фармонро бубинед.
