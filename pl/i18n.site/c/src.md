## Backend

Budowa strony dokumentacji wymaga tylko frontendu, nie backendu.

Ten backend jest backendem samej strony i18n.site, obejmującym systemy użytkowników, płatności, powiadomień itp.

* Usługi [GitHub](https://github.com/i18n-api/srv) / [AtomGit](https://atomgit.com/i18n-api/srv)

* Biblioteki podstawowe [GitHub](https://github.com/i18n-site/rust) / [AtomGit](https://atomgit.com/i18n/rust)
