# i18n.site: Czysto statyczny framework wielojęzycznej witryny

`i18n.site` – generator dokumentacji wielojęzycznej i całkowicie statycznej.

## Wprowadzenie

`i18n.site` to generator dokumentacji, a zarazem framework do tworzenia witryn.

Nowy paradygmat tworzenia stron, z `Markdown` jako centrum, z wykorzystaniem komponentów front-end do wprowadzania interaktywności.

Każdy komponent front-endowy to niezależny pakiet do instalacji.

W oparciu o separację front-endu i back-endu, wprowadzamy również podział na treści statyczne i dane dynamiczne.

Strona, którą odwiedzasz [i18n.site](/), została zbudowana na tym frameworku (w tym system użytkowników, system rozliczeń, subskrypcja e-maili itp.).
