# i18n.site: Czysto statyczny, wielojęzyczny framework dla stron internetowych

`i18n.site` – generator dokumentacji wielojęzyczny i całkowicie statyczny.

## Wstęp

`i18n.site` jest generatorem stron dokumentacyjnych oraz frameworkiem do tworzenia witryn internetowych.

Nowy paradygmat tworzenia stron, z `Markdown` jako centrum i interaktywnymi komponentami front-end.

Każdy komponent front-endowy to niezależny pakiet do instalacji.

W oparciu o separację front-endu i back-endu, mamy również oddzielenie treści statycznych od dynamicznych danych.

Strona, którą odwiedzasz [i18n.site](/), jest zbudowana na tym frameworku (w tym system użytkowników, system rozliczeń, subskrypcja e-maili itp.).
