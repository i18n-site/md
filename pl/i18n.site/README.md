## Wprowadzenie

`i18n.site` to moja refleksja nad modelem tworzenia stron internetowych.

Jest to zarówno generator dokumentacji, jak i framework do tworzenia stron internetowych.

Nowy paradygmat tworzenia stron, z `MarkDown` jako centralnym punktem i front-endowymi komponentami wprowadzającymi interaktywność.

Każdy front-endowy komponent to niezależny pakiet do zainstalowania.

W oparciu o separację front-endu i back-endu, wprowadzamy również oddzielenie treści statycznych od dynamicznych danych.

Strona, którą odwiedzasz [i18n.site](/), została zbudowana na podstawie tego frameworka (w tym system użytkowników, system rozliczeń, subskrypcja e-maili itp.).
