# i18n.site: Czysto statyczny framework wielojęzycznej strony internetowej

`i18n.site` – generator dokumentacji wielojęzycznej i całkowicie statycznej.

## Wprowadzenie

`i18n.site` to generator stron dokumentacyjnych, a zarazem framework do tworzenia witryn internetowych.

Nowy paradygmat tworzenia stron, z `MarkDown` jako centrum, gdzie interaktywność jest wstrzykiwana za pomocą komponentów front-end.

Każdy komponent front-endowy to niezależny pakiet do zainstalowania.

W oparciu o separację front-endu i back-endu, wprowadzamy również oddzielenie treści statycznych od dynamicznych danych.

Strona, którą odwiedzasz [i18n.site](/), została zbudowana na podstawie tego frameworka (w tym system użytkowników, system rozliczeń, subskrypcja e-maili itp.).
