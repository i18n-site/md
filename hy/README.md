## Հերթականություն

«Աստվածաշունչ · Ծննդոց»:

> Հինները լեզուները չէին տարբերում։ Նրանք ծնվել են հպարտությամբ և ցանկանում էին բարձր աշտարակ կառուցել, որի գագաթը հասնի դեպի երկինք
>
> <blockquote><p>Աստված դժգոհ էր մարդու ամբարտավանությունից, ուստի բոլոր կենդանի էակներին ցրեց տարբեր վայրերում՝ չկարողանալով միմյանց հասկանալ:</p></blockquote>
>

«Երեք մարմին» (չինական արտասանություն՝ `3Ti` ) գիտաֆանտաստիկ վեպը գեղարվեստական ​​է ներկայացնում այլմոլորակային քաղաքակրթություն, որը հաղորդակցվում է էլեկտրամագնիսական ալիքների միջոցով, չունի լեզվական խոչընդոտներ և տեխնոլոգիապես բարգավաճ է։

Հուսով եմ, որ այս գործիքի օգնությամբ երկրագնդի մարդիկ կնմանվեն երեք մարմնի մարդկանց, հաղորդակցությունը լեզվով չի սահմանափակվի, և ողջ մարդկությունը նորից կմիավորվի։
