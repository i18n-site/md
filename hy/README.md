## Հերթականություն

«Աստվածաշունչ · Ծննդոց»:

> Հինները լեզուները չէին տարբերում։ Նրանք կառուցել են երկինք հասնող աշտարակներ՝ Աստծո դարպասին հասնելու համար՝ բարձրացնելու մարդկային ցեղի հեղինակությունը:
>
> Աստված ասաց, որ մարդիկ իրենց ցեղն են՝ նույն մշակույթով և տեսակով: Աշտարակ կառուցելն ընդամենը նախերգանք է։ Այժմ դուք կարող եք իրականացնել այն, ինչ ցանկանում եք, և վախենալու ոչինչ չեք ունենա։
>
> Հետո Աստված եկավ ու ցրեց մարդկանց տարբեր տեղերում՝ չկարողանալով իրար հասկանալ։
>
> Այդ ժամանակվանից մարդկությունը պայքարում էր հաղորդակցվելու համար, անվերջ ծագում էին հակամարտություններ, և աշխարհն այլևս չտեսավ երկինք հասնող աշտարակ:

«Երեք մարմին» (չինական արտասանություն՝ `3Ti` ) գիտաֆանտաստիկ վեպը գեղարվեստական ​​է ներկայացնում այլմոլորակային քաղաքակրթություն, որը հաղորդակցվում է էլեկտրամագնիսական ալիքների միջոցով, չունի լեզվական խոչընդոտներ և տեխնոլոգիապես բարգավաճ է։

Հուսով եմ, որ այս գործիքի օգնությամբ երկրագնդի մարդիկ կնմանվեն երեք մարմնի մարդկանց, հաղորդակցությունը լեզվով չի սահմանափակվի, և ողջ մարդկությունը նորից կմիավորվի։
