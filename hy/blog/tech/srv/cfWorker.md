
## CNAME Կապված Տիրույթի Անուն

Cloudflare Worker անունը կապվում է `SSL/TLS` ի միջոցով → հատուկ հոսթի անունը CNAME Դոմենի անունը կապելուց հետո ուղղակի մուտքը կհայտնի սխալ `error code: 522`

Այս պահին դուք պետք է օգտագործեք HTTP երթուղիչ → երթուղի աշխատողներ՝ երթուղին կապելու համար, ինչպես ցույց է տրված ստորև։
