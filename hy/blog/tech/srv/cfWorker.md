
## CNAME Կապված Տիրույթի Անուն

Դոմենի անունն `SSL/TLS` Cloudflare Worker → Հարմարեցված հոսթինգի անուն պարտադիր է CNAME Դոմեյն անունից հետո ուղղակի մուտքը կհայտնի սխալ `error code: 522`

Այս պահին դուք պետք է օգտագործեք HTTP երթուղիչ → երթուղի աշխատողներ՝ երթուղին կապելու համար, ինչպես ցույց է տրված ստորև։
