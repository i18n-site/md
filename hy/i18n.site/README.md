# i18n.site. Մաքուր Ստատիկ Բազմալեզու Կայքի Շրջանակ

`i18n.site` , զուտ ստատիկ փաստաթղթերի կայքի գեներատոր:

Այն ներկայումս հասանելի է ի սկզբանե, բայց դեռևս մշակման փուլում են գտնվում կարևոր գործառույթները: Մանրամասների համար տես [զարգացման ծրագիրը](/i18n.site/c/roadmap) :

Խնդրում ենք <button onclick="mailsub()">սեղմել այս էլ․ հասցեն՝ ապրանքի թարմացումներին բաժանորդագրվելու</button> և <button onclick="webpush()">զննարկիչի Push-ին միացնելու համար</button> , երբ արտադրանքի թարմացումները կատարվեն:

[i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) ողջունում / հետևել մեր սոցիալական հաշիվներին [X.COM: @i18nSite](https://x.com/i18nSite)
