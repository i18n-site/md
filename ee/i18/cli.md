# Sedede Ƒe Fli Ƒe Nɔnɔmetatawo Ŋuti Numeɖeɖe Tsitotsito

## `-P` Klɔ Faɛlwo Ŋu

`-p` `--purge`

Elabena ne wole nuŋlɔɖiwo ŋlɔm la, zi geɖe la, wotrɔa asi le Markdown faɛl ƒe ŋkɔwo ŋu, si wɔnɛ be faɛl xoxo kple esiwo wogblẽ ɖi geɖe nɔa gbegɔmeɖeɖe ƒe agbalẽdzraɖoƒea.

Zã parameter sia nàtsɔ akɔ faɛl siwo wòle be woatutu le gbegbɔgblɔ bubuwo ƒe nyatakakadzraɖoƒewo me.

## `-D` Tsɔ Gbegɔmeɖeɖe Ƒe Agbalẽdzraɖoƒe

Nudzraɖoƒe si gɔme woɖe la nyea nudzraɖoƒe si me faɛl si li fifia le.

`-d` Alo `--workdir` ateŋu agblɔ gbegɔmeɖeɖe ƒe agbalẽdzraɖoƒe, abe:

```
i18 -d ~/i18n/md
```

## `-H` Kpɔ Kpekpeɖeŋu

`-h` Alo `--help` Àteŋu akpɔ sedede ƒe fli ƒe kpekpeɖeŋu.
