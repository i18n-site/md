# MarkDown Ƒe Nɔnɔme Ƒe Takpekpe

## Seke Ƒe Teƒe

Wonɔa te ɖe nuŋɔŋlɔ me nyawo dzi wɔa dekɔnu `MarkDown`

`i18n.site` Seke ƒe egbɔkpɔnu si dzi wolɔ̃ ɖoe nye be woatsɔ nuŋɔŋlɔ si sɔ kple `<a rel=id href="#xxx" id="xxx"></a>` ade `MarkDown` me be woatsɔ asi awɔ ɖoƒe ƒe seke.

`<a rel=id href="#xxx" id="xxx"></a>` `rel=id` `xxx`

Zi geɖe la, wotsɔa sekewo kpena ɖe tanya la ŋu, abe:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

Nusi woɖena fiana ƒe ŋusẽkpɔɖeamedzi le ale:

## Ŋlɔ `HTML` Le `MarkDown`

`pug` `HTML` ateŋu atsɔe ade kɔdasia me.

Womaɖe nya siwo le `<pre>`

Ne ètsɔ nya eve siawo ƒo ƒui la, àte ŋu aŋlɔ `HTML` in `MarkDown`

Àteŋu akpɔ dɔwɔwɔ le [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) kɔda la le ale :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

De dzesii be woɖe `<style>` hã gɔme le `<pre>`
