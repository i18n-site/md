
## CNAME Ilə Əlaqəli Domen Adı

Cloudflare Worker s domen adı `SSL/TLS` vasitəsilə bağlıdır → xüsusi host adı CNAME adı bağlandıqdan sonra birbaşa giriş `error code: 522` səhvini bildirəcək

Bu zaman, aşağıda göstərildiyi kimi marşrutu bağlamaq üçün HTTP marşrutlaşdırma → İşçilər marşrutlaşdırmasından istifadə etməlisiniz.
