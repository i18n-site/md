# Adehun Kika MarkDown

## Ojuami Oran

Ti aṣa `MarkDown`

Ojutu oran ti o gba ni lati fi ọrọ ti o `i18n.site` si `<a rel=id href="#xxx" id="xxx"></a>` ni `MarkDown` lati ṣeda oran ipo pẹlu ọwọ.

`<a rel=id href="#xxx" id="xxx"></a>` , nibi `rel=id` ṣe asọye aṣa oju-iwe ti aaye oran, jọwọ rọpo `xxx` pẹlu kuru Gẹẹsi gangan rẹ.

Awọn ìdákọró ni a maa n fi kun akọle naa, gẹgẹbi:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Kọ `HTML` Sinu `MarkDown`

`pug` `HTML` le ti wa ni ifibọ sinu koodu.

Akoonu ti o wa ninu nkan `<pre>` kii yoo tumọ.

Ni apapọ awọn aaye meji wọnyi, o le ni rọọrun kọ `HTML` `MarkDown` lati ṣaṣeyọri ọpọlọpọ awọn ipa ifihan.

O le tọka si imuse ni [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) koodu naa jẹ atẹle :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Ṣe akiyesi pe `<style>` tun jẹ asọye `<pre>` oke.
