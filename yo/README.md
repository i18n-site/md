Mo nírètí pé pẹ̀lú ìrànlọ́wọ́ irinṣẹ́ yìí, àwọn ènìyàn ilẹ̀ ayé yóò dà bí ẹni mẹ́ta, ìjùmọ̀sọ̀rọ̀ kì yóò jẹ́ ààlà nípa èdè, gbogbo aráyé yóò sì tún wà ní ìṣọ̀kan.
