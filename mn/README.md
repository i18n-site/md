Шинжлэх ухааны зөгнөлт зохиол &quot;Гурван бие&quot; (Хятад дуудлага: `3Ti` ) нь цахилгаан соронзон долгионоор дамжуулан харилцдаг, хэлний бэрхшээлгүй, технологийн хувьд хөгжингүй харь гаригийн соёл иргэншлийг зохиомжлон харуулсан.

### Орчуулгын Тэмдэглэл

Орчуулгын тайлбар нь \```-ын дараа ` ```rust` гэх мэт хэлийг зааж өгөх шаардлагатай.
