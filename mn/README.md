## Оршил

Шинжлэх ухааны зөгнөлт зохиол &quot;Гурван бие&quot; (Хятад дуудлага: `3Ti` ) нь цахилгаан соронзон долгионоор дамжуулан харилцдаг, хэлний бэрхшээлгүй, технологийн хувьд хөгжингүй харь гаригийн соёл иргэншлийг зохиомжлон харуулсан.

Библи · Эхлэл :

> Тэр үед дэлхийн өнцөг булан бүрт байгаа хүмүүс нэг аялга, хэл яриатай байсан.
>
> Тэнгэрт хүрэх цамхаг барьж, Бурханы үүдэнд хүрч, хүмүүсийн зүрх сэтгэлийг цуглуулж, нэр хүндээ түгээгээрэй.
>
> Их Эзэн хэлсэн нь: Ижил соёл, угсаатны хүмүүс өөр өөрийн овгийг бүрдүүлдэг. Өнөөдөр цамхаг барих нь зүгээр л оршил боловч бид ирээдүйд бүх зүйлийг хийх болно.
>
> Тэгээд ирж ойлголцож чадалгүй янз бүрийн газар тараасан.
>
> Түүнээс хойш маргаан үргэлжилсээр, дэлхий дээр Вавилоны цамхаг байхгүй.

Дэлхийн хүмүүсийг гурван биет хүн шиг болгож, хэл ярианд баригдалгүй харилцах, бүх хүн төрөлхтнийг дахин нэгтгэх хэрэгсэл бүтээнэ гэж найдаж байна.
