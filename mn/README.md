Шинжлэх ухааны зөгнөлт зохиол &quot;Гурван бие&quot; (Хятад дуудлага: `3Ti` ) нь цахилгаан соронзон долгионоор дамжуулан харилцдаг, хэлний бэрхшээлгүй, технологийн хувьд хөгжингүй харь гаригийн соёл иргэншлийг зохиомжлон харуулсан.

Энэ хэрэгслийн тусламжтайгаар дэлхийн хүмүүс гурван биет хүн шиг болж, харилцаа холбоо хэлээр хязгаарлагдахгүй, бүх хүн төрөлхтөн дахин нэгдэх болно гэдэгт итгэж байна.

### Орчуулгын Тэмдэглэл

Орчуулгын тайлбар нь \```-ын дараа ` ```rust` гэх мэт хэлийг зааж өгөх шаардлагатай.

Одоогоор rust, c, cpp, java, js, coffee, python, bash-ын тайлбар орчуулгыг дэмждэг.

[tran_md/src/comment.coffee-г](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) засварлаж, олон хэл дээрх сэтгэгдлүүдэд орчуулгын дэмжлэг нэмнэ үү.
