## Ашиглах

### Демо Төсөл

Орчуулгын тохиргоог `i18` тулд демо төслөөс үзнэ үү [github.com/i18n-site/demo.i18](//github.com/i18n-site/demo.i18)

Хятад дахь хэрэглэгчид клон хийх боломжтой [atomgit.com/i18n/demo.i18](//atomgit.com/i18n/demo.i18)

Клончилсны дараа лавлах руу ороод орчуулгыг дуусгана уу `i18`

### Лавлах Бүтэц

Загварын агуулахын лавлах бүтэц нь дараах байдалтай байна

```
┌── .i18n
│  └── conf.yml
└── en
   ├── _IgnoreDemoFile.md
   ├── i18n.yml
   └── README.md
```

`en` лавлах нь орчуулсан демо файлуудыг агуулдаг бөгөөд эдгээр нь зөвхөн жишээ бөгөөд устгагдах боломжтой.

### Орчуулга Ажиллуулах

Лавлах руу ороод орчуулга хийнэ үү `i18`

### Хадгалах Газарт Файл Нэмэх

Орчуулахаас гадна програм нь дараах файлуудыг үүсгэх бөгөөд тэдгээрийг репозитор руу нэмнэ үү.

```
.i18n/hash
.i18n/cache/.gitignore
```

Тэдгээрийн дотроос агуулга : дараах байдалтай байна `.i18n/cache/.gitignore`

```
**/*
!**/.gitignore
```

Энэ нь `.i18n/cache/` доторх бүх файлыг ( `.i18n/cache/.gitignore` бусад) үл тоомсорлодог гэсэн үг юм.

Хэрэв таны хувилбарын хяналтын програм хангамж байхгүй `git` энэ тохиргооны дагуу үүнийг үл тоомсорлоно уу!

## Тохиргооны Файл

Энэ нь `i18` мөр орчуулах хэрэгслийн тохиргооны файл `.i18n/conf.yml`

Агуулга нь дараах байдалтай байна.

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### Эх Хэл Орчуулгын Хэл &

Тохиргооны файлд `fromTo`

`en` нь эх хэл, `zh ja ko de fr` нь орчуулгын зорилтот хэл юм.

Хэлний кодыг үзнэ үү [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

Жишээлбэл, хэрэв та хятад хэлийг англи хэл рүү орчуулахыг хүсвэл энэ мөрийг дахин бичээрэй `zh: en`

Хэрэв та дэмжигдсэн бүх хэл рүү орчуулахыг хүсвэл дараа нь хоосон орхино уу `:` Жишээлбэл

```
i18n:
  fromTo:
    en:
```

### Файлыг Үл Тоомсорлох

Анхдагч байдлаар, эх хэлний лавлах дахь `.md` болон `.yml` тэмдэгтээр эхэлсэн бүх файлууд орчуулагдах болно.

Хэрэв та тодорхой файлуудыг үл тоомсорлож, орчуулахгүй байхыг хүсвэл (дүгсээгүй ноорог гэх мэт) `ignore` тохиргоог ашиглаж болно.

`.gitignore` -тэй төстэй синтакс [globset](https://docs.rs/globset/latest/globset/#syntax) `ignore`

Жишээлбэл, дээрх тохиргооны файлд `_` -ээр эхэлсэн файлууд орчуулагдахгүй гэсэн үг `_* ` .

## Орчуулгын Дүрэм

### Орчуулгын Редакторууд Мөр Нэмэх, Устгах Ёсгүй

Орчуулга нь засварлах боломжтой. Эх текстийг өөрчилж, машинаар дахин орчуулбал орчуулгад оруулсан гар аргаар хийсэн өөрчлөлтийг дарж бичихгүй (хэрэв эх текстийн энэ догол мөрийг өөрчлөөгүй бол).

Гэхдээ орчуулгын мөрүүд болон эх бичвэр нь нэг нэгтэй тохирч байх ёстойг анхаарна уу. Өөрөөр хэлбэл, орчуулгыг эмхэтгэхдээ мөр нэмж, устгаж болохгүй. Үгүй бол энэ нь орчуулгын засварлах кэшэд төөрөгдөл үүсгэх болно.

Хэрэв ямар нэг зүйл буруу болвол [шийдлийг авахын тулд Түгээмэл асуултууд хэсгээс үзнэ үү.](/i18/qa#H1)

### `Yaml`

Тушаалын мөрийн хэрэгсэл нь эх хэлний файлын лавлахаас `.yml` -ээр төгссөн бүх файлыг олж, орчуулах болно.

* Файлын нэрийн дагавар нь `.yml` ( `.yaml` биш) байх ёстойг анхаарна уу.

Энэ хэрэгсэл нь толь бичгийн түлхүүрүүдийг биш зөвхөн толь бичгийн утгыг орчуулдаг `.yml`

Жишээлбэл `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

гэж орчуулагдах болно `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

`YAML` -ийн орчуулгыг гараар өөрчилж болно (гэхдээ орчуулгад түлхүүр эсвэл мөр нэмж, устгаж болохгүй).

Орчуулга `YAML` үндэслэн та янз бүрийн програмчлалын хэлнүүдийн олон улсын шийдлийг хялбархан бүтээх боломжтой.

## Дэвшилтэт Хэрэглээ

### Орчуулгын Дэд Лавлах

Та үүсгэсэн л `.i18n/conf.yml` (тухайлбал демо төслийн загвараас эхлэх шаардлагагүй) `i18` сайн ажиллах болно.

Тушаалын мөрийн хэрэгсэл нь бүх дэд лавлахаас `.i18n/conf.yml` олж, орчуулах болно.

[monorepo](//monorepo.tools) ашигладаг төслүүд хэлний файлуудыг дэд директоруудад хувааж болно.

<img alt="" src="https://p.3ti.site/1719910016.avif">

### Тусгай Суулгалтын Лавлах

Үүнийг анхдагчаар суулгана `/usr/local/bin`

`/usr/local/bin` бичих зөвшөөрөлгүй бол `~/.bin` дээр суулгана.

Орчны хувьсагчийг : `TO`

```sh
TO=/bin sudo bash <(curl -sS https://i.i18n.site) i18
```
