# Тушаалын Мөрийн Параметрүүдийн Дэлгэрэнгүй Тайлбар

## Файлуудыг `-P`

`-p` Эсвэл `--purge` орчуулгын сан бүрт байгаа боловч эх хэлний лавлах дотор байхгүй файлуудыг цэвэрлэх болно.

Учир нь баримт бичгийг бичихдээ Markdown файлын нэрийг ихэвчлэн тохируулдаг бөгөөд үүний үр дүнд орчуулгын санд хуучин болон орхигдсон олон файлууд үүсдэг.

Бусад хэлний лавлахаас устгах ёстой файлуудыг цэвэрлэхийн тулд энэ параметрийг ашиглана уу.

## Орчуулгын Лавлахыг Зааж Өгнө `-D`

Орчуулсан лавлах нь одоогийн файл байгаа лавлах руу анхдагчаар тохируулагдана.

`-d` `--workdir` орчуулгын лавлахыг зааж өгч болно, жишээ нь:

```
i18 -d ~/i18n/md
```

## Тусламжийг Үзнэ `-H`

`-h` `--help` тушаалын мөрийн тусламжийг үзэж болно.
