# Připravte prostředí

* Nainstalujte a spusťte `docker`. (Uživatelé na Macu doporučují používat [orbstack](https://orbstack.dev) jako runtime pro `docker`)

* Instalujte [bun](https://bun.sh/docs/installation) a [nodejs](https://nodejs.org/en/download/package-manager)

## Inicializujte projekt

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
