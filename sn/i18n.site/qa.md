# Faq

## Zvakadzimwa Netsaona `.i18n/v` , Zvichiita Kuti Package `npm` Itadze Kuburitswa

Iyo yenhoroondo vhezheni yakaburitswa pasuru `npm` inochengetwa `.i18n/v/ol/v.hash` .

Kana ukadzima netsaona `.i18n/v/ol` package `npm` haiburitswe.

Panguva ino, tanga watsvaga nhamba yevhezheni yekupedzisira kuburitswa vhezheni yeprojekiti yako mu [npmjs.com](//npmjs.com) semuenzaniso, `0.1.9` .

Wobva watarisa `bash` pazasi kugadzira iyo faira.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Ziva kuti kugadzirisa nenzira iyi kucharasikirwa nenhoroondo yefaira, zvichiita kuti zvisakwanise kuvandudza kuburitswa kunotevera, uye zvese zvirimo zvichaiswa pakare uye kuiswa kamwe.
