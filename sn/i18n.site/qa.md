# Common Problem

## Zvakadzimwa Netsaona `.i18n/V` Zvichiita Kuti `Npm` Package Itadze Kuburitswa

`.i18n/v/ol/v.hash` `npm`

Kana ukadzima netsaona `.i18n/v/ol` iyo `npm` package haizoburitswa.

Panguva ino, tanga watsvaga nhamba yevhezheni yekupedzisira kuburitswa kweprojekiti yako mu [npmjs.com](//npmjs.com) semuenzaniso `0.1.9`

Wobva watarisa kune `bash` Gadzira faira pazasi.

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

Ziva kuti kugadzirisa nenzira iyi kucharasikirwa nenhoroondo yefaira, zvichiita kuti zvisakwanise kuvandudza kuburitswa kunotevera, uye zvese zvirimo zvichaiswa pakare uye kuiswa kamwe.
