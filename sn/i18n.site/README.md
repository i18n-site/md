# i18n.site: Pure Static Multi-Language Website Framework

`i18n.site` Multi-mitauro, purely static gwaro saiti jenareta.

Parizvino iripo pekutanga, asi pachine akakosha maficha pasi pekuvandudzwa Ndokumbira utarise [hurongwa hwekuvandudza](/i18n.site/c/roadmap) .

Ndokumbira <button onclick="mailsub()">udzvanye iyi email kuti unyore kugadziriso dzechigadzirwa</button> uye <button onclick="webpush()">gonesa browser kusunda</button> Isu tichakuzivisa iwe kana zvigadzirwa zvagadziridzwa.

[i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) unogamuchirwa kuti uteedzere akaundi edu / [X.COM: @i18nSite](https://x.com/i18nSite)

## Nhanganyaya

`i18n.site` ndiko kufunga kwangu patsva yemhando yekuvandudza webhusaiti.

Iri igwaro saiti jenareta uye webhusaiti yekuvandudza chimiro.

Iyo paradigm nyowani yekuvandudza webhusaiti iyo yakatarisana `MarkDown` uye inoshandisa kumberi-kumagumo zvikamu kubaya kudyidzana.

Chimwe nechimwe chemberi-yekupedzisira chikamu ipakiti inogona kuiswa yakazvimirira.

Pachikonzero chekuparadzaniswa kwepamberi-yekupedzisira uye kumashure-kumashure, kune zvakare kuparadzaniswa kweiyo static content uye ine simba data.

Zvauri [i18n.site](/) zvakavakwa zvichibva pane iyi hurongwa (kusanganisira mushandisi sisitimu, yekubhadhara system, email kunyoreswa, nezvimwewo).
