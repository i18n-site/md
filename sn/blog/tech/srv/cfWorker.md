
## CNAME Yakasungwa Zita Rezita

Cloudflare Worker 's domain name inosungwa kuburikidza `SSL/TLS` → tsika yekutambira zita CNAME mezita rezita rakasungwa, kuwana kwakananga kunozotaura kukanganisa `error code: 522`

Panguva ino, unofanira kushandisa HTTP routing → Vashandi routing kusunga nzira, sezvinoratidzwa pazasi.
