## 序言

`i18n.site`は、Webサイト開発モデルの再考です。

これはドキュメントサイトジェネレーターであり、同時にWebサイト開発フレームワークでもあります。

`MarkDown`を中心に、フロントエンドコンポーネントを用いてインタラクティブ性を導入する新しいWebサイト開発パラダイムです。

各フロントエンドコンポーネントは、個別にインストール可能なパッケージです。

フロントエンドとバックエンドの分離に加え、静的コンテンツと動的データの分離も行われています。

あなたが訪れている[i18n.site](/)は、このフレームワークを基盤に構築されています(ユーザーシステム、課金システム、メール订阅などが含まれます)。
