# i18n.site: 純粋な静的サイトの多言語フレームワーク

`i18n.site`は多言語対応で、純粋に静的なドキュメントサイトジェネレーターです。

## 序章

`i18n.site`はドキュメントサイトジェネレーターであり、またウェブサイト開発のフレームワークでもあります。

`MarkDown`を中心に、フロントエンドコンポーネントを用いてインタラクティブ性を導入する新しいウェブサイト開発パラダイムです。

各フロントエンドコンポーネントは、個別にインストール可能なパッケージです。

フロントエンドとバックエンドの分離に基づき、静的コンテンツと動的データの分離も行われています。

あなたが訪れている[ i18n.site ](/)は、このフレームワークを基盤に構築されています(ユーザーシステム、課金システム、メール購読などが含まれます)。
