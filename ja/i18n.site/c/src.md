## 後部

ドキュメント サイトの構築にはフロントエンドのみが必要で、バックエンドは必要ありません。

このバックエンドは、ユーザー、支払い、メッセージ プッシュ、その他のシステムを含む、i18n.site Web サイト自体のバックエンドです。
