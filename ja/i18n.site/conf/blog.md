# ブログテンプレート

`i18n/conf.yml`の`use: Blog`は、ブログテンプレートを使用してレンダリングすることを示しています。

ブログの記事としての`markdown`ファイルには、メタ情報を設定する必要があります。

メタ情報はファイルの冒頭に置かれ、`---`で始まり、また`---`で終了する必要があります。その中の設定情報は`YAML`形式で記述されます。

デモファイルの設定例は以下の通りです

```yml
---

brief: |
  this is a demo brief
  you can write multiline

---

# title

… …
```

`brief`は記事の概要を示し、ブログの索引ページに表示されます。

`YMAL`の助けを借りて|`構文では、複数行の要約を書くことができます。

ブログの右側に表示されるディレクトリツリーの設定は`TOC`ファイルです(前章を参照)。ブログの首页索引には、`TOC`に記載された記事だけが表示されます。

メタ情報を含まない記事はブログの首页に表示されませんが、右側のディレクトリツリーには表示されることがあります。

特定の記事をトップに固定したい場合は、`i18n.site`を実行し、`.i18n/data/blog`配下の`xxx.yml`ファイルを編集して、タイムスタンプを負の数値に変更してください(複数の負の数値は、その絶対値の小さい順に並び替えられます)。
