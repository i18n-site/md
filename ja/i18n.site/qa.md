# よくある質問

## `.i18n/v`を不注意で削除してしまい、`npm`パッケージの公開ができなくなりました

`.i18n/v/ol/v.hash`には、`npm`パッケージのリリース履歴が保存されています。

`.i18n/v/ol`を不注意で削除してしまうと、`npm`パッケージが公開できなくなります。

この場合、まずは[npmjs.com](//npmjs.com)でプロジェクトの最後のリリースバージョンのバージョン番号を見つけてください。例えば`0.1.9`などです。

その後、以下の`bash`スクリプトを参考にしてファイルを作成してください。

```bash
mkdir -p .i18n/v/ol
echo @0.1.9 > .i18n/v/ol/v.hash
```

この方法で修正を行うと、ファイルの履歴が失われるため、次回のリリース時にインクリメンタルな更新ができなくなり、すべてのコンテンツが再パッケージ化されて一度にアップロードされることになります。ご注意ください。
