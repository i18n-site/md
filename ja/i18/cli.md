# コマンドラインパラメータの詳細説明

## `-p`でファイルをクリーンアップ

`-p`または`--purge`を使用すると、各翻訳ディレクトリに存在するがソースディレクトリに存在しないファイルを削除します。

ドキュメント作成時、Markdownファイル名が頻繁に変更されるため、翻訳ディレクトリ内に不要な古いファイルが多数発生することがあります。

このパラメータを使用すると、他の言語のディレクトリ内で削除すべきファイルをクリーンアップできます。

## `-d`で翻訳ディレクトリを指定

翻訳ディレクトリのデフォルトは、現在のファイルがあるディレクトリです。

`-d`または`--workdir`を使用して、以下のように翻訳ディレクトリを指定できます

```
i18 -d ~/i18n/md
```

## `-h`でヘルプを表示

`-h`または`--help`を使用すると、コマンドラインのヘルプを表示できます。
