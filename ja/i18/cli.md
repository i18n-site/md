# コマンドラインパラメータの詳しい説明

## `-p`ファイルをクリーンアップ

`-p`または`--purge` 、各翻訳ディレクトリに存在するがソース言語ディレクトリには存在しないファイルをクリーンアップします。

ドキュメントを作成するときに Markdown ファイル名が調整されることが多く、その結果、翻訳ディレクトリ内に古いファイルや放棄されたファイルが多数残るためです。

このパラメータを使用して、他の言語ディレクトリで削除する必要があるファイルをクリーンアップします。

## `-d`翻訳ディレクトリを指定してください

変換されたディレクトリのデフォルトは、現在のファイルが存在するディレクトリです。

`-d`または`--workdir`次のように翻訳ディレクトリを指定できます。

```
i18 -d ~/i18n/md
```

## `-h`ヘルプを見る

`-h`または`--help`コマンド ライン ヘルプを表示できます。
