
## Nama Domain Terikat CNAME

Nama domain Cloudflare Worker terikat melalui `SSL/TLS` → nama host khusus CNAME nama domain terikat, akses langsung akan melaporkan kesalahan `error code: 522`

Saat ini, Anda perlu menggunakan perutean HTTP → Perutean pekerja untuk mengikat rute, seperti yang ditunjukkan di bawah ini
