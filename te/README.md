# I18N.SITE · సరిహద్దులు లేని భాష<img Src="//ok0.pw/5l" Style="float:right;width:42px;margin-Top:6px">

I18N.SITE, బహుళ-భాషా స్టాటిక్ సైట్ జనరేటర్, మార్క్‌డౌన్‌ను [వంద కంటే ఎక్కువ విభిన్న భాషల్లోకి](https://github.com/i18n-site/node/blob/main/lang/src/index.js) స్వయంచాలకంగా అనువదించగలదు.

<img src="//i-01.eu.org/i18n.lang.webp" alt="" />

ఇప్పుడు బ్రౌజర్‌లు అంతర్నిర్మిత అనువాద ఫంక్షన్‌లను కలిగి ఉన్నందున, వెబ్‌సైట్‌ను అంతర్జాతీయీకరించడం అనవసరమా అని కొంతమంది అడగాలనుకోవచ్చు.

**మొత్తం సైట్‌ను అంతర్జాతీయీకరించడం ద్వారా మాత్రమే మేము బహుళ-భాషలో సైట్ పూర్తి-టెక్స్ట్ శోధన మరియు శోధన ఇంజిన్ ఆప్టిమైజేషన్‌కు మద్దతు ఇవ్వగలమని** నేను చెప్పాలనుకుంటున్నాను.

## ఫంక్షన్ పరిచయం

### మార్క్‌డౌన్ ఆకృతిని ఉంచండి

### అనువాదాన్ని సవరించండి

అనువాదాన్ని సవరించిన తర్వాత, మీరు కాష్‌ని నవీకరించడానికి `./i18n.sh` మళ్లీ అమలు చేయాలి.

### అనువాద గమనికలు

అనువాద వ్యాఖ్యలు \``` తర్వాత ` ```rust` వంటి భాషను సూచించాలి.

ప్రస్తుతం రస్ట్, సి, సిపిపి, జావా, జెఎస్, కాఫీ, పైథాన్ మరియు బాష్ కోసం వ్యాఖ్య అనువాదానికి మద్దతు ఇస్తుంది.

మరిన్ని భాషల్లో వ్యాఖ్యలకు అనువాద మద్దతును జోడించడానికి [tran_md/src/comment.coffee](https://github.com/i18n-site/node/blob/main/tran_md/src/comment.coffee) ని సవరించండి.

### ప్రాక్సీని కాన్ఫిగర్ చేయండి

కింది ఎన్విరాన్‌మెంట్ వేరియబుల్‌లను సెట్ చేయడం ద్వారా Google Translate API కాల్‌లు ప్రాక్సీ ద్వారా వెళ్లడానికి అనుమతిస్తుంది.

```bash
export https_proxy=http://127.0.0.1:7890
```
