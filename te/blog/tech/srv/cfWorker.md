
## CNAME బౌండ్ డొమైన్ పేరు

Cloudflare Worker యొక్క డొమైన్ పేరు `SSL/TLS` → కస్టమ్ హోస్ట్ పేరు ద్వారా కట్టుబడి ఉంటుంది CNAME డొమైన్ పేరు కట్టుబడి ఉన్న తర్వాత, ప్రత్యక్ష ప్రాప్యత లోపం `error code: 522` ని నివేదిస్తుంది

ఈ సమయంలో, మీరు దిగువ చూపిన విధంగా మార్గాన్ని బంధించడానికి HTTP రూటింగ్ → వర్కర్స్ రూటింగ్ని ఉపయోగించాలి
