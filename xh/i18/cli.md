# Ingcaciso Ebanzi Yeeparamitha Zomgca Womyalelo

## `-P` Ezicocekileyo

`-p` `--purge` izakucoca iifayile ezikhoyo kuluhlu ngalunye lwenguqulelo kodwa azikho kuluhlu lolwimi lwemvelaphi.

Kuba xa ubhala amaxwebhu, amagama eefayile zikaMarkdown ahlala ehlengahlengiswa, okukhokelela kwiifayile ezininzi ezindala nezilahliweyo kuluhlu lokuguqulela.

Sebenzisa le parameter ukucoca iifayile ekufuneka zicinywe kwezinye iilwimi.

## `-D` Uluhlu Lwenguqulelo

Uvimba weefayili oguquliweyo awugqibekanga kulawulo apho ifayile yangoku ikhoyo.

`-d` `--workdir`

```
i18 -d ~/i18n/md
```

## `-H` Uncedo

`-h` Okanye `--help` uncedo lomgca womyalelo.
