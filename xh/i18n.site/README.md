# I-I18n.site: Isakhelo Sewebhusayithi Esizinzileyo Esithetha Iilwimi Ezininzi

`i18n.site` ezininzi, umenzi wesiza soxwebhu olungatshintshiyo.

Ifumaneka ngoku ekuqaleni, kodwa kusekho iimpawu ezibalulekileyo phantsi kophuhliso Nceda ubone [isicwangciso sophuhliso](/i18n.site/c/roadmap) ngeenkcukacha.

Nceda <button onclick="mailsub()">ucofe le imeyile ukuze ubhalisele uhlaziyo lwemveliso</button> kwaye <button onclick="webpush()">uvule umkhangeli zincwadi otyhalayo</button> Siya kukwazisa xa uhlaziyo lwemveliso lwenziwa.

Wamkelekile ukuba ulandele [X.COM: @i18nSite](https://x.com/i18nSite) zethu zentlalo / [i18n-site.bsky.social](https://bsky.app/profile/i18n-site.bsky.social) .

## Intshayelelo

`i18n.site` kukucinga kwam kwakhona ngemodeli yophuhliso lwewebhusayithi.

Lo ngumvelisi wesiza soxwebhu kunye nesakhelo sophuhliso lwewebhusayithi.

Iparadigm entsha yophuhliso lwewebhusayithi egxile `MarkDown` kwaye isebenzisa amacandelo angaphambili ukufaka intsebenziswano.

Icandelo ngalinye langaphambili liyiphakheji enokuthi ifakwe ngokuzimeleyo.

Ngesiseko sokuhlukana kwe-front-end kunye ne-back-end, kukho kwakhona ukuhlukana komxholo we-static kunye nedatha enamandla.

[i18n.site](/) oyindwendwelayo yakhiwe ngokusekelwe kwesi sikhokelo (kubandakanya inkqubo yomsebenzisi, inkqubo yokuhlawula, ukubhaliswa kwe-imeyile, njl.).
