# Indibano Yefomathi yeMarkDown

## Indawo Yeankile

Isiko `MarkDown`

`i18n.site` Isisombululo seankile ekuvunyelwene ngaso kukufakela okubhaliweyo okufana ne- `<a rel=id href="#xxx" id="xxx"></a>` kwi `MarkDown` ukwenza ngesandla iiankile yokubeka.

`<a rel=id href="#xxx" id="xxx"></a>` , apha `rel=id` ichaza isimbo sephepha sendawo yeankile, nceda buyisela `xxx` nge-ankile yakho yesifinyezo sesiNgesi.

Ii-ankile zihlala zongezwa kwisihloko, ezifana nezi:

```md
### <a rel=id href="#i18" id="i18"></a>i18 : MarkDown Command Line Translation Tool
```

## Bhala `HTML` Ngaphakathi `MarkDown`

`pug` `HTML` inokufakwa kwikhowudi.

Isiqulatho esingaphakathi `<pre>` asizukuguqulwa.

Ukudibanisa la manqaku mabini, ungabhala ngokulula `HTML` kwi `MarkDown`

Ungabhekisa kuphunyezo kwi [github.com/i18n-site/md/en/README.md](https://raw.githubusercontent.com/i18n-site/md/main/en/README.md) ikhowudi imi ngolu hlobo lulandelayo :

```html
<h1 style="display:flex;justify-content:space-between">i18n.site ⋅ International Solutions<img src="//p.3ti.site/logo.svg" style="user-select:none;margin-top:-1px;width:42px"></h1>
<pre class="langli" style="display:flex;flex-wrap:wrap;background:transparent;border:1px solid #eee;font-size:12px;box-shadow:0 0 3px inset #eee;padding:12px 5px 4px 12px;justify-content:space-between;"><style>pre.langli i{font-weight:300;font-family:s;margin-right:2px;margin-bottom:8px;font-style:normal;color:#666;border-bottom:1px dashed #ccc;}</style><i>English</i><i>简体中文</i><i>Deutsch</i> … …</pre>
```

Qaphela ukuba `<style>` ichazwe `<pre>` ngasentla.
