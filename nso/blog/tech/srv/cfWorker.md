
## Lebitso La Domain Name Le Tlamehileng CNAME

Cloudflare Worker ’s domain name is bound through `SSL/TLS` → custom host name CNAME ga ge leina la domain le tlemilwe, phihlelelo ya thwii e tla bega phošo `error code: 522`

Mo nakong ye, o swanetše go šomiša tsela ya HTTP → Tsela ya bašomi go tlama tsela, bjalo ka ge go bontšhitšwe ka mo tlase
