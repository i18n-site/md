## Prólogo

`i18n.site` es una reevaluación mia sobre el modelo de desarrollo de sitios web.

Es un generador de sitios de documentos, pero también un marco de desarrollo para sitios web.

Un nuevo paradigma de desarrollo web centrado en `MarkDown` e impregnado de interactividad mediante componentes front-end.

Cada componente front-end es un paquete que puede ser instalado de manera independiente.

Basado en la separación de front-end y back-end, también se distingue la división entre contenido estático y datos dinámicos.

El sitio [i18n.site](/) que está visitando se ha construido utilizando este marco (que incluye un sistema de usuarios, un sistema de facturación, suscripciones por correo electrónico, etc.).
