# i18n.site: Marco de sitio web estático multilingüe puro

`i18n.site` es un generador de sitios de documentos multilingües y estáticos puros.

## Prefacio

`i18n.site` es un generador de sitios de documentos, pero también un framework para el desarrollo web.

Un nuevo paradigma de desarrollo web centrado en `MarkDown` e inyectado con componentes front-end para la interactividad.

Cada componente front-end es un paquete que puede ser instalado de manera independiente.

Basado en la separación de front-end y back-end, también se realiza una separación de contenido estático y datos dinámicos.

El sitio [i18n.site](/) que está visitando se ha construido utilizando este framework (incluido el sistema de usuarios, sistema de facturación, suscripción por correo electrónico, etc.).
