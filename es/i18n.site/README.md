# i18n.site: Marco de sitio web multilingüe estático puro

`i18n.site` es un generador de sitios de documentos estáticos y multilingües.

## Prefacio

`i18n.site` es un generador de sitios de documentos, pero también un marco de desarrollo web.

Una nueva metodología de desarrollo web centrada en `MarkDown` e inyectada con interactividad a través de componentes front-end.

Cada componente front-end es un paquete que puede ser instalado de manera independiente.

Siendo la separación entre front-end y back-end su base, también se distingue por la división entre contenido estático y datos dinámicos.

El sitio [i18n.site](/) que estás visitando se ha construido sobre este marco (incluyendo un sistema de usuarios, sistema de facturación, suscripción por correo electrónico, etc.).
