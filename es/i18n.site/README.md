# i18n.site: Marco de sitio web multilingüe de estática pura

`i18n.site` es un generador de documentos estáticos multilingües.

## Prefacio

`i18n.site` es un generador de sitios de documentos, pero también un framework para el desarrollo web.

Una nueva metodología de desarrollo web centrada en `Markdown` e impregnada de interactividad mediante componentes front-end.

Cada componente front-end es un paquete que puede ser instalado de manera independiente.

Siendo la separación de front-end y back-end su base, también se efectúa una división entre contenido estático y datos dinámicos.

El sitio [i18n.site](/) que estás visitando se ha construido sobre este framework (incluyendo un sistema de usuarios, sistema de facturación, suscripción por correo electrónico, etc.).
