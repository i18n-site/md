# Preparar el entorno

* Instalar y ejecutar `docker` (se recomienda a los usuarios de Mac utilizar [orbstack](https://orbstack.dev) como entorno de ejecución para `docker`)

* Instalar [bun](https://bun.sh/docs/installation) y [nodejs](https://nodejs.org/en/download/package-manager)

## Inicializar el proyecto

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
