## Reynolds  Backend

Reynolds  La construcción del sitio de documentación solo requiere el frontend, no el backend.

Reynolds  Este backend es para el sitio web i18n.site mismo, que incluye sistemas de usuarios, pagos, notificaciones de mensajes, etc.

* Reynolds  Servicios [github](https://github.com/i18n-api/srv) / [atomgit](https://atomgit.com/i18n-api/srv)
