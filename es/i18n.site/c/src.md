## Backend

La construcción del sitio de documentos solo requiere el frontend, no el backend.

Este backend es para el sitio web i18n.site mismo, que incluye sistemas de usuarios, pagos, notificaciones, etc.

* Servicios [GitHub](https://github.com/i18n-api/srv) / [AtomGit](https://atomgit.com/i18n-api/srv)

* Bibliotecas base [GitHub](https://github.com/i18n-site/rust) / [AtomGit](https://atomgit.com/i18n/rust)
