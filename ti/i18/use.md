### ፋይላት ናብቲ መኽዘን ምውሳኽ

ብዘይካ ትርጉም እቲ ፕሮግራም እዞም ዝስዕቡ ፋይላት እውን ክፈጥር እዩ፣ በጃኹም ናብቲ መኽዘን ወስኹዎም።

```

```

ካብኣቶም ትሕዝቶ : `.i18n/cache/.gitignore`

```

```

እዚ ማለት ንኹሎም ኣብ `.i18n/cache/` ዘለዉ ፋይላት ሸለል በሎም (ብዘይካ `.i18n/cache/.gitignore` ).

ናትካ ቨርዥን ቁጽጽር ሶፍትዌር እንተዘይኮይኑ `git` በጃኻ ብመሰረት እዚ ውቅር ሸለል በሎ።

## ናይ ውቅር ፋይል

`.i18n/conf.yml` ናይ `i18` መስመር ትርጉም መሳርሒ ናይ ውቅር ፋይል እዩ።

ትሕዝቶኡ ከምዚ ዝስዕብ ይመስል፤

```
i18n:
  fromTo:
    en: zh ja ko de fr
    # en:

ignore:
  - _*
```

### ምንጪ ቋንቋ &

ኣብቲ ናይ ውቅር ፋይል፡ እቶም ትሕተ-ቅርጺ ናይ `fromTo`

መበገሲ ቋንቋ እዩ `en` `zh ja ko de fr` ዕላማ ቋንቋ ትርጉም እዩ።

ኮድ ቋንቋ ርአ [i18n.site/i18/LANG_CODE](https://i18n.site/i18/LANG_CODE)

ንኣብነት ቻይንኛ ናብ ትግርኛ ክትትርጉም እንተደሊኻ ነዛ መስመር ዳግማይ ጽሓፋ `zh: en`

ናብ ኩሎም ዝድገፉ ቋንቋታት ክትትርጉሙ እንተደሊኹም ድሕሪ `:` ንኣብነት

```
i18n:
  fromTo:
    en:
```

### ፋይል ሸለል ምባል

ብነባሪ መልክዑ፡ ኩሎም ብ `.md` ን `.yml` ዝጅምሩ ፋይላት ኣብቲ ምንጪ ቋንቋ ማህደር ክትርጎሙ እዮም።

ንገለ ፋይላት ሸለል ኢልካ ክትትርጉሞም እንተደሊኻ (ከም ዘይተዛዘሙ ረቂቕ)፡ ውቅር ዓውዲ `ignore`

[globset](https://docs.rs/globset/latest/globset/#syntax) `.gitignore` ዝመሳሰል ስነ-ቓል ይጥቀም `ignore`

ንኣብነት ኣብዚ ኣብ ላዕሊ ዘሎ ናይ ውቅር ፋይል `_* ` ብ `_` ዝጅምሩ ፋይላት ኣይትርጎሙን ማለት እዩ።

## ሕግታት ትርጉም

### ኣዳለውቲ ትርጉም መስመራት ክውስኹ ወይ ክድምስሱ የብሎምን።

እቲ ትርጉም ክእረም ዝኽእል እዩ። ነቲ መበቆላዊ ጽሑፍ ኣመሓይሽዎን እንደገና ብማሽን ተርጒምዎን፡ እቲ ብኢድ ዝግበር ምምሕያሻት ናይቲ ትርጉም ኣይክጽሓፍን እዩ (እዚ ሕጡበ-ጽሑፍ ናይቲ መበቆላዊ ጽሑፍ እንተዘይተቐይሩ)።

መስመራት እቲ ትርጕምን እቲ መበቈላዊ ጽሑፍን ግን ሓደ ምስ ሓደ ኺሰማማዕ ከም ዘለዎ ኣስተውዕል። ማለት ነቲ ትርጉም ክትጥርንፎ ከለኻ መስመራት ኣይትውስኽ ወይ ኣይትድምስሶ። እንተዘይኮይኑ ኣብቲ ናይ ትርጉም ኣርትዖት መኽዘን ምድንጋር ከስዕብ እዩ።

ገለ ነገር እንተተጋግዩ [፡ መፍትሒታት ንምርካብ በጃኹም ኣብቲ FAQ ተወከሱ።](/i18/qa#H1)

### `YAML` ምትርጓም

እቲ ናይ ትእዛዝ መስመር መሳርሒ ንኹሎም ብ `.yml`

* ኣስተውዕል እቲ ድሕረ-ቃል ስም ፋይል `.yml` ( `.yaml` ዘይኮነ ) ክኸውን ኣለዎ።

እቲ መሳርሒ ነቶም ናይ መዝገበ ቃላት ክብርታት ኣብ `.yml` ጥራይ እዩ ዝትርጉሞም እምበር ነቶም ናይ መዝገበ ቃላት መፍትሕታት ኣይትርጉሞምን።

ንኣብነት `i18n/en/i18n.yml`

```
apiToken: API Token
defaultToken: Default Token
```

ዝብል ክትርጎም እዩ `i18n/zh/i18n.yml`

```
apiToken: 接口令牌
defaultToken: 默认令牌
```

ትርጉም ናይ `YAML` ብኢድ እውን ክቕየር ይኽእል እዩ (ግን ኣብቲ ትርጉም መፍትሕ ወይ መስመራት ኣይትውስኹ ወይ ኣይትድምስሱ)።

ኣብ `YAML`

### ብሕታዊ ናይ ምትካል ማህደር

ብነባሪ ናብ `/usr/local/bin`

ናይ ምጽሓፍ ፍቓድ እንተዘይብሉ ናብ `~/.bin` `/usr/local/bin`

ናይ ከባቢ ተለዋዋጢ `TO` ንኣብነት ናይ ምትካል ማህደር ክትገልጾ ትኽእል ኢኻ :

```sh
TO=/bin sudo bash <(curl -sS https://i.i18n.site) i18
```
