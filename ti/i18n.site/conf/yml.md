# .i18n/conf.yml

ን `i18n.site` ዝኸውን ፕሮፋይል `.i18n/conf.yml` ዩ።

ብዘይካ ናይ [`i18`](/i18) ፣ `ignore:` ን `i18n:` ን ቅጥዕታት እቲ ናይ ውቅር ፋይል ከምዚ ዝስዕብ ይመስል፤

```yaml
upload:
  ext:
    - md
nav:
  - i18n: home
    use: Toc
    url: /
  - i18n: doc
    menu: NB demo1,demo2
    use: Doc
  - i18n: blog
    use: Doc
```

ካብዚኦም `upload` ክሳብ `ext:` ናይ ውቅር ኣቕሓ ማለት ኣብ እዋን ምሕታም `.md` ጥራይ ይስቀል ማለት እዩ።

## top navigation nav

`nav:` ናይ ውቅር ኣማራጺታት፣ ምስቲ ኣብ ላዕሊ ናይቲ መበገሲ ገጽ ዘሎ ናይ ምድህሳስ ዝርዝር ዝሰማማዕ።

<img src="//p.3ti.site/1721051426.avif" style="width:320px">

ካብዚኦም `i18n: home` ምስ `home: Home` ካብ `en/i18n.yml` ይሰማማዕ።

`en/i18n.yml` ናብ ብዙሓት ቋንቋታት ከም `zh/i18n.yml` ክትርጎም እዩ።

<img src="//p.3ti.site/1721051689.avif" style="width:320px">

እቲ ትርጉም ምስተዛዘመ፡ ዋጋ ትርጉም `yml` ክትቅይሮ ትኽእል ኢኻ፡ እንተኾነ ግን መፍትሕ ትርጉም `yml` ኣይትውስኽ ወይ ኣይትድምስሶ።

### `use: Toc` , ንጽል ፋይል ቅጥዒ (ምስ ኣውታር)

`nav` :

```
  - i18n: home
    use: Toc
    url: /
```

`use: Toc` ማለት `Toc` ቴምፕሌት ተጠቒምካ ምቕራብ ማለት እዩ፣ እዚ ድማ ንሓደ `Markdown` ቴምፕሌት ምቕራብ እዩ።

`TOC` ኣሕጽሮተ ቃል `Table of Contents` እዩ።እዚ ቅጥዒ ምስ ዝቐርብ፡ ኣውታር ናይዚ `Markdown` ፋይል ኣብ ጎድናዊ መስመር ክረአ እዩ።

`url:` ንናይ `Markdown` መንገዲ ፋይል ይውክል ( `/` ምስቲ ሱር ማህደር `/README.md` ይሰማማዕ፣ እዚ ስም ፋይል ዓቢ ቅድመ-ጥብቆን ንእሽቶ ድሕረ-ባይታን ይሓትት)።

### `use: Md` , ንጽል ፋይል ቅጥዒ (ኣውታር የለን)

እቲ `Md` ቅጥዒን እቲ `Toc` ቅጥዒን ሓደ ኮይኑ ክልቲኦም ንሓደ `Markdown` ፋይል ንምቕራብ ይጥቀሙ። እቲ `Md` ቅጥዒ ግን ኣብቲ ጎድናዊ መስመር ዘሎ ኣውታር ኣየርእን እዩ።

ኣብዚ ኣብ ላዕሊ ዘሎ ውቅር `use: Toc` ናብ `use: Md` ክትቅይሮ ትኽእል ኢኻ፣ ኣብቲ `md` ማህደር `i18n.site` እንደገና ክትጎዪ ትኽእል ኢኻ፣ ድሕሪኡ ናብቲ ናይ ምዕባለ ቅድመ-ምርኢት URL ብምኻድ ኣብቲ መበገሲ ገጽ ዘሎ ለውጢ ክትዕዘብ ትኽእል።

### ነባሪ ጽዕነት ብዘይ ናይ ውቅር መንገዲ

ሓደ ዝተወሰነ መንገዲ እንተተረኺቡን ቅድመ-ጥብቆ መንገዱ ብ `nav:` እንተዘይተዋቒሩን፡ እቲ ምስቲ መንገዲ ዝሰማማዕ `MarkDown` ፋይል ብነባሪ ይጽዓንን ነቲ `Md` ቅጥዒ ተጠቒሙ ክቐርጽን እዩ።

ንኣብነት `/test` እንተተረኺቡ፣ `nav:` ድማ ብዘይ እዚ መንገዲ እንተተዋቒሩ፣ ቋንቋ ገጽ ድማ እንግሊዝኛ እንተኾይኑ (ኮድ `en` )፣ `/en/test.md` ብነባሪ ክጽዓንን ቅጥዒ `Md` ተጠቒምካ ክቐርጽን እዩ።

`/en/test.md` እዚ ፋይል እንተዘይሃልዩ፡ እቲ ነባሪ `404` ገጽ ክረአ እዩ።

### `use: Doc` , ብዙሕ ፋይላት ዘለዎ ቅጥዒ

ኣብቲ ናይ ውቅር ፋይል፤

```
  - i18n: blog
    use: Doc
```

ንመቕረጺ ቅጥዒ `Doc` ምጥቃም የመልክት።

`Doc` template ንሓደ ወይ ንብዙሓት ፕሮጀክትታት ሰነድ ኣውታር ንምፍጣር ብዙሓት `MarkDown` ምውህሃድ ይድግፍ።

#### ሓደ ፕሮጀክት (ብዙሓት ፋይላት)

ኣብ ላዕሊ ዘሎ `blog` ናይ `Doc` ንጽል-ኣቕሓ ሞድ እዩ።

```
  - i18n: blog
    url: blog
    use: Doc
```

##### url ባዶ ምስ ዝኸውን ብነባሪ ናብ ዋጋ i18n ይኸውን።

`url` እንተዘይተጻሒፉ `url` ብነባሪ ናብ ዋጋ `i18n` ይኸውን።እዚ ሕጊ ንኻልኦት ቅጥዕታት እውን ተግባራዊ ይኸውን።

እዚ ኣብ ላዕሊ ዝተጠቕሰ ኣገባብ ጽሕፈት ማዕረ `url: blog` ኮይኑ፡ ተዛማዲ ፋይሉ ድማ `en/blog/TOC` ዩ።

#### ብዙሓት ፕሮጀክትታት

ውቅር `i18n:doc` ካብ `.i18n/conf.yml` ብዙሕ ፕሮጀክት ሞድ እዩ።

```
  - i18n: doc
    menu: NB demo1,demo2
    use: Doc
```

<img src="https://p.3ti.site/1721275191.avif" width="320px">

ኣብዚ `menu: NB demo1,demo2` ፡ ነቲ ንቑልቁል ዝወርድ ዝርዝር ንምቕራጽ ነቲ `NB` ቅጥዒ ምጥቃም ማለት እዩ።

`NB` ፡ እዚ ድማ ኣሕጽሮተ ቃል `Name Breif` ኮይኑ፡ እቲ ንቑልቁል ዝወርድ ዝርዝር ስምን ጭርሖን ናይቲ ፕሮጀክት ከርኢ ይኽእል።

`NB` ስዒቡ ድማ ናብኡ ዝሓልፍ መለክዒ `demo1,demo2` ይስዕብ።
መተሓሳሰቢ ኣብ `demo1,demo2` ቅድሚን ድሕሪን ኮማ `,` ** ክህሉ የብሉን : **

እቲ ተዛማዲ ናይ ማህደር መመልከቲ ፋይል ናይዞም ኣብ ላዕሊ ዝተጠቕሱ መለክዒታት፤

* `en/demo1/TOC`
* `en/demo2/TOC`

#### TOC ትሕዝቶ ሰሌዳ መመልከቲ

`i18n.site` ኣብቲ ዲሞ መኽዘን `js` ፕላግ-ኢን `.i18n/hook/after.tran/TOC.js` ብምፍጻም ነቲ ምስቲ `TOC` ቴምፕሌት ውቅር ዝሰማማዕ `doc` ማህደር ኢንዴክስ ፋይል ከንብብ እዩ ነቲ `json` ማህደር ኣውታር ንምፍጣር።

`doc` template እንተተጠቒምካ እዚ ፕላግ-ኢን ክህልወካ ኣለዎ።

ፕሮጀክት `i18n.site` ካብ ባዶ ማህደር እንተጀሚርካዮ፡ ዲሞ ፕሮጀክት `.i18n` ናብ ማህደርካ ምቕዳሕ ኣይትረስዕ።

እቲ `Doc` ቅጥዒ ነቲ ናይ ትሕዝቶ ሰሌዳ ኣውታር ኣብቲ ዝተፈጥረ `json` መሰረት ብምግባር ክቐርብ እዩ።
