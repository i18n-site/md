
## CNAME ዝተኣሰረ ስም ዓውዲ

Cloudflare Worker ’s domain name is bound through `SSL/TLS` → custom host name CNAME እቲ ስም ዓውዲ ምእሳሩ ቀጥታዊ ምብጻሕ ጌጋ `error code: 522` ክሕብር እዩ።

ኣብዚ እዋን እዚ፡ ከምቲ ኣብ ታሕቲ ዘሎ፡ ንሓደ መስመር ንምእሳር፡ HTTP routing → Workers routing ክትጥቀም ኣለካ
