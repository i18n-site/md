ገለ ሰባት ክሓቱ ይኽእሉ እዮም፣ ኩሎም መርበብ ሓበሬታታት ናይ ትርጉም ተግባራት ስለዘለዎም፣ ነቲ መርበብ ሓበሬታ ኣህጉራዊ ምግባር ኣየድልን ድዩ?

## መእተዊ

ሳይንሳዊ ልቢ ወለድ ልብ ወለድ &quot;ሰለስተ ኣካል&quot; (ኣደማምጻ ቻይና: `3Tǐ` ) ንሓደ ባዕዳዊ ስልጣነ ንምርድዳእ ኤሌክትሮማግኔቲክ ማዕበል ዝጥቀም፣ ብግሉጽ ኣተሓሳስባን ብልጽግና ቴክኖሎጅን ልቢ ወለድ ይገልጽ።

መጽሓፍ ቅዱስ · ዘፍጥረት። :

> ኣብቲ ዘመን እቲ ድምጽን ቃላትን ናይ ኩሎም ኣብ ትሕቲ ሰማያት ከም ሓደ እዩ ነይሩ።
>
> ክሳብ ሰማይ ዝበጽሕ ግምቢ ሰሪሖም፡ ኣፍደገ እግዚኣብሔር ዝትንክፍ፡ ልብታት ዘወሃህድ፡ ክብረቶም ዝምካሕ።
>
> ጐይታ ከምዚ ኢሉ ኣወጀ፡ ሰባት ቋንቋን መስመር ወለዶን ክካፈሉ ከለዉ፡ ንጽል ቀቢላ ይኾኑ። እቲ ግምቢ መጀመርታ ጥራይ እዩ፤ ኣብ ዝመጽእ መዓልታት፡ ዘይበጽሕዎ ነገር የለን።
>
> በዚ ድማ ወረደ፣ ህዝቢ ርሑቕን ቀረባን ፋሕ ክብል ገበሮ፣ ቋንቋታቶም ሕጂ ንሓድሕዶም ክርድኡ ኣይከኣሉን።
>
> ካብ ሽዑ ጀሚሩ ባእሲ መወዳእታ ዘይብሉ ስለ ዝነበረ፡ ዓለም ንሰማይ ዝትንክፍ ግምብታት ኣይረኣየትን።

ህዝቢ ምድሪ ከም ሰለስተ ኣካል ዘለዎም ሰባት ኮይኖም፡ ብቋንቋ ከይተኣሰሩ ክረዳድኡ፡ ኩሉ ወዲ ሰብ መሊሱ ክጥርነፍ መሳርሒ ክፈጥር ተስፋ እገብር።
