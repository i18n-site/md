**ንብምሉኡ እቲ መርበብ ሓበሬታ ኣህጉራዊ ብምግባር ጥራይ ኢና ብብዙሕ ቋንቋታት ኣብ ውሽጢ መርበብ ሓበሬታ ምሉእ ጽሑፍ ምድላይን ምምሕያሽ ሞተር ምድላይን ክንድግፍ ከም እንኽእል ክብል** እደሊ።

## ቕደም ስዓብ

"መጽሓፍ ቅዱስ·ዘፍጥረት":

> ኣብ ጥንቲ ቋንቋ ሓደ ኣብ ዝነበረሉ እዋን ሰብኣውነት ብኹርዓት እዩ ተወሊዱ። ንሰማያት ዝትንክፍ ግምቢ ዝዀነ ዓብዪ ህንጻ ኺሃንጹ ደለዩ።
>
> እንተኾነ ግን እቲ መለኮታዊ ብትዕቢቶም ስለ ዘይተሓጐሰ ነቶም ህዝቢ ኣብ መላእ ምድሪ ፋሕ ኢሎም መልሓሶም ንሓድሕዶም ከም ዘይርዳእ ገበሮም።
>
> ካብ ሽዑ ጀሚሩ ወዲ ሰብ ንምርድዳእ ተቓለሰ፣ ግርጭታት መወዳእታ ዘይብሉ ተላዒሉ፣ ዓለም ድማ ድሕሪ ደጊም ናብ ሰማይ ዝበጽሕ ግምቢ ኣይረኣየትን።

ሳይንሳዊ ልቢ ወለድ &quot;ሰለስተ-ኣካል&quot; (ኣደማምጻ ቻይና: `3Ti` ) ብኤሌክትሮማግኔታዊ ማዕበላት ዝራኸብ፣ ናይ ቋንቋ ዕንቅፋት ዘይብሉን ብቴክኖሎጂ ብልጽግና ዘለዎን ባዕዳዊ ስልጣነ ልቢ ወለድ ትገልጽ።

ብሓገዝ እዚ መሳርሒ እዚ ህዝቢ ምድሪ ከም ሰለስተ ኣካል ዘለዎም ሰባት ክኸውን፡ ርክብ ብቋንቋ ከይድረት፡ ኩሉ ወዲ ሰብ ድማ መሊሱ ክጥርነፍ ተስፋ እገብር።
