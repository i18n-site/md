# Chuẩn bị môi trường

* Cài đặt và khởi động `docker` ( Người dùng Mac được khuyến nghị sử dụng [orbstack](https://orbstack.dev) làm môi trường chạy cho `docker` )

* Cài đặt [bun](https://bun.sh/docs/installation) và [nodejs](https://nodejs.org/en/download/package-manager)

## Khởi tạo dự án

```
curl -sSL https://raw.githubusercontent.com/i18n-ops/ops/main/i18n.site/init.sh | bash
```
